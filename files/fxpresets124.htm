3:2 Pulldown

Converts timings from 24 frames per second (film) to 30 frames per second (NTSC video) or vice versa. Each four frames of film are converted to five frames of video (or vice versa) by the method shown in the following diagram.

You can select at what point in the pattern the conversion is commenced.

Requires an Input (Any).

Produces an Output (Any).

Pulldown

Source Clip Range

Specifies the frames in the source to be used as Start and End frames when retiming.

From/To

Sets whether the conversion is Add 3:2 (expand) - from 24 to 30 frames per second, or Remove 3:2 (compress) -from 30 to 24 frames per second.

A Frame

Sets the point in the conversion pattern (shown above) at which the first frame will start. Select Frame AA, Frame BB, Frame BC, Frame CD or Frame DD from the pop-up menu.

Field Dominance

Sets the field dominance. Select Even or Odd.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
