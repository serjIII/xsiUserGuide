Disconnect Component Op property editor

Controls how polygons are disconnected.

To apply: Select some polygons or polygon clusters on a polygon mesh object and choose Modify Poly. Mesh Disconnect Components from the Model toolbar.

To redisplay: Select the object with disconnected polygons, click the Selection button on the Select panel, then click the Disconnect Component Op icon.

Explode Polygons

Disconnects each polygon individually, rather than as a group.
