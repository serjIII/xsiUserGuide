Copying Animation between Characters

In Softimage, there are several ways in which you can copy animation between characters. The method you choose depends on what you need copied at what level, and which tools you prefer to, or need to, use.

  • Save the animation in an action source and copy it to another character — see Sharing Actions between Models [Nonlinear Animation in the Animation Mixer] for information.

  • Retarget the animation between characters. While you can transfer the animation from one character to another using retargeting, this is especially useful for copying motion capture data to proportionately different characters — see Overview of Retargeting Animation with MOTOR.

  • Use the dopesheet to copy all keyframe animation from one character to another — see Cutting, Copying, and Pasting Keys in the Timeline [Animation].

  • Use the Copy Animation commands in the Animation menu, which is good for individual objects within a character — see Copying Animation [Animation].
