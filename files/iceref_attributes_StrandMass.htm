StrandMass

A custom attribute that is used by some of the strand dynamics compounds to control the motion of strands. If this attribute is undefined, the default value is 1.0.

Data Type

scalar, or array of scalars

Belongs To

points

Geometry Types

  • point cloud

Attribute Type

custom

R/W or RO:

read/write
