Shot Track Property Editor

Provides options to toggle the display of Shot Track controls.

To open the Shot Track Property Editor

  • Select a Shot Track and choose Edit Properties from the Camera Sequencer Editor menu bar.

    or

  • Choose Track Track Properties.

Shot Track Controls

Ripple

Activates the Ripple effect on all selected ShotClips in a Shot Track.

Mute

Mutes the Shot Track and its contents.

Solo

Plays only the contents in the selected Shot Track.
