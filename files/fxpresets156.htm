Kaleidoscope

Creates a kaleidoscope effect. The number, angle and position of the "mirrors" can be controlled, as can the source area for the effect.

Requires an Input (Any).

Produces an Output (Any).

Kaleidoscope

Symmetry

Sets the number of "mirrors". Set any integer value using the treadmill.

Scale

Sets the scale of the image. Set any floating-point value using the slider.

Angle

Sets the angle of the "mirrors". Set any floating-point value in degrees using the treadmill.

Center X/Y/Z

Set the centre of the "mirrors" as a normalized proportion of the image. Set any floating-point values using the treadmills. The Z value has no effect.

Offset X/Y/Z

Set the offset from the original source area as a normalized proportion of the image. Set any floating-point values using the treadmills. The Z value has no effect. By default, the source area is the top-right area of the image.

Interpolate

Specifies whether or not transformed areas will be smoothed.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
