Eyelash Tool Property Editor

Create eyelash meshes for a Face Robot head.

To display:

On the Tune panel in the Face Robot layout, click the Tools tab and choose Build Eyelashes.

For more information, see Building Eyelash Meshes [Face Robot].

Eyelash Clusters - Right/Left Upper/Lower

Assign

Assigns the tagged points as a cluster for the appropriate eyelash. The cluster is automatically named accordingly as shown for Select below.

Select

Select an existing cluster for the eyelashes and click this button for the appropriate eyelash.

You must have the following naming convention for existing clusters so that the Build Eyelash tool can recognize them:

  • Left_Upper_Eyelash_Cluster

  • Left_Lower_Eyelash_Cluster

  • Right_Upper_Eyelash_Cluster

  • Right_Lower_Eyelash_Cluster

Eyelash Meshes

Build Meshes

Creates an eyelash polygon mesh for each cluster you assigned. The name of the mesh is the same as its corresponding cluster, such as Left_Lower_Eyelash. This mesh is lofted from a series of curves that correspond to each point that was tagged for the cluster.

Edit Curves

Opens a property editor in which you can edit the curves for each eyelash to adjust their shape.

See Editing the Eyelash Curves [Face Robot] for information.

Delete Meshes

Deletes the eyelash mesh objects and curves.

Eyelash Rig

Build Rig

Creates eyelash rig objects and then envelopes the eyelash meshes to them so that they are now part of the face. When you move the eyelid controls, such as for blinking, the lashes should follow correctly.

Apply Texture

Select a texture in the text box here, then select this option to apply the selected texture to the eyelash meshes when you click the Build Rig button.

View

Opens the image clip viewer in which you can view the selected texture.

Draw a render region around the eyes to see what the textures will look like when rendered
