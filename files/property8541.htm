TextProp Property Editor

A property containing a single multi-line text parameter. You can use this property to store any information you want. This property is part of a sample self-installing plug-in.

To apply: Select an object and choose Get Property TextProp.

To redisplay: Select the object, click Selection on the Select panel, and then click the TextProp icon.
