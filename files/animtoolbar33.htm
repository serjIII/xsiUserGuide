Edit Custom Parameter Dialog Box

To display: Mark one or more custom or proxy parameters, then choose Create Parameter Edit Parameter Definition from the Animate toolbar.

The original parameter name and values are displayed for reference, but you can edit them. Any connections you have set up with expressions, linked parameters, or scripts are maintained.

Parameter Name

The name used in property editors, explorers, scripting, and expressions to identify the parameter.

Script Name

Script name for the parameter. The script name is shorter than the full parameter name, but more cryptic.

Description

Short description for the parameter.

Value Type

You cannot change the value type.

Value Range

For numeric parameters, this sets the Minimum and Maximum values.

UI Range

The slider for the new parameter can have a smaller range than the Minimum and Maximum values. In such cases, you can enter values outside the slider range using the keyboard.

Enable UI Range

Allows the slider range to be smaller than the minimum and maximum values set for the Value Range. If this is off, the slider uses the Value Range.

UI Minimum, Maximum Values

The minimum and maximum values of the slider on the custom parameter set's property editor.

Characteristics

Animatable

Determines whether the new parameter can be animated.

Keyable

Determines whether the new parameter appears in the keying panel. See Keying Keyable Parameters in the Keying Panel [Animation].

Read-Only

When on, the value of the custom parameter cannot be modified interactively by users. Read-only parameters can still by modified by scripting commands or the object model.
