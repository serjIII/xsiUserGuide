Disconnect Component

Categories

  • Topology

Description

Disconnect the specified component(s). The type can be vertex, edge, polygon or polynode.

Ports

Inputs

Topo

The topology you want to use. If not connected, the default is "Self.Topology".

Component Type

The type of component to disconnect: Vertex, Edge, Polygon, or PolyNode.

Component Index

The indices of any component to delete. You can use a single integer, array, or set.

Outputs

Result

The modified topology.
