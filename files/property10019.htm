Hair Generator Operator Property Editor

Generates guide hairs on a hair object.

To apply: Select an object and choose Create Hair from Selection on the Hair panel.

To display: Select the hair object, click the Selection button on the Select panel, and then click the Hair Generator Operator icon.

For more information, see Creating Hair [ Hair ].

Emitter Subd Level

Applies only to polygon meshes with geometry-approximated subdivision surfaces.

This parameter allows you to change the subdivision level of the hairs on the polygon mesh object so that the hairs can be emitted from the subdivision surface.

For example, you can enter a value that matches the polygon mesh's subdivision level. Notice that the number of guide hairs does not change.
