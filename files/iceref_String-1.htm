String

Categories

  • Constant

  • String

Description

Outputs a string of characters. Strings can be used by custom nodes for storing file paths, for example. You can enter a value in the property editor, or you can connect a value to the input port to pass it through.

There is no predefined limit for string length, although long strings will be truncated when you display values in a viewport.

Ports

Inputs

Value

The string to output.

Outputs

Result

The output string.
