Get Polygons Inside Null

Stores an array containing the indices of the polygons of the "self" object that are inside the connected null, using the specified attribute name.

Tasks: Topology/Getters

Output Ports: Execute

Null Name

The null within which the polygons must be to be included in the array. The null's icon must be a 3D geometry shape for this to work: change the Primary Display Icon of the null in its property editor. Plug the null's Output Name output into this port.

Reference

Specify the attribute name to store the array.
