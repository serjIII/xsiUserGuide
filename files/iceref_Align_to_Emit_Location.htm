Align to Emit Location

This compound changes the orientation of the particles so that their local Y axes point in the direction of their location of emission (EmitLocation attribute).

For more information on emissions, see Setting Up Any Type of ICE Particle Emission [ICE Particle Simulations].

Plug this compound's Execute port into an Execute on Emit port of the Emit compound.

Tasks: Particles/Emission Control

Output Ports: Execute
