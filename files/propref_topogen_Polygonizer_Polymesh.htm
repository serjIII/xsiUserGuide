Polygonizer_Polymesh Property Editor

A custom property that controls a polygon mesh's contribution to the density isofield used by a Polygonizer operator. For more information about using Polygonizer, see Meshing Point Clouds (and Other Things) [Data Exchange].

To display: Select the mesh, choose Selection on the Select panel, and click the Polygonizer_Polymesh icon.

Show/Hide Weight Map Parameters

Shows or hides the options for using weight maps to control isofactor, radius, and falloff.

Active

Controls whether the object contributes to the Polygonizer's isofield. Turn this off to temporarily mute the object's contribution to the polygonized mesh without permanently removing the object as an input.

Use

Controls what components of the object contribute to the isofield.

LOD

Controls sampling along particle strands, curves, and polygon faces. This value has no effect unless Use is set to an option that includes LOD.

Size and Influence

Isofactor

Isofactor

The base contribution of the object's components to the isofield. Note that you can use a negative Isofactor value to make an input remove volume from the resulting mesh.

Multiply Isofactor by Weight Map

Multiplies the value specified by Isofactor with a weight map defined on the input mesh.

This option does not appear unless you click Show Weight Map Parameters.

Weight Map

Shows the weight map being used for the isofactor.

Use the Pick button to select an existing weight map on the input mesh.

Radius

Radius

The distance at which the isofield is equal to the Isofactor value.

Multiply Radius by Weight Map

Multiplies the value specified by Radius with a weight map defined on the input mesh.

This option does not appear unless you click Show Weight Map Parameters.

Weight Map

Shows the weight map being used for the radius.

Use the Pick button to select an existing weight map on the input mesh.

Falloff

Falloff

The thickness of the falloff region beyond the radius.

Falloff Type

Controls how the isofield value decreases in the falloff region. For example, "Square" gives an inverse-square falloff similar to many physical phenomena.

Multiply Falloff by Weight Map

Multiplies the value specified by Falloff with a weight map defined on the input mesh.

This option does not appear unless you click Show Weight Map Parameters.

Weight Map

Shows the weight map being used for the falloff.

Use the Pick button to select an existing weight map on the input mesh.
