Scene Property Editor

Renames the scene and displays the location of the scene on the hard disk or network.

To display: Set the scope to Application in an explorer, then click the icon for the Scene node.

Current Character Key Set

Name of the character key set that is currently active in the scene. This is the one on which keys are set when you press K or click the keyframe icon, regardless of what's selected in the scene.

For more information, see Keying with Character Key Sets [Animation].
