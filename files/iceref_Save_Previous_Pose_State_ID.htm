Save Previous Pose State ID

Saves the current value of actors' CrowdFX_PoseState_ID as __PoseState_ID_Previous. This attribute is used in a CrowdFX simulation for blending animation during a transition to a new pose state.

This compound should be used in an ICE tree on the Point_Cloud object of a CrowdFX simulation before changing the value of CrowdFX_PoseState_ID.

Tasks: Crowds/States

Output Ports: Execute
