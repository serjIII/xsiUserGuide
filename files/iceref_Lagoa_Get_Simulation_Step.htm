Lagoa Get Simulation Step

Gets the simulation step data that is computed in the simulation framework using the Substeps value in the Lagoa Simulate Multiphysics node. Everything computed using this number will fall on the substep time.

Tasks: Lagoa/Data

Output Ports: Value
