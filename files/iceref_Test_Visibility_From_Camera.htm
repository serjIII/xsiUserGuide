Test Visibility From Camera

This compound uses the particle's distance from the camera as a trigger for an effect. Only particles that are within visibility distance for the camera can trigger an effect.

Plug this compound's Result output into a Trigger port on a State compound or in a Condition port of an If node.

For more information on states, see ICE Particle States [ICE Particle Simulations].

Tasks: Particles/Conditionals

Output Ports: Result

Camera Name

The camera you want to use to determine the particle's distance. Plug the camera's Output Name output into this port.
