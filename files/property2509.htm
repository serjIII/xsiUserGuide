Scale Constraint Property Editor

Constrains an object so that its scaling factor is the same as that of the constraining object. If, for example, the constraining object is scaled down to 50% of its original size, the constrained object is scaled down to half its own size as well.

For more information, see Scale Constraints [ Animation ].

To apply: Select an object, Constrain Scaling, then pick the object to which the selected object will be constrained.

To redisplay: Select the constrained object, choose Explore Constraints on the MCP, and click the constraint's icon.

Active

Toggles the activeness of this constraint.

Blend Weight

Blends the weight of this constraint with other constraints on the selected object.

Scaling Offset

X/Y/Z

Sets the scale offset in global space between the constrained object's axes and those of the constraining object.
