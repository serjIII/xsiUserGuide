Get Strand Length

This compound gets the particle's strand length data as set by its StrandLength attribute. You can output the Result as scalar data.

For information in general about strands, see ICE Particle Strands [ICE Particle Simulations].

Tasks: Particles/Strands

Output Ports: Result
