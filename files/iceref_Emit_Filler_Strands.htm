Emit Filler Strands

This compound emits particle strands that fill the space between guide strands that are created on another point cloud. The filler strands are constrained to the guide strands so that they follow the guide strand movement as they respond to the dynamics.

The filler strands need to have the same number of segments (Num Segments) as the guide strands to which they are constrained. If not, their StrandPosition arrays will not match and the strand size, as well as other strand attributes, won't be considered.

You can create multiple point clouds of filler strands to be constrained to the same guide strands.

Plug this compound's Execute output into a port on a simulated ICETree node.

For more information, see Emitting the Filler Strands [ICE Particle Simulations].

Tasks: Particles/Strand Dynamics

Output Ports: Execute

Emitter1

Plug in the geometry from which you want to emit particle strands. This needs to be the same emitter object as is used for the guide strands. This must be a geometry with a surface.

You can plug multiple geometries into the Emitter ports.

Rate

The total number of particle strands that are emitted at once at the first frame of emission.

See ICE Particle Rate (Amount) [ICE Particle Simulations] for more information.

Guide Strands

Plug in the point cloud geometry to use as guide strands to which the filler strands are constrained.

This point cloud needs to have particle strands created in its ICE tree using the Emit Strands compound.

Size

Defines how large the particle strand is. This value is the particle radius, but the particle display size is the diameter. For example, if the Size value (radius) is 1, the diameter (display) size is 2.

See ICE Particle Size [ICE Particle Simulations] for more information.

Color

Defines the initial particle strand color as displayed in the viewport. Set the color value using the standard color sliders and color box.

Rendered particles do not use the particle color values unless you've set up the shader to do so, such as with an Attribute_Color shader.

See ICE Particle Color [ICE Particle Simulations] for more information.

Num Segments

The number of segments in each strand. Note that this doesn't affect the overall strand length, just how the strand is divided.

Make sure that this value matches the Num Segments defined for the guide strands.
