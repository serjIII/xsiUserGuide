Syflex Collide Capsule

This compound creates a capsule-shaped bounding collision object based on a null that you input. Any part of the cloth or curve object that is inside the bounding capsule is repelled to its outside.

The null's Size and Icon shape parameters don't matter to the size and shape of the bounding capsule that is created.

The null's translation and rotation values do affect the bounding capsule because its location and orientation is based on the null's center. You can also animate the null's translation and rotation values. The scaling values have no effect on the bounding capsule.

To apply: See Creating Collisions with Bounding Box Objects.

Tasks: Syflex/Collisions

Output Ports: The Force output plugs into the Force port on the Syflex Cloth or Syflex Curve node.

Damp

The damping force that slows down the movement of the simulated object's points that are in contact with the collision object. It is proportional to the simulated object's speed along the surface of collision object.

High values slow down the movement of the simulated object; however, increasing the damping may also increase the computation time.

Length

Length of the collision capsule in Softimage units. This size is independent of the null's Size attribute or its scale (transformation) value.

Radius

Radius of the collision capsule in Softimage units. This size is independent of the null's Size attribute or its scale (transformation) value.

Null

The null object that is used as a base for creating the bounding capsule. Plug its Get Data node's Out Name output into this port.
