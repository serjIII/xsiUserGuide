Cage Deform Operator Property Editor

Controls the cage deformation. It is created when you deform by cage, but the property editor is not displayed automatically.

To display: Select an object that is deformed by cage, click the Selection button on the Select panel, then click the Cage Deform Operator icon.

Mute

Toggles the deformation on and off without affecting the values of other parameters. For example, you can temporarily mute the operator while changing the reference pose.
