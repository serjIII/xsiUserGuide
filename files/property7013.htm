Interpolator Property Editor

Lets you set the falloff among multi-linked or orientation-linked parameter values.

For more information about the linked parameters in general, see Linking Parameters [Animation].

To display: In the explorer, expand the linked parameter's tree and click on the Expression operator's icon to open the expression's property editor, then click the Interpolator tab.

Falloff

Sets the falloff value among each parameter that contributes to the multi-linked or orientation-linked parameter output. The falloff determines at what point the controlling parameter stops having an influence on the result.
