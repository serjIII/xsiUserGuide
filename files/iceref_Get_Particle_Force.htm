Get Particle Force

This compound gets the particle's current force data as set by its Force attribute. You can output this Value as a 3D vector.

For more information, see Force Attributes [ICE Guide].

Tasks: Particles/Getters

Output Ports: Value

Reference

The particle's Force attribute.
