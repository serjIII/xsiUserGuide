Vector2 To Color

Category: Processing > Conversion

Shader Family: Texture

Output: Color

Converts 2D vector input to a color output.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Specifies the 2D vector input to be converted to a color output.

Render Tree Usage

This shader is simply placed between 2D vector output and a color input in order for the values to connect.
