Update Move Params

An internal compound used by the Define Move Animation State to update the parameters of the Move Animation State.

Tasks: Crowds/Animation

Output Ports: Execute. Connect this port to the Params Update port of the Define Animation State node.

Input

Defines the current speed of an actor.
