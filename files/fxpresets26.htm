Chroma Adjust

Adjusts the chrominance values of a clip using a graph.

Requires an Input (RGB or RGBA).

Produces an Output (RGB or RGBA).

Graph

Graph

The horizontal axis represents the chrominance values in, and the vertical axis represents the chrominance values out.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
