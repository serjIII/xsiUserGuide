Spiral Property Editor

To get a spiral, choose one of the commands on the Primitive menu. Curve objects are cubic NURBS curves. Implicit objects are non-renderable shapes that can be used, e.g., to define paths, constraints, etc.

To redisplay: Select the spiral then press Enter and click the Spiral tab.

Height

The distance between the start and end points along the central axis (perpendicular to the spiral) in SOFTIMAGE units.

Start, End Angle

The start and end positions in degrees. The spiral is drawn from the start angle to the end angle.

Start, End Radius

The distance from the central axis to the start and end points in SOFTIMAGE units.
