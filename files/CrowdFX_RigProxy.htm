CrowdFX Rig Proxy Property Editor

Custom property that sets data for a deformer in an actor's rig proxy in a CrowdFX simulation.

To display:

  • Select a CrowdFX_RigProxy icon for a rig deformer in an explorer (under a Crowd model > Actors_Proxies > actor proxy model > Rig_Proxy or Deformers_Proxy group node).

    A quick way to open this property editor is to select the deformer, press F3, then click the property editor's icon in the pop-up explorer.

For general information about rig proxies, see Modifying the Actor's Rig.

Is Center of Gravity

Specifies whether or not this deformer is used as the actor's Center of Gravity (COG) deformer. The COG deformer is the one whose horizontal speed (in the global Z direction) is used for calculating the actor's velocity for locomotion in the simulation. See The Actor's Center of Gravity (COG) Deformer.

Index

The index value of this deformer to identify it in the array used for the rig proxy.

Show/Hide Proxy Deformers

Toggles the display of the selected actor proxy's rig of deformers in an object view or viewport - see Modifying the Actor's Rig.
