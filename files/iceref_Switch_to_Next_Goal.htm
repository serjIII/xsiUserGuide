Switch to Next Goal

This compound switches from the current goal object to the next one that you have plugged into the Move Towards Goal compound, which is plugged into the Multi Goal Sequencer compound.

Plug this compound's Execute output into the Execute Every Frame port of the Move Towards Goal compound or in an Execute port of a State compound, if you're using the state system.

For more information on goals, see ICE Particle Goals [ICE Particle Simulations].

Tasks: Particles/Goals

Output Ports: Execute
