Edge Filter Op Property Editor

Selectively collapses edges based on length or angle.

To apply: Select the some edges, then choose Modify Poly. Mesh Filter Edges from the Model toolbar.

To redisplay: Select the polygon mesh, click the Selection button on the Select panel, then click the Edge Filter Op icon.

Nb Points, Edges

Displays the number of Points and Polygons in the resulting mesh. These values are for information only; they cannot be changed directly, but will change as a result of other settings.

Filter Type

Specifies the criterion for removing edges:

  • Edge Length collapses edges that are shorter than a threshold. Use the Edge Length slider to set the threshold.

  • Edge Angle collapses vertices where two edges meet at an angular difference less than a threshold. This lets you combine edges that are almost colinear. Vertices where three or more edges meet are never collapsed. Use the Edge Angle slider to set the threshold.

Edge Length

Collapses edges shorter than this value in Softimage units, when Filter Type is set to Edge Length.

Edge Angle

Collapses vertices where two edges meet with an angular difference smaller than this value in degrees, when Filter Type is set to Edge Angle.

Keep Border Edges Intact

Prevents boundary edges from being removed by the filtering operation.
