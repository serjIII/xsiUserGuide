Clearing Unused Image Sources/Clips and Materials

In addition to helping you manage external files, the external files manager allows you clear unused image sources/clips and unused materials from your scene.

The following buttons are available in the Cleanup section.

  • Delete Unused Materials — Deletes all materials that are not assigned to any object in the scene, provided they are not stored in an external material library.

  • Delete Unused Image Sources and Clips — Removes the source and clip pairs attached to any unused images, but does not delete the original images themselves.

  • Delete Unused Animation Sources — Removes unused animation sources in a scene.

  • Delete Unused Audio Sources — Removes unused audio sources in the animation.

  • Delete Unused Cache Sources — Removes unused ICE cache files in the simulation.
