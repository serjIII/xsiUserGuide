Randomize Animation State Speed

This compound helps you to easily add some randomization to the playback speed of an Animation State to make a crowd look more natural.

A playback speed of 1 means that the animation is played back in the original timing.

Tasks: Crowds/Animation

Output Ports: Execute

State

Name of the Animation State for which the playback speed should be randomized.

Base Value

The mean value around which the random Value is calculated in any XYZ direction. For example, if this value is 2 and the Value is 1, the random value possibilities would be any value between 1 and 3.

Variance

The amount of variance of the turbulence in any XYZ direction around either side of the Base Value.

Seed

Defines the sequence of random numbers used. If you require that two nodes generate different sets of values with the same parameters, simply assign them different seeds.
