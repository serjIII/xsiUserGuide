Test Particle Size

This compound uses the particle's Size value as a trigger for an effect.

Plug its Result output into a Trigger port on a State compound or in a Condition port of an If node.

For more information on states, see ICE Particle States [ICE Particle Simulations]; for information on particle size, see ICE Particle Size [ICE Particle Simulations].

Tasks: Particles/Conditionals

Output Ports: Result

Test Size

The size of the particle to use as a trigger. This uses the particle's Size attribute as a reference.

Type

Defines the condition for the Test Size value to be triggered: Equal To; Not Equal To; Less Than; Less than or Equal to; Greater Than; or Greater Than or Equal To.

Epsilon

Widens the range of acceptable values by a small amount so as to reduce rounding errors.
