BlobBlend

Determines the amount of blending between particles. A high value adds more matter between the particles, which can take longer to render if more surface area is drawn. Values are relative to BlobStrength.

To see particles displayed as blobs, you must draw a render region.

If this attribute is undefined, the default value is 0.0.

This attribute is exposed as the Blob Blend parameter in the Blobs compound.
