Plot Curve Dialog Box

Creates a curve from an object's animated transformations.

For more information about plotting in general, see Plotting (Baking) Animation [Animation].

To display: Select an animated object and choose Tools Plot Curve from the Animate toolbar.

Start Frame

The frame at which to begin plotting

End Frame

The frame at which to end plotting

Step Value

The frame step increments to use. When this value is 1, every frame is plotted; when 2, every other frame is plotted, and so on.

Curve Name

The name for the generated curve.
