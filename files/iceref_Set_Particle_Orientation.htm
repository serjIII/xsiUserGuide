Set Particle Orientation

This compound lets you set the particle's orientation, using the particle's Orientation attribute as a reference.

Plug its Execute output into the Execute on Emit port of an Emit compound to set this value once, or plug it into the ICETree node to update at every frame.

For more information, see Particle Orientation Attributes [ICE Particle Simulations].

Tasks: Particles/Setters

Output Ports: Execute

Orientation

Set the axis of rotation in the local XYZ reference frame, and then set the Angle in degrees around that axis.

Reference

The particle's Orientation attribute.
