Water Paper

The Water Paper effect renders an image with blotchy daubs that appear to have been painted onto damp paper. The painted colors blur and run along the fibers of the paper, softening and extending the original outlines of the image.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

Fiber Length

Set higher values to lengthen the fibers, or set lower values to shorten the fibers.

Brightness

Set higher values to make the image brighter, or set lower values to make the image darker.

Contrast

Set higher values to increase the contrast of the image, or set lower values to decrease the contrast.
