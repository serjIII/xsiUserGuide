Importing and Exporting User Data

You can import and export user data.

  • For information about importing and exporting using Crosswalk, see Using Crosswalk for Softimage.

  • For information about importing and exporting using older versions of the dotXSI format, see Importing and Exporting dotXSI Files. The important thing to remember is to set the dotXSI File Version to 3.6 or later. User data is not supported in version 3.5 or earlier of the dotXSI format.
