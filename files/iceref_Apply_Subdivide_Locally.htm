Apply Subdivide Locally

Subdivides polygons on the self or on the connected topology, and replaces the self's topology.

The Filter Mode options for specifying polygons are the same as Test Polygon Index.

Tasks: Topology/Modifiers

Output Ports: Execute

Rule

The algorithm used for subdivision.

Depth

Number of times that the subdivision process is iterated.
