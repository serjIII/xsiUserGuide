Setting Pass Field Rendering Options

Location:

  • The Render Manager > (select a pass)

  • Set the current pass and choose Render > Render > Pass Options or Pass > Edit > Edit Current Pass from the Render toolbar.

  • Tab: Output [Properties Reference]

  • Options: Pass Field Rendering

You can override the scene-level Field Rendering settings to activate or deactivate field rendering for specific passes only.

For a general description of field rendering and information about setting field rendering options, see To Set Field Rendering Options.

To set the pass field rendering options

  1. Make the pass for which you want to override the Field Rendering settings the current pass.

  2. From The Pass Field Rendering options, check the Override Scene Render Options box.

  3. Activate field rendering and set the options, or deactivate it as needed.
