Pre/Un Multiply RGB

Changes the image's premultiplication state from premultiplied to not premultiplied and vice-versa. In premultiplied images, the RGB values of the pixels have been multiplied by their corresponding alpha channel values

Requires an Input (RGBA).

Produces an Output (RGBA).

Premultiply

Operation

Specifies whether to premultiply or unpremultiply the image. Unpremultiplying returns the image to a not premultiplied state.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
