Lagoa Material Highly Breakable

Creates a breakable elastic structure using the point cloud that is emitted.

The parameters exposed for this compound are a subset of those in the Lagoa Main Material compound, mainly in the Inelastic Settings and Elasticity groups.

Plug this compound's Execute output into an Execute port of the Lagoa Phase node.

To create more object-like elements, emit the particles only once at a frame instead of at every frame.

To use this material, do either of the following:

  • Choose the Particles Model Library Lagoa Particles Flexible Structure command from the ICE toolbar to load a model that creates a "necklace" of rigid body particles being hung over a collision object.

  • Choose the Particles Create Lagoa Particles Throw Breakable Structure command from ICE toolbar, which lets you create your own breakable particle emission. See Creating a Breakable Particle Structure for more information.

Tasks: Lagoa/Material Type

Output Ports: Execute
