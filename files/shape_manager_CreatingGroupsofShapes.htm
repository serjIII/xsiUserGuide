Creating Groups of Shapes

Groups let you organize the shapes in the shape list. For example, for a facial animation, you could create a group that contains all shapes that relate to facial expressions and a group that contains mouth shapes for phonemes.

A group is automatically created for each cluster on an object, but you can easily create more groups as you like. Clicking on a group name expands and collapses the list of shapes it contains.

To create a group for shapes

  1. Do either of the following:

    • Choose Edit Create Group on the shape manager's menu bar.

      or

    • Select one or more shapes in the shape list (press Shift or Ctrl), then right-click one of the shapes and choose Create Group.

  2. To move shapes into groups, select one or more shapes and drag and drop them on a group name.

To rename a group

  • You can rename shape groups in the shape list in the same way as other elements in the explorer. See Renaming Scene Elements in the Explorer [Windows and Views].
