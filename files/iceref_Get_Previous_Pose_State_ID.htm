Get Previous Pose State ID

Returns the index of the previous pose state of each actor in a CrowdFX simulation, where 0 is the first imported animation cycle, 1 is the second, and so on. For example, you can use this information to get the previous pose state's deformer transformations at the current frame to perform custom animation blending after changing states.

This compound can be used only in an ICE tree on the crowd simulation Point_Cloud object.

Tasks: Crowds/Getters

Output Ports: Value
