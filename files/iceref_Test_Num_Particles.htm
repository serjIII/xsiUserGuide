Test Num Particles

This compound uses a certain number of particles emitted in a point cloud as a trigger for an effect.

Plug its Result output into a Trigger port on a State compound or in a Condition port of an If node.

For more information on states, see ICE Particle States [ICE Particle Simulations].

Tasks: Particles/Conditionals

Output Ports: Result

Reference

The point cloud's NbPoints attribute.

Test Number Particles

The number of particles in a point cloud to use as the trigger. This uses the point cloud's NbPoints attribute as a reference.

Type

Defines the condition for the Test Number Particles value to be triggered: Equal To; Not Equal To; Less Than; Less than or Equal to; Greater Than; or Greater Than or Equal To.
