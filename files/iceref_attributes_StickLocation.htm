StickLocation

The location to which a particle sticks.

This attribute is set by the Stick to Location, Stick to Surface, and Emit Trail at Surface Intersection compounds when a particle initially sticks to a location.

You can quickly get and set the StickLocation attribute with the Get Particle Stick Location and Set Particle Stick Location compounds.

Data Type

location

Belongs To

points

Geometry Types

  • point cloud

Attribute Type

custom

R/W or RO:

read/write
