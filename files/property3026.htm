Compound Clip Property Editor (Shape)

To display: Select a compound shape clip in the animation mixer and press Enter.

Instancing Mode

The mode used to instantiate shape clips within the compound. This value depends on the mode set in the Deform Shape menu of the Animate toolbar when the compound clip was first created and cannot be changed.

Controls

Auto Transition

Transition curves are automatically created for new shape clips instantiated within the compound shape clip.

Time Span

Start Time

The start time of the compound shape clip in the local time of its parent. This value is provided for information only and cannot be modified from this property editor. To change it, move or scale the clip in the animation mixer.

Duration

The duration of the compound shape clip in the local time of its parent. This value is provided for information only and cannot be modified from this property editor. To change it, move or scale the clip in the animation mixer.
