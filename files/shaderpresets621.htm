Scalars To Vector

| Render Tree Usage

Category: Processing > Conversion

Shader Family: Texture

Output: Vector

Converts scalar input values to a vector output value.

Name

The shader's name. Enter any name you like, or leave the default.

Inputs

Input X, Y, Z

Defines the scalar inputs to be converted to a vector value. Although you can define up to 3 scalar inputs, you may input a single value to be converted as well.

Channels

X, Y, Z Channels

Defines how each of the X, Y, and Z inputs are converted to vector values. Each channel can be told to have No Contribution or to use its value to Put its contribution to a specific axis: X, Y, or Z.

Math Operation

Controls how all of the channels values affect an axis. You can use the following basic math functions: Add, Subtract, Multiply and Replace.

Render Tree Usage

This shader is placed between scalar outputs and a vector input in order for the values to connect.
