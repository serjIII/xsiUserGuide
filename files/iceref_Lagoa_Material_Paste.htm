Lagoa Material Paste

Creates a thick paste material using the point cloud that is emitted, where the particles have strong attraction to each other due to the pressure settings. High viscosity settings gives the material its resistance to flow.

The parameters exposed for this compound are all of those in the Lagoa Main Material compound, just with higher Viscosity and Pressure values.

Plug this compound's Execute output into an Execute port of the Lagoa Phase node.

To use this material, you can choose the Particles Create Lagoa Particles Sticky Paste command from ICE toolbar, which lets you create your own liquid particle emission. This command uses the Lagoa Setup Sticky Paste compound, which contains the Paste material.

See Creating a Sticky Paste for more information.

Tasks: Lagoa/Material Type

Output Ports: Execute
