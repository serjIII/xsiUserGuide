Fake 3D

Allows you to simulate and control a number of 3D lighting effects in a rendered 3D sequence. This operator is designed to be used with 3D sequences whose RGBA channels, Depth channel, and normals have been rendered. Each component has a corresponding input in the operator, and provides a part of the information necessary to simulate 3D lighting.

Requires an RGB(A) Input, a Depth input (Z) and a Normal input (RGB).

Produces an RGB(A) Output

Fake 3D Setup

Depth Fading (Fog)

Activates the depth fading effect.

Start/End Distance

Specifies the start and end distance of the depth fading effect. This is best gauged by previewing the effect in the FX Viewer and experimenting with different values.

Fog Color

Specifies the color of the fog causing the depth fading. Choose any RGB value using the sliders.

Fog Transparency

Specifies the transparency of the fog. The higher the value, the more transparent the fog will be.

3Dlight

Activates the 3D light effect. This simulates a directional light in the image.

x/y/z position

Adjusts the position of the light in each axis. Proper position This is best done by previewing the effect in the FX Viewer and experimenting with different values.

Light Intensity

Specifies the light's intensity. The higher the value, the brighter the light.

Specularity

Activates the specularity effect. This modifies the image's specular highlights using the Light Color.

Light Color

When the Specularity effect is active, this value specifies the color of the light that modifies specular highlights. Choose any RGB value using the sliders.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
