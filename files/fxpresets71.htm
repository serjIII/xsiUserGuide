Craquelure

The Craquelure effect paints the image onto a high-relief plaster surface, producing a fine network of cracks that follow the contours of the image.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

Crack Spacing

Set lower values to make the spaces between cracks smaller or set higher values to make the spaces larger.

Crack Depth

Set lower values to decrease the depth of the cracks created in the image, or set higher values to increase the depth.

Crack Brightness

Set lower values to decrease the brightness of the cracks, or set higher values to increase the brightness of the cracks.
