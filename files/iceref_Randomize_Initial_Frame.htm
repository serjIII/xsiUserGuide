Randomize Initial Frame

Returns a random integer per particle, between 0 and the length of the initial animation cycle in a CrowdFX simulation. You can plug the output into the Initial Frame port of Initialize Locomotion Data or Initialize Stadium Data.

Tasks: Crowds/Initialize

Output Ports: integer

Seed

The seed for the random number generator. Change this value to generate a different random pattern.

Weight

Blends between the initial frame of the clip at 0 and a random value at 1. Decrease this value to bunch the output values more tightly around the initial frame.

To bunch the values around a different frame, add the desired frame number to the output before connecting to Initial Frame.
