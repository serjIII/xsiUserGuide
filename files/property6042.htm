Push Op Property Editor

Moves points in the direction of their normals.

To apply: Select one or more objects, clusters, or points then choose Deform Push from the Model or Animate toolbar.

To redisplay: Select the deformed object then choose Edit Properties Modeling Properties and click the Push Op tab.

Mute

Toggles the deformation on and off without affecting the values of other parameters.

Amplitude

The amount of translation in SOFTIMAGE units. Positive values move points outward in the direction of the normals, negative values move points inward in the opposite direction, and a value of 0 results in no deformation. You can type values outside the slider range.
