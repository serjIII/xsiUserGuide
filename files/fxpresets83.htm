Watercolor

The Watercolor effect paints the image on smooth paper using a medium brush loaded with water and color. The color appears to have dried leaving dark concentrations of pigment around the edges of the daubs.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

Brush Detail

Set lower values to paint with less detail, or set higher values to paint with more detail.

Shadow Intensity

Set lower values to add fewer shadows to the image, or set higher values to add more shadows.

Texture

Select the desired texture level 1 is the smoothest texture and 3 is the roughest.
