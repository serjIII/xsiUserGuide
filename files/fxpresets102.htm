Cutout

The Cutout effect portrays an image as though it was made from roughly cut pieces of colored paper. High-contrast images appear as if in silhouette, while colored images are built up from several layers of colored paper. The Cutout effect is also reminiscent of lino block prints, especially when it is applied to high-contrast original images.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

No. of Levels

Set higher values to increase the number of intensity levels in the original image used to create the pieces of card, or set lower values to decrease the number of levels. The more levels used, the greater the detail in the resulting cutout image.

Edge Simplicity

Set higher values to make the cutout edges smoother, or set lower values to make them rougher.

Edge Fidelity

Set higher values to have the cutout edges represent the actual shapes in the image with greater fidelity, or set lower values for a more distorted representation.
