New Grain

Lets you control the grain size of images at a subpixel level.

Requires an Input (RGB or RGBA).

Produces an Output (RGB or RGBA).

Grain

Red Size, Green Size, Blue Size

Sets the size of the filter applied to each component; these should be set to the size of the grain in each component.

Red Intensity, Green Intensity, Blue Intensity

Set the intensity with which edges are drawn in each component. The higher the value, the greater the intensity.

Film Speed

Choose from among several default film speeds.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
