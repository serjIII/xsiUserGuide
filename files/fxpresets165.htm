Sparkler

Creates a random spark or firework effect. The number and nature of the sparks can be controlled.

Requires an Input (Any).

Produces an Output (Any).

Generation

Birth Rate

Sets the number of sparks generated every frame. Set any floating-point value using the treadmill.

Life Time

Sets how long each spark lives in frames. Set any floating-point value using the treadmill.

Length

Sets the cumulative exposure of the sparks; if set to 3 Â­ you will get what would be seen if a notional camera shutter had been left open for the last 3 frames. Set any floating-point value using the treadmill.

Speed Bias X/Y

Set the bias to the direction of the sparks; these values are added to the initial random directions. Set any floating-point values in pixels per frame using the treadmills.

Link Speed

If animating the center, causes the movement of the center to be added on to the movement of the sparks.

Center X/Y/Z

Set the center of the effect as a normalized proportion of the image. Set any floating-point values using the treadmills. The Z value has no effect.

Movement

Speed

Sets the value around which the speed of the sparks is randomized. Set any floating-point value using the treadmill.

Gravity

Sets the attraction of the sparks downwards or upwards. Set any floating-point value using the treadmill; positive values attract the sparks downwards, negative values attract them upwards.

Air Resistance

Set the air resistance. Set any floating-point value using the treadmill; the higher the value, the less the sparks will radiate outwards.

Wind X/Y

Set a notional directional wind, which affects the trajectory of the sparks. Set floating-point values between -100 and 100 using the treadmills.

Air Resistance must be greater than 0 for Wind to have an effect.

Turbulence

Adds randomness to the movement of the sparks. Set a floating-point value between 0 and 100 using the treadmill.

Glow

Glow Radius

Sets the radius of the glow in pixels. Set a floating-point pixel value between 0.1 and 40 using the treadmill.

Glow Color

Sets the color of the glow.

Core

Core Radius

Sets the radius of the core in pixels. Set a floating-point pixel value between 0.1 and 10 using the treadmill.

Core Color

Sets the color of the core.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
