Boolean Map Lookup

Category: Data > Map Lookup

Shader Family: Texture

Output: Boolean

Extracts or creates a texture map, weight map, or vertex RGBA property and outputs it as a boolean value which can be used to drive other shaders' parameters.

Name

The shader's name. Enter any name you like, or leave the default.

Map

Specifies the property to use. See Specifying Parameter Maps or Vertex Colors in Property Editors [Scene Elements].

Threshold

The map intensity necessary for this shader to output a "true" value.

Render Tree Usage

This shader can be used in conjunction with various tool and mixer shaders, like blend shaders or mix shaders, to drive any other shader's parameters.
