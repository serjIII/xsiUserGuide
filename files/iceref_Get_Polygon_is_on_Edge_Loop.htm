Get Polygon is on Edge Loop

Returns the value of Self.CCWPolygonIsOnEdgeLoop, Self.CWPolygonIsOnEdgeLoop, or their union as set by Get Edge Loop.

See Working with Loops.

Tasks: Topology/Getters

Output Ports: CCW_Polygon, CW_Polygon, Both
