mib_photon_basic

Category: mental ray > Photon

Shader Family: Photon Material

Output: Color

This photon shader supports diffuse reflection and specular reflection and transmission/refraction.

Name

The name of the shader node displayed in the render tree. Enter any name you like, or leave the default.

diffuse

The diffuse color.

specular

The specular color, which controls reflectivity.

transp

The fraction of light transmitted.

ior_frac

The refraction index ratio. It is used if the photon material is transparent.
