Build Rectangle

Builds rectangle shapes for display. To use, connect the output into the Point Array port of Display Line.

Category: Debugging

Output Ports: Result

Center

The position of the shapes' centers in the object's local reference frame. Connect an array of positions for multiple shapes.

Base Vector

The direction that the shapes' X axis points toward.

Axis

A vector that constrains the spin. That vector will be in the YZ plane of the shape.

Dimensions

The X and Z dimensions of the rectangle. Y is not used.

Start and End in Center

Adds a line to the center of the shape.
