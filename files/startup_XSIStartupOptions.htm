Softimage Startup Options

There are a number of startup options that can be used for starting Softimage. These options allow you to perform a variety of tasks. For example, you can control which license token gets grabbed at startup with the licensing options, and which component of Softimage to run, such as batch rendering or scripting. You can also specify a project list or scene to load, install and uninstall add-ons, and set workgroup paths: all at the command line.

To get a list of command-line options you can use when starting Softimage, type:

xsi -h

Refer to the following topics for information about using these command line options:

  • Project lists - See To import a project list on startup.

  • Scene files - See Toopen a scene from the command line.

  • Auxiliary data files - See To apply auxiliary data from the command line.

  • Preference migration - See Migrating from the Command Line.

  • Batch rendering - See Batch Rendering [Rendering].

  • Command line scripting - See Issues in Batch Scripting [SDK Guide].

  • Workgroups - See To connect to a workgroup from the command line [Customizing Softimage].

  • Install and uninstall .xsiAddon and .spdl files using the various plug-in and filter options. For more information about add-ons and how to install and uninstall add-on elements, see Building and Deploying Customizations [SDK Guide].

  • User interface and documentation language - See Language in the general preferences [Preference Reference].
