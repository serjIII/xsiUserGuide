Vertex Color

Category: Data > Map Lookup

Shader Family: Texture

Output: Color value

Extracts or creates a Vertex RGBA map and outputs it as a color value which can be used to drive other shaders' parameters.

Name

The shader's name. Enter any name you like, or leave the default.

Vertex Property

Specifies the property to use. See Specifying Parameter Maps or Vertex Colors in Property Editors [Scene Elements].

Alpha Only

When enabled, only the Vertex Property's alpha channel is used.

Render Tree Usage

This shader can be used in conjunction with various utility shaders, like blend shaders or mix shaders, to drive any other shader's parameters.
