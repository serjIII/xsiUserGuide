Make Control Spline

To display: Select one or more curves, then choose Create Skeleton Create Control Splines from the Animate toolbar.

For more information, see Creating Control Splines for Facial Rigs [Character Animation].

Controller Setup

Divisions

Number of control objects to be created per control spline.

Scale

Size of the control objects.

Control Icon Types

Each of these control types can be set with No Controller, or with Nulls, Cubes, Squares, Pyramids, or Diamond objects:

Point Type

Creates control objects that move the points of the curve and smoothly interpolate without skin stretching.

Path Type

Creates control objects that are path-constrained to the splines at length percentages.

Offset Type

Creates control objects that are used for the translation (for pinching of wrinkles and facial expressions) and rotation (twisting along the curve) of the splines for areas like the lips.

Control Icon Colors

Point/Path/Offset Color

Use the color controls to set the display color for each control icon type.
