Bend Strands Towards Vector

This compound bends the strands towards a vector that you define.

Create strands, then plug this compound's Execute output into a PostSimExecute port on the Simulation Root node or in a port on the ICETree node.

For information, see Shaping the Strands [ICE Particle Simulations].

Tasks: Particles/Strands

Output Ports: Execute

Bend Amount

The amount of the bending to apply to the strand.

Target Vector

The target XYZ direction toward which all the strands try to bend.

Bend Profile

This curve defines how the bend amount is applied to the strand, starting from the strand's base (key on the left) to the tip (key on the right). By default, not much bend is applied to the strand base, but then the full amount is applied by the tip of the strand.
