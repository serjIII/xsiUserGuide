Set Particle Speed

This compound lets you set the particle's speed, using the particle's PointVelocity attribute as a reference. The particle's speed is its distance travelled over time.

Plug its Execute output into the Execute on Emit port of an Emit compound to set this value once, or plug it into the ICETree node to update at every frame.

For more information on particle speed, see ICE Particle Speed [ICE Particle Simulations].

Tasks: Particles/Setters

Output Ports: Execute

Speed

Sets the particle's speed value, which is the number of Softimage units it travels per second.
