Create Symmetry Mapping Template Dialog Box

Creates a symmetry mapping template between deformers on an envelope. This is required if you want to mirror envelope weights symmetrically.

To display: See Mirroring Envelope Weights Symmetrically [Character Animation].

Object Symmetry Template Generation

Controls the symmetry correspondence between deformer objects.

Fill Symmetry Template

Automatically populates the template with mapping rules between deformers. Whether this option is on or off, you can always add, change, and delete rules later.

Symmetry Axis

The desired plane of symmetry: YZ (X = 0), XZ (Y = 0), XY (Z = 0).

Geometry Symmetry Map Generation

Controls the symmetry correspondence between points on the envelope.

Create Symmetry Map

Automatically creates a symmetry map for the points. Uncheck this option if your envelope already has a symmetry map that you want to use for establishing the correspondence between points.
