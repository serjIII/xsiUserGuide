Verlet Wind Force

Applies a wind force to points. This node is designed to be plugged into the Force input of Verlet Framework. See Verlet Integration [ICE Deformations].

Tasks: Deformation/Verlet Forces

Output Ports: Force

Base Value

A 3D vector representing the average wind speed and direction.

Variance

The variance of wind speed and direction around the base value.

Turbulence Scale

The scale of the turbulence field. Larger values spread changes in speed and direction over longer distances.

Animation Speed

The speed at which the turbulence field changes.

Complexity

The "detail" of the turbulence field. Lower values create smoother fields. Larger values create noisier fields but take longer to calculate.
