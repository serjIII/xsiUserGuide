Importing and Exporting FBX Files

The Crosswalk for FBX plug-in allows you to import, export, and merge back scene data using the FBX (*.fbx) file format.

Importing FBX

  1. In Softimage, choose File Crosswalk Import FBX.

  2. The Import FBX File dialog box opens:

    • Select Autodesk (*.fbx) as the file type.

    • Use the file browser to locate the FBX file you want to import.

    • Select the name of the FBX file.

    • Click Ok.

  3. Set the options in the Import FBX Options Dialog Box [Properties Reference]:

    • Set the appropriate import method from the Load Options area on the Import tab.

    • Select the options that apply to your scene from the Import, Animation, and Advanced tabs.

    For more information about what scene elements are supported, see Supported Scene Elements and Attributes for FBX.

  4. Click Ok.

Exporting FBX

  1. Create/Open a scene in Softimage.

  2. Choose File Crosswalk Export FBX.

  3. The Export FBX File dialog box opens:

    • Use the file browser to locate where you want to export your FBX file.

    • Type the name of the .fbx file in the File Name text box.

    • Click Ok.

  4. Set the options in the Export FBX Options Dialog Box [Properties Reference]. For more information about what scene elements are supported, see Supported Scene Elements and Attributes for FBX.

  5. When you are ready to export, click Ok.

Exporting Point Positions as a Geometry Cache

You can plot point positions, such as the result of an envelope or other animated deformation, and export them as a geometry cache.

  1. Before you export, create a group and add the objects that you want cached. For more info about groups in general, see Grouping Objects [Scene Elements].

  2. Export as described in the procedure above. In step 4, make sure to activate Group Cache File(s) and specify the desired Group.
