Transform Group Options

Defines where the transform group null is placed when you select objects and create a transform group that contains and parents them.

For information on transform groups, see Transform Groups (Parenting with Null Objects) [Animation].

To display: Choose File Preferences from the main menu bar, then select Commands Transform Group Options.

Group Pivot

Bounding Box Center

Positions the transform group null in the geometrical center of the selection bounding box of the objects that will be in the transform group. This is the default.

Parent Origin

Positions the transform group null at origin of the selected objects' parent. If the selected objects have no parents, or if they have different parents, the transform group null is placed at the scene origin.
