Set Particle Mass

This compound lets you set the particle's mass, using the particle's Mass attribute as a reference.

Plug its Execute output into the Execute on Emit port of an Emit compound to set this value once, or plug it into the ICETree node to update at every frame.

For more information on particle mass, see ICE Particle Mass [ICE Particle Simulations].

Tasks: Particles/Setters

Output Ports: Execute

Mass

Sets the particle's mass value in kilograms (kg).

Reference

The particle's Mass attribute.
