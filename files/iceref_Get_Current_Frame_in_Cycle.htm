Get Current Frame in Cycle

Shows each actor's current frame in an animation cycle, based on the actor's speed compared to the action's speed in a CrowdFX simulation. It sets the CrowdFX_CurrentFrameInCycle attribute, which you can get and use in an ICE tree.

Tasks: Crowds/Debugging

Output Ports: Result

Show

Toggles the display of the current frame in viewports.

Cycle Index

The ID of the animation cycle, where 0 is the first imported action source, 1 is the second, and so on.
