Get Array Sum

Categories

  • Array

Description

Outputs the sum (addition) of the members of the specified numerical input array.

Ports

Inputs

Array

The numerical array for which the sum of its members is computed.

Outputs

Result

The sum of the members of the specified numerical input array.
