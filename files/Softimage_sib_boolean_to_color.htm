Boolean To Color

Render Tree Usage

Category: Processing > Conversion

Shader Family: Texture

Output: Color

Converts a Boolean input to a color (RGBA) output value which can be used to drive other shaders' parameters.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Specifies the Boolean value to be used.

Render Tree Usage

This shader is simply placed between a Boolean output and a Color input in order for the two values to connect.
