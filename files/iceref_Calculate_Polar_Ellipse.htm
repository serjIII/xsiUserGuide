Calculate Polar Ellipse

Returns the radius corresponding to the input angle on an ellipse.

Category: Trigonometry

Output Ports: Radius

Angle

The angle for which to calculate the radius.

X SemiAxis

The radius of the ellipse along the X axis.

Y SemiAxis

The radius of the ellipse along the Y axis.
