Light Control Property Editor

This property editor is a global control for the intensity and saturation of lights in a light rig.

Coarse Intensity

Alters the intensity of all of the lights in a rig, in large increments. Use this parameter to set the overall intensity level of the light rig and then use the Fine Intensity to tune it.

Fine Intensity

Provides finer control over the intensity level defined by the Coarse Intensity parameter.

Desaturate

Increase this value to remove color from lights in the rig.
