Handles from Velocity Dialog Box

To display: Select a biped guide whose elements have been tagged using the Create Character Biped Tags command, then choose Create Character Biped - Define Step Marks, test out the velocity threshold, and then choose Create Character Biped - Deform Motion from the Animate toolbar.

For more information, see Converting a Biped's Motion to Deformable Tracks [ Character Animation ].

Start and End

The first and last frames of the animation you want to be deformed with the Deform Motion command.

Velocity Threshold

The threshold of the velocity of the animated elements. You need to find the lowest velocity of the animated elements, such as where a foot is on the ground, going from one step to the next. This value needs to be the closest to zero, but not zero.
