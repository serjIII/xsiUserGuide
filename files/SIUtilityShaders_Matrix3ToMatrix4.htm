Matrix3 To Matrix4

Category: Processing > Conversion

Shader Family: Texture

Output: 4x4 Matrix

Converts 3-by-3 matrix input to 4-by-4 matrix output.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Specifies the 3-by-3 matrix input to be converted to 4-by-4 matrix output.

Render Tree Usage

This shader is simply placed between 3-by-3 matrix output and 4-by-4 matrix input in order for the values to connect.
