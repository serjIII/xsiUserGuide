Clean Surface Property Editor

Resamples a surface, reducing the number of control points while retaining the overall shape within the specified tolerance.

To apply: Select a surface and choose Modify Surface Clean from the Model toolbar.

To redisplay: Select the surface, click the Selection button on the Select panel, then click the Clean Surface icon.

Clean in U, V

These options control the directions in which to clean. You can toggle these options to clean only in U, only in V, or both.

U, V Tolerance

The allowable error for resampling the surface. Higher values produce fewer control points but reproduce the original shape less accurately. A value of zero produces no change. You can set separate tolerances for U and V independently.
