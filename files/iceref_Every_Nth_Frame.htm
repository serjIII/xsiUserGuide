Every Nth Frame

This compound uses a specific interval of frames as a trigger for an effect. For example, you can have an effect triggered at every 30th frame.

Plug its Result output into a Trigger port on a State compound or in a Condition port of an If node.

For more information on states, see ICE Particle States [ICE Particle Simulations].

Tasks: Particles/Conditionals

Output Ports: Result

N Value

The interval of frames that is used as a trigger.
