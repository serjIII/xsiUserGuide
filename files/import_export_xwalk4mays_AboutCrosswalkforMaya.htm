About Crosswalk for Maya

Reliable interoperability is the most important part of a mixed-tool pipeline — Crosswalk for Maya helps you seamlessly transfer Softimage content into and out of an Autodesk® Maya® pipeline using the latest dotXSI file format.

Crosswalk for Maya is optimized for dotXSI version 6.0. Supported versions of Maya can read the dotXSI format using the Crosswalk for Maya plug-in. Maya does not need to be installed on the same system as Softimage to enable data transfer.

Getting Crosswalk Updates

To get the latest updates to the Crosswalk for Maya integration, you can download the Autodesk Crosswalk toolset for free from autodesk.com/softimage-crosswalk. The Crosswalk for Maya plug-in files are components of this toolset and they can be installed on any system running a supported Maya installation.

For information on how to enable the full Crosswalk capabilities for a Maya installation, see Crosswalk Setup [Advanced Setup and Configuration].
