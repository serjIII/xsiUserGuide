Rollup

Rolls up the foreground clip to reveal the background clip. The size and direction of the roll-up can be controlled.

Requires a Background (Any) and a Foreground (Any).

Produces an Output (Any).

Rollup

Size

Sets the size of the roll as a normalized proportion of the image. Set a floating-point value between 0 and 1 using the treadmill.

Amount

Sets the proportion of each clip to be displayed. Set a floating-point value between 0 and 1 using the slider; 0 is all of the foreground clip, 1 is all of the background clip. The normal animation would be from 0 at the first frame to 1 at the last frame.

Direction

Sets the direction of the roll-up. Set any floating-point value in degrees using the treadmill; 0 degrees is 3 o'clock, positive values move counter-clockwise.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
