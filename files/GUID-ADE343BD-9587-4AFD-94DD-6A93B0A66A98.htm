Euler to Quaternion

Categories

  • Conversion

  • Rotation

Description

Outputs a new quaternion built from converting the input 3D vector according to the rotation order specified in the Rotation Order parameter.

Parameters

Rotation Order

Rotation order

Ports

Inputs

Rotation XYZ

The 3D vector to convert.

Outputs

Quaternion

The new quaternion.
