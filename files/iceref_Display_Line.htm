Display Line

Displays lines between the positions in an array. To use, plug one of the Build Shape compounds into Point Array. The shapes are displayed if the output of the Result port is pulled, or alternatively you can connect the Execute output port.

Category: Debugging

Output Ports: Result, Execute

Display

Toggles the display on and off.

Select Color

Selects a predefined color.

Point Array

Connect an array of positions in the object's local reference frame.
