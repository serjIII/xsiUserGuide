Rotation to Scalar

Categories

  • Conversion

  • Rotation

Description

Outputs the X, Y, Z and angle components of the input rotation.

Ports

Inputs

rotation

The input rotation.

Outputs

X

The X component of the input rotation.

Y

The Y component of the input rotation.

Z

The Z component of the input rotation.

Angle

The angle component of the input rotation.
