Text Property Editor

| File | Edit | Formatting Controls | Auto | Apply | Text Entry Box | Lines

Creates formatted text to be converted to curves or polygon mesh.

To apply: Choose any item from the Create Text menu on the Model toolbar.

To redisplay: Select the converted curve or polygon mesh, click the Selection button on the Select panel, then click the Text icon (under the Text to Curve Converter).

File

Manages Rich Text Format (RTF) files.

New

Clears the text entry box.

Open

Select an RTF file to import.

Save

Save the contents of the text entry box as an RTF file.

Save As

Save the contents of the text entry box as an RTF file with a new name.

NOTE:Softimage does not maintain a reference to RTF files on disk. If you change an RTF file outside of Softimage, it is not automatically reflected here. You must re-open the changed file.

Edit

Standard editing commands: Undo, Cut, Copy, Paste, Clear.

Formatting Controls

 

Typeface of selected text. TrueType and OpenType fonts (*.ttf) are supported.

On Linux, font files must be located in the $MWHOME/fonts directory. For WYSIWYG display in the text editing box, you also require a corresponding .bdf file.

 

Size of selected text in points.

 

Bolds selected text.

 

Italicizes selected text.

 

Alignment of the selected lines: Left, Centered, or Right.

Auto

Applies changes in the Text property editor to the scene automatically as you work. There is a brief delay to ensure you have finished typing, to avoid performance problems.

Apply

Applies changes in the Text property editor. Use this button to manually update the scene if Auto is off.

Text Entry Box

This is where you type your text. If Multiple Lines is on, press Enter to start a new line.

Lines

Controls how lines are treated.

Multiple Lines

Allows text on multiple lines.

Single Line

Allows text on a single line only. If you import an RTF file, only the first line is shown and character formatting stored in the paragraph mark is not displayed.

Width

The width of lines in points, when Multiple Lines is on.
