Filter by Volume

This compound reduces the number of particles based on them being within an object's volume. The volume object must intersect with the emitter in order to emit particles.

Plug this compound's Execute on Emit output into an Execute on Emit port of the Emit compound or into a port on the ICETree node. If you do the latter, the emission is updated at each frame.

For more information, see Filtering Particles Based on an Object's Volume [ICE Particle Simulations].

Tasks: Particles/Emission Control

Output Ports: Execute_on_Emit

Volume1

The object within which the particles must be to be emitted. This must be an object that has volume (3D geometry). Plug the object's Value output into this port.

Invert

Inverts the falloff effect (Distance and Profile curve).

Use Falloff

Uses the Falloff Distance and Falloff Profile curve information to determine how particles are filtered.

Falloff Distance

The maximum distance (in Softimage units) from the volume object in which the particles are emitted.

Falloff Profile

This graph determines the falloff of the particle emission starting from the volume object.
