Interoperability with Mudbox

The "Send to" commands let you transfer data between Softimage and Autodesk Mudboxâ„¢. You can send a polygon mesh from Softimage to Mudbox, then paint textures and sculpt, and finally send the result back to Softimage. The following map types are supported:

  • normal
  • displacement
  • vector displacement
  • ambient occlusion
  • specular
  • shiny
  • incandescence
  • bump
  • reflectivity
  • diffuse

NOTE:Softimage does not support Ptex.

Overview of Using Mudbox to Create Maps for Softimage

Here is an overview of a typical way of using Softimage and Mudbox together:

  1. Send base meshes from Softimage to Mudbox.

  2. Use Mudbox to paint textures, sculpt the geometry, and/or define shapes.

  3. Send the objects and their maps back to Softimage.

  4. Continue to send data back and forth as you refine the result. You can keep multiple versions of the Mudbox and Softimage scenes as you test different settings by saving the scenes under different names, loading them manually, and updating.

NOTE:When extracting any type of map, activate Smooth Target UVs if both of the following are true:
  • The object will be subdivided in Softimage using a geometry approximation property.
  • Smooth when subdividing is activated in the object's Texture Projection property.

Working with Base Meshes

Sending Base Meshes from Softimage to Mudbox

  1. Use Softimage to create base meshes for the objects that you want to paint or sculpt.

  2. Apply a material to the objects. Phong, Lambert, and Blinn are supported.

  3. Apply texture projections to the objects. Mudbox requires that objects have texture UV properties.

  4. For best results, freeze the objects and their projections.

  5. Select the objects and choose File Send to Mudbox Send as new Mudbox scene.

    If Mudbox is not already running then it starts, or if it is already running then it gets focus and you are prompted to save the current scene as necessary. After that, the object sent from Softimage is imported into a new scene in Mudbox. If the Mudbox window does not come to the front automatically, then you may need to switch to it manually.

Updating Base Meshes from Softimage

You can make changes to the base meshes in Softimage and then update them in Mudbox. You can also add base meshes to the Mudbox scene.

  1. In Softimage, select the base meshes to update or add. Don't forget to make sure that they have texture projections, and freeze them for best results.

  2. Do one of the following:

    • Click Update or choose File Send to Mudbox Update current Mudbox scene.

      If the objects already exist in the Mudbox scene, they are updated. If they do not exist, they are added.

    • Choose File Send to Mudbox Add to current Mudbox scene. If any of the selected objects already exist in the Mudbox scene, additional copies are created.

  3. In Mudbox, adjust the Transfer Details if necessary and click Import.

Using Base Meshes from Mudbox

If you want to start with any of the base meshes from Mudbox, such as the tRex, you can create them in Mudbox and perform some initial painting or sculpting, and then send them to Softimage using either Send Selected as New Scene or Add Selected to Current Scene from the File Send to Softimage menu.

Sending Painted Textures from Mudbox to Softimage

If you just want to use Mudbox to paint texture maps and don't want to do any sculpting, follow this procedure.

  1. In Mudbox, create paint layers on the base meshes and paint on them as desired.

  2. To send the result back to Softimage, click Update at the lower left of the Mudbox interface or choose File Send to Softimage Update Current Scene.

    If you just want to send the painted maps and not any changes to the mesh, then choose Update Textures in Current Scene instead. However, if you have done any sculpting, note that modifications to the base mesh are also sent back to Softimage.

TIP:Choose File Send to Softimage Preferences to control how paint layers are flattened.

Sending Sculpt Maps from Mudbox to Softimage

Because Mudbox modifies the base mesh when sculpting, you should extract maps onto a copy of the base mesh for best results.

  1. In Softimage, select the base mesh and choose File Send to Mudbox Send as new Mudbox scene. The object opens in Mudbox.

  2. Switch back to Softimage, and with the object still selected choose File Send to Mudbox Add to current Mudbox scene. There are now be two copies of the object in Mudbox.

  3. In Mudbox, lock or hide the first object so that you don't accidentally alter the base mesh.

  4. Select the second object and add subdivision layers as necessary.

  5. Sculpt the second object.

  6. Extract the desired maps using the second object as the source and the first object as the target.

    Vector displacement and normal maps require tangents. For vector displacement maps, choose one of the following options for Vector Space:

    • World can be used for objects that are completely static.
    • Object can be used for objects with animated transformations but no deformation.
    • Absolute Tangent should be used for objects with deformations.
    Relative Tangent is not supported in Softimage.

    For normal maps, set Coordinate Space to Tangent. The other options are not supported.

  7. Select the first object and click Update at the lower left of the Mudbox interface. The object is updated in the current Softimage scene with the maps applied.

Sending Sculpt Layers as Shapes to Softimage

IMPORTANT:Any existing shape clips and sources in Softimage will be lost.

  1. In Mudbox, create sculpt layers and modify their shape as desired.

  2. Choose File Send to Softimage Preferences and make sure that Export sculpt layers on current mesh level as Shapes is enabled.

  3. Select the first object and click Update at the lower left of the Mudbox interface. The object is updated in the current Softimage scene with a separate shape clip for each sculpt layer. You can use the shape manager to animate the blend between the shapes.
