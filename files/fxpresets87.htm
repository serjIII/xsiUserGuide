Colored Pencil

This effect draws the image with colored pencils on a solid background. Strong edges are retained and give a rough cross-hatched appearance, while the solid background is allowed to show through in the smoother areas of the image.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

Pencil Width

Set higher values to make the pencil stroke wider, or set lower values to make the pencil stroke narrower.

Stroke Pressure

Set higher values to increase the pressure and intensity of the pencil stroke, or set lower values to decrease the pressure and intensity.

Paper Brightness

Set higher values to make the background paper brighter, or set lower values to make it darker.

Foreground Color

Sets the color for foreground regions of the effect. Set a color using the sliders.
