Vector4 To Vector2

Category: Processing > Conversion

Shader Family: Texture

Output: 2D Vector

Converts 4D vector input to 2D vector output.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Specifies the 4D vector input to be converted to a 2D vector output.

Render Tree Usage

This shader is simply placed between 4D vector output and 2D vector input in order for the values to connect.
