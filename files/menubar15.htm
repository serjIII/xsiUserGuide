dotXSI Import Options Dialog Box

Specifies what attributes to include when importing dotXSI (.xsi) files. This importer supports dotXSI versions 3.0, 3.5, and 3.6.

To import: Choose File Import dotXSI from the main menu.

Animation

Imports animation included in the dotXSI file.

Classical Scaling

When on, uses "Classical Scaling" on all imported objects. When off, uses "Hierarchical (Softimage) Scaling" on all imported objects. [Default: On]

Material

Imports materials defined in the dotXSI file.

Texture

Imports textures contained in the dotXSI file.

Normal

Imports user-defined normals into a UserNormals user data map.

Import linear Fcurves as RAW

Converts linear function curves to raw function curves when importing dotXSI files. This can significantly improve the performance and memory usage, and is especially useful when importing files that were exported from SOFTIMAGE|Behavior.
