Fast Blur

Creates a blur that is faster than a Gaussian blur, but of a higher quality that a Box blur.

Requires an Input (Any).

Produces an Output (Any).

Filter

Size

Sets the size of the blur box. The larger the box, the greater the blurring effect of the filter.

Blur Type

Specifies the blur's direction.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
