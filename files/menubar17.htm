Acclaim Import Dialog Box

Specifies which Acclaim Skeleton (ASF) and Motion Capture files (AMC) to import and how they are to be imported.

The ASF file contains hierarchy information and the base pose of the skeleton. This information is used to create a hierarchy of nulls (the skeleton) within a model when imported into Softimage.

The AMC file contains the motion capture data as keyframed animation that can be applied to the skeleton created by the ASF file. This AMC file must be compatible with the ASF file you are importing.

To display: Choose File Import Acclaim File from the main menu bar.

For more information, see Importing Acclaim Files [Character Animation].

File Input

Skeleton File

Specify the path and file name for the Acclaim Skeleton (ASF) file to be imported. Click the Browse (...) button to select a specific location and file name from a browser.

Motion File

Specify the accompanying Acclaim Motion Capture (AMC) file to be imported along with the ASF file. The AMC file must be compatible with the ASF file you're importing. Click the Browse (...) button to select a specific location and file name from a browser.

Skeleton Description Mode

Bone Chains

Creates a standard skeleton hierarchy made of bones. You can then set the Bones Skeleton Options to define the skeleton setup.

Hierarchy of Nulls

Creates a skeleton hierarchy of nulls.

Bones Skeleton Options

If you selected Bone Chains as the Skeleton Description Mode, you can choose how you want the skeleton's bones to be connected and what type of chain structure to create.

Hierarchy

Choose how you want the skeleton's bones to be connected:

  • Connect with unanimated bones: All bones, animated or not, are connected as part of the skeleton hierarchy.

  • Leave unconnected: Bones that are not animated are not connected to the skeleton hierarchy.

Chains

Select which structure you want for the resulting skeleton hierarchy:

  • SI|3D Mode: Creates SOFTIMAGE|3D-style chains whose effectors are children of the last bone of the chain.

  • XSI Mode: Creates default XSI-style chains whose effectors are children of the chain root.

Import Options

Crop Motion Values to Limits

Applies the DOF (degrees of freedom) limits in the ASF file to the animation in the accompanying AMC file. The DOF limits are the minimum and maximum values allowed for the XYZ translation and rotation, as well as movement along the bone. Any values outside the minimum/maximum range are cropped.

Override Bone Length Multiplier

Overrides the bone length multiplier information in the ASF file to be applied to the animation in the accompanying AMC file.

If you select this option, you must then set the value with the Length Multiplier slider.

Length Multiplier

This slider is active only when you select Override Bone Length Multiplier. This is the length (in Softimage units) of the bones.

Add Motion Action to Mixer

Adds the accompanying keyframed motion as an action clip in the animation mixer.

Group Animated Objects

Creates a group that includes all animated objects in the skeleton hierarchy.
