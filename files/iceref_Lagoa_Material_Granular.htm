Lagoa Material Granular

Creates a grain-like inelastic material using the point cloud that is emitted, allowing for rigid bodies that can pile up on each other because of high surface friction.

The parameters exposed for this compound are a subset of those in the Lagoa Main Material compound, basically all parameters except for those in the Elasticity group.

Plug this compound's Execute output into an Execute port of the Lagoa Phase node.

To use this material, choose the Particles Model Library Lagoa Particles Inelastic command from the ICE toolbar, which loads a model that creates rigid body particles being poured onto a collision object's surface.

As well, the Lagoa Setup Grains compound contains this material.

Tasks: Lagoa/Material Type

Output Ports: Execute
