Vector To Scalar

| Render Tree Usage

Category: Processing > Conversion

Shader Family: Texture

Output: Scalar

Converts vector input information to a scalar output.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Specifies the vector inputs to be converted to a scalar output.

Render Tree Usage

This shader is simply placed between a vector output and a scalar input in order for the two values to connect.
