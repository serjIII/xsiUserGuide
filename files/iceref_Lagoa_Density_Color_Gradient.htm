Lagoa Density Color Gradient

Sets the gradient color of particles based on their density. This helps to visualize the particle density for testing and troubleshooting purposes. You can also transfer this color to the polygonized mesh so that the borders of the fluid and its interior have a different color according to their density.

To apply: See Applying Cached Post Sim Effects [Lagoa Multiphysics Simulations].

Tasks: Lagoa/Cached Sim Post Effect

Output Ports: Execute

Gradient

The color gradient used for the particles based on what is set for the Min/Max Density values.

See the Gradient node for more information.

Min Density

Minimum particle density value to be affected by the color gradient. This value corresponds to the leftmost color on the gradient.

Max Density

Maximum particle density value to be affected by the color gradient. This value corresponds to the rightmost color on the gradient.
