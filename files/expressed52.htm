Floor (Expression Function Reference)

The Function Value Floor command in the expression editor inserts the token floor( at the insertion point in the editing pane. This token returns a value that is rounded down. Enter a value after the opening bracket ( then enter a closing bracket ).

Example

floor(2.4)

Rounds the value down to 2.
