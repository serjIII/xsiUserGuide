Creating Key Maps

You can create and modify your own key maps. However, you cannot modify or remove the default key maps installed with Softimage.

To create a key map

  1. Select the Key Map on which you want to base your new map.

  2. Click New and give the key map a name.

    A new key map is created and automatically selected.

To remove a key map

  1. Select the Key Map you want to remove.

  2. Click Remove.
