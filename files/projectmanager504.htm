The Project Manager

| Projects

A project is a file folder containing subfolders for your scene files and related elements. The Project Manager is the tool that lets you manage multiple projects and scenes. You can create new projects and scenes, open existing projects and scenes, scan your system for existing projects, and manage project lists. To work in Softimage, you must create a project or open an existing one.

To display the Project Manager

  • Choose File Project Manager from the main menu.

    A

    Select a project from your project list.

    B

    Create and name a new project folder.

    C

    Sets the selected project as default. When Softimage is started, the default project is automatically opened.

    D

    Open scenes associated to the selected project.

    E

    Sort projects in the list by Name, Origin (factory [F], user [U], and workgroup [W]), or none (no sorting).

    F

    Indicates the path to the selected project. The project path is displayed for information purposes only and cannot be modified.

    G

    Sets the selected project in the project list as the active (current) project. The active project is displayed in the titlebar at the top of the Softimage interface.

    H

    Deletes the selected project.

    The associated folder and its files are removed from the hard disk.

    I

    Use the Project List options to manage your project lists.

    For example, Scan Disk searches for projects in a specified path and adds them all to the project list.

    J

    Create a new scene within the selected project folder.

    K

    Delete the selected scene from the project folder.
