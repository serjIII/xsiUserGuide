Apply Slice Between Two Vertices

Tasks: Topology/Modifiers

Output Ports: Execute

Adds a range of edges intersecting all other edges between two points on a polygon mesh.

To apply:

  1. Select two vertices on a polygon mesh.

  2. Choose Modify Poly. Mesh Apply Slice Between Two Vertices from the Model toolbar.

    If you did not select two vertices in the previous step, you are now prompted to pick an object and then pick two vertices.

    The operator is applied. In addition, a null named Slice_Up_Vector is added as a child of the mesh and automatically selected.

  3. Transform the null to control the path of the new edges.

       

To redisplay:

  • Expand the object's construction stack in an explorer and click the Apply_Slice_Between_Two_Vertices icon.

Start, End Vertex Index

The indices of the two vertices.

Up Vector Name

The object used to define the cutting plane.

Limit to Left and Right of Cut Plane

Limits the new edges to those that are completely between the start and end vertices in the cutting plane. When this option is off, the new edges might not reach the specified vertices if they take the "long way" around the mesh.

Limit to Bottom Half of Cut Plane

Limits the new edges to those that are on the same side of the line between the vertices in the cutting plane as the up vector.

Display options

Various options to help visualize the effect.
