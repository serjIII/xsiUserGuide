Material Panel Preferences

The settings you make in the material panel are stored in its preferences and re-used the next time you open it. For more information about the material panel, see Using the Material Panel [Textures and Maps].

If you make changes directly in the preferences, they will not be used until you restart Softimage.

Mode

  • Materials and Layers displays the layer panel beneath the selected objects' materials.

  • Materials and Image Clips displays the list of clips below the selected objects' materials.

Thumbnail Size

Sets the size of the thumbnails for the selected objects' materials.

Name Display

Determines the names listed next to the thumbnails for the selected objects' materials.

Hide Clusters Without Material

Displays only clusters with locally applied materials.

Hide Layer Panel

Hides the panel for editing layers of the selected material.

Hide Image Clip Panel

Hides the panel that list image clips of the selected object.

Hide Texture Space Panel

Hides the panel that allows you to see and modify the texture projection used by the selected layer or clip.

Hide Layer Ports Panel

Hides the panel that allows you to see and edit the ports used by the selected layer.
