2D Depth Cue

DEPRECATED. This shader is unsupported, but it may still get installed with Softimage to provide compatibility with older scenes that use it. It is recommended that you replace unsupported shaders in your scenes with equivalent shaders from the current Softimage shader library.

| Render Tree Usage

Shader Type: Output

Output: Color (RGB) value

Creates a 2D layer of fog between two specified distances. This effect does not support layered fog.

Fog

Bias

Defines the fog's contrast.

Strength

Controls the fog's intensity.

Color Background

Defines the background color. Note that you can use the RGBA, HSVA, or HLSA color models to set the background color.

Near

Near Distance

Specifies the distance between where the fog begins (Near Color). Fog appears only between the near and far distances.

Near Color

Sets the fog's start color.

Far

Far Distance

Specifies where the fog ends (Far Color).

Far Color

Sets the fog's far color.
