Spread From Controller

This compound emits particles away from a controller object or a specific location (no need for a controller object) that you select. The effect created by this compound is useful for creating particle explosions, water hoses, and similar types of simulations. You may want to use it in conjunction with the Emit Blast compound.

Plug this compound's Execute on Emit output into the Execute on Emit port on the Emit compound.

For more information on particle direction, see Spreading the Particles Away from a Controller [ICE Particle Simulations].

Tasks: Particles/Emission Control

Output Ports: Execute_on_Emit

Spray Origin Method

The method by which you want to control the particle emission:

  • Spray Away from Controller Object Center. You must plug in that object in the Spray Origin Controller port.

  • Spray Away from Controller Object Surface. You must plug in that object in the Spray Origin Controller port.

  • Spray Away from Specified Position. You must enter the values below for Spray Origin Position.

Spray Origin Controller

The object to use as the controller. This object is used only if you selected Spray Away from Controller Object Center or Surface as the Spray Origin Method. Plug the object's Out Name output into this port.

Spray Origin Position

The XYZ coordinate in global space that determines the point from which the particles are repelled. These values are used only if you selected Spray Away from Specified Position as the Spray Origin Method.

Amplitude

The strength of the force moving the particles away from the controller object or specified position.
