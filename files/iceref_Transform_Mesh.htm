Transform Mesh

Deforms an object by transforming its point positions.

Despite its name, this compound can be applied to any geometry type including polygon meshes, curves, NURBS surfaces, lattices, and point clouds.

Tasks: Deformation/Effects

Output Ports: Execute

Scale

The scaling to apply to the point positions.

Rotate

The rotation to apply to the point positions.

Rotation Order

The order in which to apply the rotation.

Translate

The translation to apply to the point positions.

Scale Translation by bounding box

Adjusts the amount of translation relative to the object's overall size.
