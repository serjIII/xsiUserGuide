Soft Min

Returns the input value down to the specified minimum value, with an ease-in as the input approaches the limit.

Category: Basic

Output Ports: Result

Value

The input value.

Min

The minimum value to return.

Softness

The ease-in of the result as the input value approaches the limit. When this value is 0.0, the result becomes clamped abruptly when the input value is less than the minimum.
