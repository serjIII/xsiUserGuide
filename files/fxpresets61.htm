Bump Shade

Creates a relief texture on an image by applying light and shade to the image according to changes of intensity in the mask.

Requires an Input (RGB or RGBA) and a Bump Matte (A).

Produces an Output (RGB or RGBA).

Bump

Bump Scale

Sets the scale of the effect. The larger the value, the greater the effect. If a negative value is given, the bump will be recessed.

Light Color

Sets the color of the light.

Light Dir

Sets the angle of the light to be used when calculating the light and shade. At zero degrees the light comes from 9 O'clock, positive values move counterclockwise.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
