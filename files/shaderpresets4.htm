Color Interpolate

DEPRECATED. This shader is unsupported, but it may still get installed with Softimage to provide compatibility with older scenes that use it. It is recommended that you replace unsupported shaders in your scenes with equivalent shaders from the current Softimage shader library.

| Render Tree Usage

Shader Type: Mixer (Tool)

Output: Color (RGB) value

Takes a scalar value and performs an interpolation of a color map. The map can be up to eight colors or textures and is set by a specific parameter.

Name

The shader's name. Enter any name you like, or leave the default.

Output: Color value

Input

The scalar value used to compute an intensity level.

Number of Colors

Defines how many colors the shader will calculate (1-8).

Color_0, 1, 2, 3, 4, 5, 6, 7

Defines the color for each position from 0 to 7.

Weight_1, 2, 3, 4, 5, 6

Defines the location of colors 0-7 within the color spread.

0 = low end; 1 = high end.

Render Tree Usage

This shader can be used almost anywhere in a render tree to interpolate the output of up to eight shaders to create a single value. It is most often used to blend several surface shaders or many textures image clips. You can also use a texture image clip to drive the various Weight parameters in order to mix the surfaces or textures in a predictable or specific pattern.
