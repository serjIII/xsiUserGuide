Test Particle Age

This command uses the particle's age value as a trigger for an effect.

Plug its Result output into a Trigger port on a State compound or in a Condition port of an If node.

For more information on states, see ICE Particle States [ICE Particle Simulations]; for information on particle age, see ICE Particle Age [ICE Particle Simulations].

Tasks: Particles/Conditionals

Output Ports: Result

Select Value Type

Particle age value (Age attribute) is in seconds or frames.

Test Age

The particle's age in seconds or frames to use as a trigger.

Type

Defines the condition for the Test Number Particles value to be triggered: Equal To; Not Equal To; Less Than; Less than or Equal to; Greater Than; or Greater Than or Equal To.

Epsilon

Widens the range of acceptable values by a small amount so as to reduce rounding errors.
