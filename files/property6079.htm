Inverse Curve Property Editor

Inverts a curve so that its maximum U endpoint becomes its minimum U endpoint.

To apply: Select a curve then choose Modify Curve Inverse from the Model toolbar.

To redisplay: Select the curve, click the Selection button on the Select panel, then click the Inverse Curve icon.

Inverse Curve

Swaps the start and end points of the curve. When this is on, the curve is parameterized in the opposite direction.
