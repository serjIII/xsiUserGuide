Rig to Rig Property Editor

Retargets the animation from a tagged source rig to a tagged target rig.

To display: Select any part of the tagged source skeleton or rig, press Ctrl and select any part of the tagged target skeleton or rig, then choose Tools MOTOR Rig to Rig from the Animate toolbar.

For more information, see Retargeting Animation from Rig to Rig [ Character Animation ].

Model Rigs

Source

The name of the source rig/model that has the animation that you want to retarget. Click the Pick button and pick a different model if the one displayed here is not correct.

Target

The name of the target rig/model to which you want to retarget the animation. Click the Pick button and pick a different model if the one displayed here is not correct.

Range Settings

Start and End

The range of frames of the source rig's animation that you want to retarget onto the target rig.

Apply

Retargets the animation from the source rig to the target rig.

Adjust

Opens the Retarget Adjust Property Editor in which you can adjust the retargeted motion on the target rig.

Remove

Removes the retargeted animation from the target rig.

Plot

Opens the Plot Retarget Dialog Box in which you can plot the retargeted motion on the target rig into fcurves.
