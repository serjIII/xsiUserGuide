Set Animation State Parameters

Categories

  • Crowds > Animation

Description

Sets the parameters of an Animation State. For example, if the Move state is defined with two animations (Walk and Run), then the parameters are:

  • Weights — array of two scalar values representing the weights of Walk and Run. These weights are used in blending the two animations.
  • Speed — single scalar value representing the playback speed of the Move state

Parameters

Reference

Reference to simulation Point_Cloud

State

String value representing the name of the Animation State

Ports

Inputs

In Name

Name of the simulation Point_Cloud

Weights

Array of scalar values representing the weight of each animation used for the Animation State

Speed

Playback speed of the Animation State

Outputs

Execute

Sets the animation states

Out Name

Output name of the simulation Point_Cloud
