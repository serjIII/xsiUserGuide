Surface Deform Op Property Editor

Deforms an object by mapping its local XZ space is mapped to the UV space of a surface.

To apply: See Deforming by Surfaces [Data Exchange].

To redisplay: Select the object, click the Selection button on the Select panel, then click the Surface Deform Op icon.

Transforms

Mute

Toggles the deformation on and off without affecting the values of other parameters.

Scaling, Rotation, Translation XYZ

These parameters define offsets for the deformed object relative to the deforming surface. Adjust them until the deformed object is in the correct position relative to the deforming surface.

Constraint

These options are useful if either the deformed object or the deforming surface were moved from the global scene center before the deformation was applied.

Constrain to Deformer

Select this option if the deformed object was moved from the global center before the deformation was applied.

Constrain to Deformee

Select this option if the deforming surface was moved from the global center before the deformation was applied.
