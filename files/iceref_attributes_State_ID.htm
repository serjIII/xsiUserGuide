State_ID

The current state of a particle.

This attribute is set and used by the States compounds, and is exposed in the State compound property editor as the State ID parameter.

You can quickly get the State_ID attribute with the Get Particle State ID compound.

Although you can get and set this attribute manually, it's probably easier and less error-prone to use the state system implemented by the factory compounds.

Data Type

integer
