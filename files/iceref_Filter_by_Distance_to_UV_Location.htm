Filter by Distance to UV Location

This compound reduces the number of particles based on their distance to a UV location on a NURBS surface or curve object. Particles are emitted only when they are within a certain distance of this location on the object.

Plug this compound's Execute on Emit output into an Execute on Emit port of the Emit compound or into a port on the ICETree node. If you do the latter, the emission is updated at each frame.

For more information, see Filtering Particles Based on Distance to a UV Location [ICE Particle Simulations].

Tasks: Particles/Emission Control

Output Ports: Execute_on_Emit

Geometry

The NURBS surface or curve object within which the particles must be to be emitted. Plug the object's Value output into this port.

UV

The location of the UV on the object NURBS curve or surface.

Distance Threshold

The maximum distance (in Softimage units) in which the particle considers the object for particle emission.
