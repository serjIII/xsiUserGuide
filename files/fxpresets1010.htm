Wipe Shape Property Editor

| Shape | Mask | Display

To Display: Select a wipe shape in the FX Viewer or in the explorer and press Enter.

In vector paint layers, the FX Viewer Paint menu's Draw > Clear Image command deletes all shapes in the layer and creates a wipe shape. By default, the layer is wiped to black, but you can edit the wipe shape's properties to change the color and other properties.

Shape

Color

Specifies the color of the wipe. This color fills the entire image. Set any RGBA value using the sliders.

Mask

Paint On

RGB

When active, the wipe shape is painted on the image's RGB channels.

Alpha

When active, the wipe shape is painted on the image's alpha channel.

Obey Paint Mask

The paint mask is a matte or grayscale image that defines which areas of an image are paintable, and which areas aren't (sort of like an obey matte for painting). You can set any operator as the paint mask from the operator's contextual menu.

Enable

When active, parts of the image that correspond to black parts of the mask are protected from the wipe shape.

Invert

When the Obey Paint Mask > Enable option is activated, activating this option inverts the mask so that parts of the image that correspond to white parts of the mask are protected from the wipe shape.

Display

Name

Displays the shape's name in the explorer. You can change it by typing a new name in the text box.

Visible

Controls the shape's visibility. Disabling this option mutes the shape in the FX Viewer.
