Spine Maker Keys Dialog Box

To display: Choose Create Skeleton Create Spine from the Animate toolbar, then pick the hip and chest control objects you have created.

For more information, see Creating a Spine [ Character Animation ].

Number of vertebrae

Number of vertebrae nodes to be created in the spine control.

Implicit vertebrae

Creates the vertebrae nodes from little implicit cubes instead of polygon cubes.
