Importing and Exporting DirectX Files

You can import and export scenes saved in the DirectX (*.x) file format.

Importing DirectX Files

To import a DirectX file

  1. Choose File Import DirectX from the main menu.

  2. Set the options in the DirectX Import Dialog Box.

  3. In the File text box, enter the name and path of the file you wish to import. Alternatively, you can click the More button (...) to browse for the file.

  4. Click OK. The imported objects are added to the current scene as a new model.

Exporting DirectX Files

The DirectX exporter exports the entire scene, or at least those elements of it specified in the export options. This means that you do not have to select specific objects to export.

To export a DirectX file

  1. Choose File Export DirectX from the main menu.

  2. Set the options in the dotXSI Export Options Dialog Box.

  4. Click OK. The scene is exported to the specified .x file.
