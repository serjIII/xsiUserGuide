Lagoa Setup Sticky Paste from Volume

Contains all the necessary Lagoa nodes to create a pasty particle emission. The default results are a thick paste that sticks easily to a collision object.

Plug this compound's Execute output into a port on the ICETree node or in an Execute port on a Simulation Root node.

Tasks: Lagoa/Presets

Output Ports: Execute

This compound is the same as Lagoa Setup Sticky Paste except that you need to use a volume object as the emitter object instead of a null object.

Emitter In Name

Plug in a volumetric object's Out Name output here. The particles fill this volume when emitted.

You can also plug a group node here, with each volumetric emitter object defined in the group.
