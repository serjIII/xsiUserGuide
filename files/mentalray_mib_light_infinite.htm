mib_light_infinite

Category: mental ray > Light

Shader Family: Light

Output: Color

Infinite (directional) lights cast parallel rays in the light direction. The origin is infinitely far away (and left unspecified), and there is no distance attenuation.

Name

The name of the shader node displayed in the render tree. Enter any name you like, or leave the default.

color

The light color.

shadow

When on, enables shadows. The light will be diminished or blocked by occluding objects between the light and the illuminated object.

factor

Only has an effect if shadows are enabled. Blends from the shadow color (0, the default) to the light color. Values greater than 0 (zero) let the light penetrate occluding objects. At 1, no shadows appear.
