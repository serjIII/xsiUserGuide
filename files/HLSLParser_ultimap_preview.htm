ultimap_preview

Category: Realtime > HLSL

Shader Family: Realtime

Output: Realtime

The ultimap_preview shader is used for previewing Ultimapper results. The maps are applied to the low-resolution object via an automatically generated render tree. The HLSL version of the ultimap_preview shader is used when you set the Ultimapper Display option to DirectX. For more information, see Transferring Surface Attributes (Ultimapper) and Previewing Ultimapper Results [Textures and Maps].
