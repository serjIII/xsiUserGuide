Color Mix

Mixes the clip input with a color wash.

Requires an Input (RGB).

Produces an Output (RGB).

Mix Color

Mix Color

Sets the color to mix with the input clip.

Mix Setup

Mix Level

Specifies the amount of mixing, depending on the Mix Type setting.

Mix Type

Choose one of the following:

Mix Up: mixes the color with the image such that higher Mix Level values increase the amount of mixing.

Mix Down: mixes the color with the image such that lower Mix Level values increase the amount of mixing.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
