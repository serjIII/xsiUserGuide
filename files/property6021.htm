Fillet Curves Property Editor

Creates an arc-shaped fillet between two intersecting curves.

To apply: Select the first curve then choose Create Curve Fillet Intersection from the Model toolbar and pick the second curve.

To redisplay: Select the fillet curve, choose Edit Modeling Properties and click the Fillet Curves tab.

Inputs

Hide/Unhide

Hides all the inputs for the generated object to simplify the display of your scene in the 3D views, or unhides the inputs when you need to modify them.

Delete

Freezes the generator and deletes the inputs. This simplifies the scene once you are satisfied that you won't need to modify the inputs again.

Radius

The radius of the arc that joins the two curves in SOFTIMAGE units. When this value is too large, the fillet is a unit circle at the global center.
