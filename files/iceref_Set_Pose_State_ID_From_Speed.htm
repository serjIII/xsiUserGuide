Set Pose State ID From Speed

Sets CrowdFX_PoseState_ID (the actors' current animation) based on the actors' speed in a CrowdFX simulation. It uses Walk Min Speed and Run Min Speed set in Initialize Locomotion Data.

This compound should be applied in the Simulation region of the Point_Cloud object of a CrowdFX simulation, before Pose State. Make sure to use Save Previous Pose State ID before this compound.

For more information, see How Action Sources Are Used in the Crowd Simulation.

Tasks: Crowds/States

Output Ports: Execute
