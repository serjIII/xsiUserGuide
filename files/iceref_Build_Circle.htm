Build Circle

Builds circles, arcs, or pie slices for display. To use, connect the output into the Point Array port of Display Line.

Category: Debugging

Output Ports: Result

Center

The position of the shapes' centers in the object's local reference frame. Connect an array of positions for multiple shapes.

Base Vector

The direction that the shapes' X axis points toward.

Axis

A vector that constrains the spin. That vector will be in the YZ plane of the shape.

Num Segments

The number of segments used to draw the shapes.

Angle

The angle of the arc or pie slice. Use 360.0 for a complete circle.

Radius

The radius of the shape.

Build Pie Sides

Draws straight lines from the ends of the arcs to the centers of the shapes. Turn this off to build arcs instead of wedges.
