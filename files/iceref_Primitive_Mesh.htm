Primitive Mesh

Categories

  • Topology

Description

Return the topology of some basic polygon mesh primitives.

Ports

Inputs

Primitive Type

The basic shape of the primitive.

Length or Radius

The size of the primitive.

Subdivision

The number of subdivisions in the various directions, depending on the type of primitive.

Outputs

Result

The new topology.
