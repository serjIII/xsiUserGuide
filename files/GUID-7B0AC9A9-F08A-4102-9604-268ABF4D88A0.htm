Move Towards Goal

This compound makes particles move toward one or more goal objects.

Plug this compound's Execute port into a Port on the ICETree node or in an Execute port of a State compound, if you're using the state system.

For more information on goals, see ICE Particle Goals [ICE Particle Simulations].

Tasks: Particles/Motion Control,Particles/Goals

Output Ports: Execute

Goal Definition

Select Goal Type

The part of the goal object to which the particles move:

  • Use Random Location on Geometry

  • Use Input Position Value

  • Use Closest Location on Geometry

  • Use Collision Location

  • Use Emit Location

  • Use Input Location.

See Particle Surface Interaction Attributes [ICE Particle Simulations] for more information on these locations.

Goal Position

If you selected Use Input Position Value as the Select Goal Type option, enter the global XYZ coordinates here for the goal position.

Geometry1

The goal object to which the particles move. You can plug in more objects in the other Geometry ports below this to have several goal objects. Plug the object's Value output into this port.

Motion Towards Goal

Weight

The amount that the particles are attracted to the goal objects.

Select Speed Method

The type of velocity used to calculate how the particles reach the goal objects: Set New Velocity; Multiply by Existing Velocity; or Multiply by Emit Velocity.

Velocity over Distance to Goal Profile

This graph determines the particle velocity over the distance to the goal objects.

Speed Multiplier

If you selected Multiply by Existing Velocity or by Emit Velocity as the Select Speed Method option, set the multiplier value here.

Limit Acceleration

Toggles the limiting of the acceleration to the value you specify for the Max Acceleration. Turn this option off so that there are no restrictions to the particle's acceleration.

Max Acceleration

The maximum velocity acceleration that you want the particles to reach while moving toward the goal.

Limit Max Turning Rate

Limits the particle turning rate to the Max Turning Rate value below.

Max Turning Rate

The maximum rate of turning that you want the particles to do while moving toward the goal.

Align Particle with Goal Normal

Rotates particles so that they are aligned in the direction of the normals of a object.

Particle Alignment Vector

If you selected Align Particle with Goal Normal, select this local particle axis you want to use for aligning it to the goal's normals. The Y axis is typically the vector that points along the length of a particle. If you change your particle type to cone, you can see that the Y axis is the direction of the cone point. The X and Y axes are the sideways axes.

Orientation over Distance to Goal Profile

This graph determines the particle orientation over the distance to the goal objects.

Execute Each Frame

Executes whatever compound is plugged into this port at each frame.

Goal Reached

Test Distance

The distance in Softimage units to the goal object.

Execute When in Range of Goal

If the Test Distance is reached by the particle, whatever compound or node is plugged into this port is executed. For example, if the Stick to Location compound is stuck here, the particles stick to the goal objects.
