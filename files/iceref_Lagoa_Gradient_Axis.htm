Lagoa Gradient Axis

Maps a gradient color in the direction of the desired axis of the element's area.

If you want to map a different gradient color for all axes, use the Lagoa Gradient XYZ compound.

Plug the Execute output into any Execute port, such as in the Lagoa Phase node; plug the Result output into any Color port, such as in the Lagoa Material node.

You can also select a point cloud and choose Particles On Emission Lagoa Particles Color Gradient Axis from the ICE toolbar to have this node set up in the point cloud's ICE tree.

Tasks: Lagoa/Data

Output Ports: Execute, Result

Axis

Select the element's axis in which you want to map a gradient color: X, Y, or Z.

Gradient

The color gradient used for the particles based on the Min/Max values.

See the Gradient node for more information.

Fitting Mode

Fits the gradient to the point cloud:

  • Automatically — you don't have to do anything.

  • Manually — set the Min/Max values below.

Min

The minimum position (in Softimage units) in global space along the selected axis of the element's area to be mapped to the color gradient.

This value corresponds to the leftmost color on the gradient.

Max

The maximum position (in Softimage units) in global space along the selected axis of the element's area to be mapped to the color gradient.

This value corresponds to the rightmost color on the gradient.
