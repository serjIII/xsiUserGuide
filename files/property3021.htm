Volumic Scene Property Editor

Defines the volumic sampling, smoke, and depth-fading properties for a scene.

To display: Open an explorer at the Scene_Root level (only visible when a volumic light has been defined) and select the Volumic icon.

Sampling

Step

Controls the steps of a volume map. Use this to "smooth" the volumic effect.

Max Ray Length

Specifies how deep a ray travels within the volumic structure, based on the first intersection point.

Smoke

Enabled

Enables the smoke effect and parameters.

3D

When on, the smoke effect will use the Z axis.

Scaling

The size of the turbulence pattern. If the smoke is made very large, it will be necessary to increase this value to keep the same size ratio between the smoke and its turbulence.

Intensity

Use this parameter to control the overall intensity of the smoke effect.

Speed

The speed at which the turbulence is moved. SOFTIMAGE units per second.

Depth Fading

This feature useful for reducing the apparent detail and contrast as an object recedes from the camera.

Enabled

When on, depth fading is enabled.

Start Distance

Defines the distance (from the camera to the surface) at which fading begins.

End Distance

Defines the distance at which depth fading reaches full intensity.

Transparency

Defines the transparency of the depth fading. This parameter applies to the Transmitted Color parameter. It's best to keep this value low to make the depth fading more visible.

Transmitted Color

Defines the amount of depth fading to apply to the object's color.
