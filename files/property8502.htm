Reference Plane Property Editor

| Pose | Static Kinestate Property Editor

Controls various options of the reference plane used for snapping and for transformations in Ref and Plane modes.

To display: Right-click on Ref or Plane in the Transform panel (you must have clicked on a transform property for it to be active, then choose Current Reference Properties. Alternatively, choose Transform Reference Planes Current Reference Properties.

Plane

Name

The name of the active or last active reference plane. You cannot change the names of default or transient reference planes. You can change the name only of a permanent reference plane that you create.

Cell Size

U, V

Sets the display size of the cells of the visible reference grid in Softimage units. Used for translations in Plane mode only.

Snap Size

U, V

Sets the distance between snapping targets when snapping to the grid in Softimage units. Used for translations in Plane mode only. (Ref mode uses the Snap Increments that you can access using Transform > Transform Preferences â€” see Snapping Preferences [Preference Reference].)

For more information about snapping in general, see Snapping [Working in 3D Space].

Display

Controls how the reference grid is displayed in the 3D views when it is active. These options have no effect on snapping.

Infinite

Disables the Extent controls. Note that to save resources and preserve visibility in the viewports, the grid isn't actually drawn to infinity, but instead fades off.

Extent U, V

Displays the grid up to the specified distance in both positive and negative directions with respect to the global scene center.

Pose

The transformation of permanent and transient reference planes in global space. You can modify these options if desired, but it's probably easier to position an object in your scene and use it to define a new reference plane. These options are not available for default planes.

Scaling Orientation X, Y, Z

Not used for reference planes.

Scaling X, Y, Z

Not used for reference planes.

Rotation X, Y, Z

The orientation of the plane with respect to the global scene center.

Position X, Y, Z

The position of the plane with respect to the global scene center.

Plane Normal

The axis that is considered as the plane's normal when rotating in Plane mode. This also controls which plane is considered UV when translating in Plane mode. Choose a value: X Axis (UV = YZ), Y Axis (UV = XZ), or Z Axis (UV = XY).
