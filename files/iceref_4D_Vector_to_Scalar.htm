4D Vector to Scalar

Categories

  • Conversion

Description

Outputs the X, Y, Z and W components of the input 4D vector.

Ports

Inputs

vector

The 4D vector to convert.

Outputs

X

The X component of the input 4D vector.

Y

The Y component of the input 4D vector.

Z

The Z component of the input 4D vector.

W

The W component of the input 4D vector.
