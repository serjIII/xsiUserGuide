Set All Actors Cycles

Initializes data for a CrowdFX simulation, and sets the animations for all actors.

This compound should be used in the Modeling region on the Point_Cloud object of a CrowdFX simulation, before particles are emitted. Connect Load Actor Cycles compounds to get the animations from the actor proxies. Use a single Load Actor Cycles compound to get all the animations of one actor, and use separate Load Actor Cycles compound for multiple actor sources. Plug all Load Actor Cycles compounds into the same Set All Actors Cycles compound.

Tasks: Crowds/Initialize

Output Ports: Execute

Load Cycles Actor1

Connect a Load Actor Cycles compound with the animations from one actor source. Use the New port to connect additional Load Actor Cycles compounds for other actor sources.
