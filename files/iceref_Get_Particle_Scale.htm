Get Particle Scale

This compound gets the particle's current scale (XYZ) data as set by its Scale attribute. You can output the Scale value as a 3D vector.

For more information, see ICE Particle Size [ICE Particle Simulations].

Tasks: Particles/Getters

Output Ports: Scale

Reference

The particle's Scale attribute.
