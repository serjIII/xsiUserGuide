Get Particle Collision Location

This compound gets the particle's location at the point of collision with an obstacle. You can output the particle's Collision Location, Position, Normal, and Collision Force data.

For more information on collisions, see Motion Control and Surface Interaction Behavior for ICE Particles [ICE Particle Simulations].

Tasks: Particles/Getters

Output Ports: Location, Position, Normal, Collision_Force

In Name

Plug a collision object's Out Name output into this port to give it a reference.

Reference

The particle's CollisionLocation attribute.

See Particle Surface Interaction Attributes [ICE Particle Simulations] for more information on this location.
