Vertex Color Change Data Type Property Editor

Controls whether Color at Vertices data is stored in a short integer (2 bytes, or 16 bits) or a floating point value (4 bytes, or 32 bits).

To apply: Click a vertex color property's icon to display its property editor, and then click Change.

Desired Datatype

Set the desired value:

  • Short (2 Bytes) is an unsigned short integer.
  • Float (4 Bytes) is a single precision floating-point value.
