Set Crowd PointPosition

Categories

  • Crowd

Description

Converts an array of positions per point of one object (typically the Point_Cloud object of a CrowdFX simulation) to a single position per point of the Actor_Copies mesh.

Ports

Inputs

Actors Point Position

An array of positions per point representing the envelope deformation of each actor instance.

Outputs

Result

Aa single position per point of the Actor_Copies mesh.
