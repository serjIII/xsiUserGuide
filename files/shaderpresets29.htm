Photon Irradiance

| Render Tree Usage

Category: Illumination

Shader Family: Texture

Output: Color

Returns photon illumination (caustics or global illumination) as soon as light hits an object.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Specifies the input color to return at the intersection point.

Render Tree Usage

This shader is usually connected to a Surface shader in order to reveal photon (global illumination and/or caustic) effects on the object's surface. It can also be connected to the Photon input of the Material node.
