Constrain Between 2 Points

Constrains an object's position to lie between two other objects. This node should be plugged into an Add Constraints compound.

Category: Kinematics

Output Ports: Constraint

Mute

Deactivates the constraint.

Constrainee

The constrainee specified here must match the constrainee specified in the Add Constraints compound.

Constrainee In Name

A scene reference to prepend to Constrainee Reference to fully resolve the name.

Constrainee Reference

The object to constrain.

Constrainer1

Constrainer1 In Name

A scene reference to prepend to Constrainer1 Reference to fully resolve the name.

Constrainer1 Reference

The first constraining object.

Constrainer2

Constrainer2 In Name

A scene reference to prepend to Constrainer2 Reference to fully resolve the name.

Constrainer2 Reference

The second constraining object.

Blend

Controls the constrained objects's position between the two constrainers. At 0.0, the object is at the same position as Constrainer1 and at 1.0 the object is at the same position as Constrainer2.

Tangency

Affect Tangency

Rotates the constrained object to align it along the line between the constrainers.

Axis to Align

Specifies which axis to align with the line between the constrainers.
