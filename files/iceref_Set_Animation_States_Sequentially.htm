Set Animation States Sequentially

This compound provides an easy possibility to playback two Animation States in a sequence, means one after another. As soon as the referenced animation in Animation State 1 is played completely, the Animations State 2 will be played back.

If the type of the first Animation State is set to Loop, then this loop will only be played back once.

Tasks: Crowds/Behavior

State 1

The first Animation State to be played back.

State 2

The second Animation State to be automatically played back, right after the first Animation State is finished.

The ICE tree for this compound displays how the two animation states are played sequentially.
