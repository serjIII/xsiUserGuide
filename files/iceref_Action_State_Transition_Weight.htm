Action State Transition Weight

Returns the proportion of the Transition Duration set in Action State that has elapsed in a CrowdFX simulation since an actor entered the current action state.

Use this compound in a branch connected to the Execute Every Frame1 port of an Action state compound to control the blending of the modifications you make to the CrowdFX_Actor_Deformers transformation values in that branch.

For more information, see Modifying Deformer Transformations for the Animation States.

Tasks: Crowds/States

Output Ports: Value
