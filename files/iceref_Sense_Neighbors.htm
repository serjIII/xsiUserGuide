Sense Neighbors

Detects neighboring actors that one actor might collide with, which is required in calculating the avoidance data. This compound is used in the Set Velocity Using Collision Avoidance compound.

Tasks: Crowds/Behavior

Output Ports: Execute

Max Number Points to Query

Maximum number of points to query for proximity information.

Max Number Points to See

Maximum number of points to store as neighbors.
