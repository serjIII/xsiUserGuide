Set Velocity By Speed and Direction

Sets the actor velocity from a speed and a direction.

Tasks: Crowds/Behavior

Output Ports: Velocity. Connect this port to the Velocity Input on the Basic Crowd Simulation compound.

Speed

Defines the speed of an actor.

Direction

Defines the angle from Z-axis to the direction.

Coordinate System

Coordinate system in which the direction is calculated.

  • Emitter Local Space — Direction is calculated from the local Z axis of the emitter
  • Scene Global Space — Direction is calculated from the global Z axis
