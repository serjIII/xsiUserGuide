Importing and Exporting Softimage Models

Models are like "mini scenes" that can be easily reused in scenes and projects. In fact, exporting and importing models is how you can copy and share objects and object hierarchies between scenes.

Importing Softimage Models

You can import a Softimage model (File Import Model) for reuse in any scene. When you import the model it is saved internally to the scene file.

You can also import a model as a referenced model (File Import Referenced Model), so that any changes made to the model will be reflected in your scene the next time you open the scene or update the reference.

For details about importing models and working with models in general, see Models.

Exporting Softimage Models

You can export models created in Softimage for use in other scenes. When you export a model, a copy is saved as an independent file.
