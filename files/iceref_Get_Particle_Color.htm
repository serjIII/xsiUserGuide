Get Particle Color

This compound gets the particle's current color data as set by its Color attribute. You can output this Color data.

For more information on color, see Particle Color Attributes [ICE Particle Simulations].

Tasks: Particles/Getters

Output Ports: Color

Reference

The particle's Color attribute.
