Control Displacement Instance Animation

This compound lets you scale the instanced object's animation according to its size when it becomes instanced geometry. Instanced objects adopt the particle Size, so it may be different from what its real size is. As a result, the instanced object's animation changes in accordance to the particle size. For example, if an object is smaller as instanced geometry, its animation must cycle at a faster rate to keep the same distance covered as its original animation.

Plug this compound's Execute output into a port on the ICETree node.

For more information, see Controlling the Instance's Animation [ICE Particle Simulations].

Tasks: Particles/Geometry Instancing

Output Ports: Execute

Distance Covered

The number of Softimage units that the original instanced object travels in its animation cycle, such as with a walk or run cycle.

Start Frame

The first frame of the instance's animation cycle that you want to use for the particle simulation.

End Frame

The last frame of the instance's animation cycle that you want to use for the particle simulation.
