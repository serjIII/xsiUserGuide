Invert Polygon

Categories

  • Topology

Description

Inverts one or more polygons.

Ports

Inputs

Topo

The topology you want to use. If not connected, the default is "Self.Topology".

Polygon Index

The indices of any polygon to invert. You can use a single integer, array or set.

Outputs

Result

The modified topology.
