Luma Adjust

Adjusts the luminance values of a clip using a graph.

Requires an Input (Any).

Produces an Output (Any).

Graph

Graph

The horizontal axis represents the luminance values in, and the vertical axis represents the luminance values out.

Mode

Specifies whether you edit the graph in HSV or RGB mode.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
