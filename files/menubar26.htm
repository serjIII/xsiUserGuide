DirectX Import Dialog Box

Imports a DirectX (*.x) file into Softimage.

To Import: Choose File Import DirectX Import from the main menu.

File

Path

Specifies the path and file name of the .x file that you wish to import. You can type a path and file name into the text box or click the Browse button (...) to browse to the desired path.

Options

Import Animation

Controls whether or not animation data stored in the .x file is imported.

Import Textures

Controls whether or not textures identified in the .x file are imported.

Invert UVs

Some programs show .x files UV inverted. If you are importing a .x file created in such a program, toggling this option causes the UVs to display properly.

Reorient Scene

Converts the scene to a right-handed coordinate system (Y is up and positive Z moves away from the screen).

Animation Frame Offset

Specifies a frame offset for the animation. For example, if your animation runs from frame 1-100 in Softimage, but you need it to run from frame 50-150, you would enter an offset value of 50.
