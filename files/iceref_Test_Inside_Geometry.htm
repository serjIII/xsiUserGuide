Test Inside Geometry

This compound tests for particles being inside a geometry volume. When the particles are inside the volume, they trigger the effect.

Plug its Result output into a Trigger port on a State compound or in a Condition port of an If node.

For more information on states, see ICE Particle States [ICE Particle Simulations].

Tasks: Particles/Conditionals

Output Ports: Result

Geometry

The object within which the particles must be to trigger the effect. This must be an object with a surface. Plug the object's Value output into this port.

Negate

Triggers an effect only if particles are outside the volume instead of inside it.
