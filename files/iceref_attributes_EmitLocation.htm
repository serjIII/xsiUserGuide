EmitLocation

The location on a surface at which a particle was emitted.

When emitting from a volume, the location is an offset from the object center and is not affected by deformation.

This attribute is set automatically by the Add Point node as well as compounds that contain it, such as the Emitters. However, note that if positions instead of locations are connected to Add Point, as is the case for Emit from Null and Emit from Position, then this attribute is meaningless.

This attribute is also defined in the Get Particle Emit Location compound.

You can use this attribute with some of the Surface Interaction compounds that allow you to use the Emit Location for particles.

You can also use this attribute with different properties of an object, such as weight maps or texture maps. For example, you can use a texture map on a particle emitter object to color each particle based on its emit location.
