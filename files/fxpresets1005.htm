Flood Fill Shape Property Editor

| Flood Fill | Mask | Transform | Display

To Display: Select a flood fill shape in the FX Viewer or in the explorer and press Enter.

In vector paint layers, using the Flood Fill tool creates a flood fill shape. You can edit the shape's properties to change the fill after you've applied it.

Flood Fill

Mode

Choose one of the following fill modes.

RGB: neighboring pixels are filled if their RGB values match those of the selected pixel.

HSV: neighboring pixels are filled if their Hue, Saturation, and/or Value values match those of the selected pixel.

Anti Alias

Softens the edges of the filled area. The fill fades out at the edges over a distance defined by the Tolerance setting.

Feather

Blurs the edge of the filled area. The higher the value, the larger the blurred area.

RGB

Tolerance

When Mode is set to RGB, specifies how closely a neighboring pixel's RGB values must match those of the selected pixel before the pixel is filled.

A value of 0 means that only pixels that match the selected pixel exactly are filled. Raising the Tolerance value broadens the range of "matching" RGB values such that pixels with increasingly dissimilar RGB values are filled. Setting the value to 100 fills/selects the entire image.

HSV

Hue/Saturation/Value

When Mode is set to HSV, specifies how closely a neighboring pixel's individual Hue, Saturation, and/or Value values must match those of the selected pixel before the pixel is filled.

As with the RGB mode's Tolerance setting, the lower the values, the more closely a pixel must match the selected pixel's Hue, Saturation, and/or Value value before it is filled.

Fill Color

Specifies the color of the fill. This color fills the region defined by the Mode and Tolerance options. Set any RGBA value using the sliders.

Opacity

Specifies the opacity of the fill. The higher the value, the more opaque it is.

Mask

Obey Paint Mask

The paint mask is a matte or grayscale image that defines which areas of an image are paintable, and which areas aren't (sort of like an obey matte for painting). You can set any operator as the paint mask from the operator's contextual menu.

Enable

When active, parts of the image that correspond to black parts of the mask are protected from the flood fill.

Invert

When the Obey Paint Mask > Enable option is activated, activating this option inverts the mask so that parts of the image that correspond to white parts of the mask are protected from the flood fill.

Transform

Move

X/Y

Translates the flood fill shape from its initial position in X and/or Y. The values are in pixels.

Display

Name

Displays the shape's name in the explorer. You can change it by typing a new name in the text box.

Visible

Controls the shape's visibility. Disabling this option mutes the shape in the FX Viewer.
