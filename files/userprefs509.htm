Uninstall Add-On Dialog Box

Lets you uninstall .xsiaddon files containing various customizations such as scripts, presets, custom commands, toolbars, layouts, and so on.

To display: Choose File Add-Ons Uninstall.

Add-Ons

Lists the Name and File Path of the add-ons installed on your machine. Select the add-on to uninstall here.

You cannot uninstall the individual components of the add-on files. The entire .xsiaddon file and its components will be removed.

Shared Plug-Ins

Lists the add-ons installed on the workgroup path. Uninstalling a workgroup add-on has the effect of unregistering it for the current Softimage session. All workgroup add-ons will be automatically installed and registered again the next time you start Softimage.

Uninstall

Uninstalls the selected add-ons.

Properties

Click to display the Add-On Information dialog box, which may contain additional information provided by the add-on author. See Add-On Information Dialog Box.
