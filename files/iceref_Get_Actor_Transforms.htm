Get Actor Transforms

Returns CrowdFX_Actor_Deformers, an array containing the deformer transformations per actor instance in a CrowdFX simulation.

This compound can be used only in an ICE tree on the crowd simulation Point_Cloud object.

Tasks: Crowds/Getters

Output Ports: Value, Out Name
