Get Element Index

Categories

  • Data Access

Description

Returns the index of the element in the set. For example, if the data set contains the point normals of a polygon mesh primitive, the return values will be the indices of the points of the mesh.

Ports

Inputs

Value

Deprecated. The set is defined by the output connection.

Outputs

Indices

The element indices of the data set.
