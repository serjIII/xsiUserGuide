Object View Preferences

To display: Choose File Preferences, expand the Editors node, and click Object View.

These preferences control the default behavior of the object view. Changes to any of the preferences are only applied to object view opened after the changes are made. Currently open object views are not affected.

General

Display All Objects

Displays all visible scene objects in the object view, instead of just the selection.

Use Local Camera View (relative to object)

Controls whether the top/bottom/front/back/left/right views are based on the reference object (i.e., the first object in the display list) or whether they are global views.

Modify Scene Cameras

When the View is set to a scene camera or spotlight, changing the selection will modify the camera/light's position based on changes in the object view.

Update

Auto Frame Camera

When enabled, the camera will automatically frame the selection whenever the view is updated.

Auto Align Camera

When enabled, the camera automatically aligns along the default view axis (see below) whenever the view is updated.

Auto Select View to Align (use bounding box)

When enabled, this option modifies the behavior of "Auto Align Camera" so that the camera alignment axis is controlled by the maximum area of the bounding box. In other words, the camera view is set to front/top/right based on the area of the bounding box in that view.

Hold Last Selection (when nothing is selected)

Continues to show the last selection even if you deselect it and the view is not locked. The view updates normally when you select a different object unless the view is locked.

Default Align Direction

Sets the default camera direction in new object views to one of the following: Top, Bottom, Right, Left, Top, or Bottom.

Compensation

Use Camera Compensation

This option controls whether the camera is adjusted to compensate for changes to the selection. In other words, when this option is enabled, the camera will be locked to the selection while it is animated, transformed, and so on.

Compensate for Scaling

When Use Camera Compensation is activated, this option toggles compensation for scaling.

Compensate for Rotation

When Use Camera Compensation is activated, this option toggles compensation for rotation.

Compensate for Translation

When Use Camera Compensation is activated, this option toggles compensation for translation.
