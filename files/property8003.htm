SubdividePolygon Op Property Editor

Divides one or more polygons evenly.

To apply: See Subdividing Polygons and Edges {Modeling].

To redisplay: Select the modified polygon mesh object then choose Edit Properties Modeling Properties from the Edit panel and click the SubdividePolygon Op tab.

Subdivision Type

The pattern used to subdivide the selected polygons:

  • Plus adds edges from the geometric center of a polygon to the midpoint of each edge.

  • Diamond draws edges connecting the midpoints of a polygon's edges.

  • X draws edges from the midpoint of a polygon to each of its vertices.

  • Triangles simply triangulates all quad and n-sided polygons.

Iterations

The number of times to subdivide the selected polygons iteratively. A value of 1 subdivides each polygon once; 2 subdivides each polygon and then subdivides each subdivision; and so on. This option has no effect when Subdivision Type is Triangles.

Tile

When on, all but the last iteration is of type Plus and the last iteration is of the type specified. When off, all iterations are of the type specified. This option has no visible effect when Iterations is set to 1 or when Subdivision Type is set to Plus.
