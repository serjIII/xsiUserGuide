Materials

An array containing the names of materials that can be used on a polygon mesh object, such as "Sources.Materials.DefaultLib.Blue". The names in this array can be referenced by MaterialID.

Materials can be specified in any of these forms:

  • MaterialName
  • MatLibName.MaterialName
  • Sources.Materials.MatLibName.MaterialName

The material names are resolved at render time, so undefined material names will not show as errors in the ICE tree. If a material name is not available at render time, the object's material is used instead.

Data Type

array of strings

Belongs To

objects

Geometry Types

  • polygon mesh

Attribute Type

predefined

R/W or RO:

read/write
