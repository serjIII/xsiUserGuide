Chain End Effector Property Editor

Adjusts a chain effector's display type and size, including shadow icons.

For more information, see Changing the Shape and Size of Chain Elements [Character Animation].

To display: Select a chain's end effector and press Enter.

Primary Display

Icon

Select from a list of shapes to use for the effector's display — see Shapes for Effector, Root, and Null Icons and Their Shadows.

Size

Sets the size of the display icon for the effector on all axes (uniform scaling).

When you change this value, the effector's accompanying shadow always scales proportionally with it, regardless of the state of the Lock Shadow Size When Scaling option.

Shadow Display

A shadow is a visual reference that makes it easier to see and select chain elements. When you select a shadow, its chain element is also selected.

Icon

Select from a list of shapes to use for the effector's shadow — see Shapes for Effector, Root, and Null Icons and Their Shadows.

Select an option other than None to see the following parameters.

Color

Sets the color of the shadow. Select the Custom option to set a custom color using the color sliders. The resulting color appears in the color chip.

You cannot change the color of the shadow if Wireframe is selected: the standard display color is used.

Position

Offsets (translates) the shadow from the effector on the X/Y/Z axes.

Size

Specifies the scale of the shadow. The default value of 1 means a 1:1 ratio with the effector.

Lock Shadow Size When Scaling

Keeps the shadow at a constant size when the effector is scaled (using the Scale transform commands, not the Size parameter above).

Shapes for Effector, Root, and Null Icons and Their Shadows

Select from any of these shapes for effectors, roots, and nulls and their shadows from the Icon list in their property editor.

 

Null - Default style.

 

Rings - Three concentric rings, each aligned with an axis (X,Y,Z) representing the centre of the root/effector.

 

Arrow Rings - Same as Rings except with an arrow in each ring to mark the direction of the axis.

 

Box - centered at the root/effector.

 

Circle - centered at the root/effector.

 

Square - centered at root/effector.

 

Diamond - centered at root/effector.

 

Pyramid - bottom is centered at root/effector.

 

Pointed Box - bottom is centered at root/effector.

 

Arrow - centered at root/effector.
