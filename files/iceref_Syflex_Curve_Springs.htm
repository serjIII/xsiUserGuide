Syflex Curve Springs

This compound defines the springs that are used to link a curve's set of points in a deformation simulation. There are three different types of springs that controlling a different kind of deformation on the curve: stiffness, shearing, and bending.

For more information on springs, see Using the Syflex Spring Forces.

This compound is used in the Syflex Curve compound.

Tasks: Syflex/Forces

Output Ports: The Force0/1/2 outputs each plug into the Force ports on the syflexISimul node, which controls the Syflex curve simulation.

The parameters in this compound are identical to the parameters in the Syflex Cloth Springs compound.
