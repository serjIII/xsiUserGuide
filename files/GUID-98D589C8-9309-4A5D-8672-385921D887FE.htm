Producing Lip-sync Animation for Characters Created Using Character Generator

This topic explains how to produce lip-sync animation for 3D animated characters created using Autodeskâ€™s Character Generator.

Character Generator Overview

Character Generator is a free technology preview that leverages Autodesk's powerful 3D design and animation tools. Using a web browser and Character Generator, you can create your own or modify existing stock characters using a variety of pre-defined body types, facial features, and clothing styles for inclusion in 3D games, animations, and scenes.

For more information, see the Autodesk Character Generator Website.

Producing lip-sync animation involves the following steps:

  • Customizing a character in Character Generator
  • Downloading a character from Character Generator
  • Producing lip-sync for the downloaded character in Softimage

Customizing a Character in Character Generator

  1. Log in to the Character Generator Website.

    If you do not have an account, click Create Account to create one.

  2. Click the New character icon.

    The Choose a Character to customize page appears. This page displays the existing catalog of characters.

  3. From the Ancestor Catalog section, choose a character.

    If you want your character to look like a child, then from the Artistic Style section, choose Junior. In this example, the a Standard style character is created.

  4. Click Customize.

    The Customize your Character page appears. This page provides options to customize the Face, Skin, Eye Color, Hair, Body, and Clothing of your character.

  5. Click on any of the options and customize your character as you like.
  6. Click Done.

    The Save Character dialog box appears.

  7. Enter a name for the customized character, and click OK.

    The My Characters page displays your customized character. There are options to Delete, Edit, and Publish your character.

  8. Click the Publish icon.

    The Publish Character dialog box appears.

  9. Click the Options button to modify the settings of your character such as height, orientation, texture, and so on.
  10. Under Facial Expressions, choose the Facial blend shapes option.

    This option creates facial animation for your character. Only the 3D characters with facial animation can produce lip-sync animation when imported in Softimage.

  11. Click Publish.

    In the My Characters page, click Published Characters to view all the published characters.

Downloading a Character from Character Generator

  1. In the published character, click the Download icon on the published character.

    The Opening <file_name>.zip dialog box appears.

  2. Click OK to save the file.

    The file is saved in the .fbx format.

Producing Lip-sync for the Downloaded Character in Softimage

  1. In Softimage, choose File Import Import FBX.
  2. In the Import FBX file dialog box, navigate to the folder where you saved the .fbx file, click on the file, and click OK.

    The Crosswalk FBX Import dialog box appears.

  3. Click OK.

    If there are errors and warnings while importing the .fbx file, the FBX Import/Export Warning! Dialog box displays them.

  4. Click OK to ignore warnings.

    The .fbx file created in Character Generator is imported in Softimage.

  5. Open the Script Editor in Softimage and run the GenerateLipSync command.

    Softimage prompts to pick a session.

  6. Open the Explorer and click on the name of the newly imported character.

    The Select a File dialog box appears.

  7. Navigate to the directory where you have saved an audio (.wav) file.
  8. Click on the audio file, and click OK.

    The lip-sync animation is now applied to the imported character.

    For more information about the supported audio file formats, see this Help topic.

    TIP:You can run the following commands to import the .fbx file to generate lip-sync using an audio file: 
    FBXImport "<FBX_file_path>:\<FBX_file_name>.fbx"
GenerateLipSync "<FBX_file_name>", Application.InstallationPath( siFactoryPath ) & "\<audio_file_path>\<audio_file_name>.wav", null, "en", "hmlc", null
  9. Click the Play Forwards from the Start Frame button in Softimage to view the lip movements that match with the text in the audio file.

    You can adjust the timing of the phonemes to match the dialogue better. See Adjusting the Phonemes.
