Weight Map Generator Property Editor

Generates a weight map. Weight maps can be used to modulate parameters such as the amplitude of a deformation across an object or cluster.

To apply: Select an object or cluster and choose Get Property Weight Map.

To redisplay: Select the object then choose Explore Property Maps on the Select panel, click the icon of the corresponding weight map, and click the Weight Map Generator tab. If the weight map is connected to a parameter, you can also display these parameters by clicking the red connection icon to the right of the connected parameter's slider on the corresponding property page.

Weight Map Properties

Map Type

Specifies the basic type of the weight map before you paint any strokes on it. You can select Constant or one of a variety of gradients.

Base Weight

The base value for the points on the map. For example, if you select a Constant type, all points have this weight until you paint strokes on the map.

Invert Weights

When on, high weight values are treated as low values and vice versa.

Negate Weights

Causes weight values to be considered as negative.

Weight Value Range

Minimum

The minimum value for any point on the map.

Maximum

The maximum value for any point on the map.

Reference Position X, Y, Z

These values define an origin for the base Weight Map Type on the surface of an object. Modifying these values "slides" the weight map along the object's surface. Note that modifying these values does not affect the position of any paint strokes applied on top of the base map.
