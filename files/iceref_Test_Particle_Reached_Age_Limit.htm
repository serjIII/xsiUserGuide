Test Particle Reached Age Limit

This compound uses the reaching of the particle's age limit value (AgeLimit attribute) as a trigger for an effect, as defined in the Set Particle Age Limit compound. You need to first have the Set Particle Age Limit compound plugged into the Emit compound's Execute on Emit port so that the state knows what the particle's age limit is.

Plug this compound's Result output into a Trigger port on a State compound or in a Condition port of an If node.

For more information on states, see ICE Particle States [ICE Particle Simulations]; for information on particle age limit, see ICE Particle Age [ICE Particle Simulations].

Tasks: Particles/Conditionals

Output Ports: Result
