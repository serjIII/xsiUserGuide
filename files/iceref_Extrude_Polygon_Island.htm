Extrude Polygon Island

Categories

  • Topology

Description

Extrude a group of polygons as one extrusion.

Ports

Inputs

Topo

The topology you want to use. If not connected, the default is "Self.Topology".

Polygon Index

The indices of any polygon to extrude. You can use a single integer, array, or set.

Duplicate

If this option is on, polygons are duplicated, that is, the copied polygons are disconnected from the original geometry.

Nb Subdivs

Number of subdivisions along the extrusion.

Inset

Shrinks the boundary of the new polygons by the specified amount in Softimage units. Negative values expand the boundary instead of shrinking it, and the resulting polygons will not overlap even when the extrusion length is 0.

Transform

Apply Scaling, Rotation, and Translation to the extruded components in the X, Y, and Z axes. The transform input is applied for each extrusion section. If you leave these values at the default Identity matrix, the new components are not moved and remain superimposed on the original components.

Outputs

Result

The modified topology.
