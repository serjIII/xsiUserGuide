Get Copy Index

Returns a set of integers that corresponds to which copy of a mesh each vertex on the "self" object belongs to. This node can be used to differentiate between copied instances of a mesh after using Create Copies from Polygon Mesh.

Tasks: Topology/Getters

Output Ports: Out
