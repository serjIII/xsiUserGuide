Hierarchy From Guide Dialog Box

Creates a new hierarchy based on the selected guide of any type (biped, biped dog leg, or quadruped).

For more information, see Creating Character Hierarchy Setups from a Guide [ Character Animation ].

To display: Select a guide and choose Create Character Hierarchy from Guide from the Animate toolbar.

Hierarchy

Type

Null Hierarchy

Hierarchy of nulls with a null at the root of each chain.

Box Hierarchy

Hierarchy made of boxes.

SI|3D Skeleton

SOFTIMAGE|3D-style skeleton hierarchy with the effectors as children of the last bone.

XSI Skeleton

XSI-style skeleton hierarchy with the effectors as children of the chain's root.

Lumbar Divisions

Number

Number of vertebrae in the spine.
