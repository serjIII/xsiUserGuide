Scalar To Color

| Render Tree Usage

Category: Processing > Conversion

Shader Family: Texture

Output: Color

Converts a scalar (numeric) input value to a color output.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Specifies the input scalar value (value between 0 and 1) to be copied to RGB channels.

Alpha

Specifies the input scalar value to be copied to the alpha channel of the output.

Render Tree Usage

This shader is simply placed between a scalar output and a color input in order for the two values to connect.
