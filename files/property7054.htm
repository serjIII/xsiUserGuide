Environment Cache Property Editor

Creates a cached simulation source when you select Cache on Playback and Save Cache to Mixer options in the rigid body simulation's Simulation Time Control Property Editor.

To display: In the explorer, open the current rigid body simulation Environment and click the Environment Cache icon.

For more information, see Creating ICE Simulation Environment Cache Files.

Select a Storage type from the list, then enter a File name for the cached rigid body simulation.

The options here are the same as the ones found in the Action or Animation Layer Source Property Editor.
