Standalones

Autodesk® Softimage® ships with a number of command line utilities called standalones that allow you to accomplish specific tasks outside the Softimage interface. In some cases, you can also perform these tasks from within the interface, but may choose to use a standalone for more control or speed — the choice is yours.
