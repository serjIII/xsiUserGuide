Set Particle Instance Animation Time

This compound uses the point cloud's ShapeInstanceTime attribute to set the time (in seconds) at which the instance is evaluated for rendering. The evaluation happens at the current time (instead of the instance's animation time), so you still see the object animate.

This attribute may cause issues when rendering â€” see Rendering Particle Instances [ICE Particle Simulations] for information and workarounds.

Plug its Execute output into an Execute on Emit port of the Emit compound.

For more information on using animated instances, see Controlling the Instance's Animation [ICE Particle Simulations].

Tasks: Particles/Geometry Instancing

Output Ports: Execute

Select Value Type

Select whether the animation time is in Frames or Seconds.

Animation Time

Sets the time at which the instance is evaluated in either Frames or Seconds, depending on what you selected as the Value Type.

Reference

This compound uses the point cloud's ShapeInstanceTime attribute as a reference.
