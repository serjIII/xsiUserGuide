Limit Num Particles

This compound limits the number of particles that are emitted to create a one-time particle emission (that is, this is not a continuous number of particles per second). This is useful for things like blasts or intermittent emissions.

Plug this compound's Enable Emitter output into the Enable port of the Emit compound.

For more information, see Creating Intermittent Particle Blasts (Limit Number of Particles) [ICE Particle Simulations].

Tasks: Particles/Emission Control

Output Ports: Enable_Emitter

Max num Particles

The maximum number of particles that are emitted at once. Animate this parameter's values to create intermittent emissions.

Reference

The point cloud's NbPoints attribute.
