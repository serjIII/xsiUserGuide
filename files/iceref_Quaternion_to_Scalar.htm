Quaternion to Scalar

Categories

  • Conversion

  • Rotation

Description

Outputs the W, X, Y and Z components of the input quaternion.

Ports

Inputs

Quaternion

The quaternion to convert.

Outputs

X

The X component of the input quaternion.

Y

The Y component of the input quaternion.

Z

The Z component of the input quaternion.

W

The W component of the input quaternion.
