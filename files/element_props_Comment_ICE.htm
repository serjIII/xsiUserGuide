Comment Property Editor (ICE)

Specifies the text and color of a comment in the ICE trees.

For more information about applying and editing comments, see Documenting Trees: Groups and Comments [ICE Guide].

Comment

Apply

Applies changes without closing the property editor

Comment

The text of the comment.

Word Wrap

Automatically wraps long line to the beginning of the next line when they reach the end of the comment area.

Auto Resize

Automatically resizes the comment area in the tree based on the comment text.

Color

Title Color

The color of the background for the comment text.

Title Color Text

The color of the comment text

Background Color

The background color of the group area. Available for group comments only. To make the color show more clearly in the Bird's Eye View, increase the Background Color's Alpha value.
