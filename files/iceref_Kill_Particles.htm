Kill Particles

Deletes particles upon reaching the age limit that you define.

Category: Point Cloud

Output Ports: Execute, Out_Name

Age Limit

The maximum age of a particle in seconds. This sets the AgeLimit attribute.

In Name

Plug in the Out Name output of an object.
