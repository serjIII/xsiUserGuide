Hair Pop Operator Property Editor

Extends guide curves on hair straight out along the normals of the hair emitter, and sets a standard length.

To apply: Either select a hair object and choose Modify Length Pop Zero Hairs from the Hair panel, or select one or more points on each of the guide curves you want to pop and choose Modify Length Pop Selected Hairs.

To display: Select the hair object, click the Selection button on the Select panel, and then click the Hair Pop Operator icon.

For more information, see Resetting the Hairs [ Hair ].

Mute

Toggles the deformation on and off without affecting the values of other parameters.

Pop

The type of operation: Zero Hairs or Selected Hairs depending on the command you chose when applying the operator. This is for information only and cannot be changed after the operator has been applied.

Zero Threshold

The minimum length in Softimage units for determining when a hair's length is close enough to zero. Hairs shorter than this value are popped and hairs longer than this value are not popped. This parameter is available only when Pop is "Zero Hairs".
