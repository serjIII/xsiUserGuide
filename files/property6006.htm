Clean Curve Property Editor

Resamples a curve, reducing the number of control points while retaining the overall shape within the specified tolerance.

To apply: Select a curve and choose Modify Curve Clean from the Model toolbar.

To redisplay: Select the curve, click the Selection button on the Select panel, then click the Clean icon.

Tolerance

The allowable error for resampling the curve. Higher values produce fewer control points but reproduce the original shape less accurately. A value of zero produces no change.
