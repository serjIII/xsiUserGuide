Logarithm (Expression Function Reference)

The Function Exponent Logarithm command in the expression editor inserts the token log( at the insertion point in the editing pane. This token evaluates the natural (base e, or Napierian) logarithm of an expression. Enter an expression after the opening bracket ( then enter a closing bracket ).

Example

log(sphere.kine.local.posy)

Returns the natural logarithm of the local Y translation of sphere.
