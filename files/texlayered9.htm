Layer/Port Property Editor

This property editor contains parameters that control a texture layer's effect on one of a material/shader's ports.

To display, do any one of the following:

  • Open a material/shader's property editor. If the material/shader has texture layers, a layer property editor is appended to the main property editor for each layer. A layer/port property editor is appended to the main layer property editor for each port to which each layer is applied.

  • From the texture layer editor, select a layer and press Enter to open its property editor. A layer/port property editor is appended to the main layer property editor for each port to which the layer is applied.

  • Locate the material/shader in the explorer. Expand its Texture Layers Layername Ports node and choose the port to open its layer/port property editor.

  • Select a layer in the material panel. The property editors of its ports are displayed in the lower portion of the material panel.

M

Mutes the layer for this port, causing the layer to have no effect.

(Slider)

Port scaling factor. This value is multiplied with the layer's weight to produce the layer's weight for the port.

A

Determines the color source.

  • When this option is on (blue), the port uses the alpha channel of the layer's Color input as its color source.

  • When this option is off, the port uses the layer's Color input as its color source.

I

Inverts the layer's color for the port. When this option is on, the icon is blue.

If you are using the layer's alpha as the color for the port and also inverting the color, the invert operation is performed last so as to affect the grayscale texture generated from the alpha.
