Set Particle Scale

This compound lets you set the particle's scale, using the particle's Scale attribute as a reference.

Plug its Execute output into the Execute on Emit port of an Emit compound to set this value once, or plug it into the ICETree node to update at every frame.

For more information, see Particle Size and Scale Attributes [ICE Particle Simulations].

Tasks: Particles/Setters

Output Ports: Execute

Scale

Sets the particle's scale value in its local XYZ coordinates.

Reference

The particle's Scale attribute.
