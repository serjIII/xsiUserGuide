Keying Panel Preferences

To display: Choose File Preferences from the main menu and click Editors Keying Panel or its icon, or choose Parameters Preferences in the keying panel.

For more information on the keying panel in general, see Keying Keyable Parameters in the Keying Panel [Animation].

Common to Both Views

Use Script Names

Displays the parameter names in both the keyable parameters editor and the keying panel using the script names. These are the shortened names used for parameters in scripting, but are sometimes more cryptic than the full parameter names.

Toggling this preference updates the parameter names in both the keyable parameters editor and the keying panel.

Keying Panel

Show Proxy Parameters

Toggles the visibility of proxy parameter sets for the selected object in the keying panel.

Show Marking Psets

Toggles the visibility of marking sets for the selected object in the keying panel.

By turning off this preference, you can hide all marking sets from the Keying Panel without hiding all proxy parameters. All CustomPSets with the prefix "MarkingSet" to their name are affected by this preference, since you can have many inoperative marking sets by just modifying the names.

Keyable Parameters Editor

Change for all Objects in the List

Makes changes to the keyable parameters for all objects in the multiple selection list in the keyable parameters editor.

If this option is off, changes to the keyable parameters are made only for the object currently selected in the multiple selection list.
