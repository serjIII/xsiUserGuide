Scene Debugging Preferences

To display: Choose File Preferences from the main menu and click Editors SceneDebugging or its icon. Alternatively, open the Scene Debugger (View General Scene Debugger) and choose File Preferences.

For more information on using the Scene Debugger in general, see Analyzing Scene Performance and Memory.

Performance

Show the consolidated categories

When operators are consolidated under higher nodes in the call stack, they are divided into categories (Animation, Deformation, and so on). When this option is off, the operators are consolidated as a flat list.

Filtering threshold (in ms)

Nodes that take less time than this value in milliseconds to evaluate are hidden in consolidated lists. This simplifies the data when you are looking for slow nodes.

Memory

Filtering threshold (in k)

Nodes that take less than this amount of memory in kilobytes are hidden. This simplifies the data when you are looking for large nodes.
