Color Balance

Category: Processing > Image Processing

Shader Family: Texture

Output: Color

The Color Balance shader modifies the color balance of the input color selectively for Shadow, Highlight, and Midtone ranges.

Input

Specifies the input color. This parameter is normally connected to an image whose colors you wish to balance.

Highlights/Midtones/Shadows

Each of these sets of controls affects the corresponding range of colors. You can change the colors within the range by adjusting the RGB sliders.

Red/Green/Blue

Adjust the color balance of the input color in each color range (Highlights, Midtones and Shadows).

Preserve Brightness

Ensures that the brightness of the input color is maintained, regardless of changes to color balance.

Render Tree Usage:

This shader can be used nearly anywhere in a render tree. Connect the color output of one Render Tree node to the Input Color port of the Color Balance shader.
