Foreground Mask

DEPRECATED. This shader is unsupported, but it may still get installed with Softimage to provide compatibility with older scenes that use it. It is recommended that you replace unsupported shaders in your scenes with equivalent shaders from the current Softimage shader library.

Shader Type: Lens

Output: Color (RGB) value

Is a mask placed in the foreground of the image. You can either use the default intensity or the alpha channel to create the mask, or load your own.

Name

The shader's name. Enter any name you like, or leave the default.

Mask

Uses the default intensity mask. (When off, the alpha channel is used.)

Pre-Multiplication

Multiplies the foreground mask and the resulting picture.

Map

Opens a browser that lets you use a predefined or saved map as a foreground mask.
