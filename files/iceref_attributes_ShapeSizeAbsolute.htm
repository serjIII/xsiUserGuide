ShapeSizeAbsolute

When this attribute is set to 1 (True), particles' Size values are interpreted as Softimage units instead of pixels. This applies only when Shape is set to Point or Segment.

For example, this attribute can be used to render strand segments as hair and the relative thickness will not be view-dependant.

For more information about rendering strands in general, see Viewing and Rendering Strands [ICE Particle Simulations].
