syflexIWind

Categories

  • Syflex Force

Description

Applies a directional force with intensity and noise (perturbation) that adds variation to the wind's movement.

This node is the same as the Syflex Wind compound.

For information on Syflex ICE forces, see Syflex ICE Forces.

Ports

Inputs

Direction

The direction of the wind force in XYZ global space.

Perturb

Perturbation (noise) in XYZ global space that matches one or more defined Direction axes.

This value is the maximum percentage of perturbation that is added to the direction of the wind.

Intensity

The strength of the wind force on each of the defined Direction axes.

Time Freq

The time interval (in frames) between each change in the perturbation.

One Side

The wind force can be applied on either one or both sides of the cloth.

Select this option to have wind applied to only the outside of the cloth. This is the side in which direction the normals are pointing.

When this option is off, the wind is applied to both sides of the cloth.

Outputs

Force

The output force.
