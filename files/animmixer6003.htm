Tracks Display Options Dialog Box (Animation Mixer)

Switches the display of various track properties in the animation mixer. Some of these options can also be set on the animation mixer's View menu.

To display: Right-click on an empty area of the track and choose Display Options.

Name

The name displayed at the left of tracks. The model name is shown before the track name.

Transition

Colored bars representing transitions between clips: purple for standard, red for Cardinal, and light orange for bridge.

Markers

Rust-colored outlines representing markers on tracks. This option is currently unavailable.
