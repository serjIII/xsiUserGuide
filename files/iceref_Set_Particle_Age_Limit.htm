Set Particle Age Limit

This compound sets a limit to the number of seconds that a particle lives from when it is born. If you want to kill off the particles at the end of this time, you need to use the Delete Particle at Age Limit compound in conjunction with this compound.

Plug this compound's Execute output into the Execute on Emit port of the Emit compound or in a Port on the ICETree node.

For more information, see ICE Particle Age [ICE Particle Simulations].

Tasks: Particles/Setters, Particles/Emission Control

Output Ports: Execute

Reference

The particle's AgeLimit attribute.

Select Value Type

Sets the age limit in either seconds or frames.

Age Limit

The number of seconds or frames of the particle's age, depending on which Select Value Type you chose. This uses the particle's AgeLimit attribute as a reference.
