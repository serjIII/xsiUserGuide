Symmetry Map Property Editor

Controls attributes of a symmetry map. Symmetry maps establish correspondence between points on either side of a geometrical object for symmetrical manipulation.

To apply: Select an object and choose Get Property Symmetry Map from any toolbar.

To redisplay: Select the object, click the Selection button on the Select panel, expand the Clusters\SymmetryMapCls node, then click the symmetry map's icon (by default named Symmetry_Map).

Name

The name of the symmetry map.

Map Color

The color used for the points directly on the plane of symmetry that do not get affected by symmetrical manipulation. This color also automatically determines the distinguishing colors used for points on opposite sides of the object. You can adjust the color if you need to distinguish between multiple symmetry maps on the same object.
