Comment Property Editor (Render Tree)

Specifies the text and color of a comment in the render trees.

For more information about applying and editing comments in the render tree or ICE tree, see Documenting Trees: Groups and Comments [ICE Guide].

Comment

Name

The name of the comment node.

Apply

Applies changes without closing the property editor

Comment

The text of the comment.

Word Wrap

Automatically wraps long line to the beginning of the next line when they reach the end of the comment area.

Auto Resize

Automatically resizes the comment area in the tree based on the comment text.

Color

Title Color

The color of the background for the comment text.

Title Color Text

The color of the comment text

Background Color

The background color of the group area. Available for group comments only.
