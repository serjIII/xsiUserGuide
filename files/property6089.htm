Deform by Spine Operator Property Editor

The Deform by Spine Op property editor is created when you apply a deformation by spine, but it is not displayed automatically.

To display: Select the deformed object then choose Edit Properties Modeling Properties and click the Deform by Spine Operator tab.

Mute

Toggles the deformation on and off without affecting the values of other parameters.
