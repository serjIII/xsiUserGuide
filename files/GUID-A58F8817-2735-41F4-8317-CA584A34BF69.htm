mental ray Progressive Rendering

The mental ray renderer in Softimage supports progressive rendering.

In the progressive rendering mode, a full image is rendered at a low resolution, then enhanced to higher resolution as more image data becomes available. The initial low-resolution display is a big fuzzy thumbnail image, and each update makes the image progressively sharper.

Progressive rendering displays an approximation of the final result quickly in cases when the full resolution will take a long time to download or compute.

By default, the progressive rendering mode is not active. You can activate it by:

  • Clicking the Progressive button in the View Rendering/Texturing Render Preview window. See Previewing a Single Frame.

or

  • Choosing the Progressive Rendering option from the Render Region menu. See Previewing Interactively with the Render Region.

For more information about the progressive rendering options, see mental ray Render Options Property Editor.
