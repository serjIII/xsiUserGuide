The Image Clip Viewer

The image clip viewer is a handy tool that allows you to view and crop image clips. You can also play back animated image clips, and export images in a different image format.

TIP:The SDK Example workgroup includes a relational view called RV - Image Clip Viewer that can also be used to explore the clips in your scene. For information about connecting to the SDK Example workgroup, see SDK Example Workgroup [Customizing Softimage].

To view image clips in the image clip viewer

  1. Choose Views Rendering/Texturing Image Clip Viewer from the main menu.

  2. Do one of the following:

    • Choose File Import from the viewer's command bar and choose an image from the browser.

      or

    • Select a clip to view from the Clips menu in the viewer's command bar.

To export images from the image clip viewer

  1. Choose File Export from the viewer's command bar.

  2. From the browser that opens, specify a name, destination, and file format for the image.

To crop images in the image clip viewer

  1. Choose View Crop Tool from the viewer's command bar. The cropping guide appears.

  2. Drag diagonally across the image to define the uncropped region.

    A

    Drag diagonally to crop the image.

  3. At any time, you can undo the cropping by choosing View Uncrop from the command bar.
