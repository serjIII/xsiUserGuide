HSV To RGB

| Render Tree Usage

Category: Processing > Conversion

Shader Family: Texture

Output: Color

Converts a color input from an HSV color space to RGB values.

Name

The shader's name. Enter any name you like, or leave the default.

HSVA

Defines the input HSVA color.

Render Tree Usage

This shader acts somewhat like a color correction node. You can connect it between an image clip's output and a color input of a surface, mixer or similar shader. It is useful to edit the RGB values of an image clip before being applied to an object.
