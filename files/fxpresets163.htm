Pattern Generator

Creates a clip of a specified solid color, or with a grid or a variety of color bars. The frames can be automatically numbered, and a circle can be animated; these features are useful for testing purposes.

Pattern Generator does not require an input.

Produces an Output (Any).

Pattern

Type

Sets the pattern to be used. Select the required option from the pop-up menu.

Axis

Sets whether color bars are drawn horizontally or vertically. Select X axis or Y axis from the pop-up menu.

Base Color

Sets the color of the clip when the Type parameter is set to Solid Color, and the color of the grid lines when the Type parameter is set to Grid.

When Type is set to Grid, the background color of the grid will be black if the Base Color has 50% or greater luminance, or white if the Base Color has less than 50% luminance. Select the required color from the color menu.

Numbering

X/Y Offset

Set the position of the center of the number as a normalized proportion of the image. Set any floating-point values using the treadmills.

Text Size

Sets the size of the number as a normalized proportion of the horizontal size of the image. Set any floating-point value using the treadmill.

Stroke Width

Sets the width of the number stroke. Set any floating-point pixel value using the treadmill.

Number Offset

Sets the offset of the number from the current frame number. Set any integer value using the treadmill.

Text Color

Sets the color of the number.

Opacity

Sets the opacity of the number. Set any floating-point value between 0 and 1 using the slider.

Circle

X/Y Offset

Sets the position of the center of the circle as a normalized proportion of the image. Set any floating-point values using the treadmill.

Circle Size

Sets the radius of the circle as a normalized proportion of the vertical size of the image. Set any floating-point value using the treadmill.

Circle Width

Sets the width of the circle outline. Set any floating-point pixel value using the treadmill.

Circle Color

Sets the color of the circle.

Opacity

Sets the opacity of the circle. Set any floating-point value between 0 and 1 using the slider.

Output

Controls this operator's Output Options.
