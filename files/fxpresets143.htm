Twist

Distorts the image by twisting the two ends in opposite directions. The position and the amount of the twist can be controlled.

Requires an Input (Any).

Produces an Output (Any).

Twist

Amount

Sets the amount of twist. Set any floating-point value.

Axis

Sets whether the twist is applied about the X axis or the Y axis.

Center X, Y

Set the center of the effect as a normalized proportion of the image. Set any floating-point values.

Interpolate

Sets whether or not transformed areas will be smoothed.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
