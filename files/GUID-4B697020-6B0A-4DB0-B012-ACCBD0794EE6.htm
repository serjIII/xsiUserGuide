Merge Polygon Meshes

Replaces the Topology of the "self" object with the result of merging the topologies of the specified meshes. For more information, see Merging Topologies in ICE.

Tasks: Topology/Generators, Topology/Copy and Merge

Output Ports: Execute

In Name

Connect the name of a polygon mesh to merge. Connect additional meshes to New (In Name).

Transform Mode

  • No Transform performs to transformation on the inputs' point positions.

  • From Topo Input transforms the point positions by the input meshes' global kinematic values.

Execute on Merge

Connect any execute-type nodes you want to run immediately after merging.
