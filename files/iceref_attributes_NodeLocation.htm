NodeLocation

The locations of polygon nodes (also called polynodes or texture sample points) on a polygon mesh. This is useful for getting values of vertex color properties (CAVs).

Data Type

location

Belongs To

samples

Geometry Types

  • polygon mesh

Attribute Type

inherent

R/W or RO:

read-only
