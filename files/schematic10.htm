Schematic View Preferences

| Display | Visibility

To display: Choose File Preferences, expand the Editors node, and click Schematic View.

Display

Layout

Layout Orientation

Controls the overall position of parents and children:

  • Vertical Layout moves child nodes below their parents.

  • Horizontal Layout moves child nodes to the right of their parents.

Hierarchy Sorting

Sorts the root and child nodes according to one of the following criteria:

  • None (creation) does not sort the hierarchies at all. The default order is based on the order of creation of the root nodes.

  • by Type puts cameras first, followed by lights and then other 3D objects.

  • by Height/Width places the hierarchies in order of tallest to shortest. When two hierarchies have the same height, the one that is wider is placed before the other.

  • by Width/Height places the hierarchies in order of widest to narrowest. When two hierarchies have the same width, the one that is taller is placed before the other.

Clean Hierarchy

Controls whether nodes in the schematic view get rearranged when you change a hierarchy by parenting or cutting children.

  • If it is on, nodes in the affected hierarchy are automatically rearranged. The root of the hierarchy is not moved.

  • If it is off, nodes are not moved automatically.

Clean All

Controls whether nodes in the schematic view get rearranged when you display or hide different types of nodes using the options in the Show menu or context menu.

  • If it is on, nodes are automatically rearranged when you display or hide different types of node. However, nodes are not moved when you display or hide links.

  • If it is off, nodes are not moved automatically when you display or hide nodes and links.

Node Color

Sets the colors used to draw elements in the schematic:

  • Use Standard Colors displays unselected objects in black.

  • Use Wire Colors displays unselected objects using their wire color.

  • Use Diffuse Colors displays unselected objects using the diffuse component of their material. Objects without their own material inherit the material of their parent or use the scene's default material.

Draw Parallel Links

Controls whether parent-child links are drawn as straight lines or as vertical and horizontal segments.

Show Grid

Displays a grid background on the schematic view.

Show Links for Selection Only

Displays links only for nodes that are selected. This simplifies the view because links aren't displayed for all objects at once.

Show Icons

Displays icons for different element types, like in the explorer. Click an icon to open an element's property editor.

Visibility

Nodes

Models

Displays the roots of model hierarchies.

Simulation

Displays particle types and events.

Materials

Displays materials.

Node Sub-types

Lights

Displays lights.

Cameras

Displays cameras.

3D Geometry

Displays 3D objects with points, such as polygon meshes, NURBS surfaces, hair objects, point clouds, lattices, and so on.

Curves

Displays curve objects.

Nulls

Displays nulls.

Chains

Displays roots, bones, and effectors of kinematic chains.

Control Objects

Displays control objects like forces, waves, and volume deformers.

Texture Controls

Displays texture control objects.

Links

Shared Material

Displays links between objects and materials that have been applied to groups that contain those objects.

Constraint

Displays links between objects that are constrained and those they are constrained to.

Associated Model

Displays links between selective lights and any included or excluded objects, as well as between models and their instances.

Expression

Displays links between objects that have an expression on one of their transformation parameters and any objects involved in the expression.

Operator

Displays links between objects that have an operator dependency, for example, between a lofted surface and the curves used as inputs.

Simulation

Displays links between objects that are involved in a simulation, force example, between a force and any affected objects.
