syflexIMimic

Categories

  • Syflex Constraint

Description

Constrains a simulated cloth or curve's vertices to those on another object that has the same topology (the same object type and same number of vertices). When the constraining object is animated (using any transformation or deformation), the vertices of the cloth or curve attached to it will mimic it.

This node is used in the Syflex Mimic and Syflex Mimic Null compounds.

For general information on constraints, see Syflex ICE Constraints.

Ports

Inputs

Points

The points on the cloth or curve object that you want to constrain.

Matrix

The matrix of points of the constraining object.

The vertices of the cloth or curve are attached directly to the constraining object's vertices using the same index (that is, vertex 37 on the cloth or curve is constrained to vertex 37 on the constraining object).

Position

The XYZ global position of the constraining object.

Reset

Forces the node to recompute where the simulated object's points are attached to the constraining object.

Distance

Distance (in Softimage units) between the cloth or curve's vertices and the constraining object. When this value is:

  • Greater than 0, the vertices are constrained to the constraining object with a spring.

  • Equal to 0, the vertices of the cloth are fully constrained to the constraining object without a spring. This is faster to compute.

  • Less than 0, the springs have no compression force.

Stiff

Stiffness of the spring between the cloth or curve vertices and the constraining object.

This parameter is available only if the Distance value is greater than 0.

Damp

The damp coefficient, which controls the intensity of the damping effect on the spring.

This parameter is available only if the Distance value is greater than 0.

Outputs

Force

The ouput force.
