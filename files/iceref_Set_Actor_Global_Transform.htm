Set Actor Global Transform

Sets Self.CrowdFX_Actor_Deformers.

This compound can be used in an ICE tree on the Point_Cloud object of a CrowdFX simulation.

Tasks: Crowds/Animation, Crowds/Setters

Output Ports: Execute

Actor Transform

The array of 4x4 matrices representing the deformer transformations per particle to set.

Reference

The attribute name. This must be Self.CrowdFX_Actor_Deformers to work with the default crowd compounds.
