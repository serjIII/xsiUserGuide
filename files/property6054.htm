Wave Op Property Editor

| Wave Op | Time Control

Deforms objects, clusters, or tagged points by a wave.

To apply: First get a wave control object by choosing Primitive Control Object Wave. Next select the elements to be deformed then choose Deform Wave from the Model or Animate toolbar and pick the wave object.

To redisplay: Select the wave-deformed object then choose Edit Properties Modeling Properties and click the Wave Op tab.

Wave Op

Mute

Toggles the deformation on and off without affecting the values of other parameters.

Amplitude

A multiplier for the height of the wave profile curve defined on the Profiles tab of the Wave Property Editor of the wave control object. This can be used to control decay.

Spread

A modifier for the length of the wave profile curve. Lower values produce longer waves.

Time Control

Start offset

The first frame for which the wave begins to affect the object.

End

The last frame for which the wave affects the object.
