Simple State

This compound is the same as the State compound except that it doesn't contain the Simulate Particles node. This allows you to use it with either the Simulate Bullet Rigid Bodies or Simulate Rigid Bodies node.

Plug any of the Simulate nodes into the Simulate port of this node - see Setting Up States for more information.

Tasks: Particles/States

Output Ports: Execute_State

See the ICE Particle States for general information on using states.
