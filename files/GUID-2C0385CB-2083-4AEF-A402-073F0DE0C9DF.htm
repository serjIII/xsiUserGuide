Export Alembic

Using the Export Alembic option, you can export all objects or selected polygon and NURBS objects in a Softimage scene to an Alembic cache file.

To export all or selected objects to an Alembic cache file

  1. Open a Softimage scene and select the object(s) that you want to export.
  2. Choose File Alembic Export Alembic.

    The Alembic Export dialog box appears.

  3. Specify the Start Frame and End Frame in a sequence to determine the cache time range.
  4. From the Archive Format drop-down list, choose the Ogawa or the HDF5 archiving file formats to store and organize large amounts of cache data.
  5. Turn on the Include hierarchy up to Scene Root option to export the parent hierarchy of a scene.

    The ICE Attributes for PointClouds and ICE Attributes for Polygon Meshes text boxes display the standard ICE attributes as in the cache manager. If you do not want to export all these pre-populated attributes, then click Delete All. If you want to append all the pre-populated attributes to the other attributes that you type in, then click Add Default Attributes. To export all attributes, type *.

    In the ICE Attributes for other primitives text box, specify any additional attributes related to primitives that you want to export

  6. Click OK.
NOTE:Softimage supports exporting only the following object(s) to Alembic format:

  • PointCloud — Shape instancing, particle animation through ICE attributes (position, color, size, orientation), and most of the ICE data types

  • PolyMesh — Shape animation, UVs, FaceSets

  • Null rig
