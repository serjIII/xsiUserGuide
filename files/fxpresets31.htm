Comp Extract

| Graph | Output | Output

Extracts a matte based on any primary (red, green, blue) or secondary (cyan, magenta, yellow) component. The matte can be adjusted using a graph.

Requires an Input (RGB).

Optionally takes a Combine Matte (A).

Produces an Output (A).

Graph

Graph

The horizontal axis represents the selected component values in, and the vertical axis represents the matte values out.

Comp

Sets the component that will be used when extracting the matte. Select Red, Green, Blue, Cyan, Magenta or Yellow from the pop-up menu.

Make Cutout

Forces the output image to RGBA mode.

Premultiply RGB

Premultiplies the output image by the generated matte.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
