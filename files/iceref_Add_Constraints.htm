Add Constraints

Connect this node to the root or another execute-type port, and connect other constraint nodes into it.

You can have only one Add Constraints node per constrainee per tree.

The constrainee specified here must match the constrainee specified in all connected constraint compounds.

Category: Kinematics

Output Ports: Execute

Constrainee In Name

A scene reference to prepend to Constrainee Reference to fully resolve the name.

Constrainee Reference

The object to constrain.

Constraint

Connect a constraint compound to this port to apply it.

To apply additional constraints, plug them into New (Constraint).
