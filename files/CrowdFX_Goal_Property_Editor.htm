CrowdFX Goal Property Editor

Custom property that sets data used for goals in a CrowdFX simulation.

To display: Select a CrowdFX_Goal icon in the explorer (under a Crowd model node).

For general information, see Moving Toward Goals.

Index

The index of the goal when using Goal Sequencer. Actors go first to the goal with the lowest index in their assigned goal group, then the next lowest, and so on.

Group Index

Specifies which group that the goal belongs to. Different actors can be assigned to different goal groups using Goal Group Index in Initialize Collision Avoidance.

Min Distance

The size of the goal. With Goal Sequencer, this is the minimum distance from the goal at which actors decide that they have reached it and proceed to the next goal. With Closest Goal, this is the maximum radius for a random position around the goal.
