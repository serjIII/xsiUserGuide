Dynamics Initial State Property Editor

| Pose | Velocity

Determines the position, orientation, and linear and angular velocity of a rigid body or constraint object at the start of the simulation.

NOTE:Scaling is not considered as part of the initial state properties because rigid body simulation does not drive scaling. The scale of an object is automatically taken into consideration for the initial state.

To apply: Select an object and choose Create Rigid Body Active or Passive Rigid Body from the Simulate toolbar, or create a rigid body constraint from the Create Rigid Body Constraint menu or the Multi Constraint Tool command.

To redisplay: Select the rigid body or constraint object and choose Modify Rigid Body Edit Rigid Body from the Simulate toolbar. You can also click the rigid body's Dynamics Initial State icon in the explorer.

For more information, see Setting a Rigid Body's Initial State [ Simulation and Effects].

Pose

Position

PosX/Y/Z

Sets the position in X/Y/Z of the rigid body or constraint object at the start of the simulation (initial position).

You can also change these values by translating the object in a view at or before the first frame of the simulation. The Pos values here are automatically updated to reflect the current transformation values.

Position From Animation

Uses the values from the rigid body or constraint object's animated position parameters as the initial position for the simulation.

This option is automatically activated if you select the rigid body or constraint object and choose Modify Rigid Body Use Animation as Initial State from the Simulate toolbar.

Rotation

RotX/Y/Z

Sets the rotation in X/Y/Z of the rigid body or constraint object at the start of the simulation (initial orientation).

You can also change these values by rotating the object in a view at or before the first frame of the simulation. The Rot values here are automatically updated to reflect the current transformation values.

Rotation from Animation

Uses the values from the rigid body or constraint object's animated rotation parameters as the initial orientation for the simulation.

This option is automatically activated if you select the rigid body or constraint object and choose Modify Rigid Body Use Animation as Initial State from the Simulate toolbar.

Velocity

Linear

LinVelX/Y/Z

Sets the linear velocity in X/Y/Z of the rigid body or constraint object at the start of the simulation.

Linear Velocity From Animation

Automatically calculates the linear velocity at the start of the simulation based on the rigid body or constraint object's animated position parameters.

This option is automatically activated if you select the rigid body or constraint object and choose Modify Rigid Body Use Animation as Initial State from the Simulate toolbar.

Angular

AngVelX/Y/Z

Sets the angular velocity in X/Y/Z of the rigid body or constraint object at the start of the simulation.

Angular Velocity from Animation

Automatically calculates the angular velocity at the start of the simulation based on the rigid body or constraint object's animated position parameters.

This option is automatically activated if you select the rigid body or constraint object and choose Modify Rigid Body Use Animation as Initial State from the Simulate toolbar.
