Average (Expression Function Reference)

The FunctionValue Average command in the expression editor inserts the token av( at the insertion point in the editing pane. This token returns the average value of a list of expressions. After the opening bracket ( enter a series of expressions separated by commas then enter a closing bracket ).

Example 

av(sphere.kine.local.posx, sphere.kine.local.posy)

Returns the average of the local X and Y translations of sphere.
