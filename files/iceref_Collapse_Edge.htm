Collapse Edge

Categories

  • Topology

Description

Weld the two vertices of the edge(s) together.

Ports

Inputs

Topo

The topology you want to use. If not connected, the default is "Self.Topology".

Indices

The indices of the edges to collapse.

Ratio

Controls the position of the resulting vertex as a ratio from the position of the first vertex to the second vertex in the edge.

Outputs

Result

The modified topology.
