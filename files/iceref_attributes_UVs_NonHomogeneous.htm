UVs_NonHomogeneous

The non-homogeneous equivalents of the texture UV coordinates at a location in a texture projection.

If you get this attribute on a spatial projection, the W value is dropped. If you get this value on a homogeneous (camera) projection, the values (U/q, V/q) are returned where q represents the distance to the camera (stored in the W coordinate).

Data Type

2D vector

Belongs To

samples

Geometry Types

  • texture UV coordinates

Attribute Type

inherent
