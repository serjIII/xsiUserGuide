Get Weight Map Value

This compound gets the values that are defined in the weight map on the "self" object.

Make sure to first create a weight map for the object (Get Property Weight Map).

This compound is designed to reference the default weight map (called Weight_Map) on the default cluster created on an object; however, if you want to use a different weight map, you must specify it.

To do this, click the Explorer button in this property editor to select a weight map from the list that appears, or click the Pick button and pick a weight map from another explorer. If you pick the weight map, make sure to remove the object name from the beginning of the string and add the Weights attribute to the end of the string so that the weight map name is displayed as shown below.

Tasks: Deformation/Getters

Output Ports: Value

Weightmap Reference

The Weights attribute of the selected weight map on the object. The syntax should look like this, depending on the weight map's name:

.cls.WeightMapCls.Weight_Map.Weights
