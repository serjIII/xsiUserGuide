Set Particle Force

This compound lets you set the particle's force, using the particle's Force attribute as a reference.

Plug its Execute output into the Execute on Emit port of an Emit compound to set this value once, or plug it into the ICETree node to update at every frame.

For more information on forces in general, see ICE Forces [ICE Guide].

Tasks: Particles/Setters

Output Ports: Execute

Force

Sets the particle's force value in any XYZ direction.

Reference

The particle's Force attribute.
