WeldPoints Op Property Editor

Merges two or more vertices on a polygon mesh into a single vertex.

To apply: See Welding Vertices {Modeling].

To display: Select the polygon mesh object, then right-click on the Selection button on the Select panel and choose Modeling Properties.

Max. distance

Welds only those points that are no further apart than the value set by the slider. If this option is off, all selected points are welded. This option is not available if a target was specified when applying the operator.

Collapse new double edges

Collapses double edges created by the welding process. You can turn this option off, for example, if you want to weld two rows of non-boundary points without collapsing them all to a single vertex.
