Build Array from Per Point Data

Builds an array of point locations in the object context. You can then plug its output into a Get Data node to get an array of point attribute values. This is useful for looking up point values using the point index as the array index, for example, if you want to set data on one point based on the previous point's value.

Category: Array

Output Ports: Result

In Name

A scene reference to prepend to Reference to fully resolve the name.

Reference

The object whose point locations will be placed in an array.
