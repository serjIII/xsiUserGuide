Accented Edges

The Accented Edges effect adds accents to the edges in an image.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

Edge Width

Set higher values to make the edges wider, or set lower values to make the edges narrower.

Edge Brightness

Set higher values to make the edges bright (more like white chalk), or set lower values to make the edges dark (more like black ink).

Smoothness

Set higher values to make the overall image smoother, or set lower values to retain more detail.

The Smoothness control in this effect is useful for making a photograph appear more painterly.
