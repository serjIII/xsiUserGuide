Limit Envelope Deformers Operator Property Editor

Limits the number of deformers to which envelope points can be weighted by unassigning the deformers with the lowest weights and normalizing the values among the remainder. See Limiting the Number of Deformers per Point [Character Animation].

To apply: Click Enforce Limit in the weight editor.

Maximum Deformers

The maximum number of deformers to which a point can be weighted.
