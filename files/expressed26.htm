Exponential (Expression Function Reference)

The Function Exponent Exponential command in the expression editor inserts the token exp( at the insertion point in the editing pane. This token evaluates the value e^expression where e is the natural (Napierian) base of logarithms, or approximately 2.718. Enter an expression after the opening bracket ( then enter a closing bracket ).

Example

exp(sphere.kine.local.posy)

Returns the value 2.718 raised to the power of the value of the local Y translation of sphere.
