Fit Curve Property Editor

Creates a new curve by resampling an existing one. The parameterization of the new curve is uniform.

To apply: Select a curve and Create Curve Fit on Curve.

To redisplay: Select the created curve then choose Edit Properties Modeling Properties and click the Fit Curve tab.

Inputs

Hide/Unhide

Hides all the inputs for the generated object to simplify the display of your scene in the 3D views, or unhides the inputs when you need to modify them.

Delete

Freezes the generator and deletes the inputs. This simplifies the scene once you are satisfied that you won't need to modify the inputs again.

Subdivision

The number of segments between knots in the resulting curve. Higher values follow the shape of the original curve more closely but result in denser, heavier geometry.

Degree

Whether the curve is a Cubic or Linear NURBS.

Close

Closes the resulting curve, if the input curve is not closed. If the input is already closed, the result always closed and this option has no effect.

Maintain Discontinuity

Maintains sharp points created by multiknots in the original curve. When off, multiknots are smoothed out.

Note that when this option is on, the number of segments in the result is never less than the number of segments in the input curve, even if the Subdivision parameter specifies a lower value.
