Set Particle Size

This compound lets you set the particle's size, using the particle's Size attribute as a reference.

Plug its Execute output into the Execute on Emit port of an Emit compound to set this value once, or plug it into the ICETree node to update at every frame.

For more information, see Particle Size and Scale Attributes [ICE Particle Simulations].

Tasks: Particles/Setters

Output Ports: Execute

Size

Sets the particle's size in Softimage units when drawn on screen. Some compounds use the size value to help with collision detection or to define motion, such as Stick to Surface, for example.

Reference

The particle's Size attribute.
