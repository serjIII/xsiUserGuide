Video Noise Removal

Removes or extracts noise from video-based clips. If the noise is extracted, edges can be prevented from being extracted as noise. The noise can be composited back in after processing using the Grain Composite operator.

Requires an Input (RGB or RGBA).

Produces an Output (RGB or RGBA).

Noise Filter

Mode

Sets whether the image or the noise is displayed and rendered. Select Image to remove the noise from the image, or Noise to extract the noise for re-compositing later.

Edge Threshold

When extracting the noise, Edge Threshold limits the range of high frequency information to be included in the noise. Increase this value until the edges disappear.

Edge Scale

When extracting the noise, Scale sets the intensity with which the noise is drawn. The higher the value, the greater the intensity.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
