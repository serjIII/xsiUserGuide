Every Nth Particle

This compound uses specific particle intervals as a trigger for an effect. For example, you could have an effect triggered for every 10th particle that is born.

Plug its Result output into a Trigger port on a State compound or in a Condition port of an If node.

For more information on states, see ICE Particle States [ICE Particle Simulations].

Tasks: Particles/Conditionals

Output Ports: Result

N Value

The interval of particles that is used as a trigger.
