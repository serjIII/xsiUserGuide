Define Animation State

Categories

  • Crowds > Animation

Description

Calculates the transforms of the actor deformers in global space. This node is used by the Convert Deformer Transforms to Global compound .

Parameters

Reference

Reference to the simulation Point_Cloud

State type

Name of the Animation State

Ports

Inputs

In Name

Name of the simulation Point_Cloud. Should be Self, when the node is used in the Animation_Blending ICE Tree.

Actor Proxy

Proxy name of the actor

State Name

Name of the Animation State

Pose Modifier

Here, you can connect an Ice tree to change the prerecorded animation by changing transformation values for single deformers. You can achieve this by using the Get Self Pose and Set Self Pose nodes. You can add even complex logic such as an IK solving system here.

Params Update

Updates the parameters of an animation state.

Name1

The animation source.

No of loops1

Number of loops in the Animation Source

Offset1

Offset to the start of the loop in the Animation Source

Outputs

Define

Execute

Out Name

Out name of the simulated Point_Cloud
