Constant (Texture)

DEPRECATED. This shader is unsupported, but it may still get installed with Softimage to provide compatibility with older scenes that use it. It is recommended that you replace unsupported shaders in your scenes with equivalent shaders from the current Softimage shader library.

| Render Tree Usage

Shader Type: Texture

Output: Color (RGB) value

Allows you to set the returned color and the alpha component separately.

Name

The shader's name. Enter any name you like, or leave the default.

Color

Defines the RGB components used in the output value; default is white.

Transparency

Defines the alpha channel according to the set output Color.

Render Tree Usage

Used to control a parameter of a surface shader that you wish to remain constant. For example, you can connect it to the Specular parameter of a surface shader if you do not want the specular value affected by the light. The Color parameter can be connected to another or an Image processing shader.
