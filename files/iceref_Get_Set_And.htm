Get Set And

Categories

  • Math > Statistics

Description

Outputs the logical conjunction (AND) of all Boolean values in the set. In other words this node returns true if all the elements in the set are true.

Ports

Inputs

Value

The data set on which perform the logical conjunction (AND).

outputs

Result

The logical conjunction (AND) of the data set.
