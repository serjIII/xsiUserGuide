Start Timer

This compound uses a timer to determine the amount of time that a particle spends in a certain state. This works in conjunction with the Test Timer and Init Timers compounds.

You need to use the Init Timers compound to initialize the timers before you can use them.

Plug its Result output into an If True port of an If node (for example) or any node or compound that causes an execution.

If you're using states, plug its Result output into an Execute on Trigger port on a State compound or the Simulation Root node.

For more information, see Setting Particle Timers.

Tasks: Particles/Timers

Output Ports: Execute

Select Timer

Select a name for this timer. You can have several timers in an ICE tree.

Global Time

Uses the scene's Global In/Out setting (as displayed in the timeline range) for the time value.

Timer Duration in Seconds

The number of seconds that the particles stay in this state.

Reset Timer if Already Started

Resets the timer.
