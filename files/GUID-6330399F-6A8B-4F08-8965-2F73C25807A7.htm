Get Particle Goal

This compound gets the goal object's point position and normal values as set by its GoalLocation attribute. This is used in conjunction with the compounds found in the Goals group. You can output the goal object's Location, Position, and Normal data.

For more information on goals, see ICE Particle Goals [ICE Particle Simulations].

Tasks: Particles/Getters,Particles/Goals

Output Ports: Location, Position, Normal

Reference

The GoalLocation attribute.
