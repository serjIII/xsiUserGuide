Set Goal By Walls

Sets the wall exit point as temporary goal, so that the actors can avoid a wall before continuing the course.

Tasks: Crowds/Behavior

Output Ports: Execute. Connect this port to the Goal group on the Set Velocity Using Collision Avoidance compound.
