Set Crv Knot Multiplicity Op Property Editor

Sets the number of control points associated with knots.

To apply: Select one or more knots on a curve, then choose Modify > Curve > Set Knots Multiplicity.

To redisplay: Select the curve, click the Selection button on the Select panel, then click the Set Crv Knot Multiplicity Op icon.

Multiplicity

The number of control points associated with the knot. A knot will full multiplicity (3) behaves like a Bezier point.
