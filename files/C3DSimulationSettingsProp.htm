Simulation Settings Property Editor

Sets the number of times per frame that an ICE simulation is sampled.

To display:

Click the Simulation Settings property under an ICE-simulated object.

See ICE Simulation Subframe Sampling for more information.

NOTE:Subframe sampling is not currently supported for non-ICE simulations (such as rigid bodies, soft body, cloth, and non-ICE Syflex Cloth).

Subframe Sampling

The number of times per frame that the ICE simulation is sampled. The higher the value, the more stable and accurate the simulation and collisions will be, but also slower.

See Setting the Number of Subframes for a Simulation for more information.

Cache All Simulation Samples

Caches each subsampled frame when you cache the ICE simulation.

If this option is off, each "whole" frame is cached without the Subframe Sampling data.

See Caching Subframe Sample Data for more information.

Open Cache Manager

Opens The Cache Manager in which you can write (and read) cache files for the ICE simulation.
