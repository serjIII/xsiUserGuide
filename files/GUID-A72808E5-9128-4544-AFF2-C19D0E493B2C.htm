Display Strand Shape

Working with strands can be difficult because the actual shape of the strand is unknown until render time. This compound builds hull lines to approximate the actual shape of the strand volume as it will be rendered.

The compound cannot determine the shape used by the particle, so you must manually set it.

Create strands, then plug this compound's Execute output into a PostSimExecute port on the Simulation Root node or in a port on the ICETree node.

Category: Debugging

Tasks: Particles/Strands

Output Ports: Execute1

Strand Shape

The shape used for the display strand shape. This does not need to match the Shape you've set for the particle.

Draw Cross Sections

Creates a three-dimensional shape around the hull lines that are drawn based on the Strand Shape you set.
