Hair Split Operator Property Editor

Creates a single new interpolation group for the selected guide curves, for example, allowing you to create parts in hair. Render hairs are not interpolated between guide curves that belong to different interpolation groups. Instead, half the render hairs between the guides follow one guide and the other half follow the other guide.

To see which guide curves belong to which interpolation group, turn on Hair Interpolation Group IDs in the Objects.

To apply: Select at least one point on each of the guide curves you want to split, and then choose Modify Interp Shatter from the Hair panel.

To display: Select the hair object, click the Selection button on the Select panel, and then click the Hair Split Operator icon.

For more information, see Working with Groups of Guide Hairs [ Hair ].

Mute

Toggles the deformation on and off without affecting the values of other parameters.
