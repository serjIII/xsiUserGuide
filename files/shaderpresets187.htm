Kaleidoscope UV Remap

Render Tree Usage

Category: Texture Space Controllers

Shader Family: Texture

Output: Scalar

Remaps a 2D-kaleidoscope texture space based on coordinate input.

Name

The shader's name. Enter any name you like, or leave the default.

Coordinates

Defines the X, Y, and Z texture coordinates.

Angle

Defines the current rotation of the texture coordinate. To make the texture turn, animate this value.

Number of Reflections

Defines the number of reflections within the remapping. A value of 1 results in a mirror-like effect, with two halves reflected. As you increase the value, the texture takes on a star shape with angular, reflected edges around the center.

Render Tree Usage

Use this shader to edit a Coordinates input. Place it between a Texture Space Generator and a Texture Generator shader to create highly customizable effects.

Connect a Texture Space Generator shader to drive the Coordinate value of this shader.
