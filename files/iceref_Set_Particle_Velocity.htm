Set Particle Velocity

This compound lets you set the particle's velocity data, using the particle's PointVelocity attribute as a reference.

Plug its Execute output into the Execute on Emit port of an Emit compound to set this value once, or plug it into the ICETree node to update at every frame.

Tasks: Particles/Setters

Output Ports: Execute

Velocity

Sets the particle's velocity value in the global XYZ direction. This is the speed of the particle with a direction vector.

Reference

The particle's PointVelocity attribute.
