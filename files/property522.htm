Motion Blur Property Editor

To display: Choose Get Property Motion Blur

When you activate motion blur for the scene, in the pass, and the render region, all moving scene objects are rendered with motion blur.

Local motion blur properties are useful for turning motion blur off for specific objects, which you can do from this property editor.

Blur

Switches the standard SRT motion-blur effect on or off.

Deformation Blur

When on, this parameter activates a motion blur that is recomputed every time a vertex is animated.

Because this option is very memory-hungry, it is off by default.
