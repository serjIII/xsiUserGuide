Sense Walls

Detects walls and calculates the exit point. This compound is used by the Set Goal By Walls compound.

Tasks: Crowds/Behavior

Output Ports: Execute. Connect this port to the Avoid Neighbors compound.

Geometry

Connect to the wall geometry.

Mode

Mode for Raycast query.

Blocked Exit Point Factor

Push factor to apply when the exit point is blocked by other actors.

Recompute Exit Point Time

Timer for recomputing the exit point. An actor might choose to use the other exit point, if the current one is blocked for too long.
