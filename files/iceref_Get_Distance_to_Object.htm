Get Distance to Object

This compound gets the distance from the current particle position to an object. You can output the Distance and Distance Vector values.

Tasks: Particles/Getters

Output Ports: Distance, Distance_Vector

Global Pos

The global position of an object whose distance you want to consider. Set the XYZ values in global space here or plug in 3D vector data from another node to get that information.
