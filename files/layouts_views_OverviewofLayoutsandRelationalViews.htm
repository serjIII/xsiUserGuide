Overview of Layouts and Relational Views

Here's a quick overview of how to create your own layouts and views. More details can be found in the sections that follow.

  1. Open the layout editor or relational view editor as described in Creating and Editing Layout and Relational Views.

    The editor has a floating command bar that you can resize and move so that it doesn't limit the size of the window you are editing. However, most of the time you use the context menus in the layout or view window itself.

  2. Divide the window into panes to hold different views. See Working with Panes and Framesets.

  3. Choose the contents for the panes. See Setting Views and Panels and Setting Toolbars and Shelves.

  4. If desired, you can create trays with buttons to switch between panes. See Setting Tray Switchers.

  5. For relational views, you can define relationships that allow the selection in one pane to control the display in another pane or perform other actions. See Defining Relationships.

  6. Close the layout editor or relational view editor, and then display your handiwork using the View Others or View Layouts menu.

  7. If desired, you can open the .xsily or .xsivw file in a text editor or XML editor to define more advanced features. See Layout and View XML Syntax.

TIP:You can display a relational view by dragging the .xsivw file into a viewport from a browser, the Tree tab of the Plug-in Manager, or the My Views tab of the Application Shelf. This is often more convenient than choosing it from the View menu while you are editing and testing it.
