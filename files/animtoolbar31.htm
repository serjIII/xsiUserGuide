Set Envelope Dialog Box

To display: Select an object that has already been enveloped and choose Deform Envelope Set Envelope from the Animate toolbar. Set the option as desired, then click OK and use the left mouse button to add deformers or the middle mouse button to add deformers in branch mode. Right-click to end picking.

When Adding Deformers

Automatically Reassign Envelope

Controls whether weights are recalculated when adding deformers:

  • When on, the existing weights will be recalculated and distributed among all previous and new deformers.

  • When off, any objects you pick are added to the list of deformers but any envelope points are not automatically reassigned. You must reassign points to the new deformers manually.

This option has no effect on envelopes that have been frozen or imported from other software. Envelope weights cannot be automatically recalculated in these cases.

NOTE:If you have modified weights manually, you should leave this option off. Otherwise, the result of the new initial weighting combined with the manual modifications will probably not give good results.
