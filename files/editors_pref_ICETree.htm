ICE Tree Preferences

To display: In the ICE tree view, choose Show Preferences.

Node Creation Mode

Specifies the default state of new nodes in the render tree. Choose one of the following options:

Create Nodes as Expanded: New nodes are expanded by default.

Create Nodes as Collapsed: New nodes are collapsed by default.

Create Nodes as Show Connections Only: New nodes are collapsed except for connected parameters.

Selection Locked

Prevents the view from updating when you select other objects in the scene.

Allow Selection in Explorer

Allows selection in the hosted explorer on the right of the ICE tree to affect the global selection in the scene.

Character to resolve shortcut

Specify the shortcut character that you want to append with "g" or "s" while adding the Get Data or the Set Data node, respectively using the Tab key. See Specifying References for Get Data and Set Data Nodes using the Tab key for more information.

NOTE:If you want blank space to be the shortcut character, then specify <Space> in this text box.
