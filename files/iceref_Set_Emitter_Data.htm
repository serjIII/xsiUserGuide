Set Emitter Data

Sets data on emitter objects in CrowdFX simulation, and optionally creates the emitter's geometry. Instead of adding this compound manually, use either the CrowdFX Crowd Draw Directional Emitter or Add Emitter(s) from Selectioncommand on the ICE toolbar to make sure that emitters get added to the correct model and group of the CrowdFX simulation.

For more information, see Crowd Particle Emitters.

Tasks: Crowds/Setters

Output Ports: Execute

CrowdFX Emitter Bidirectional

Sets CrowdFX_Emitter_Bidirectional, which controls whether a directional emitter generates actors facing both forward and backward.

Display Direction

Displays arrowheads indicating the emitter's direction (forward, or forward and backward) in the 3D views.

ID

An identifier for the emitter. By default, this is 0 but you can set it to a unique value to identify a particular emitter for use with your own custom logic. You can get an actor's emitter by looking up CrowdFX_Emitter_EmitterID at the actor's EmitLocation.

Negative values are used for non-directional emitters.

Grid

Create Grid

Generates a simple grid topology for the emitter. This is typically used only by directional emitters.

U Subdiv, V Subdiv

The number of subdivisions in the U and V directions.
