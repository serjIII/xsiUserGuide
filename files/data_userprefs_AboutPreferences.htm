About Preferences

Autodesk Softimage has many preferences to control various aspects of its behavior. This chapter describes how to set and manage your preferences in general. Descriptions of individual options can be found in Preference Reference as well as interspersed throughout the Softimage documentation.

You manage preferences from the Preferences window. You can set, export, and import your preferences, as well as restore the defaults.

Preferences are saved in the *.xsipref format. These are text files that can be edited manually in any text editor.

At startup, Autodesk Softimage loads all *.xsipref files that it finds in the Data\Preferences folder of your factory or user path. If there are conflicts, the settings in your user path have priority.

Autodesk Softimage saves only those preferences that you modify. Modified preferences are always saved in the default.xsipref file in the Data\Preferences folder of your user path.
