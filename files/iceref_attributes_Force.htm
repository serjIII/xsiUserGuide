Force

The force applied to a point. This value, together with the point's mass, is used by the Simulate Particles and Simulate Rigid Bodies nodes to calculate the acceleration and update the velocity.

Note that this attribute is set to 0.0 at the end of every frame.

You can quickly get and set the Force attribute with the Get Particle Force and Set Particle Force compounds.

Data Type

3D vector

Belongs To

points

Geometry Types

  • point cloud

  • polygon mesh

  • NURBS surface

  • NURBS curve

  • lattice

Attribute Type

predefined

R/W or RO:

read/write
