NbSubSamples

Represents the number of subsamples that were executed for one frame. It is set by the EvolveOp, which relies on the Simulation Settings property to get the number of subframe samples used. See ICE Simulation Subframe Sampling for more information.

Data Type

scalar

Belongs To

objects

Geometry Types

  • point cloud

  • polygon mesh

  • NURBS surface

  • NURBS curve

  • lattice

Attribute Type

inherent

R/W or RO:

read-only
