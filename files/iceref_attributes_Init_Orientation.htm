Init_Orientation

A particle's original Orientation value when it was emitted.

This attribute is set by the Emitters compounds, and is exposed in their property editors as the Orientation parameters.

It can also be used by your own compounds.

Data Type

rotation

Belongs To

points

Geometry Types

  • point cloud

Attribute Type

custom

R/W or RO:

read/write
