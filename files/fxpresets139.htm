Radial Blur

Blurs the image inwards or outwards from a user-defined point, and about a user-defined rotation. The zoom factor can be controlled.

Requires an Input (Any).

Produces an Output (Any).

Radial Blur

Center X, Y

Set the center of the effect as a normalized proportion of the image. Set any floating-point values between 0 and 1.

Zoom

Sets the scaling of the blur. Set any floating-point value. 1 is no zoom, values less than 1 will zoom in, values greater than 1 will zoom out.

Angle

Sets the rotation of the blur. Set a value in degrees. The greater the angle, the slower the effect.

Mode

Sets whether the effect is rendered, or just the center marker shown. Select Locate while setting up the center, and Compute to render the effect.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
