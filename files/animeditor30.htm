Key Properties Editor

Controls how selected keys on function curves are manipulated in the fcurve editor. Each function curve corresponds to a parameter's animation, and each point corresponds to a key in that animation.

To display: Open the fcurve editor, select keys on fcurves, and choose Keys Key Properties from the fcurve editor command bar.

For more information about the fcurve editor in general, see Editing Function Curves [Animation].

Frame

The frame number of the selected key. To move the key to another frame, type a frame number and press Enter.

Value

The value of the selected key. To change it, enter a new value and press Enter.

Click the Previous Key or Next Key buttons to select other keys on the same function curve.

Interpolation

The Interpolation options control how values are calculated for frames between the selected key and the next key.

Constant

Repeats the value of a key for each of the frames after it. The movement is characterized by sudden changes and then static positions.

Linear

Connects keys by straight line segments. The movement is uniform, with sudden changes at each key.

Spline

Uses a curved spline interpolation. The interpolation accelerates and decelerates to ease into and out of each key, resulting in a smooth transition. The degree of acceleration and deceleration near a key is determined by the slope and length of the slope handles. This is the default.

Slope Orientation

Controls how the orientation of slope handles on key points change when they are manipulated. The slope handle controls the sharpness of a function curve's acceleration or deceleration near the key.

Unified

The slopes move together, keeping a constant angle between them. This is the default.

Broken

The slopes move freely as you drag each handle individually.

Slope Length

Controls how the length of slope handles on key points change when they are manipulated. The length of the slope handle helps you control the angle of the slope.

Unified

The slope handles move together, keeping the same length. This is the default.

Broken

The slopes move freely as you drag each handle individually.
