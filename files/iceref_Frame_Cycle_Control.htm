Frame Cycle Control

This compound lets you cycle frames when you play back a cached file. Plug its Frame output into the Frame port of the Cache on File node.

Category: Data Access

Output Ports: Frame_Value

Global Time

Uses the scene's Global In/Out setting (as displayed in the timeline range) for the time reference for the frame cycle.

Invert

Reverses the cycle animation.

Cycle

Cycles the frames from the First Frame to the Last Frame that you specify.

Hold

Holds the Last Frame that you specify after the cycle has finished.

Cycle Length

Number of frames in the cycle.

First Frame

The first frame that you want to use for cycling the frames.

Last Frame

The last frame that you want to use for cycling. If you select Hold, this is the frame that is held for the rest of the timeline after the frames have cycled.

Offset

Offsets the cycling by this number of frames.
