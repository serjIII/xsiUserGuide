Adding Nodes to ICE Trees

You can add nodes to an ICE tree in a variety of ways:

  • Drag a node from the preset manager. For more information about using the preset manager, see The Preset Manager.

  • Select an item from the Nodes menu.

  • Choose Compounds Import Compound to select an .xsicompound file.

  • Drag an .xsicompound file from a Softimage file browser or from a folder window of your operating system.

  • Drag an element from an explorer into the ICE tree view to add a pre-filled Get Data node. For more information, see Getting Data.

Adding Nodes and Compounds using the Tab key

In the ICE Tree editor, you can now quickly add ICE nodes and compounds by pressing the Tab key.

To quickly add ICE nodes and compounds

  1. Press the Tab key in the ICE Tree editor.

    A text field appears at the position of your mouse cursor.

  2. In the text field, type any part of a node or a compound name. For example, to add the Constrain Between 2 Points compound, you can type any part of the compound name such as "constrain", "strain", "cons", "rain", "point", and so on.
    NOTE:The text field is not case-sensitive.

    The autocompletion functionality fetches the matching nodes and compounds and displays them in the alphabetical order. For more information about autocompleting ICE attributes, see Accessing ICE Attributes Using Autocompletion.

    The first node or compound in the list is automatically selected. To choose another node or compound use the mouse pointer or the up or down arrow keys.

Copying and Pasting Nodes

To copy and paste nodes

  1. Select the nodes to copy.

  2. Right-click on one of the selected nodes (not over a port) and choose Copy Nodes. Alternatively, press Ctrl+C.

  3. If necessary, open another scene or switch to the ICE tree in which you want to paste the nodes. You can also copy and paste nodes between separate running instances of Softimage (however, you cannot copy and paste between different versions of Softimage).

  4. Right-click on the background and choose Paste. Alternatively, press Ctrl+V.

Deleting Nodes

To delete nodes

  1. Select the nodes to delete.

  2. Press the Delete key.

Removing Unconnected Nodes

To remove all unconnected nodes

  • Choose User Tools Delete Unconnected Nodes.

This affects nodes in the displayed level only. It does not affect nodes in nested compounds.
