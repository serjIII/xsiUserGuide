About Syflex Constraints

Using Syflex constraints, you can create a number of common effects for cloth simulations. This is useful for constraining simulated cloth or curves, such as for laundry hanging on a line, a flag attached to a flag pole, or clothing attached to a part of the body.

NOTE:Edge and polygon clusters on the cloth object are not supported for constraints - only point (vertex) clusters.

Types of Constraints

There are several types of constraints that you can create for cloth or curve simulations, using these Syflex ICE compounds:

  • Syflex Pin/Pin Null creates constraints between the cloth or curve's vertices and a polygon mesh object, usually one whose deformations are animated - see Pinning Cloth or Curve Vertices to an Object.
  • Syflex Nail/Nail Null creates constraints between the cloth or curve's vertices and another object - see Nailing Cloth or Curve Vertices to an Object.
  • Syflex Mimic/Mimic Null creates constraints between the cloth or curve's vertices and another object that has the same topology and usually has animated deformations - see Making a Cloth or Curve Mimic an Object.

Creating Clusters of Vertices on a Cloth or Curve Object

You can create clusters of tagged points (vertices) to define specific areas of a cloth or curve to be constrained. For example, you can create a cluster of points at one corner of a blanket that is constrained to a character's hand as he grabs it.

For general information on clusters, see Clusters.
