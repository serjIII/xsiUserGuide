Crowd Transform Offset Constraint

Tasks: Crowds/Constraints

Output Ports: Execute. Connect this port to the Pose Modifier on the Define Animation State or Define Move Animation State node.

Proxy Deformer Name

Name of the deformer. This should be a deformer under Crowd Actors_Proxies [ActorProxyModel] Rig_Proxy.

Children Transform Compensation

Turn on to keep the offset constraint on the deformer only.

Turn off to have the offset constraint propagated to the children.

Scale Active

Turn on to activate the Scale offset.

Scale (X, Y, Z)

Scale offset value

Rotation Active

Turn on to activate Rotation offset.

Rotation (X, Y, Z)

Rotation offset value

Translation Active

Turn on to activate Translation offset.

Translation

Translation offset value

Blend

Blend factor between the original pose and the constrained pose.
