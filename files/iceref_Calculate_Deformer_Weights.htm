Calculate Deformer Weights

Creates a weights table associating each point in a geometry with N members of the specified deformer group based on distance. The weights are stored per point in the Self.EnvelopeWeights array and the corresponding deformer indices are stored in Self.EnvelopeDeformerIndices. These attributes can be used by custom skinning compounds.

Tasks: Deformation/Skinning

Output Ports: Execute

Num Deformers Per Point

The number of deformers among which to divide each point's weight.

Bone Group In Name

A scene reference to prepend to the group specified by Bone Group Reference to fully resolve the group's name.

Bone Group Reference

Specifies the group of bones to use for weighting.
