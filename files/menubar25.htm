DirectX Export Dialog Box

Exports the scene to the DirectX (*.x) format.

To export: Choose File Export DirectX Export from the main menu.

File

Path

Specifies the path and file name of the exported .x file. You can type a path and file name into the text box or click the Browse button (...) to browse to the desired path.

Format Type

Specifies whether the .x file is output in Text, Binary, or Compressed Binary format.

Animation

Export Animation

Controls whether or not animation is exported.

Frame Offset

Specifies a frame offset for the animation. For example, if your animation runs from frame 1-100 in Softimage, but you need it to run from frame 50-150, you would enter an offset value of 50.

Type

Specifies whether the animation is output as Matrix Keys or as scaling, rotation and translation (SRT) transformation values.

Textures

Export Textures

Controls whether or not paths to textures are included in the exported .x file.

Absolute Paths

When on, paths to exported textures are absolute. Otherwise they are relative.

Sequences

When on, exports animated texture sequences. When off, exports the texture as a static texture using the first picture of the sequence. Default [off].

Copy Textures

When on, textures used by the exported scene are copied to the directory to which the .x file is saved.

Resize X/Y

If the Copy Texture option is on, these parameters allow you to resize the copies. Choose the textures' new X and Y resolutions form their respective drop-downs.

Choosing Original keeps the original resolution for each texture, while choosing Automatic allows Softimage to determine a reasonable resized resolution.

Format

If the Copy Texture option is on, this option allows you to specify a new file format for the copied textures. Choose a format from the drop-down, or choose Original to keep the original file format of each texture.

Options

Triangulate

When on, meshes are triangulated prior to export. Mesh data in the resulting .x file will specify triangles rather than quads or n-sided polygons.

Plot Animation

When on, exported animation is plotted to function curves.

Export Hidden Objects

When on, objects hidden in the Softimage scene are included in the resulting .x file. Otherwise, they are not.

Compress Mesh

When activated, vertices, normals and UVs are indexed, which reduces the size of the exported .x file.

Chains

Plot IK to FK: Plots the scene's IK to FK, which is exported in the .x file.

Preserve FK: If an IK chain is animated using FK (by adding rotation animation directly on the bones), the chain is exported the FK animation preserved.

OK

Once the other export options are set, click this button to export the scene in .x format and close the dialog box.

Cancel

Cancels the export operation and closes the dialog box.
