Plot Retarget Dialog Box

Plots the retargeted mocap data that is applied on a rig or skeleton and turns it into fcurves.

To display: Select a rig that retargeted mocap data on it, then click the Plot button in the Mocap to Rig Property Editor, the Rig to Rig Property Editor, or the Motor File Property Editor.

The plotting process removes the mocap data and retargeting operators from the animation controls, turns the motion data into fcurves, and fits those curves using the best tangency possible given the key Interval Size information. The plotted fcurves are then applied to the animation controls.

For information on retargeting in general, see Overview of Retargeting Animation with MOTOR [Character Animation].

Preserve Source Keys

Keeps the original keys from the animation on the rig, if the retargeted animation was created from keyframes.

Key Peak, Valley, and Inflection Points

Sets keys on these points of the animation data. You need to select this option if you have a large Interval Size value which causes fewer keys to be created.

Key at Interval Spacing

Set keys only at certain intervals in the plotted fcurves. If you select this option, you need to set the Interval Size.

Interval Size

The interval value for setting keys at interval spacing. For example, if you specify 2 as the Interval Size, a keyframe is created on every other frame on the plotted fcurve.

Start and End Frame

Determines the frame range of animation that you want to plot.
