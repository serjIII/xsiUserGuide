Mixer Animation or Layer Track Property Editor

To display: Select an animation or layer track and choose Track Track Properties, or right-click an empty area of an animation or layer track in the animation mixer and choose Track Properties.

Controls

You can also control these options using the track controls to the left of the tracks in the mixer.

Ripple

Toggles the activeness of ripple mode. When ripple is on and you move a clip by dragging it, it pushes the next or previous clips along the track if necessary so that clips do not overlap on the same track.

If ripple is off, you cannot drag a clip past the position of the next or previous clip. Note that adding new tracks will always ripple existing clips if necessary.

Auto Transition

Toggles the activeness auto-transition mode. When this option and Mix Auto Transition are both active, transitions are automatically created between all neighboring clips (on any track that has auto-transition active) when you add a clip. Any existing transitions there were before you selected auto-transition are replaced with the automatic ones.

Ghost

Toggles the activeness of ghosting mode. When an object is driven by two or more clips at a time, you can see the contribution of each clip's animation on the object as a ghost in the viewport.

To view ghosting, activate both this option and Mixer Ghosting in a viewport's display menu. This option is the same as clicking the g button at the left end of an animation track.

Mute

Mutes the track so that it does not contribute to the animation when you play back the scene or scrub the timeline.

Solo

Makes only this track contribute to the animation when you play back the scene or scrub the timeline: other tracks do not contribute.

Ghost Color

Sets the ghost color of the selected track and its accompanying ghost image in the viewport. Click the color chip beside this option to open the color editor and select a color from the color graph, or make this property editor a little wider until the color sliders appear, then set the color with them.

Time Span

These values are provided for information only. You cannot change them.
