PSD Layer Extract

Allows you to extract a specific layer from a .psd image.

Operator Name

The Operator's name. Changing it will update the operator's icon in the FX Tree.

Layer Name

Specifies the name of the selected layer.

Rectangle

Left, Right, Top, Bottom: Specifies the bounding box of the selected layer relative to the original image. The values indicate how far away (in pixels) the edges of the layer's bounding box are from the edges of the original image.

Blend: Specifies how the selected layer is blended with other layers in the original image.

Opacity: Specifies the selected layer's opacity within the original image.

Visible: Specifies whether the selected layer is visible in the original image.

Layer Number

Specifies the layer that you want to extract. Layers are numbered sequentially from back to front, the background layer being number 0, the next layer being number 1, and so on.
