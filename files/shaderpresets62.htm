Fire

DEPRECATED. This shader is unsupported, but it may still get installed with Softimage to provide compatibility with older scenes that use it. It is recommended that you replace unsupported shaders in your scenes with equivalent shaders from the current Softimage shader library.

| Flame Controls | Peekaboo

Shader Type: Material (Soft3D)

Output: Color (RGB) value

Simulates a surface projection fire effect. This shader does not create a volumic effect.

Name

The shader's name. Enter any name you like, or leave the default.

Flame Colors

Hot

Defines the fire's most intense color, usually at the base of the flame.

Medium

Defines the color of the flame's body.

Cool

Defines the color of outside edge of the Medium flame color.

Cold

Defines the coolest, or most peripheral areas of a flame.

Flame Controls

Burn Speed

Controls the flicker intensity over time.

Rise Speed

Controls the height fluctuation of the flames over time.

Scaling

Controls the relative size of the flames.

Maximum Height

Defines the highest point the flames will reach. Measured in SOFTIMAGE units.

Streak

Defines the contrast between the flame colors to create a streak effect.

Complexity

Controls the blending complexity of the various flame colors.

Intensity

Controls the intensity of the effect's colors. Careful, however, using too high a value may heat your monitor unnecessarily.
