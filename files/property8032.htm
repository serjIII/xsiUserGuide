Split Polygon Op Property Editor

Splits a polygon in one of several ways.

To apply: See Splitting Polygons {Modeling].

To redisplay: Select the polygon mesh object, click the Selection button on the Select panel, then click the SplitPolygon Op icon.

Type

Controls how the polygon is split:

  • Cross splits the polygon using a cross pattern centered on the point you originally clicked.

  • By the Vertices splits the polygon by connecting the point you originally clicked to each vertex.

  • By the Edges Midpoints splits the polygon by connecting the point you originally clicked to the midpoint of each edge.
