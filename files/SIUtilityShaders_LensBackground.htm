Lens Background Color

| Render Tree Usage

Category: Lens

Shader Family: Lens

Output: Color

Renders a lens effect against a specified color background.

Name

The name of the shader node displayed in the render tree. Enter any name you like, or leave the default.

enable

Activates the effect.

color

Specifies a background color.

Render Tree Usage

This shader can be applied directly to the camera in the lens shader stack. See Applying Lens Shaders to Cameras [Cameras and Motion Blur].
