Scalar to Quaternion

Categories

  • Conversion

  • Rotation

Description

Outputs a new quaternion built from the input W, X, Y and Z components.

Ports

Inputs

X

The X component for the new quaternion.

Y

The Y component for the new quaternion.

Z

The Z component for the new quaternion.

W

The W component for the new quaternion.

Outputs

Quaternion

The new quaternion.
