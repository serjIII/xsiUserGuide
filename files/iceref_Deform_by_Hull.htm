Deform by Hull

Demonstrates one way to implement a cage deformation in ICE. This compound is designed to work in conjunction with Find Hull Surface Point. See Hull Deformation [ICE Deformations].

Tasks: Deformation/Hull Deformer

Output Ports: Execute

Cage Geometry

Specifies the object to be used as a deforming cage.

Retain Distance from Mesh

Preserves the original distance from each point to its associated location on the cage.
