Mixer Animation or Layer Clip Property Editor

To display: Select an action or animation layer clip in the animation mixer, then press Enter and click the tab with the layer's name or the action's name with "_Clip" appended

These values are provided for information only. To change them, use the mouse in the animation mixer to move, scale, or cycle an action clip. You cannot change a layers clip's values in this way.

Time Span

Start Time

The start time of the action or layer clip in the global time of the scene.

Duration

The duration of the action or layer clip in the global time of the scene.
