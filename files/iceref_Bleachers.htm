Bleachers

Creates a simple topology resembling the benches of a stadium. This compound is used by the CrowdFX Simulation Stadium Crowd command on the ICE toolbar to create an emitter if no emitters have been added beforehand.

Tasks: Topology/Generators

Output Ports: Execute

All lengths are in Softimage units.

Rows

Nb of Rows

The number of rows.

Rows Height Offset

The vertical distance between adjacent rows.

Rows Depth Offset

The vertical distance between adjacent rows.

Seats

Nb of Seats

The number of seats (polygons) per row.

Width

The width of each seat.

Depth

The depth of each seat.

Enable Thickness

Applies thickness to each bench.

Thickness

The amount of thickness to add.

Bleachers

Use Two Bleachers

Creates a pair of facing bleachers.

Space Between Bleachers

The distance between the pair of bleachers.
