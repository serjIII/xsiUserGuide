Spline Interpolation (Expression Function Reference)

The Function Constants Spline Interpolation command in the expression editor inserts the token SplineInterp at the insertion point in the editing pane. This token is used to specify spline interpolation with the Ease In (Expression Function Reference), Ease Out (Expression Function Reference), and Link (Expression Function Reference) functions.
