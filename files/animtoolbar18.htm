Apply Path Dialog Box

To display: Select an object and choose Create Path Set Path from the Animate toolbar. Set the option as desired, then click OK and pick a curve to use as a path.

For more information, see Setting Objects on Paths [ Animation ].

Start Frame

The frame at which to start the path animation.

End Frame

The frame at which to end the path animation.

Linear

Specifies whether or not the object will move along the path at a constant speed. If Linear is off, the path will ease into and out of the path animation.

Tangent

Specifies whether to set a tangency constraint on the object. The default tangency setting will cause the object's X axis to point in the direction of the path's slope.
