Constraining Multiple Rigid Bodies to One

If you want to constrain multiple objects to one, you use the Multi Constraint Tool command. After you choose a constraint command, a picking session begins in which you need to pick the constrained and constraining objects.

In this example, you will constrain objects A, B, C (active rigid bodies) to object D (a passive rigid body).

To constrain many rigid bodies to one

  1. Choose a constraint from the Create Rigid Body Multi Constraint Tool menu in the Simulate toolbar: Hinge, Ball and Socket, Spring, Slider, or Fixed.

  2. Press Shift while you left-click to pick all objects to be constrained (A, B, and C).

  3. Middle-click to pick object D (the passive object).

  4. Right-click to end. A constraint object is created for each object and is attached at the center of each of these objects.

    TIP:You can also select all objects to be constrained (A, B, C) before choosing the constraint command.
