Synoptic Views

Synoptic views are image maps that serve as visual toolbars and are associated to particular elements in your scene.

  • You create synoptic files by using the synoptic editor to import images, draw hotspots, and enter scripts.

  • You attach a synoptic file to an object by applying a synoptic property.

For information about using synoptic views, rather than creating your own, see Synoptic Views [Commands and Tools].
