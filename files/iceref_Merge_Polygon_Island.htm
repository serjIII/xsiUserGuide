Merge Polygon Island

Categories

  • Topology

Description

Merges contiguous polygons.

Ports

Inputs

Topo

The topology you want to use. If not connected, the default is "Self.Topology".

Polygon Index

The indices of any polygons to merge. You can an array or a set of polygon index. Any contiguous polygons within those specified are merged.

Outputs

Result

The modified topology.
