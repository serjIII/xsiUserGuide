Fade Color

Fades any image into any color, or vice versa.

Requires an Input (Any).

Produces an Output (Any).

Fade Color

Fade Color

Sets the color to or from which the input clip is faded. Select a color using the sliders.

Fade Level

Sets the opacity of the second element (either the color or the input clip, depending on whether the fade is set to Up or Down). Set a value between 0 and 1 using the slider.

Sets the direction of the fade. Select Down to fade from the color to the input clip, or Up to fade the input clip to the color.
