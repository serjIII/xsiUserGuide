Lagoa Material Soft Loose

Creates a soft loosely-connected elastic material using the point cloud that is emitted. This is similar to Lagoa Material Soft Loose except that it doesn't return to its shape after collision. The elastic structure has a little stretch and shear resistance, but more damping.

Plug this compound's Execute output into an Execute port of the Lagoa Phase node.

The parameters exposed for this compound are a subset of those in the Lagoa Main Material compound, mainly in the Inelasticity and Elasticity Settings groups.

For more information on setting up a Lagoa ICE tree, see Creating a Lagoa Effect from Scratch.

Tasks: Lagoa/Material Type

Output Ports: Execute
