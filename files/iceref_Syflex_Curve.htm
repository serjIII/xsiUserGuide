Syflex Curve

This compound creates a Syflex deformation simulation on a curve object. The stiffness of the springs and the mass of the points define the properties of the curve deformation, while gravity and global damping add forces.

To apply: See Creating Syflex ICE Simulations.

Tasks: Syflex/Basic

Output Ports: The Value output plugs into a port on the ICETree node.

The parameters found in this compound are the same as in the Syflex Cloth compound except that the Syflex Curve Springs compound is used inside it instead of cloth springs.

For more information on the curve springs used, see Using the Syflex Spring Forces.
