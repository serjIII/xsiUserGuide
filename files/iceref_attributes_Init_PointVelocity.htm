Init_PointVelocity

A particle's original PointVelocity value when it was emitted.

This attribute is set by the Emitters compounds and is exposed in their property editors as the Speed parameter.

It can also be used by Move Towards Goal, Modulate Velocity Over Time, or your own compounds.

Data Type

3D vector

Belongs To

points

Geometry Types

  • point cloud

Attribute Type

custom

R/W or RO:

read/write
