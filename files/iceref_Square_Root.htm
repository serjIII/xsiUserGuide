Square Root

Categories

  • Math > Basic

Description

Outputs the square root of the input value.

Ports

Inputs

Value

The value from which to retrieve the square root.

Outputs

Result

The square root of the input value.

Valid

False if the result is undefined for the specified input; true otherwise.
