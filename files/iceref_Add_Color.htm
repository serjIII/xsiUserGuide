Add Color

Categories

  • Color

Description

Outputs a new color made from the sum (addition) of the set of the specified input colors.

Parameters

Clamp Mode

Specifies how to clamp color values outside the [0, 1] range.

  • Unit [0,1]

  • Positive [0,Inf

  • Don't Clamp

Ports

Inputs

Alpha

Determines whether to consider the alpha components of the specified input colors during the addition operation or not.

Value

The value (color) used for the addition operation.

Outputs

Result

The new color made from the sum (addition) of the set of the specified input colors.
