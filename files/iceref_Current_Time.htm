Current Time

Categories

  • Data Access

Description

Returns the global or local time in seconds at which the ICE tree is being evaluated. There can be a difference between the local and global time only with simulated ICE trees. In this case, the local time corresponds to the time elapsed since the beginning of the simulation. The beginning of the simulation is defined in the Simulation Time Control Property Editor under the simulation environment.

Ports

Inputs

Global Time

Specifies whether to return the global time or the local time.

Outputs

Time

The time in seconds at which the ICE tree is being evaluated.
