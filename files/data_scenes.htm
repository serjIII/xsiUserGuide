Scenes

A scene file contains all the information necessary to identify and position all the models and their animation, lights, cameras, textures, and so on for rendering. All the elements of a scene are compiled into a single file with an .scnextension.

The compiled scene file can be further modified by the information set in its corresponding scene TOC (scene table of content) or by opening the scene with auxiliary data.
