Polygonizer_PointCloud Property Editor

A custom property that controls a point cloud's contribution to the density isofield used by a Polygonizer operator. For more information about using Polygonizer, see Meshing Point Clouds (and Other Things) [Data Exchange].

To display: Select the point cloud, choose Selection on the Select panel, and click the Polygonizer_PointCloud icon.

Active

Controls whether the object contributes to the Polygonizer's isofield. Turn this off to temporarily mute the object's contribution to the polygonized mesh without permanently removing the object as an input.

Use

Controls what components of the object contribute to the isofield.

LOD

Controls sampling along particle strands, curves, and polygon faces. This value has no effect unless Use is set to an option that includes LOD.

Size and Influence

Isofactor

The base contribution of the object's components to the isofield. Note that you can use a negative Isofactor value to make an input remove volume from the resulting mesh.

Radius

The distance at which the isofield is equal to the Isofactor value.

Multiply Radius by Point Size (if available)

Multiplies the value specified by Radius by each particle's Size attribute (if it is defined).

Note that if you are using the default values for the emitter compounds, Size is set to 0.1 â€” you may need to turn off this option or increase particles' Size to get a generated mesh.

Falloff

The thickness of the falloff region beyond the radius.

Falloff Type

Controls how the isofield value decreases in the falloff region. For example, "Square" gives an inverse-square falloff similar to many physical phenomena.

Multiply Falloff by Point Size (if available)

Multiplies the value specified by Falloff by each particle's Size attribute (if it is defined).
