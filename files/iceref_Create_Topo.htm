Create Topo

Categories

  • Topology

Description

Create a new polygon mesh topology using two input arrays.

Ports

Inputs

Vertex Position Array

An array of 3D vectors representing the vertex positions.

Polygonal Description

An array of indices representing the vertices of the polygons to build.

Each positive integer in this array is a reference to the index of an element in the Vertex Position Array. You should specify the indices of each polygon in counter-clockwise order when the normal is facing you. Any negative value means "End of the polygon".

For example to describe one triangle and one quad it could be like this: "0, 1, 2, -2, 1, 3, 4, 2, -2" where "0, 1, 2" is a triangle and "1, 3, 4, 2" is a quad.

Outputs

Result

The new topology.
