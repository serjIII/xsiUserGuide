Fold Op Property Editor

Wraps the extremities of an object towards an axis.

To apply: Select an object or cluster and choose Modify Deform Fold from the Model toolbar.

To redisplay: Select the object then choose Edit Properties Modeling Properties and click the Fold Op tab.

Mute

Toggles the deformation on and off without affecting the values of other parameters.

Direction

Axis

The axis toward which the object is folded.

Type

The shape of the deformation. Linear creates a sharp crease with no smoothing. Spherical creates a cylindrical fold (like rolling something into a tube) or spherical wrapping with distortion.

Amplitude

Angle

The amount of folding along the primary plane in degrees. For spherical folds, this value is in degrees. For linear folds, this value is in 0.25 degrees.

Distortion

The amount of folding along the secondary plane. For spherical folds, this is a percentage of the Angle. For linear folds, this is a percentage of a complete fold.

Deformation Center

Use Bounding Box Center

Uses the center of the object's bounding box as the deformation center. If this option is not checked, the object's center is used.

Offset X, Y, Z

Offset of the deformation center from either the bounding box center or the object center, in SOFTIMAGE units, in the object's local space. You can type values outside the slider range.
