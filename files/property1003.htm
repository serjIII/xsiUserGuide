Volumic Light Property Editor

| Volumic | Shard

Assigns a volumic property to the selected light(s). You can also define the look and effects of a volumic light.

To Display: Select a light and choose Get > Property > Volumic from the Render toolbar. See Applying a Volumic Light Property for more information.

Volumic

Enabled

Activates a volumic effect on the light the property is assigned to.

Volumic Settings

Map Size

Defines the volumic map size. A high value will increase the size of the volumic effect.

Reflectance

Controls the reflectance emitted by a volumic light.

Minimum Distance

Defines the minimum distance from the light source that the volumic effect will be visible.

Force Volume Shadows

Causes the volumic effect to create shadows. Although more realistic, using this option will increase render times.

Transparent

Defines a transparent quality for the volumic light.

Shard

Enabled

When on, the light produce rays, or shards, of light.

Intensity

Defines the intensity of the light shards. A very high value will resemble sun rays poking through clouds.

Scaling

Defines the size of the beams/rays.

Animation

Defines the rate of change of the beams/rays.

Complexity

Controls the random jitter used when rendering the shards. Complexity adds a random, natural quality to the shards.

Rotation

When on, the shard effect will automatically rotate as the light source is moved.
