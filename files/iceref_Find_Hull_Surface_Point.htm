Find Hull Surface Point

Demonstrates one way to implement a cage deformation in ICE. This compound is designed to work in conjunction with Deform by Hull. See Hull Deformation [ICE Deformations].

Tasks: Deformation/Hull Deformer

Output Ports: Execute

Cage Reference Geometry

Specifies the object to be used as a reference cage.

Search for Ideal Location

Refines the way that points are assigned to locations on the cage. Activating this option increases the computation time. If this option is off, Angle Threshold and Max Repeat have no effect.

Angle Threshold

Controls the accuracy used to find the ideal location.

Max Repeat

Limits the amount of time spent searching for the ideal location in case the threshold is not reached.
