Facade

Category: mental ray > Materials

Shader Family: Surface Material

Output: Color

The Facade shader places virtual cardboard-cutouts in a scene. This allows you to replace complex geometry with simple 2D faces, saving you lots of render time.

The shader then "rotates" the cutout so that it faces the camera, lights, and reflective surfaces all at the same time. By doing this, stretching in the image, shadows, and reflections is eliminated.

The texture's scaling, rotation, and translation are overridden by the Facade shader.

Name

The name of the shader node displayed in the render tree. Enter any name you like, or leave the default.

Bitmap

The image you want to use for the facade. You can Edit the image clip, or get a New image clip from file or from source. For more information about working with images, see Managing Image Sources & Clips [Textures and Maps].

Size

Sets the height of the facade image.

Up Direction

Sets which direction to use for the "up" direction of the image.

Cylindrical Rotation

The facade is rotated only around the Y-axis.

Spherical Rotation

The facade is rotated around all axes.
