To display: edit a paint operator and choose Brush Properties from the FX Viewer paint menu, or press 2.

The Brush Properties property editor displays and controls the properties of the current brush, selected from the FX Paint Brush List. You can modify any brush's properties to define the look of your paint strokes. The available properties vary depending on which brush is selected.

  • The default brush properties are available for all brushes.

  • The other brush properties listed above appear in various combinations when you edit certain brushes' properties.

Paint Brush

Name

Displays the brush name. You cannot change this parameter.

Input

Changes the paint source. Choose one of the following sources.

Paint: uses the current foreground color.

Current Layer: uses the paint operator currently displayed in the FX Viewer.

Merge Source: uses the merge source (see page 270)

Erase Source: Erases paint strokes. In raster paint clips, erasing reveals the last saved state of the image.

Profile

List the available brush shapes. The brush profile can be Solid, Soft, Gaussian, Linear, Curve, or Brush Image.

Opacity

Controls the opacity level of paint strokes. The higher the value, the more opaque the strokes.

Radius

Sets the maximum brush size radius in pixels. As you change the radius value, the paint cursor grows and shrinks accordingly.

To quickly change the radius of the current brush press [ to lower the value and ] to raise it.

Air Brush

When activated, the brush simulates an airbrush. Rather than stamping paint onto an image, the brush applies paint gradually. The longer you hold the paint cursor over a given point on the image, the more opaque the paint becomes at that point.

Spacing

Specifies the distance between adjacent brush stamps. The units are percentage of brush radius, so a spacing of 100% would result in a new stamp each time the brush reaches the edge of the last stamp and a spacing of 1% would put down a new stamp when the brush moves 1% of the radius. Increasing Spacing increases the distance between stamps, decreasing Spacing decreases the distance.

Profile

Profile Type

List the available brush shapes. The brush profile can be Solid, Soft, Gaussian, Linear, Curve, or Brush Image.

Flatness

Sets the flatness of the brush profile. As you raise the value, the brush profile appears flatter and the paint cursor is flattened accordingly.

Angle

Sets the angle of the brush. Set any value from -360 to 360% by dragging the Angle slider. As you change the Angle value the brush cursor is rotated accordingly.

Merge/Clone Offset

X/Y

Sets the source offset for merging/cloning in X and Y. Positive values offset to the right, negative values offset to the left.

Offset in Time (Frames)

Offsets the merge/clone source, in time, by the specified number of frames. For example, using the clone brush with a time-offset of -1 allows you to paint the previous frame of a sequence into the current frame.

Curve

Use the curve editor on this tab to edit the brush profile.

Texture

When you paint with a texture, the paint color is applied using the selected texture, in the shape defined by the other brush settings (opacity, radius, flatness, and so on).

Texture list

Choose a texture to paint with from the list.

Texture preview area

The texture that you choose in the list is displayed in this pane.

Use

When activated, the brush applies the paint color using the texture.

Invert

When activated, the texture is inverted before the brush applies the paint color using it.

Add

Adds the selected paint source to the image using normal (pixel) or diffuse (patch) applications.

Type

Specifies how the effect is applied when you click.

Normal: applies the effect individually to each pixel under the brush stamp.

Diffuse: applies the effect to the entire brush stamp area.

Src

Specifies the source that is added to the image.

Paint: use the current paint foreground color.

Current Layer: uses the current image in the FX Viewer.

Merge Source: uses the merge source image.

Binary

Converts the image to either black or white according to a luma threshold.

Thresh

The luma threshold according to which the image is converted to black or white.

Pixels whose luma value falls below the threshold are converted to black. Pixels whose luma value exceeds the threshold are converted to white.

Blur

Blurs the image using a simple box blur.

Width/Height

Specifies the width and height of the blurred area in pixels.

Drag

Smudges an area of an image by dragging pixels a short distance from their original position and painting with them.

Decay

Controls the speed at which the current color decays over the length of the brush stroke. The higher the value, the faster the decay.

Elastic RGB

Causes the pixel at the center of the brush stamp to influence the color of the other pixels. The color of the other pixels will move toward the color of the center pixel if the value is positive, and away from the color for the center of the pixel if the value is negative. The sample is weighted so that the pixels nearer the center are influenced more than the pixels nearer the edge.

Use

All: applies the effect to all channels at once.

RGB: has no effect.

All%

When Use is set to All, this value controls the strength of the effect.

Elastic XY

Causes the pixels to move toward or away from the center of the brush sample, according to whether the value is positive or negative. The sample is weighted so that pixels nearer the center are influenced more than pixels nearer the edge.

Use

All: applies the effect to all channels at once.

RGB: has no effect.

All%

When Use is set to All, this value controls the strength of the effect.

Emboss

Creates an embossed effect by applying light and shade to emphasize edges in the image. You can set the angle of the light and add a proportion of the original image back in.

Angle

Specifies the angle of the light in degrees.

Orig

Specifies the percentage of the original image that is added back into the embossed result.

Jitter

Randomizes various brush attributes.

Size

Controls the degree to which the brush size is randomized.

Flat.

Controls the degree to which the brush flatness is randomized.

Angle

Controls the degree to which the brush angle is randomized.

Opacity

Controls the degree to which the brush opacity is randomized.

Jitter HSV

Randomizes the color values of each pixel or the overall patch.

Type

Specifies whether the effect is applied to each pixel in the brush stamp area individually (pixel) or to the brush stamp area as a whole (patch).

Use

All: applies the effect to all channels at once.

HSV: has no effect.

All%

When Use is set to All, this value controls the strength of the effect.

Jitter RGB

Randomizes the color values of each pixel or the overall patch.

Type

Specifies whether the effect is applied to each pixel in the brush stamp area individually (pixel) or to the brush stamp area as a whole (patch).

Use

All: applies the effect to all channels at once.

RGB: has no effect.

All%

When Use is set to All, this value controls the strength of the effect.

Min Max

Applies a minimum or maximum filter to the image, depending on the filter settings you have chosen.

Type

Specifies whether to apply a Min or Max filter to the image.

Width/Height

Specifies the width and height of the filter.

Mix

Lets you choose the percentage of the mixing source that will be mixed with the image, using normal (pixel) or diffuse (patch) applications.

Type

Specifies how the effect is applied when you click.

Normal: applies the effect individually to each pixel under the brush stamp.

Diffuse: applies the effect to the entire brush stamp area.

Src

Specifies the source that is mixed with the image.

Paint: use the current paint foreground color.

Current Layer: uses the current image in the FX Viewer.

Merge Source: uses the merge source image.

Mix Val

Controls the percentage of the mixing source that gets mixed with the image.

Multiply

Lightens or darkens the image, depending on whether positive or negative values are set.

Use

All: applies the effect to all channels at once.

RGB: has no effect.

All%

When Use is set to All, this value controls the strength of the effect. Positive values lighten the image, while negative values darken it.

Noise

Adds noise to each brush stamp.

Level

Controls the amount of noise. The higher the value, the more noise is introduced.

Offset HSV

Offsets the color value of the brush stamp area in HSV mode.

Use

All: applies the effect to all channels at once.

HSV: has no effect.

All%

When Use is set to All, this value controls the strength of the effect.

Offset RGB

Offsets the color value of the brush stamp area in RGB mode.

Use

All: applies the effect to all channels at once.

RGB: has no effect.

All%

When Use is set to All, this value controls the strength of the effect.

Oil Dab

Applies an OilDab filter to the image. This creates blocks of color randomized around the original colors which creates the appearance of a painted image.

Width/Height

Specifies the width and height of the OilDab filter in pixels.

Pick and Mix

Takes the color from the selected area (the point where you click) and mixes it with the other colors in the image.

Decay

Controls the speed at which the current color decays over the length of the brush stroke. The higher the value, the faster the decay.

Pickup

Lets you pick a sample area from the image and paint with it.

Action

Specifies the brush stamp will Pick color or Paint picked color.

Typically, you set this parameter to pick, make a brush stamp to pick colors, then change this parameter to Paint and paint the colors on the image.

Posterize

Posterizes the image.

Use

Specifies whether the number of levels is set in terms of All channels or in terms of the RGB channels individually.

Levels

Specifies the number of levels.

ReplaceLum

Applies paint to the image to darken or lighten it, depending on the Type setting.

Type

Lighten applies paint to the image, but only in those areas where the result would be lighter than the original.

Darken applies paint to the image, but only in those areas where the result would be darker than the original.

ReplaceRGB

Paints only those pixels whose color lies within a specified range of a color picked from the paint source.

Rotate

Rotates the brush sample.

Angle

Specifies the amount of rotation in degrees.

Rub

Paints the current color into the image where edges exist in the alpha channel. An edge exists where there is change in mask level. The side of the alpha channel that is "picked up" depends on the direction of the brush stroke.

Scale

Scales the rub effect to make it more or less prominent.

Scatter

Scatters the pixels by the set distance in X an Y.

Use

Specifies whether the effect is applied in terms of All components, or individual RGB components.

X/Y Val

Specify the X and Y scatter distance in pixels.

Swirl

Swirls the image.

Angle

Specifies the angle of the swirl per brush stamp, in degrees.

Threshold

Paints only those pixels whose color in the paint source lies above or below the chosen threshold.

Type

Specifies whether to use a Min or Max threshold filter.

Use

All: applies the effect to all channels at once.

RGB: has no effect.

All%

When Use is set to All, this value controls the strength of the effect.

Tint

Type

Specifies the color model to use when tinting the image. Choose HSV or HLS.

Src

Specifies the source for the color with which the image is tinted. Choose the Paint color or the Merge Source image.

Zoom

Magnifies the image under the brush.

Zoom

Controls the zoom factor. The higher the value, the greater the zoom.
