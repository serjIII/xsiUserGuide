Inherit Emitter Velocity

This compound sets the particle velocity based on the emitter's velocity. Plug this compound's Execute on Emit output into an Execute on Emit port of the Emit compound.

For more information on emissions, see Setting Up Any Type of ICE Particle Emission [ICE Particle Simulations].

Tasks: Particles/Emission Control

Output Ports: Execute_on_Emit

Percentage

The percentage of the emitter's velocity that you want the particles to inherit.
