Get Particle Orientation

This compound gets the particle's current rotation data as set by its Orientation attribute. You can output the Orientation as rotation data.

For more information on orientation, see ICE Particle Orientation [ICE Particle Simulations].

Tasks: Particles/Getters

Output Ports: Orientation

Reference

The particle's Orientation attribute.
