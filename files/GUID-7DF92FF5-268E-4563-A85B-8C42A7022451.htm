Quaternion to Axis and Angle

Categories

  • Conversion

  • Rotation

Description

Outputs the axis and angle components calculated by converting the input quaternion to a rotation.

Ports

Inputs

Quaternion

The quaternion to convert.

Outputs

Axis

The axis component.

Angle

The angle component.
