Directional Blur

Blurs the image in a specified direction. The direction and length of the blur can be controlled.

Requires an Input (Any).

Produces an Output (Any).

DirBlur

Direction

Sets the direction of the blur. Set a value in degrees.

Length

Sets the length of the blur. The higher the value, the longer it takes to render the effect.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
