Filter by Object Intersection

This compound reduces the number of particles based on the emitter intersecting with another object. Particles are emitted only within that intersection area.

Plug this compound's Execute on Emit output into an Execute on Emit port of the Emit compound or into a port on the ICETree node. If you do the latter, the emission is updated at each frame.

For more information, see Filtering Particles Based on Object Intersections [ICE Particle Simulations].

Tasks: Particles/Emission Control

Output Ports: Execute_on_Emit

Filter Geom1

The object that intersects with the emitter. Plug the object's Value output into this port.

Distance Threshold

The maximum distance (in Softimage units) of the objects intersecting.

Invert

Inverts the particle emission so that particles are emitted only outside of the object intersection area.
