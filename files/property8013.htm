Add Edge Op Property Editor

Created automatically when you use the Add Edge tool to draw new edges on a polygon mesh.

To apply: See Adding Edges {Modeling].

To display: Select the polygon mesh, click the Selection button on the Select panel, then click the AddEdge Op icon.

Ratio Edge

The position of the new edge's endpoint, as a proportion of the length of the edge on which you originally clicked.
