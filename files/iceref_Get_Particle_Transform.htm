Get Particle Transform

This compound gets the particle's transformation as defined by its Scale, Orientation, and PointPosition attributes.

You can output the transform data as a 4x4 matrix.

For more information on transforms, see Converting Between Local and Global Reference Frames [ICE Guide].

Tasks: Particles/Getters

Output Ports: Matrix
