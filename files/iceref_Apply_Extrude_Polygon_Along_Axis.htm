Apply Extrude Polygon Along Axis

Extrudes a single polygon island on the self or on the connected topology along its average normal, and replaces the self's topology. To extrude along a different axis, set Length to 0.0 and use the Position values.

Tasks: Topology/Modifiers

Output Ports: Execute

The Filter Mode options for specifying polygons are the same as Test Polygon Index.

Length

Distance to extrude in SOFTIMAGE units.

Duplicate

Controls how polygons are copied.

  • If this option is off, polygons are extruded, that is, the copied polygons are connected to the original geometry.
  • If this option is off, polygons are extruded, that is, the copied polygons are connected to the original geometry.

Nb Subdivs

Number of subdivisions.

Inset

Shrinks the boundary of the new polygons by the specified amount in Softimage units. Negative values expand the boundary instead of shrinking it, and the resulting polygons will not overlap even when the extrusion length is 0.

You can achieve similar effects by scaling using the options on the Transform tab, but with insetting the new edges are always parallel to the originals. In addition, scaling does not always work well with concave polygons whose edges might self-intersect. Another difference is that the inset always applies to the entire extrusion, rather than being repeated per subdivision.

Transform Per Subdivision

Applies the Transform settings iteratively at each subdivision. When this option is off, the transformation is applied to the extrusion as a whole.

Transform

Transforms the extruded geometry by the Scaling, Rotation, and Position values. The reference from is the average local frame of all polygons in the island.
