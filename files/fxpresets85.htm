Angled Strokes

The Angled Strokes effect paints an image with diagonal strokes. The lighter areas of the image are painted with strokes of one orientation, while the darker areas are painted with strokes of the opposite orientation.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

Direction Balance

Set higher values to have more of the image painted with right diagonal strokes, or set lower values to have more of the image painted with left diagonal strokes.

Stroke Length

Set higher values to make the strokes longer, or set lower values to make the strokes shorter.

Sharpness

Set higher values to make the resulting image sharper and more detailed, or set lower values to make the image smoother and softer.
