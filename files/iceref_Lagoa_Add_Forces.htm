Lagoa Add Forces

This compound is used as a type of "hub" when applying forces to particles, similar to the Add Forces compound in the Particles > Forces group in the preset manager.

All Lagoa forces can understand the interface between air and points by considering that there is air wherever there is no point with mass.

You plug other force outputs into its force ports (such as any of the forces in the Particles > Forces group), and then it adds up all those forces and applies them to the particles.

The particle's mass value (based on the particle's Mass attribute) is assumed to be 1 so that forces can act upon them accordingly.

Plug this compound's Execute output into a port on the ICETree node or into an Execute port on the Simulation Root node.

See Setting Up Forces for Lagoa Effects [Lagoa Multiphysics Simulations] for more information.
