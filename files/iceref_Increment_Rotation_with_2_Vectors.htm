Increment Rotation with 2 Vectors

Increments a rotation with the rotation needed to align a vector (expressed locally with respect to the input rotation) to a vector expressed in world space. Only one axis is used for the rotation so, for example, you can use this node to align the direction of a spinning bullet without affecting its spin.

Category: Rotation

Output Ports: Result

Rotation

The input rotation to modify.

Local Vector

The vector to align, expressed relative to the input rotation.

To Vector

The vector to align with, expressed in world space.

Weight

The amount to increment by. A value of 0 leaves the input rotation unaffected, and a value of 1 increments it fully.
