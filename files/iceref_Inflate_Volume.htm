Inflate Volume

Tries to restore the original volume of a mesh as set by Init Verlet Geometry Data. It uses a very simple algorithm that works with very simple shapes. See Verlet Integration [ICE Deformations].

Tasks: Deformation/Verlet

Output Ports: Execute

Weight

The strength of the effect.
