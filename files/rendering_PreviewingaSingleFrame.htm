Previewing a Single Frame

The Render Preview command in the Render toolbar lets you preview the current frame at fully rendered quality in a floating window. The frame is rendered using the render options for the current render pass or using the render region options defined in any of the four viewports. The preview always shows the main channel, rather than a specific framebuffer.

To preview a single frame

  1. Choose Render Preview from the Render toolbar.

  2. From the Options menu, choose which render options to use when previewing:

    • Current Pass Render Options — see Pass Rendering Options for details on how to set these options.

    • Viewport A, B, C, D Region Options — see Setting Render Region Options. These are especially useful if you want to render a viewpoint (such as the top, front and right views) rather than a camera view.

  3. You can also do the following in the render preview window:

    • Click the Update button to force a re-render of the frame.

    • Click the Lock button to stop the viewer from automatically updating when the scene changes. See Comparing Render Previews.

    • Select the Autofit Viewer option to resize the image to fit the preview window. See Autofitting A Render Preview.

When you hover the mouse cursor on the rendered content, the Render Preview window status bar displays the RGBA values of the corresponding color.

Comparing Render Previews

You can open more than one render preview and compare separate results in each window.

  1. Choose Render Preview from the Render toolbar to open a viewer and generate a preview.

  2. Click the Lock button to stop the viewer from updating when the scene changes.

  3. Choose View Rendering/Texturing Render Preview from the main menu to open a second preview window.

  4. Make modifications to your scene. You can then compare the result in the locked preview window with the result in the current preview window.

Autofitting A Render Preview

The Autofit Viewer option fits the image to the maximum size the viewer can be, without requiring scrollbars (the aspect ratio of the image is maintained). The maximum size allowed by the viewer is dependent on the resolution of your screen. For example, if the image was 1600 x 1600, it would end up as 800 x 800 when your screen resolution is 1280 x 1024.

When Autofit Viewer is off, the complete image is displayed in its 1:1 pixel ratio, but the entire image is only viewable using the scrollbars.

Scaling the Render Preview Content

By default, the Options Autofit viewer option is on.

When you scroll the mouse, the Autofit viewer option automatically turns off to allow zooming the content.

You can scale and pan the frame preview to magnify and examine the details of rendered content.

When the scaling mode is active, you can perform the following tasks:

  • Scaling — Scroll the mouse wheel up or down to zoom in or zoom out, respectively. The main menu bar in the Render Preview window also provides the Zoom option. You can adjust the slide bar in this field or double-click the field and enter a scaling value for the rendered image.

    A reset icon is available next to the Zoom field. Clicking the icon for the first time resets to the default size of the rendered image. Clicking the icon again resets to the previously defined zoom value.

  • Panning — Press the left-mouse button to pan the rendered image.

Setting the Exposure and Gamma Correction Values

The Render Preview window menu bar contains the Exposure and Gamma fields to define the Exposure and Gamma correction values for the rendered image.

The Exposure and Gamma values are not permanent. You can use the slider or double-click these fields to define the values. A reset icon is available next to the Exposure and Gamma fields. Clicking the icon for the first time resets to the default value. Clicking the icon again resets to the previously defined value.

The neutral value of Exposure is zero. Increasing or decreasing its value by 1 point means a double or half the exposure time for photography.

The Gamma correction is added before you define its value in the display preferences.

Saving a Frame Preview

The Save menu provides options to save the original and the scaled versions of the rendered image for comparison.

Choosing the Original Size or the Scaled Size options displays a browser, where you can select a directory, format, and file name for your image.
