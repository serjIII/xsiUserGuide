Sphere Volume Property Editor

A control object used to pull points within its volume with a proportional fall-off.

To create: Choose Get Primitive Control Object Volume Deform.

To redisplay: Select the volume deform object and press Enter.

Radius

The radius of the spherical volume in Softimage units.

Remember Attached Objects

If this option is on, the deformer attaches to only those geometric objects to which it was previously attached. If off, the deformer affects all visible geometric objects within its volume.

Inspect Volume Deform

Adjusts the falloff profile and other attributes of the current operation. See Proportional Volume Operator Property Editor.
