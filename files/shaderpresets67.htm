Peekaboo

DEPRECATED. This shader is unsupported, but it may still get installed with Softimage to provide compatibility with older scenes that use it. It is recommended that you replace unsupported shaders in your scenes with equivalent shaders from the current Softimage shader library.

Shader Type: Material (Soft3D)

Output: Color (RGB) value

Creates surface transparency by using the light hitting it, allowing you to "see" into objects. The degree of transparency caused by a light's rays hitting a surface increases with the intensity of the light.

Name

The shader's name. Enter any name you like, or leave the default.

Strength

Defines the maximum effect of the shader. A value of 1 makes a surface transparent by a light. Lowering this value diminishes the effect of the shader.

Light List

Contains the list of lights affecting transparency.
