Shape Source Property Editor

Lets you modify a shape key's source.

To display this property editor, do either of the following:

  • Right-click on a shape clip or shape compound clip in the animation mixer and choose Source.

  • In an explorer, click the shape source node's icon in the model's Mixer > Sources > Shape folder.

For more information, see Using the Animation Mixer for Shape Animation.

While you cannot change any of these parameters, you can activate or deactivate the shape item.

Type

The type of animation on the parameter; for example, SHP (shape compound) or Key (shape key).

Active

Whether the parameter is active. You can toggle individual parameters on and off. This affects any new clips based on the source action.

Item Name

The name of the affected cluster.

Value

The name of the shape source.

Toggle Activeness

Toggles the activeness of the selected parameters. This is equivalent to clicking in the Active column of the corresponding rows.

Clip Color

Sets the color of all clips in the animation mixer that are instantiated (created) from this source.

Click the color chip and select a color from the color editor. When you click the Close button in the color editor, all clips that are instances of this source change to this color.

NOTE:To share a clip color between sources, drag the color chip from one source property page and drop it on a color chip in another source property page.

For general information about setting colors, see Defining Color Properties.
