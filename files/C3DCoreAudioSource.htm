Audio Source Property Editor

Lets you edit the audio source that has been loaded into the animation mixer.

For general information, see Audio and Animation.

To display this property editor, do any of the following:

  • Right-click on an audio clip in the animation mixer and choose Source.

  • In an explorer, click the audio source node's icon in the model's Mixer > Sources > Audio folder.

  • Select an audio clip in the animation mixer and press Enter, or right-click on an audio clip and choose Clip Properties, then click the tab with the source's name in the property editor that appears.

File Name

The path and file name of the original audio source file.

Name

The name of the audio source as it's loaded into Softimage. You can change this name without changing the original audio source name or the audio clip name.

Sampling Rate

The audio sampling rate in Hz. This is displayed for information purposes only: you cannot change it.

Channel Count

The number of channels used for the audio: 1 for mono, 2 for stereo. This is displayed for information purposes only: you cannot change it.

Duration

The duration of the audio source in seconds. This is displayed for information purposes only: you cannot change it.

Clip Color

Sets the color of all clips in the animation mixer that are instantiated (created) from this source.

Click the color chip and select a color from the color editor, or drag the color sliders to select a color.

NOTE:To share a clip color between sources, drag the color chip from one source property page and drop it on a color chip in another source property page.

For general information about setting colors, see Defining Color Properties.
