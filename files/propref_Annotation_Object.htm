Annotation Object Property Editor

An annotation object is a text balloon that can be used to communicate messages to colleagues who are working on the same scenes and models. Annotation objects can be selected, transformed, and parented like any other 3D object in the scene, so the balloon can point at specific elements in the viewport.

To add:

  • Choose Get Primitive Annotation from any toolbar.

To redisplay:

  • Select the annotation object and press Enter.

Annotation

Purpose

A text field that you can use for a brief description of the purpose of the message. This can be displayed in viewports using Show Only Purpose.

Sender

A text field that you can use to indicate the author of the message.

Recipient

A text field that you can use to indicate the intended recipient of the message.

Importance

Choose low, normal, or high priority.

Message

The content of the message.

Balloon Color

The background color of the text balloon in the viewports.

Layout

Maximum Width, Height

The maximum dimensions of the text balloon before Size Scaling is applied, in pixels.

Size Scaling

Scaling factor for the size of the balloon.

Line Spacing

Controls the spacing between lines of text.

Offset X, Y, Z

Offset of the text balloon from its position in the scene. Values are in screen coordinates. A negative offset in Z is useful when annotations are occluded by other objects. By default, the units are pixels for X and Y, and Softimage units for Z. However if Offset in View is on then all axes use Softimage units.

Display

Show Only Purpose

Display Purpose in the balloon, instead of Message.

Distance Scaling

Scales the balloon based on the distance to the camera.

Distance Fading

Fades the balloon based on the distance to the camera.

Offset in View

Uses Softimage units for Offset X and Offset Y, instead of pixels.
