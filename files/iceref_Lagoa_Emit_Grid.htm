Lagoa Emit Grid

Creates and emits particles from one or more null emitter objects. The null's shape and size then define the space from which the particles are emitted.

Plug this compound's Execute output into a port on the ICETree node or in an Execute port on a Simulation Root node.

See Setting Up Emissions in Lagoa Simulations [Lagoa Multiphysics Simulations] for more information.

Tasks: Lagoa/Element Creation

Output Ports: Execute

NOTE:The parameters for this compound are the same as for the Lagoa Emit Volume compound, except for the ones listed below.

Null In Name

Plug in a null object's Out Name output here. The particles are emitted from this object.

If no object is plugged in here, the particles are emitted from a global position.

You can also plug a group node here, with each null emitter object defined in the group.

You can change the shape of the null in its property editor to change the emission volume.

Fitting

Height

The size in Y of the emission space.

Width

The size in X of the emission space in Softimage units, regardless of the null's size.

Depth

The size in Z of the emission space.

Center Offset

Offsets the point cloud from the null's center in any direction (XYZ).
