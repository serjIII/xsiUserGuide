Standalone Crosswalk Setup (Windows)

The standalone Crosswalk setup program lets you install and update all or some of your Crosswalk integrations.

The Crosswalk Setup detects whether your computer is running Windows 32-bit or Windows 64-bit and install the plug-in files that correspond to the appropriate operating system.

You can perform two types of installations:

  • Choose Typical to automatically install or update any Crosswalk integrations that are detected on your system. During installation, all occurrences of the supported applications (Softimage, 3ds Max, and Maya) are detected and the setup program copies the required plug-in files to the appropriate destination folders.

  • Choose Custom to select the integrations that you want to install and deselect those that you do not want to install. When you choose Custom you can only select from the integrations that are detected on the computer where you are installing.

In either case, if an earlier version of a plug-in is detected during installation, the setup program will backup the file using a .keep extension. For example:

C:\Program Files\Autodesk\Maya8.5\bin\plug-ins\dotXSISceneConverter.mll.keep0

The Crosswalk setup will display a warning message if the version of the plug-in being installed is older than the existing plug-in. You can choose whether to overwrite the files or not.

Crosswalk General Files Installed with Windows Setup Program

Files

Destination Folder

Description

Assorted setup files

<CrosswalkPath>\Setup

Contains the setup files if ever a re-install or an un-install is needed.

Crosswalk integration plug-in files

<CrosswalkPath>\Plugins

Contains copies of all the plug-in files available for this platform. You can use these files to manually update your integrations.

Documentation (*.htm) files

<CrosswalkPath>\Documentation

Contains all the documentation files for the Crosswalk toolset.
