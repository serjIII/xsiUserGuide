Sponge

The Sponge effect creates an image that appears to have been painted with a sponge. The resulting image has highly textured areas of contrasting color.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

Brush Size

Set higher values to increase the size of the sponge, or set lower values to decrease the size.

Definition

Set higher values to increase the definition of the textured areas, or set lower values to decrease the definition.

Smoothness

Set higher values for an overall smoother image, or set lower values for a sharper image.
