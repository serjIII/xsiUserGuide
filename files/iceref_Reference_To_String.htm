Reference to String

Categories

  • Conversion

Description

Converts a reference to a scene element into a string value. This is especially useful when building Material attribute arrays.

Ports

Inputs

Reference

A reference to a scene element.

In Name

A scene reference to prepend to Reference to fully resolve the name.

Outputs

Out String Name

The string name of a scene element.

Out Name

The fully resolved reference to the scene element.
