Get Self Pose

Categories

  • Crowds > Animation

Description

Gets the transforms of deformers produced by the active Animation State. This node, together with the Set Self Pose, is useful when you need to do more modification to the pose than just blending between Move and Idle states.

Parameters

Reference

Reference to simulation Point_Cloud

Ports

Inputs

In Name

Name of the simulation Point_Cloud

Outputs

Pose

The transforms of deformers

Out Name

Output name of the simulation Point_Cloud
