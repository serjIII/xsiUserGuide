Adding and Inserting a Shot Track

You can add a Shot Track directly above an existing Shot Track or to the bottom of all Shot Tracks.

To add a Shot Track above an existing Shot Track

  • Select a Shot Track and choose Track Add Shot Track.

    or

  • Right-click on a Shot Track and choose Insert Shot Track.

To add a Shot Track to the bottom of all Shot Tracks

  • From the Track menu, choose Add Shot Track.

    or

  • Right-click on any Shot Track and choose Add Shot Track.
TIP:To add many Shot Tracks quickly, choose the Track Add Shot Track command once, then middle-click the Track menu name to repeat the last command.
