Delete Particles by Volume

This compound deletes particles if they are within the volume of a 3D object.

Plug this compound's Execute output into a port on the ICETree node.

For more information on deleting particles, see ICE Particle Age [ICE Particle Simulations].

Tasks: Particles/Deleting Particles

Output Ports: Execute

Volume1

The object within which the particles are deleted. This must be an object that has volume (3D geometry). Plug the object's Value output into this port.

Invert

Inverts the falloff effect (Falloff Distance and Profile curve).

Use Falloff

Uses the Falloff Distance and Profile curve information to determine how particles are deleted.

Falloff Distance

The maximum distance (in Softimage units) in which the particles are deleted.

Falloff Profile

This graph determines the shape of the falloff of the particle deletion starting from the volume object.
