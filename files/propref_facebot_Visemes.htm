Visemes Dialog Box

Create and adjust viseme libraries for lip syncing in Face Robot.

To display:

On the Act panel in the Face Robot layout, click the Library > LipSync tab and choose Setup Visemes.

For more information, see Working with Visemes [Face Robot].

TIP:See Lip-Sync Workflow Overview [Face Robot] for an overview of the Face Robot lip sync workflow.

Viseme Library

Path

The Viseme Library name. Click the browse button (…) to open a browser in which you can select the library.

The Face Robot viseme library path is located in:

Softimage install path\FaceRobot\Data\FaceRobot\Viseme

Create New Library

Opens the Create New Library Dialog Box in which you can create a new viseme library based on an existing one.

Viseme

Language

The language for the current viseme library that is associated to the phonemes. Although Japanese is also supported, there is only one .csv file required (called USEnglish.csv) which maps the phonemes that can be used with both languages — see English and Japanese Phoneme Charts [Face Robot] to see the supported phonemes.

Phoneme

Select a phoneme from this list and click the Apply button to make the animation controls on the face take on the viseme pose for that phoneme.

You can can edit the viseme by moving and rotating the animation controls in the position you want. To return to the original mouth shape, click the Apply button again.

See Editing the Viseme Poses [Face Robot] for information.

Variation

The list of viseme variations. Select an option from the list and click the Apply button to apply that viseme variation of the phoneme to the current one.

To creates a separate variation of the current viseme pose, click the New button. This opens the New Variation Dialog Box in which you can enter the name of a new variation to be added to list of variations.

See Creating Variations of the Visemes [Face Robot] for information.

Save Viseme Pose

Overwrites the existing viseme with the changes you've made for the the current phoneme.

Create Corrective Shape

Creates a corrective shape for the current phoneme selected in this dialog box.

Clicking this button automatically stores a shape key using the selected phoneme name for the cluster on the face's geometry and opens the shape manager with that corrective shape key selected. By default, only the points of the cluster that are affected are highlighted in green.

See Creating or Editing Corrective Shapes [Face Robot] for more information.

Import Visemes

Opens the Import Visemes Dialog Box in which you can select the viseme libraries to import for the Face Robot head.
