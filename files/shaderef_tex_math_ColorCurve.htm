Color Curve

Category: Processing > Math

Shader Family: Texture

Output: Color

Creates three profile curves (four profile curves if Use Alpha is enabled) that you can use to control (and animate) the falloff of color values for each color channel specified by the Input parameter. If a shader is plugged into the Input port, its values are driven by the values on the curves.

Name

The shader name. Enter any name you like, or leave the default.

Input

The color values being controlled by the profile curves.

Curve

Controls the falloff of the Input value for each color channel.

Use Alpha

When enabled, the alpha channel specified by the Input parameter is used in the calculations and you can control (and animate) its falloff by using a profile curve.

Render Tree Usage

You can plug any shader with color input into the Input port and plug the shader's output into any other color port.
