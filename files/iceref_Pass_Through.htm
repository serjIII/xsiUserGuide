Pass Through

Categories

  • Execution

Description

Outputs the input data without any modification. This node is useful when one needs to branch out one compound input port to drive many ports inside the compound. A Pass Through node is automatically created when reusing an existing exposed port to drive another node port that is in the compound (when editing a compound, click and drag from an already existing exposed input port to a new port of the same type.

Ports

Inputs

In

The input values or execution to pass through.

Outputs

Out

The same values or execution connected to the input port.
