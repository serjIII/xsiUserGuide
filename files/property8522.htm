Symmetry Mapping Template Property Editor

Establishes the symmetrical correspondence between envelope deformers. You can add, change, and delete mapping rules.

To apply: See Mirroring Envelope Weights Symmetrically [Character Animation].

To redisplay: Select the model to which the envelope belongs, click the Selection button on the Select panel, and then click the Symmetry Mapping Template icon.

Map From

The name of a deformer.

Map To

The name of a symmetrically corresponding deformer.

Note that the rules are bidirectional. For example, if you have a rule mapping LShin to RShin, then you do not need a second rule mapping RShin to LShin.

Active

Whether this template is active. When points are mirrored, the rules in the last-applied active connection mapping template are used.

Insert Rule

Adds a new blank rule to the template.

Delete Rule

Deletes the selected rule.

Validate

Checks the rules for validity. Invalid strings are displayed in red.
