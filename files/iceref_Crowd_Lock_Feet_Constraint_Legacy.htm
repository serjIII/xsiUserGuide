Crowd Lock Feet Constraint Legacy

This is a legacy compound, which you can use for the old CrowdFX system.

You can use this compound for specifying the frame intervals during which the feet position are locked. There is one interval for each foot, and the frame number refers to that in the animation source.

The new version uses loop ratio (from 0.0 to 1.0) to specify the lock-free intervals.

For more information on the input and output parameters, see Crowd Lock Feet Constraint.
