Frames to Seconds

This compound converts time values from frames to seconds.

Plug its Seconds output into any time-related port that accepts a scalar value, such as the Age Limit port on the Set Particle Age Limit compound.

Category: Conversion

Output Ports: Seconds

Frames

The time value in frames.
