Apply Split Edge

Splits edges on the self or on the connected topology, and replaces the self's topology.

The Filter Mode options for specifying edges are the same as Test Edge Index.

Tasks: Topology/Modifiers

Output Ports: Execute

Split Settings

Ratio

Controls the position of the resulting vertex as a ratio from the position of the first vertex to the second vertex in the edge.

Offset

An offset for the position of the new vertices, in the local coordinates of the "self" object.
