Tail Maker

To display: Select a curve or a number of objects, then choose Create Skeleton Create Tail from the Animate toolbar.

A tail is a chain where each bone is controlled by a spring so that it reacts to the skeleton's animation (secondary animation).

For more information, see Creating a Tail [ Character Animation ].

Number of bones

Number of bones to be created in the tail control.
