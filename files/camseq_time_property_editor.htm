Time Control Property Editor

Provides options to change the first and last frames of a ShotClip in the Scene (Global) and Sequencer timelines, and its Scale value.

To open the Time Control Property Editor

  • Select a ShotClip and choose Clip Time Properties from the Camera Sequencer Editor menu bar.

    or

  • Right-click on a ShotClip and choose Time Properties.

    or

  • Press Ctrl+T.

Time Control Options

Scene Time

In

Start frame of a ShotClip in the Global timeline.

Out

End frame of a ShotClip in the Global timeline.

Sequence Time

In

Start frame of a ShotClip in the Sequencer timeline.

Out

End frame of a ShotClip in the Sequencer timeline.

Info

Scale

Displays the current scaling value of a ShotClip. See Scaling ShotClips.
