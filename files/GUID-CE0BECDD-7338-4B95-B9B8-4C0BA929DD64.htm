Randomize Rotation by cone

This compound lets you define the range of angle values that are randomly applied to objects or particles as they are emitted. The particles' orientation stays the same over their lifetime.

Plug this compound's Result (Rotation) output into any compound's port that accepts Rotation data, such as the Orientation port of the Emit compound.

For more information, see Randomizing the Orientation [ICE Particle Simulations].

Tasks: Particles/Modifiers,Deformation/Modifiers

Output Ports: Result

Cone Axis

The particle's or object's local XYZ axis around which the randomization of the rotation occurs.

Twist Min Value

The minimum amount of rotation the particles does around its Y axis.

Twist Max Value

The maximum amount of rotation the particles does around its Y axis.

Start Angle

The Start and End Angles create the angle range around the Cone Axis in which the particle or object rotation can vary (0 - 360 degrees is full rotation range).

End Angle

The End Angle of the range of possible degrees that can be used for randomizing the orientation.

Animated

Toggles the animation of the randomness; that is, whether it varies over time or not.

You can use this option when using an unsimulated ICE tree (such as if the point cloud's or object's ICETree node is in the Modeling region).

Seed

Defines the sequence of random numbers used. If you require that two nodes generate different sets of values with the same parameters, simply assign them different seeds.
