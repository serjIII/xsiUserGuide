Curve Boundary Adaptor Property Editor

Identifies a boundary of a curve. This property is created automatically when you perform certain modeling operations (e.g., Blend Curves, Merge Curves).

To redisplay: Select a generated curve, click Select on the Select panel, expand the appropriate operator (e.g., Blend Curves, Merge Curves), then click the Curve Boundary Adaptor icon.

Boundary

The boundary to use:

  • Min U Boundary (Red). When NURBS Boundaries on the Attributes tab of the Camera Visibility Property Editor is on, this boundary is displayed in red.)

  • Max U Boundary

Subcurve Index

The index of the disjoint subcurve that includes the point. This parameter is useful only if a single-curve object is composed of discontinuous segments as can happen when, for example, a curve is generated from the intersection of two surfaces.
