Lagoa Material Dirt Pile

Creates an inelastic material using the point cloud that is emitted, allowing for rigid bodies that can pile up on each other because of high surface friction.

The parameters exposed for this compound are a subset of those in the Lagoa Main Material compound, mainly in the Inelastic Settings group.

Plug this compound's Execute output into an Execute port of the Lagoa Phase node.

For more information on setting up a Lagoa ICE tree, see Creating a Lagoa Effect from Scratch.

Tasks: Lagoa/Material Type

Output Ports: Execute
