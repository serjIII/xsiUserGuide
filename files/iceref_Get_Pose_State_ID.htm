Get Pose State ID

Returns the value of CrowdFX_PoseState_ID which is the index of the current pose state of each actor in a CrowdFX simulation, where 0 is the first imported animation cycle, 1 is the second, and so on.

This compound can be used only in an ICE tree on the crowd simulation Point_Cloud object.

For more information, see How Action Sources Are Used in the Crowd Simulation.

Tasks: Crowds/Getters, Crowds/States

Output Ports: Value
