Creating a Data Map

A data map associates a cluster with a template. Once a map is created, you can set map values for each component in the cluster.

To create a data map

  1. Select a cluster. You cannot use components directly: you must make a cluster first.

  2. Choose Get Property User Data Map New Templated Data Map. A picking session starts.

  3. Pick the desired template. (A template is really just a custom parameter set, so you can actually pick any custom parameter set.)

    A user data map is created for the cluster.

NOTE:When you create a data map, the components are not initialized with the default parameter values from the template. You must set a component's map value to initialize it.
