Visibility Property Editor (Clusters)

Controls the visibility of a cluster of polygons.

This property is applied automatically to the InvisiblePolygons cluster when you hide polygons using the commands on the View menu or their keyboard equivalents.

To apply manually: Select one or more polygon clusters, then choose Get Property Polygon Cluster Visibility from any toolbar.

To display: Select a cluster with a Visibility property and press Enter, or click the icon of the Visibility property located beneath a cluster in an explorer.

General

View Visibility

Controls whether polygons are drawn in the OpenGL display modes (Wireframe, Hidden Line Removal, Constant, Shaded, Textured, Realtime Shaders, and so on).

Render Visibility

Controls whether polygons are drawn when rendering with mental ray, such as when rendering to file or using the render region.
