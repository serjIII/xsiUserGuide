Sets the position, rotation, and scaling coordinates of an object's X, Y, and Z axis in global space.

When you create an expression on any of the parameters in this property editor, an Expression page is added to the property editor so that you can easily edit the expression without opening the expression editor.

For general information on animating global transformations, see Animating Transformations [ Animation ].

Velocity & Acceleration

Calculates the object's linear or angular velocity at the current frame. You cannot modify the velocity and acceleration directly, but you can use the computed values in expressions, scripted operators, and elsewhere.

Computation of Velocity and Acceleration

Active

Velocity and acceleration are only calculated when this parameter is active. Click Compute Vel/Acc to activate it, or Remove Vel/Acc to deactivate it.

Velocity

Displays the object's linear and angular velocity at the current frame in Softimage units per second. Velocity and acceleration are only calculated when the Computation of Velocity and Acceleration parameter is active in the Velocity & Acceleration tab.

Acceleration

Calculates the object's linear and angular acceleration at the current frame in Softimage units per second squared. Velocity and acceleration are only calculated when the Computation of Velocity and Acceleration parameter is active in the Velocity & Acceleration tab.
