Blobs

This compound blends particles together when you have specified the Blob as the shape type for the particles, such as you do in an Emit compound.

Plug its Execute output into a port on the ICETree node.

To see the particles as blobs, you must create a render region because the blending is done at render time. In a viewport, particles with the blob shape are displayed as spheres.

For more information, see ICE Particle Blobs.

Tasks: Particles/Blobs

Output Ports: Execute

Blob Blend

Determines the amount of blending between particles using the BlobBlend attribute. A high value creates more blending between the particles, but takes longer to render because more surface area is drawn.

Particle size also has a big effect on rendering time. To speed up rendering, make sure that the particle size is not too large. Instead, you may be able to get a similar blob look with a smaller particle size but higher Blob Blend value.

Blob Strength

Controls how strongly the surfaces of blobs are pulled toward each other using the BlobStrength attribute. At 0.0, there is no attraction; at 1.0, blob surfaces are pulled together regardless of distance. The default value is 0.5.

A high value creates more blending between the particles, but takes longer to render because more surface area is drawn.
