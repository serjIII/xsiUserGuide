Deform by Curve

The ICE-based Deform by Curve compound uses the shape of a curve to bend and twist an object's geometry.

To apply: See Deform by Curve [ICE Deformations].

Tasks: Deformation/Effects

Output Ports: result

Deformer

The name of the curve to deform by.

Mute

Toggles the deformation on and off without affecting the values of other parameters.

Axis

The axis that follows the curve.

Scaling Along

Scale Curve

Scales the object along the length of the curve.

Scale Normal

Scales the object along the curve's normal, or along the Up Vector if you are using it.

Scale Binormal

Scales along a vector perpendicular to the tangent and either the normal or the Up Vector if you are using it.

Roll

Roll

Rotates the object around the curve's length.

Translate

Along Curve

Moves the object along the curve.

Along Normal

Offsets the object along the curve's normal, or the Up Vector if you are using it.

Along Binormal

Offsets the object along a vector perpendicular to the tangent and either the normal or the Up Vector if you are using it.

Constraint

Constrain To Deformer

Select this option if the deformed object was moved from the global center before the deformation was applied.

Constrain To Deformee

Select this option if the deforming curve was moved from the global center before the deformation was applied.

Profiles

Use Scale Profiles

Multiplies the Scale Normal and Scale Binormal values by the Scale Profile curve.

Scale Profile

Defines the values along the U length of the curve used to multiply Scale Normal and Scale Binormal.

Use Roll Profile

Multiplies the Roll value by the Roll Profile curve.

Roll Profile

Defines the values along the U length of the curve used to multiply Roll.
