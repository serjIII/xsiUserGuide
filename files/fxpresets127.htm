Shape Cutter

Lets you create and manipulate simple shapes to add an alpha channel to an image. Because Shape Cutter only affects the alpha channel, you will not be able to see its effect unless you enable the alpha channel display in the FX viewer.

Requires an Input (RGB).

Produces an Output (RGBA).

Shape Toolss

Allow you to draw, edit, and manipulate one or more shapes that can be used as a garbage matte.

Select Shapes

Manipulates the entire shape outline.

Rotate Shapes

Rotates one or more entire shape outlines about their respective pivot points.

Edit Points

Allows you to edit the control points around a shape outline so you can change its shape and size.

Add Points

Creates a new point. Select Add Point then click on the outline where you want the new point to appear.

Draw Freehand

Draws a line as you drag around the image. When you release the mouse button, the start and end points of the line are joined, control points are placed near the traced line and a bezier curve is created through them.

Draw Polygon

Creates polygonal shapes; these are drawn as a series of straight lines between mouse clicks. When you click again at the start of the first line, the last line is joined to the first. Each click point becomes a control point.

Draw Bezier

Creates bezier shapes; these are drawn as a series of straight lines between mouse clicks. When you click again at the start of the first line, the last line is joined to the first. Each click point becomes a control point.

Draw B-Spline

Creates B-Spline shapes; these are drawn as a series of curves between mouse clicks. When you click again at the start of the first curve, the last curve is joined to the first. Each click point becomes a control point.

Key Points

All

Sets keys for all points on the selected shape(s)

Selected

Sets keys for the selected points on the selected shape.

AutoKey

Enables autokey mode, but only for shapes.

Delete

Deletes the selected shape(s).

Inspect

Opens the selected shape's property editor, from which you can transform the shape and/or invert it.

...

Opens a pop-up explorer, with the selected shape's node expanded. This is useful when you want to mark several parameters for animation.

Opacity

Changes the opacity of all the shapes. Set a value between 0 (transparent) and 1 (opaque) using the slider.

Shape Blur

Controls a defocusing effect applied to the shape outlines. Use the slider to set a value between 0 an 50. The higher the value, the greater the blur.

Pre Multiply RGB

When enabled, the output image is premultiplied.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
