Weight Painter Property Editor

Represents one or more successive paint strokes on a weight map.

To display: Select an object with a weight map, choose Explore Property Maps from the Select panel, click the icon of a weight map, then click the Weight Painter tab.

Mute

Toggles the effect of the paint strokes on and off.
