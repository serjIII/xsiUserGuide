Integer To Scalar

Render Tree Usage

Category: Processing > Conversion

Shader Family: Texture

Output: Scalar

Converts an integer value to a scalar value.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Defines the integer to be converted to a scalar value. Decimals are rounded, and fractions that are lower than 1 are changed to 1.

Render Tree Usage

This shader is placed between an integer output and a scalar input in order for the two values to connect.
