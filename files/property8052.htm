Character Designer Property Editor

A polygon mesh human body is created when you choose the Character - Man command. The polygon mesh is designed to be used as the basis of a subdivision surface, and is suitable for enveloping.

Each well-named page in this property editor contains a set of sliders that control the different aspects of the body, from basic body type to skin color and facial features. This allows you to create any number of different bodies and heads with the same topology. Using the Reset buttons in this editor, you can easily reset the parameters for specific body parts or for the whole character.

For more information, see Designing Your Own Character [Character Animation].

To apply: Choose Get Primitive Model Character - Man from any toolbar.

To redisplay: Tree-select the body, then click the Selection button on the Select panel and click the Character Designer custom parameter set icon.
