Get Particle Stick Location

This compound gets the particle's position and normal data at the point of contact with an obstacle when sticking using the Stick on Surface compound. This is defined by the particle's StickLocation attribute.

You can output the particle's Location, Position, and Normal data.

For more information on sticking, see ICE Particles Sticking to Obstacles [ICE Particle Simulations].

Tasks: Particles/Getters

Output Ports: Location, Position, Normal

Reference

The particle's StickLocation attribute.
