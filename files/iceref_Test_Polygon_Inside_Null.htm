Test Polygon Inside Null

Returns a set of Booleans per polygon which are true if the polygon is inside the volume defined by a null's size.

Tasks: Topology/Conditionals

Output Ports: Result

Null Name

The null within which the polygons must be to trigger the effect. The null's icon must be a 3D geometry shape for this to work: change the Primary Display Icon of the null in its property editor. Plug the null's Output Name output into this port.

Use Falloff Profile Curve

Toggles the use of the Falloff Profile Curve below. If you turn this off, the polygons are simply tested as to whether they are inside the null shape or not: no falloff.

Profile

This graph determines the falloff of the polygons inside the null to create a more gradual effect.
