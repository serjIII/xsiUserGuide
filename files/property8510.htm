Annotation Property Editor

Adds comments, keywords, or flags to the selected element. You can use the annotations to share information within a workgroup, or as another method of grouping objects together for spreadsheet queries.

To apply: Select an object and choose Get Property Annotation from any toolbar.

Text

Type comments in this box.

Search Keywords

Type in key words for the information in the annotation for which you can search later, such as by using the Annotation query in the spreadsheet.

Search Flags

Flag 1, 2

Use these check boxes to flag objects for any purpose you want. You can use the Annotation query in the spreadsheet or scripting to see which objects are flagged.
