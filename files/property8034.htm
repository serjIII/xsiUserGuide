Face Maker Property Editor

A low-resolution polygon mesh head. The low-resolution mesh is designed to be used as the basis of a subdivision surface, and is suitable for enveloping. Each slider controls a different aspect of the face, allowing you to create any number of different faces with the same topology.

To apply: Choose Get Primitive Model Face - Man/Woman from any toolbar, or choose Primitives Model Face - Man/Woman from the main-menu bar.

To redisplay: Tree-select the head, then click the Selection button on the Select panel and click the Face_Maker custom property icon.
