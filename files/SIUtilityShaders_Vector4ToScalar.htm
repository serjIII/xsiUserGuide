Vector4 To Scalar

Category: Processing > Conversion

Shader Family: Texture

Output: Scalar

Converts 4D vector input to a scalar output.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Specifies the 4D vector input to be converted to a scalar output.

Render Tree Usage

This shader is simply placed between 4D vector output and a scalar input in order for the values to connect.
