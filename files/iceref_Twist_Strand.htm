Twist Strand

This compound twists the strand starting from its base at the particle. This creates a twist around either the particle's local axis or the strand's axis, like a spiral.

Create strands, then plug this compound's Execute output into a PostSimExecute port on the Simulation Root node or in a port on the ICETree node.

For information, see Shaping the Strands [ICE Particle Simulations].

Tasks: Particles/Strands

Output Ports: Execute

Select Twist Type

Select the method by which the strand is twisted:

  • Twist Around Strand Local Axis uses each strand's vector to twist the shape around the strand.

  • Twist Around Particle Local Axis uses each particle's local orientation axis as the vector to twist the shape around the strand.

Twist Amount

The number of degrees in which the strand is rotated around the vector type you selected. A value of 360 is a full rotation.

Twist Distribution Profile

This curve defines how the twist amount is distributed along the strand, starting from the strand's base (key on the left) to the tip (key on the right).
