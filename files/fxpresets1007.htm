Drop Shadow Property Editor

To display: open a color shape's property editor and activate the Drop Shadow parameter on the Shape tab.

Adjusts various properties of a color shape's drop shadow.

Color

Defines the drop shadow's color. Use the sliders to set any RGB value.

Offset

X/Y

Offsets the drop shadow from the shape fill by the specified amount in X and Y. Positive values move the drop shadow up and to the right, negative values down and to the left.

Opacity

Controls the opacity level of the paint stroke. The higher the value, the more opaque the stroke.

Softness

Softens the drop shadow by blurring its edge. The higher the value, the greater the blur.
