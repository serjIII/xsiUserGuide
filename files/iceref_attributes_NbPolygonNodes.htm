NbPolygonNodes

The total number of polygon nodes (texture samples) on a polygon mesh. There is one polygon node for every corner of a polygon. Each vertex has as many polygon nodes as there are adjacent polygons.

Data Type

integer

Belongs To

objects

Geometry Types

  • polygon mesh

Attribute Type

inherent

R/W or RO:

read-only
