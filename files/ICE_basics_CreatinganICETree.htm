Creating and Editing ICE Trees

This section describes how to create an ICE tree manually.

As an alternative to creating ICE trees manually, several commands in the ICE module automatically build ICE trees for specific effects.

Creating an ICE Tree Manually

  1. Display the ICE tree view. See The ICE Tree View.

  2. Select the geometric object to which you want to apply an ICE tree.

    If the ICE Tree view is locked, you must update it to show the selection.

  3. Do one of the following:

    • Choose Create Simulated ICE Tree to create an ICE tree in the Simulation region. The Simulation and Post-Simulation regions are created automatically if required.

      or

    • Choose Create ICE Tree to create an ICE tree in the current construction region.

    An ICE tree is created and the root node appears in the ICE tree view.

    For more information about simulated versus non-simulated trees, see Understanding the ICE Framework.

  4. Add nodes and connect them to achieve the effect you want. See the following sections:

    • Adding Nodes to ICE Trees

    • Connecting ICE Nodes.

    • Getting Data

  5. If the tree is a simulation, add the appropriate node to make sure that it updates properly at each frame: Simulate Particles or Simulate Rigid Bodies.

  6. If desired, you can create a compound node and export it for reuse in other trees and scenes. See Building ICE Compounds.

NOTE:If a Simulation region already exists, all new ICE trees are added to it. You can move the tree to another position in the stack as described in Changing the Order of Operators [Data Exchange].

Editing an Existing ICE Tree

  1. Select the object with an ICE tree.

  2. If necessary, click Refresh in the ICE Tree view.

  3. If there are multiple trees on the object, select the appropriate tree node in the explorer on the right of the view. If the explorer is not visible, click Hide/Show Explorer View (the right arrow button).

  4. Add nodes and connect them to achieve the effect you want.

Editing ICE Trees Offline

You can edit ICE tress offline, without updating the scene, by clicking the Enable/Disable ICE editing offline mode button (). This button is available in the ICE Tree Editor menu bar.

Editing ICE trees offline helps to save the computation time of Softimage for each modification (modifying parameter values, changing connections between ICE nodes, and so on) that you perform in an ICE tree. Moreover, there is no waiting time to make successive modifications.

When the offline editing mode is active, the Enable/Disable ICE editing offline mode button appears as . Toggle-click this button to enable or disable the offline editing mode.

NOTE:Disabling the offline editing mode forces Softimage to evaluate all modifications that you performed in the ICE tree and update the scene.

Applying Offline Modifications in the ICE Tree

After completing all modifications offline in the ICE tree, click the Refresh button () in the ICE Tree Editor menu bar to update the scene. Softimage evaluates all your modifications in the ICE tree and updates the scene.

The Refresh button is also enabled only when the Offline editing mode is active.

Both the Enable/Disable ICE editing offline mode and the Refresh buttons are also available globally on the Softimage main menu bar. These buttons are helpful if you want to edit the PPG editor parameter values offline when you invoke it from the Explorer and not from the ICE Tree Editor.

NOTE:The functionality of the Enable/Disable ICE editing offline mode and Refresh buttons apply at a global-level. This means, if you click the Enable/Disable ICE editing offline mode button in one ICE Tree Editor, then it enables the offline editing mode on all the other ICE Trees in one scene. Similarly, clicking the Refresh button in one ICE Tree Editor updates modifications in all the other ICE trees.

Muting ICE Trees

You can mute ICE trees like any other operator on the operator stack in a scene. Muting helps you to hide or temporarily disable one or more effects created by ICE trees in a scene.

Muting complex ICE trees also improves the performance of a scene.

In the following example, a Fireworks scene contains two ICE trees to display the Sparkle and Flower effects.

If you do not want to view the sparkle effect, then you can mute the Sparkles ICE tree.

You can make modifications to a muted ICE tree. Softimage does not evaluate your modifications, until you unmute the ICE tree.

NOTE:When you mute an ICE tree, the ICE tree effect is no longer visible and Softimage evaluates all operators on top of that ICE tree. Depending on the complexity of these operators, evaluation might take a long time. Therefore, do not mute an ICE tree to perform quick edits. Use the offline editing mode, instead.

To mute an ICE tree

  1. In the Explorer, double-click on the name of an ICE tree, click on the icon, or double-click on the root ICE tree node in the ICE Tree Editor.

    The PPG for the root ICE Tree appears.

  2. Turn on the Mute option.

    Alternatively, right-click on the ICE Tree in the Explorer and choose Mute from the contextual menu.

    In the Explorer, the letter M appears on a muted ICE Tree.
