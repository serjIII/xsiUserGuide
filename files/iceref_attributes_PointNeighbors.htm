PointNeighbors

An array containing the locations of the vertices that are connected to a vertex by edges.

If you get this attribute at a surface location, it returns the neighbors of the vertex closest to the location.

Data Type

array of locations

Belongs To

points

Geometry Types

  • polygon mesh

Attribute Type

inherent

R/W or RO:

read-only
