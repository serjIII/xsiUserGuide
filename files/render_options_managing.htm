Managing Rendering Options

Before you do any kind of rendering, you'll need to set some render options. Rendering options are set for the scene, for your renderer of choice (by default this is set to mental ray), and for each render pass you define.

For interactive preview renders, the render region has its own set of renderer options. See Previewing Interactively with the Render Region.
