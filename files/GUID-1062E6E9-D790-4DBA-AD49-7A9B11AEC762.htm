Collapse Layers Dialog Box

All layers that are not muted are collapsed into the base layer's animation, and an FCurve is plotted.

See Collapsing or Storing the Animation Layers for more information.

To display: Select an animated object and choose View Animation Animation Layer Manager. Click the Collapse Animation Layers icon.

Collapse Layers Options

Time span options

Start Frame

The frame at which the plotting begins.

End Frame

The frame at which the plotting ends.

Step value

The frame step increments to use. When this value is 1, every frame is plotted; when 2, every other frame is plotted, and so on.

FCurve options

Fit plotted values with an fcurve

Reduces the number of keys on the fcurves resulting from the plot.

Fit Tolerance

NOTE:This option is enabled only if you turn on the "Fit plotted value with an fcurve" option.

Use this option to adjust the closeness of the fit of the curve. Smaller values respect the original shape of the curve more by using more keys, resulting in a fit that keeps closer to the original curve.
