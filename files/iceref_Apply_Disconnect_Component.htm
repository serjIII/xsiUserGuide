Apply Disconnect Component

Disconnects components on the self or on the connected topology, and replaces the self's topology.

Tasks: Topology/Modifiers

Output Ports: Execute

Topo

The input topology. If nothing is connected, Self.Topology is used.

Component Type

The type of component to disconnect.

Filtering Mode

Mode

How to specify which components to disconnect:

  • Is Element checks whether the component is a member of a cluster or against a per-component Boolean attribute.
  • String Array checks whether the component index is included in a string.
  • Comparison tests the index against a single value.

Is Element Context Type

If Mode is Is Element, connect a per-component Boolean value to the port corresponding to the Component Type set above.

Cluster Name

Cluster Name

If Mode is Is Element, connect the name of a cluster to disconnect. When a cluster is connected, it overrides the per-component Boolean value connected to the input in the Is Element Context Type group.

String

String

If Mode is String Array, enter a comma or whitespace-separated list of integers. You can also use a range notation.

Compare

If Mode is Comparison, compares the component indices against the specified integer.

Type

Choose the comparison to perform.

Vertex Index

Specify a value to compare with the component index.
