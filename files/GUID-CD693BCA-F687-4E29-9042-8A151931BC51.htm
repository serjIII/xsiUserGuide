Lagoa Init Emitter Volume Data

This compound initializes volume emitters that are in a group.

You need to plug this node into an ICETree node on each volume emitter object in a group so that the Lagoa Emit Volume compound can properly compute the proper transform and bounding box limits of each emitter.

Then you can plug the group's Get Data node into the Lagoa Emit Volume compound as usual.

You do not need to use this node if you're using a single volume emitter object.

See Setting Up Emissions in Lagoa Simulations for more information.

Plug this compound's Execute output into a port on the ICETree node for each volume emitter object in a group.

Tasks: Lagoa/Element Creation

Output Ports: Execute
