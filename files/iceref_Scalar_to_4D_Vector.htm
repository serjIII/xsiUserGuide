Scalar to 4D Vector

Categories

  • Conversion

Description

Outputs a new 4D vector built from the input X, Y, Z and W components.

Ports

Inputs

X

The X component for the new 4D vector.

Y

The Y component for the new 4D vector.

Z

The Z component for the new 4D vector.

W

The W component for the new 4D vector.

Outputs

Vector

The new 4D vector.
