OGLMulti

DEPRECATED. This shader is unsupported, but it may still get installed with Softimage to provide compatibility with older scenes that use it. It is recommended that you replace unsupported shaders in your scenes with equivalent shaders from the current Softimage shader library.

Shader Type: RealTime

Output: RealTime

Location: This shader is still supported as a legacy shader, but has been removed from menus in Softimage. Scenes containing this shader will display correctly. Load this shader into the render tree from Nodes RealTime OpenGL More.

The Multiple Textures shader combines four Single Texture shaders OGL_Single_Texture in one property editor. You can set their basic attributes, and blend them using the modulation functions. However, the modulation functions offer very limited blending capabilities, so for full control over blending functions, it's best to use multiple Single Texture shaders, as oppose to a single Multiple Texture shader. The advantage of the multiple texture node is that it does multi texturing in a single pass.

Render Tree Usage

This shader accepts inputs from up to four different image clip nodes, as well as from any other realtime shader. It can be output to any other realtime shader.
