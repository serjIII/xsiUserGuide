Cleanup Painted Weights

Fixes existing envelope weights by removing deformers that are not in the specified group and renormalizing the weights among the remaining deformers. The weights are stored per point in the Self.EnvelopeWeights array and the corresponding deformer indices are stored in Self.EnvelopeDeformerIndices. These attributes can be used by custom skinning compounds.

This can be useful in certain cases if some deformers are missing (or if you want to reduce the number of deformers) and the results of renormalizing are acceptable.

Tasks: Deformation/Skinning

Output Ports: Execute

Deformer Group In Name

A scene reference to prepend to the group specified by Deformer Group Reference to fully resolve the group's name.

Deformer Group Reference

Specifies the group of deformers to use for weighting.
