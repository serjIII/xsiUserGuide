Particle Gradient Fcurve Compound

The Particle Gradient Fcurve compound creates a curve that you can plug into any gradient port of a shader (such as gradient color or gradient density) to control the gradient's falloff. It contains a Gradient Control shader, the Scalar Math Curve shader, and a Scalar to Color converter.

For general information about ICE particle shaders, see ICE Particle Shading.

Parameters

Name

The compound's name. Enter any name you like, or leave the default.

Curve

Controls the profile curve for the gradient's falloff shape over distance.

Render Tree Usage

Plug the Particle Gradient Fcurve output into any Gradient port, such as those on the Particle Density shader or the Particle Renderer compound.
