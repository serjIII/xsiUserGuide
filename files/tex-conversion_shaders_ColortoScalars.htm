Color To Scalars

| Color To Scalars

Category: Processing > Conversion

Shader Family: Texture

Multiple Outputs: Scalars

Converts a color (RGBA) input into separate scalar outputs for each channel.

Input

Specifies the input color to be converted to scalar values (values between 0 and 1).

Render Tree Usage

This shader is placed between a color output and scalar inputs in order for the values to connect.
