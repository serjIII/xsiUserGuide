Set Flags

Stores a set of Boolean values as bits in a bit field. The returned value is an integer.

Category: Binary Flags

Output Ports: Result

Flag0

The first Boolean value. To create additional values, right-click over the port in the ICE tree view and choose Insert PortBefore or After, or connect a value into New (Flag0).
