SCM Fixer Op2 Property Editor

Controls the continuity at the seams between the subsurfaces of an assembled surface mesh.

To apply: Select two or more surfaces and choose Create Surf Mesh Assemble from the Model toolbar. This both assembles the surfaces into a single mesh and applies the Surface Continuity Manager (SCM).

To reapply SCM, for example if you have frozen an assembled surface mesh, select the mesh and choose Create Surf. Mesh Continuity Manager from the Model toolbar.

To redisplay: Select the surface mesh then choose Edit Properties Modeling Properties and click the SCM Fixer Op2 tab.

Always Evaluate

Activates continuity control at the seams. When this option is off, the operator does not fix the seams. This allows you to adjust the subsurfaces before turning this back on again.

Continuity

The degree of continuity maintained at the seams:

  • C0 (Position) The point is joined by a straight line segment.

  • C1 (Tangent) The point is joined by a smooth curve segment and the tangent does not change abruptly.

Don't Fix the Tagged Points

Controls whether continuity is maintained for the NonFixingPointsCluster:

  • When this option is on, continuity is not maintained for points in the NonFixingPointsCluster. This lets you create holes like mouths in seamless surfaces.

  • When this option is off, continuity is maintained for all points.

You can manually add and remove points from the NonFixingPointsCluster after the surface mesh has been assembled.

Allow Modification of Boundary Points

Controls how you can select and manipulate points at the boundary between subsurfaces:

  • When on, you must select both "ghost" points on opposite sides of a boundary point to move it.

  • When off, you can select and move a boundary point directly.
