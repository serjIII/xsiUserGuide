Set Particle Transform

This compound sets the particle's transformation attributes: Scale, Orientation, and PointPosition.

For more information on transforms, see Converting Between Local and Global Reference Frames [ICE Guide].

Tasks: Particles/Setters

Output Ports: Execute

Value

The 4x4 matrix data used to define the particle's transform data.
