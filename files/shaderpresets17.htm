Panoramic

| Render Tree Usage

Category: Lens

Shader Family: Lens

Output: Color

Renders an image of up to 360 degrees around the camera (sometimes called a cylindrical or anamorphic projection) to which it is applied. Used primarily for rendering an image to be used with QuickTime VR or Microsoft Surround Video.

Horizontal Field of View

Defines the horizontal field of view to be rendered. A value of 360 will render an image all around the camera.

Render Tree Usage

This shader can be applied directly to the camera in the lens shader stack. See Applying Lens Shaders to Cameras [Cameras and Motion Blur].
