Square Root (Expression Function Reference)

The Function Exponent Square Root command in the expression editor inserts the token sqrt( at the insertion point in the editing pane. This token evaluates the square root of an expression. Enter an expression after the opening bracket ( then enter a closing bracket ).

Example

sqrt(sphere.kine.local.posy)

Returns the square root of the local Y translation of sphere.
