Color Switch

Category: Processing > Switch

Shader Family: Texture

Output: Color

Enables one of two input values using a Boolean (on or off) switch. The Switch parameter is animatable, so you can then switch between the inputs by animating the Switch parameter (see Keying Parameters in Property Editors).

Name

The shader's name. Enter any name you like, or leave the default.

Switch On: Color 2 Switch Off: Color 1

This Boolean switch is animatable. When on, Color 2 is displayed instead of Color 1; when off, Color 1 is displayed.

Color 1

Defines Color 1 input.

Color 2

Defines Color 2 input.
