Device Driver Property Editor

Allows you to set up and activate/deactivate the channels of the selected device driver.

For more information about device drivers in general, see Device Drivers [ Animation ].

To display: Select a driver from the list and click the Inspect Selection button in the Device Manager.

Device

ID

ID of the device driver. This is for information purposes only and cannot be changed.

Active

Toggles the activeness of the device driver.

Options

Opens a property editor in which you can set options for the selected device driver. This is only supported for drivers that have options created for them, such as the MIDI Keyboard driver.

Channels

Displays information about the device driver's channels that you can edit. You can also edit the individual channels of the device driver in its Channel Property Editor.

  • Active: Toggles the activeness of the channel.

  • ID: ID of the channel.

  • Name: name of each channel input.

  • Value: You cannot change the value here.

  • Offset: Offsets the channel's value.

  • Scale: Scales the channel's value.

  • Action: Select the type of action you want the channel to drive: Non, Drive, Command, or Special. Then specify the Target for this action.

  • Target: Specify what you want the action to drive. If Drive is the Action, enter a parameter name; if Command is the Action, enter a command name; and if Special is the Action, click Channel Target Special Values for a list of possible commands.
