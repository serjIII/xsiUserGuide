Field Filter

Filters individual fields of images.

Requires an Input (Any).

Produces a Output (Any).

Filter

Filter Type

Sets the filter type. Select Interpolate, Replicate, Swap or Blur.

Interpolate and Replicate are useful for removing field blur from grabbed images

Interpolate replaces the pixel in the other field with the average value of the pixels above and below it in the dominant field. Thus if the dominant field is Odd, a pixel in line 2 will become the average value of the pixels above and below it in lines 1 and 3.

Replicate copies the dominant field into the other field.

Swap is useful for correcting an image which has accidentally been rendered to the wrong fields; it swaps the contents of the two fields over.

Blur is useful for smoothing out field blur while retaining movement. It replaces each pixel with a value made of half the original pixel value and a quarter of each of the values of the pixels directly above and below it.

Field

Sets the dominant field for use with the Interpolate and Replicate filters. Select Even or Odd.

Mask

Controls this operator's Mask Options.
