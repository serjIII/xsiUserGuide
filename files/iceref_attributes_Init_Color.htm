Init_Color

A particle's original Color value when it was emitted.

This attribute is set by the Emitters compounds, and is exposed in their property editors as the Color parameters.

It can also be used by Modify Particle Color compound or your own compounds.

Data Type

color

Belongs To

points

Geometry Types

  • point cloud

Attribute Type

custom

R/W or RO:

read/write
