Scene Colors Property Editor

| Objects | Components | Geometry Views

Display colors of objects and components in their selected, unselected, and unselectable state. You can also specify the color of viewports, background area, and grid. You specify the color simply by clicking on the color chip of an element then using the color selection controls that subsequently appear.

To display this property editor, see Setting Scene Colors [Basics].

Objects

Selected

Node

Color of selected scene elements.

Branch (children)

Color of selected branches of objects.

Owner of Subselection

Color of the owner of selected elements. This would, for example, affect the wireframe color of an object whose property or cluster has been selected.

Active Geometry

Color of objects that are active for component selection (points, edges, polygons, and so on).

Non-Selectable

All Objects

Color of unselectable objects.

Currently Inspected Nodes

NOTE:These settings don't apply unless Highlight Property Set Owners is on in your Display preferences.

Owned By

Color of objects whose properties are currently being inspected in a property editor, if the objects aren't selected themselves.

Inherited By

Color of objects who inherit properties that are currently being inspected in a property editor, if the objects aren't selected themselves.

Unselected

Geometry

Color of unselected geometric objects.

Null

Color of unselected nulls.

Light

Color of unselected lights.

Camera

Color of unselected cameras.

Control Object

Color of unselected control objects.

Components

Controls the color of various types of components in the viewports. To see component type, make sure that it is on in the viewport's Objects.

Selected

Component

Color of selected components.

Cluster

Color of selected clusters.

Unselected

Point

Color of unselected points.

Knot

Color of unselected knots.

Cluster

Default color of unselected clusters.

Normal

Color of normals.

Tag

Color of selected points on unselected objects, or selected points on selected objects or active geometries when the selection filter is not Points.

Isoline

Color of unselected isolines.

Boundary Curve

Color of unselected boundary curves.

Surface Curve

Color of unselected surface curves.

Sampled Point

Color of unselected sampled points (texture UVs).

Stereo

Zero Parallax Plane

Color/transparency of the zero parallax planes drawn in a viewport for the left and right cameras in a stereo camera rig.

See Stereoscopic (3D) Camera Rigs [Cameras and Motion Blur] for more information.

Geometry Views

Background

Color/transparency of the viewport background.

Film Gate

Color/transparency of the film gates displayed for camera views in the viewport or Object view.

See Film Gates in Camera Views [Basics] for more information.

Grid

Color/transparency of the grid or "floor" in viewports.

Viewport

Color/transparency of the viewport in the Camera view.
