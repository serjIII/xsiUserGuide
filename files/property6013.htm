Curve Deform Property Editor

| Curve Deform Operator | Constraint

Distorts an object or cluster by mapping its local Y axis onto a curve.

To apply: See Curve and Surface Deformations [Data Exchange].

To redisplay: Select the object then choose Edit Properties Modeling Properties and click the Curve Deform tab.

Curve Deform Operator

Mute

Toggles the deformation on and off without affecting the values of other parameters.

Axis

The local axis of the object which is deformed by the curve.

Scaling

Along Curve

The scaling factor of the deformation along the direction of the curve.

Along Normal

The scaling factor of the deformation in the direction of the curve's normal (perpendicular to the tangent of the curve).

Along Binormal

The scaling factor of the deformation in the direction of the curve's binormal (perpendicular to both the tangent and the normal).

Roll

The rotation of the object or cluster around the curve in degrees. If a deformed cluster appears twisted, try adjusting this value.

Translation

Along Curve

The translation offset of the object or cluster along the length of the curve in Softimage units.

Along Normal

The translation offset of the object or cluster in the direction of the curve's normal (perpendicular to the tangent of the curve).

Along Binormal

The translation offset of the object or cluster in the direction of the curve's binormal (perpendicular to both the tangent and the normal).

Constraint

These options are useful if either the deformed object or the deforming curve were moved from the global scene center before the deformation was applied.

Constrain to Deformer

Select this option if the deformed object was moved from the global center before the deformation was applied.

Constrain to Deformee

Select this option if the deforming curve was moved from the global center before the deformation was applied.
