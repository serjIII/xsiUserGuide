EmitPosition

The position at which a particle was originally emitted.

This attribute is set automatically by the Add Point node as well as compounds that contain it, such as the Emitters. Unlike EmitLocation, it does not move with an animated or deformed geometry.

You can use this attribute with some of the Surface Interaction compounds that allow you to specify the Emit Position for particles.

Data Type

3D vector

Belongs To

points

Geometry Types

  • point cloud

Attribute Type

custom

R/W or RO:

read/write
