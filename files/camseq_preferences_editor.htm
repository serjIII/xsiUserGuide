Camera Sequencer Editor Preferences

Sets the display preferences for Shot Tracks and ShotClips in the Camera Sequencer Editor. See Camera Sequencer Editor Overview for more information.

To display the Camera Sequencer Editor Preferences:

  • Choose View Preferences from the Camera Sequencer Editor menu bar.

    or

  • Choose File Preferences from the Softimage main menu, and then click Editors Camera Sequencer Editor.

Tracks

Display Tracks

Shot Tracks

Toggles the display of Shot Tracks in the Camera Sequencer Editor.

Show Track Options

Track Name

Toggles the name of a Shot Track displayed at its left.

For more information on Shot Tracks, see Shot Track.

Clips

Toggles the display of various ShotClip properties in the Camera Sequencer Editor.

Show Labels

Name

The name that appears in the center of ShotClips.

Clip In+Out

In and out frame numbers relative to the source in the bottom corners of ShotClips.

Scale

Scale factor displayed at the top of ShotClips. If you scale a ShotClip to twice its length, then the scale factor is 0.50.

Define Length of Newly Created ShotClip

Enables you to define the length of a newly created ShotClip.

Until End Frame of Camera Sequencer

The length of the newly added ShotClip is equal to the total duration of the Sequence Time. .

Absolute Duration

Choosing this option activates the Number of Frames field.

Number of Frames

Set its value to define the length of the new ShotClip.

For more information on ShotClips, see ShotClip.
