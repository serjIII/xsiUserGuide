Boolean Logic

Category: Processing > Math

Shader Family: Texture

Output: Boolean value

Applies a logical operation between 2 Boolean inputs.

Name

The shader's name. Enter any name you like, or leave the default.

Input 1

Defines the first Boolean switch input to be considered in the operation.

Operation

And

Or

Equals

Input 2

Defines the first Boolean switch input to be considered in the operation.

Negate Result

Reverses the polarity of the result.

Render Tree Usage

Used to connect one or two shaders (with Boolean outputs) to the Input parameters. Once a function is selected, the output can be used on any shader that accepts a Boolean input.
