Get Particle Velocity

This compound gets the particle's current velocity data (the speed plus a vector that determines direction) as defined by its PointVelocity attribute. You can output the Velocity Vector and Speed data.

For more information, see ICE Particle Speed [ICE Particle Simulations].

Tasks: Particles/Getters

Output Ports: Velocity, Speed

Reference

The particle's PointVelocity attribute.
