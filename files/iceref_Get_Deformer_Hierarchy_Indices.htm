Get Deformer Hierarchy Indices

Returns the indices of a rig element, its parent, and its children in a Rig_Proxy hierarchy of a CrowdFX simulation. You can use these indices to get the transformations of the deformers from the actors' CrowdFX_Actor_Deformers array, for example, for building custom crowd constraints.

For more information, see Modifying the Actor's Rig.

Tasks: Crowds/Getters

Output Ports: Parent Index, Index, Children Indices, Children Found

Deformer Proxy Name

A name to prefix to Reference to fully resolve the name.

Reference

A reference to an object in the actor's Rig_Proxy hierarchy.
