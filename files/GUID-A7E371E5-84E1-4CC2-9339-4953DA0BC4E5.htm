Interoperability with Other Autodesk Products

The "Send to" commands on the File menu allow for easy interoperability between Softimage and other Autodesk products. The commands work in a similar fashion in all products:

  • Send as new scene starts a new scene in the other program and adds the selected objects. If the other program is not already running then it starts, or if it is already running then you are prompted to save the current scene as necessary. If nothing is selected, then you are prompted to send the entire scene.
  • Update current scene resends the selection to the other program. Objects that already exist in the other scene sent get updated, and additional objects get added. If nothing is selected, then you are typically prompted to resend the previous selection or send the entire scene depending on the situation.

    You can keep multiple versions of the scenes as you test different settings by saving them under different names, loading them manually, and updating.

  • Add to current scene adds the selected objects to the scene in the other program. If the objects already exist in the other scene, additional copies are created.

Note that if the other program's window does not come to the front automatically, then you may need to switch to it manually.
