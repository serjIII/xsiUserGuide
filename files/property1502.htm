Flare Property Editor (Camera)

Defines how the active camera will "see" the flare. You can use this property page if a scene has several cameras but you do not want all of them to render the flare effect.

To Display: Expand the camera node in the explorer view and click the Flare Properties icon (will only be visible if a flare has been applied).

Brightness/Saturation

Saturation

Defines what color the camera sees the flare. At 0, the flare changes to gray scale.

Brightness

Defines the brightness or intensity of the flare effect.

Flare Controls

Lens Guard Margin: Defines a rectangle around the view plane with width equal to margin. The flare intensity falls off linearly within this region. This parameter blocks out direct rays from light sources outside the view angle. In screen space, this means that light(s) outside the view area should not cause lens effects.

Focus Pull

Allows a lens flare to be animated. The effects individual components rotate and translate to simulate the camera's focus being changed. You can also use this parameter to select a specific flare configuration or pattern.

Safe Overlay

Causes a lens flare to take the underlying color into account and tries to make sure that the result does not exceed the maximum color intensity.

Render Flare Only

Renders the flare only on a black background. Saves rendering time.
