To Clip

The To Clip operator is one half of the Image Clip operator pair. It is a terminal operator, meaning that it has only an input. By default, it receives its input from the From Clip operator. However, if you modify the image clip by inserting effects between the From Clip and To Clip operators, the To Clip operator represents the image clip with any and all changes applied.

Receives an input from the From Clip operator by default.

Operator Name

Specifies the operator's name.

Image Clip Name

Specifies the name of the Image clip represented by the operator.
