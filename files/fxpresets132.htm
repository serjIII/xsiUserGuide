Bit Depth

Allows you to take an input image of any bit depth and output it at another bit depth.

Requires an Input (Any).

Produces an Output (Any).

Bit Depth

Destination

Sets the destination bit depth of the input image. Choose 8-bit, 16-bit, or floating point depth (Float).

Output

Controls this operator's Output Options. Note that this operator uses limited output options.
