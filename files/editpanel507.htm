Duplicate/Instantiate from Animation Dialog Box

Specifies the start frame, end frame, and step value of the original object's animation in order to create the duplicates or instances. The step value determines which frames should be used to create the new objects. For instance, if the start value is 10, the end value is 40, and the step value is 15, the new duplicated or instantiated objects will be created from frames 10, 25, and 40.

To display: Choose Edit Duplicate/Instantiate Duplicate/Instantiate from Animation from the Edit panel.
