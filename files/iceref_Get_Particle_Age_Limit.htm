Get Particle Age Limit

This compound gets the particle's current Age Limit data as set by its AgeLimit attribute. You can output this AgeLimit data.

For more information, see ICE Particle Age [ICE Particle Simulations].

Tasks: Particles/Getters

Output Ports: Age

Reference

The particle's AgeLimit attribute.
