Get Nearby Points

For each point on the "self" object, returns an array of points on the same object.

Tasks: Deformation/Getters

Output Ports: Num_Points, Points, Point_Positions

Enable Cutoff Distance

Restricts the returned points to those within a specified distance (set below).

Cutoff Distance

The maximum distance of the returned points.

Enable Max Number Points

Restricts the number of points returned.

Max Number Points

The maximum number of points to return.
