StickOffsetMatrix

The offset matrix of a particle from its StickLocation.

The Stick to Location and Stick to Surface compounds apply this matrix to set the particle's final position and orientation at each frame.

Data Type

4x4 matrix

Belongs To

points

Geometry Types

  • point cloud

Attribute Type

custom

R/W or RO:

read/write
