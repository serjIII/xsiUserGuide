Animating along Paths and Trajectories

A path provides a route in global space for an object to follow in order to get from one point to another. The object stays on the path because its center is constrained to the curve for the duration of the animation.

The plane on the top uses path animation. Its position is measured as a percentage along the curve.

The plane on the bottom uses trajectory animation. It jumps from knot to knot at each frame.

TIP:To store the path animation in an action to use in the animation mixer, see Storing Path Animation [Nonlinear Animation in the Animation Mixer] for more information.
