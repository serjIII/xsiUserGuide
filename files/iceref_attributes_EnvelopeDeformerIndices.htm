EnvelopeDeformerIndices

The indices of the deformers in the deformer group to which a point is weighted.

Data Type

array of integers

Belongs To

points

Geometry Types

  • envelope weights property

Note that the Calculate Deformer Weights and Cleanup Painted Weights compounds also set a custom attribute with the same name directly on geometric objects. This can be used by custom skinning compounds.

Attribute Type

inherent

R/W or RO:

read/write
