From Clip

The From Clip operator is one half of the Image Clip operator pair. It represents the original image clip. Applying effects between this operator and the To Clip operator lets you modify image clips in place.

Produces an output to the To Clip operator by default.

Operator Name

Specifies the operator's name.

Image Clip Name

Specifies the name of the Image clip represented by the operator.
