Synoptic Property Editor

Associates an object to a synoptic file, allowing you to use synoptic views to control the object.

To apply: Select an object and choose Get Property Synoptic from any toolbar.

To redisplay: Select the object, click the Selection button on the Select panel, then click the Synoptic_Property icon.

Synoptic Name

Displayed on the title bar of the synoptic view.

Path

Specifies the synoptic file to display. You can type a location and file name directly in the box, or use the browse button (...) to select a file. You can use a relative path if the synoptic file is saved in your scene, or an absolute path if it is saved in an application data path.
