Lagoa Advect Simulated Mesh

Deforms a mesh based on a Lagoa particle simulation, allowing the deformation to interact with fluids in the same cloud.

To apply: See Advecting Geometry with Lagoa [Lagoa Multiphysics Simulations].

Tasks: Lagoa/Simulate

Output Ports: Execute

In Name

The Out Name of the mesh to be advected. This mesh has to created before in the simulation framework.

Plug the Out_Name of a Get Data node that points to the mesh.
