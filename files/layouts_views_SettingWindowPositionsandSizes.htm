Setting Window Positions and Sizes

Each layout stores information about the default size and position of each type of floating window, including views, toolbars, and shelves. You can modify a window and store its new size and position in a custom layout. Whenever you start Softimage and open that window, it always appears in its new location.

NOTE:If the size and position of a window is not present in the current layout (for example, if a new view has never been opened in that layout before), then the window uses the default size and position stored in the window's XML file. You can set those defaults by editing the .xsivw or .xsitb file manually in an XML or text editor.

To save window sizes and positions

  1. Make sure that you are viewing a custom layout. If necessary, choose a layout marked u from the View Layouts menu.

  2. Open a view, toolbar, or shelf and change its size and position.

  3. Close the floating window. This stores the size and position internally.

  4. Choose View Layouts Save Layouts. The Edit Layout dialog box opens.

  5. If window size and position is the only change you have made to a layout, then by default there is no checkmark next to the layout. Simply click to add a checkmark if necessary and then click Save.
