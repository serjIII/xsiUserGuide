HLSA to Color

Categories

  • Color

  • Conversion

Description

Creates and outputs a new color by converting the hue, lightness, saturation and alpha components taken from the input HLSA color space.

The specified input alpha component has no effect on the conversion. The alpha component of the resulting color equals the specified input alpha component.

Ports

Inputs

Hue

The hue component taken from the HLSA color space.

Lightness

The lightness component taken from the HLSA color space.

Saturation

The saturation component taken from the HLSA color space.

Alpha

The alpha component.

Outputs

Color

The new color.
