Randomize Color

This compound adds a random value to specific channels of the particle color so that each particle uses a different color from the range you set. You set a mean value around which you determine the range of variance of color values.

Plug this compound's Random Color output into any Color port, such as that of the Emit compound.

For more information, see Randomizing and Turbulizing the Color [ICE Particle Simulations].

Category: Color

Tasks: Particles/Modifiers

Output Ports: Random_Color

Randomized Channel

The color channels you can choose from to randomize: RGB, Red, Green, Blue, Hue, Lightness Saturation, or Alpha.

Seed

Defines the sequence of random numbers used. If you require that two nodes generate different sets of values with the same parameters, simply assign them different seeds.

Animated

Toggles the animation of the randomness; that is, whether it varies over time or not.

You can use this option when using an unsimulated ICE tree (such as if the point cloud's ICETree node is in the Modeling region).

Mean Value

The base value used to calculate the range of random variation on either side of it.

Variance

The amount of randomness calculated around the Mean Value.
