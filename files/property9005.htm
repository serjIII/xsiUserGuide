MatchUVW Property Editor

The match command matches a selection of sample points by snapping them to one or more specified targets. This is useful for aligning selections of points, or bringing two identically shaped groups of points (halves of a face, for example) to the exact same location.

A tolerance value specifies the range within which targets are detected. This tolerance can be disabled, if necessary, in which case the selection is matched to the closest available target

Tolerance

Use Tolerance

When enabled, selected points are snapped to targets (selected points, unselected points, grid corners or pixel corners) within the range specified by the tolerance slider.

When disabled, selected points are snapped to the closest target (selected points, unselected points, grid corners or pixel corners).

Tolerance Slider

The threshold within which selected points are matched (snapped to their targets). Selections that are farther apart require a higher tolerance to match them, and vice-versa. If the tolerance value is set to high, the selected points will sometimes appear to be collapsing.

Targets

The Targets options specify the targets to which the selected points are matched. These options are not mutually exclusive, meaning that any combination of the available target types can be enabled simultaneously.

Selected Points

When enabled, selected points are matched to one another.

Unselected Points

When enabled, selected points are matched to unselected points.

Grid

When enabled, selected points are matched to a virtual grid. By default, this grid matches the grid displayed in the texture editor work area, but you can adjust it using the Grid Info options (see below).

Pixel Corners

When enabled, selected points are matched to physical pixel corners. By default, pixel corner snapping is based on the physical resolution of the texture image, but you can adjust the resolution, for snapping purposes only, using the Image Info options (see below).

Grid Info

When Grid is enabled in the Targets options, these options control the resolution and offset of the virtual grid to which selected points are matched.

Grid U Res

Specifies the virtual grid's resolution in the U direction.

Grid V Res

Specifies the virtual grid's resolution in the V direction.

Grid U Offset

Offsets the virtual grid, by the specified amount, in the U direction.

Grid V Offset

Offsets the virtual grid, by the specified amount, in the V direction.

Image Info

When Pixel Corners is enabled in the Targets options, these options control the resolution the virtual image to whose pixel corners selected points are matched.

Image U Res

Specifies the virtual image's resolution, in pixels, in the U direction.

Image V Res

Specifies the virtual image's resolution, in pixels, in the V direction.
