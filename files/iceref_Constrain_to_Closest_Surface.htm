Constrain to Closest Surface

Constrains an object to the closest surface location on a polygon mesh or NURBS surface. This node should be plugged into an Add Constraints compound.

Category: Kinematics

Output Ports: Constraint

Mute

Deactivates the constraint.

Constrainee

The constrainee specified here must match the constrainee specified in the Add Constraints compound.

Constrainee In Name

A scene reference to prepend to Constrainee Reference to fully resolve the name.

Constrainee Reference

The object to constrain.

Constrainer

Constrainer In Name

A scene reference to prepend to Constrainer Reference to fully resolve the name.

Constrainer Reference

The constraining object.

Find Closest Location

Proximity Method

Method used to compute the proximity:

  • Closest Surface

  • Closest Vertex or Knot

  • Closest Smoothed Surface

Enable Cutoff Distance

Allows to search only within given distance.

Cutoff Distance

The distance within which to search when the Enable Cutoff Distance option is turned on.

Alignment

Align to Normal

Aligns the constrained object with the normal at the closest location.

Axis to Align

Specifies which axis to align with the normal.
