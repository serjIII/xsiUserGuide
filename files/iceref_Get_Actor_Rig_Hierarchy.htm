Get Actor Rig Hierarchy

Gets the rig hierarchy data from the CrowdFX_Deformers_ParentIndices ICE attribute.

Tasks: Crowds/Animation

Parent Indices

Parent indices of the deformers in the 4x4 matrix array Self.CrowdFX_Actor_Deformers.

Root Index

Index of the root deformer in the 4x4 matrix array Self.CrowdFX_Actor_Deformers.

Root Indices

(Used rarely) Indices of multiple roots in the deformer hierarchy.
