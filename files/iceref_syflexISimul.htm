syflexISimul

Categories

  • Syflex Simul

Description

Updates each the position and the velocity of each point on an input object based on its previous velocity, position, mass and force attributes.

This node is the simulator node in the Syflex Cloth and Syflex Curve compounds.

For information on creating Syflex ICE simulations, see Syflex ICE Basics.

Ports

Inputs

Current Frame

The current frame at which the ICE tree is being evaluated.

First Frame

The first frame at which this simulation begins.

Precision

This is the percentage of error allowed that determines the precision of the simulation. It doesn't depend on the Softimage unit scale.

Mass

Mass is the quantity of matter of an object. This value defines the mass of the cloth object in whatever units you have specified.

Geometry

The geometry of the simulated object: polygon meshes for cloth and linear curves for curve deformations.

In Position

The global XYZ position of the input object.

Force

Calculates the input force for the simulation.

Outputs

Out Position

The output global XYZ position of the simulated object.
