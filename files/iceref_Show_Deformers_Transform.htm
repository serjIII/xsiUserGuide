Show Deformers Transform

Displays axes representing the transformations of the actors' deformers in a CrowdFX simulation.

For more information, see Modifying the Actor's Rig.

Tasks: Crowds/Debugging

Output Ports: Result

Enable

Toggles the display on and off.

Mode

  • All shows the transforms of all deformers for all actors.
  • Use Index shows the transforms of the deformer with the specified Index for all actors.

Index

The index of the deformer.

Size

Controls the size of the displayed axes.
