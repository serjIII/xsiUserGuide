misss_mia_skin2_phen

This shader phenomenon is similar to misss_fast_skin_phen with two important differences:

  • It uses the misss_fast_shader2 fast scattering shader which uses per-color scattering, giving a much more realistic looking effect.
  • It uses the mia_material shader for diffuse, reflections, highlights etc., adding many of the highly-advanced shading features from that shader into this shader phenomenon.

For information on this shader, see the mental ray Documentation at www.autodesk.com/mentalray-help-2013-enu.
