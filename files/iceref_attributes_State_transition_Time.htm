State_transition_Time

The time at which a particle last changed states, in seconds.

This attribute is set by the States compounds and is used by the Get Time in State and Test Time in State compounds.

You can also get this attribute and use it to do other things based on when a particle changed state.

Data Type

scalar

Belongs To

points

Geometry Types

  • point cloud

Attribute Type

custom

R/W or RO:

read/write
