BlobStrength

Controls how strongly the surfaces of blobs are pulled toward each other. At 0.0, there is no attraction and at 1.0, blob surfaces are pulled together regardless of distance.

If this attribute is undefined. the default value is 0.5.

This attribute is exposed as the Blob Strength parameter in the Blobs compound.

Data Type

scalar

Belongs To

points

Geometry Types

  • point cloud

Attribute Type

predefined

R/W or RO:

read/write
