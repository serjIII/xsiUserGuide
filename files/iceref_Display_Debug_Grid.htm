Display Debug Grid

Displays a grid that may be useful when debugging ICE trees.

Category: Debugging

Output Ports: Execute

Grid Count

The number of subdivisions.

Grid Size

The size of the grid in the object's local reference frame.

Grid Position

The position of the grid in the object's local reference frame.

Display Grid

Toggles the display of the grid on and off.
