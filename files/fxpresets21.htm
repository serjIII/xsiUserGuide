Swap Component

| Swap | Mask | Output

Maps any combination of input component channels onto any combination of output component channels. For example, the green channel can be mapped onto the red channel, and/or the red and blue channels can be mapped onto the alpha channel. Where more than one input channel is mapped onto an output channel, the result can be calculated as either the average or the sum of the input channels.

Requires an Input (Any).

Produces an Output (Any).

Swap

Red, Green, Blue, Alpha

Each column represents one input component channel.

R, G, B, A, 1

Each input component column has a checkbox for each of these output channels. Checking an output channel maps the input channel to it. Checking 1 forces the input component channel to maximum.

Mode

Where more than one input channel is mapped onto an output channel, Mode sets whether the result is the average or sum of the input components.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
