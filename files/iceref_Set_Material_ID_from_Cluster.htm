Set Material ID from Cluster

Sets the MaterialID attribute of polygons on the "self" object based on membership in the specified cluster. The MaterialID specifies which material in the Materials array to use for which polygon.

Tasks: Topology/Setters

Output Ports: Execute

Mode

Controls whether the MaterialID is set for polygons that are in the cluster or not in the cluster. The MaterialID for other polygons is unchanged â€” the default is 0 if it has never been set before.

Material ID

The Material ID to apply. Use a single integer or a set of integers per polygon.

Cluster Name

Connect the name of the cluster to use.
