Create Framebuffer From Render Channel Dialog Box

Adds a channel to the Render Channels Output grid in the Render Pass Property Editor. You can edit this channel to define its framebuffer (describes how the channel will be written out).

To display:

  1. The Render Manager > (select a pass) > Output

  2. Click the Add... button under the Render Channels Output grid.

Render Channel

Choose the render channel for which you want to define a framebuffer.
