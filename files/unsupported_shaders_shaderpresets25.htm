Vector2scalar

DEPRECATED. This shader is unsupported, but it may still get installed with Softimage to provide compatibility with older scenes that use it. It is recommended that you replace unsupported shaders in your scenes with equivalent shaders from the current Softimage shader library.

| OGLTCTrans

Shader Type: Conversion (Tool)

Output: Scalar value

NOTE:This shader is no longer supported. Scenes using it will still load, but it is recommended that you use the newer Vector To Scalar and Vector To Scalars shaders.

Lets you convert the X, Y, and Z vector values to a single channel.

Name

The shader's name. Enter any name you like, or leave the default.

Input (XYZ or UVW)

Specifies the input vector information.

Component

Lets you select which component will be extracted from the vector and used as a scalar value.

Render Tree Usage

This shader is simply placed between a vector output and a scalar input in order for the two values to connect.
