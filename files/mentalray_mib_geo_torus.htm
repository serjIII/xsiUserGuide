mib_geo_torus

Category: mental ray > Geometry

Shader Family: Geometry

Output: Custom

This shader generates a polygonal torus lying on the XY plane, centered at the origin. The torus is defined by two circles: the smaller one revolves around the center of the larger circle.

Name

The name of the shader node displayed in the render tree. Enter any name you like, or leave the default.

radius

The radius of the large circle.

thickness

The radius of the small circle.

uSpans/vSpans

Specifies the number of subdivisions in each parametric range.

If omitted (set to 0), parameters default to the following values: radius 2.0, thickness 1.0, uSpans 32, vSpans 32.
