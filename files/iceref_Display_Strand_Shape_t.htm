Display Strand Shape

Displays an approximation of a strand's shape for visual feedback. To use, plug the output into the root ICETree node after the simulate node or any other node that updates particle positions.

Category: Debugging

Tasks: Particles/Strands

Output Ports: Execute1

Strand Shape

Select a shape to use to approximate the strand. Note that hull lines are drawn only for certain shapes.

Draw Cross Sections

Toggles the display of cross-sectional lines. Cross-sections are only drawn for certain shapes.
