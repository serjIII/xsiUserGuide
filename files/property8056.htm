Offset Polygon Op Property Editor

Expands or shrinks the outline of selected polygons. Each contour edge is shifted by a specified amount in the plane of its polygon and remains parallel with its original orientation.

To apply: Select some polygons or polygon clusters on a polygon mesh object and choose Modify Poly. Mesh Offset Polygons Contour from the Model toolbar.

To redisplay: Select the object with offset polygons, click the Selection button on the Select panel, then click the Offset Polygon Op icon.

Offset

The amount of offset in Softimage units. Positive values expand the outline and negative values shrink the outline. A value of 0 has no effect.
