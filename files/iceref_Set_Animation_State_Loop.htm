Set Animation State Loop

Categories

  • Crowds > Animation

Description

You can use this node to playback specific time ranges of an Animation State. For an active Animation State, this node is setting the current time to be used for the referenced animation.

The complete length of all animations referenced in an Animation State is mapped to a range from 0 to 1. For example, if the length of the referenced animations is 40 frames, then setting the Animation State Loop to 0.25 lets the system to use the animation values defined at frame 10.

Parameters

Reference

Reference to the simulation Point_Cloud.

State

Defines the requested animation state name.

Ports

Inputs

In Name

Name of the simulation Point_Cloud.

Loop

Defines the current time for an Animation State.

Outputs

Execute

Set the loop position.

Out Name

Output name of the simulation Point_Cloud.
