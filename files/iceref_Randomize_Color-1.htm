Randomize Color

Randomizes a color or a channel within a range of the specified mean value.

Category: Color

Output Ports: Random Color

Randomized Channel

Channel to be randomized

  • RGB
  • Red
  • Green
  • Blue
  • Hue
  • Lightness
  • Saturation
  • Alpha

Seed

Defines the sequence of random numbers used. If you require that two nodes generate different sets of values with the same parameters, then assign them different seeds.

Animated

Turn on to generate a new set of random numbers at each frame. Otherwise, the same numbers are always returned when the current time changes.

Mean Value

The color around which the random colors will be generated.

Variance

Specifies the range in which the random values will be generated. The variance is a distance from the mean value.

Random Color

Random color generated
