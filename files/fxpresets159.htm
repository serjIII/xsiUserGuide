Melt Wipe

Melts away the foreground clip to reveal the background clip. The foreground clip "melts" downwards off the screen, while being distorted in a random "liquid" manner along the horizontal lines; the vertical lines are not distorted. The strength of the distortion can be controlled.

Requires a Background (Any), and a Foreground (Any).

Produces an Output (Any).

Melt Wipe

Amount

Sets the progression of the effect. Set a floating-point value using the treadmill. A value of 0 has no effect, and therefore all of the foreground clip is displayed. The higher the value, the more the foreground clip is melted away, and therefore the more of the background clip is displayed. The normal animation would be from 0 at the first frame to 1 or greater at the last frame.

Strength

Sets the strength of the distortion. Set any floating-point value between 0 and 1 using the slider; the higher the value, the greater the distortion.

Seed

This is a base number upon which all random values are calculated. Set any integer value using the treadmill to achieve a different random effect.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
