Tongue Setup Property Editor

Create, set up, or remove a tongue for the Face Robot head.

To display:

On the Tune panel in the Face Robot layout, click the Face > Tools tab and choose Build Tongue.

For more information, see Adding a Tongue [Face Robot].

Mesh

Hide/Unhide Face

Toggles the display of the Face Robot head mesh. Hiding the head makes it easier to see and adjust the tongue.

Remove Tongue Controls

Deletes the tongue controls for the currently selected and solved tongue, but not the actual tongue mesh.

Delete Tongue Mesh

Deletes the selected tongue mesh when you click the Remove Tongue Controls button.

Tongue Mesh

Displays the name of the selected tongue mesh.

Import Tongue

Automatically selects a tongue in the scene. The name of the tongue appears in the Tongue Mesh text box.

Set to Selected

Select a tongue object in your scene, then click this button to set this as the current tongue.

Setup

Fit

Create four markers (nulls) on the selected tongue. You can then move these markers to fit inside the tongue mesh as is done on the Fit panel — see Fitting the Face [Face Robot].

Solve

After the tongue is fitted, click this button to process the tongue as is done when you click the Solve button on the Fit panel — see Fitting the Face [Face Robot].
