Scene Layer Manager Preferences

Sets basic preferences that control what attribute columns are displayed in the scene layer manager and how layers are sorted.

To Display: Choose View Preferences from the scene layer manager menu, or open the Preferences view and click the Editors Scene Layer Manager in the left-hand pane.

Default Ordering

Layer Ordering

Specifies how scene layers should be sorted within groups: None (creation) or Alphabetical (where uppercase letters come before lowercase letters).

Note that groups are always sorted alphabetically.

Default Visible Columns

Contains a check box for each layer attribute. When an attribute's box is checked, the attribute's column appears in the scene layer manager. Clearing the check box removes the column from the scene layer manager.
