Get Particle Emit Location

This compound gets the particle's point position and normal values at emission time as set by its EmitLocation attribute. You can output the particle's Location, Position, and Normal data.

For information, see ICE Particle Emission Attributes [ICE Particle Simulations].

Tasks: Particles/Getters

Output Ports: Location, Position, Normal

Reference

The particle's EmitLocation attribute.
