Integer To Color

Category: Processing > Conversion

Shader Family: Texture

Output: Color

Converts an integer value to color values.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Specifies the input integer value (value between 0 and 1) to be copied to RGB channels.

Render Tree Usage

This shader is simply placed between an integer output and a color input in order for the values to connect.
