Toon and Non-Photoreal Effects

Autodesk® Softimage® has a library of toon shaders that allows you to create a limitless variety of cartoon-style and non-photorealistic rendering effects. These shaders work like any other shaders in Softimage and can be modified and extended using the full shader library in the render tree.
