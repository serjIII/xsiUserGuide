Picker

| RGBA Split

Category: Processing > Color Channels

Shader Family: Texture

Output: Color

Lets you isolate a specific color model and channel from a color, texture, or shader.

Name

The shader's name. Enter any name you like, or leave the default.

Invert

Inverts the input.

Alpha Multiply

Multiplies the input's RGB channels by its alpha channel.

Input

Defines the image, texture, or shader that will have a color picked from it.

Color Space

Determines which color model you will select a color from: RGBA, HLSA, or HSVA.

Channel

Selects the color channel. A separate drop down is available for each color space.

Render Tree Usage

Used to connect any image clip or material to the input and isolate a channel from any color model to output to a color input. The output can be used to drive any color input parameter (diffuse, transparency, etc.)
