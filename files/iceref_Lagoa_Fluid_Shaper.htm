Lagoa Fluid Shaper

Modifies the shape of polygonized meshes.

To apply: See Shaping Fluids [Modeling].

Tasks: Lagoa/Cached Sim Post Effect

Output Ports: Result

Enable

Activates and deactivates the deformation.

Global Iterations

The number of times to repeat the Smooth and Remove Intersections operations.

Mesh Flattening

Flatten

Flattens the mesh to make it less blobby.

Intensity

The amount of flattening.

Threshold

Thickness above which there is no flattening.

Profile

Not implemented.

Mesh Smoothing

Smooth

Smooths the geometry.

Iterations

The number of times to repeat only the smoothing operation.

Smoothing Factor

The amount of smoothing to apply on each iteration.

Contact Geometry

Geometry

The object or group that you want to move points outside of, for example, the collision objects of the original point cloud.

Remove Intersections

Moves points outside of the specified Geometry.

Closed Volume

Allows for faster and simpler calculations if the specified Geometry is a closed volume.

Mesh Offset

Specifies a buffer between the polygonized mesh and the specified Geometry.

Backtrack

The amount by which points can backtrack along their normals towards the specified Geometry. This can be used to allow a certain amount of liquid to hang on to the underside of the geometry.
