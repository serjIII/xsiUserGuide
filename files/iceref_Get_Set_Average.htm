Get Set Average

Categories

  • Math > Statistics

Description

Outputs the average value in the data set. For example, if the data set contains the positions of each point on a polygon mesh, Get Set Average will return the average point position of the mesh (in other words the geometric center of the point positions).

Ports

Inputs

Value

The data set from which to calculate the average value.

outputs

Result

The average value of the set.
