OGL13Shade

DEPRECATED. This shader is unsupported, but it may still get installed with Softimage to provide compatibility with older scenes that use it. It is recommended that you replace unsupported shaders in your scenes with equivalent shaders from the current Softimage shader library.

Shader Type: RealTime

Output: RealTime

Location: This shader is still supported as a legacy shader, but has been removed from menus in Softimage. Scenes containing this shader will display correctly. Load this shader into the render tree from Nodes RealTime OpenGL More.

OGL 1.3 shader that defines the ambient, diffuse, specular, and emissive RGBA values of an object, as well as the size of the object's specular highlight.

General

Enable Lighting

Toggles the lighting components listed on this tab.

Colors

Ambient

Determines how much the global ambience (indirect illumination) affects the object.

Diffuse

Defines the Diffuse surface color in the illuminated area of the object. The value is blended with the object's Ambient value.

Specular

Switches on or off the Specular illumination component of the shading model.

Emissive

Controls the object's emitted light intensity. Not affected by external lights.

Shininess

Controls the size of the object's specular highlight

Render Tree Usage:

This shader controls an object's basic surface shading characteristics. It can be used anywhere in a RealTime shader tree but, like any other RealTime shader, it must be connected to a draw shader before it can be drawn.
