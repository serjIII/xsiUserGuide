Duplicate Symmetry Dialog Box

To display: Select an object or branch and choose Edit Duplicate/Instantiate Duplicate Symmetry from the Edit panel or Create Skeleton Duplicate Symmetry from the Animate toolbar.

Duplicate Options

Duplicate Constraints

Duplicates the constraints that exist on the element you're duplicating. If the element you want to duplicate is constrained to a null, for example, and you select only the element with Duplicate Constraints selected, the new symmetric element is also constrained in position to the original null.

If you select both the element and the null, the null is duplicated and the new symmetric element is constrained to the new null.

Share Parent

The new element has the same parent as the element you're duplicating. If you don't select this option, the new element will be a child of the scene root.

Freeze Negative Scaling

Freezes the negative scaling on duplicated elements.

  • When on, there is no negative scaling on duplicated elements.

  • When off, the scaling for the duplicated element is negated, switching its handedness. For example, if the original element is right-handed, the duplicated element will have a left-handed coordinate system. Note that negative scaling may not be compatible with game engines or motion capture data. In addition, mixed handedness can cause problems in hierarchies of kinematic chains.

Symmetry Plane

Plane of Symmetry

Duplicates across the XY, YZ, or XZ plane.
