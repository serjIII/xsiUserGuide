Graphic Pen

The Graphic Pen effect uses fine linear ink strokes to capture details in the original image. The result appears to have been drawn on paper laid against a fine-grained wood surface. The pen strokes are denser in darker areas, but are not visible in lighter areas.

By default, this effect produces a monochrome (gray-scale) image.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

Stroke Length

Set lower values to shorten the ink strokes, or set higher values to lengthen the strokes.

Stroke Direction

Select a direction in which to draw ink strokes Right Diagonal, Horizontal, Left Diagonal, or Vertical.

Light/Dark Balance

Set lower values to give the image a higher proportion of white (background color), or set higher values for a higher proportion of black (foreground color).

Background Color

Sets the color for background regions of the effect. Set a color using the sliders.

Foreground Color

Sets the color for foreground regions of the effect. Set a color using the sliders.
