Get Particle Shape

This compound gets the particle's current shape data as set by its Shape attribute. You can output the Shape data.

For more information, see ICE Particle Shapes [ICE Particle Simulations].

Tasks: Particles/Getters

Output Ports: Shape

Reference

The particle's Shape attribute.
