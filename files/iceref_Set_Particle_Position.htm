Set Particle Position

This compound lets you set the particle's position, using the particle's PointPosition attribute as a reference.

Plug its Execute output into the Execute on Emit port of an Emit compound to set this value once, or plug it into the ICETree node to update at every frame.

Tasks: Particles/Setters

Output Ports: Execute

Position

Sets the particle's position value in its local XYZ coordinates.

Reference

The particle's PointPosition attribute.
