Mocap to Rig Property Editor

Retargets the animation from motion capture files (BVH or C3D) to a tagged target rig.

To display: Select the tagged target rig to which you want to apply the mocap data, then choose Tools MOTOR Mocap to Rig from the Animate toolbar.

For more information, see Retargeting Animation from Rig to Rig [Character Animation].

Step 1: Specify Source Mocap File Type

File Types

Type of motion capture file you want to retarget: Biovision BVH or C3D. Depending on which file type you choose, the options in this property editor are a little different.

For C3D options, see Fit Cloud - C3D Files.

Step 2: Specify BVH File Location

Mocap File

For BVH files only. Click the browser button (...) to load a BVH mocap data file.

If you need to create a tag template (.tagt) file, this should be a BVH file where the skeleton is in a T-pose.

Step 3: Load or Create Tag Template for Your BVH File

For information on the process of retargeting BVH mocap data, see Retargeting Biovision (BVH) Mocap Data [Character Animation].

Create

Creates a .tagt (tag template) file from the current Mocap File you've specified. This should be the BVH skeleton in a T-pose for tagging.

Load

Click this button to load an existing tag template file.

Step 4: Target Model

Target Model

Name of the rig to which you want to apply the tag template and Mocap File you're specified. Click the Pick button, then select the target rig in the scene.

Step 5: MOTOR Controls

Apply

Applies the specified BVH mocap data file to the target rig.

Adjust

Opens the Retarget Adjust Property Editor in which you can adjust the retargeted mocap data on the target rig.

Remove

Removes the mocap data from the target rig.

Plot

Opens the Plot Retarget Dialog Box in which you can plot the retargeted mocap data on the rig into fcurves.

Fit Cloud - C3D Files

For information on the process of retargeting C3D mocap data, see Retargeting C3D Mocap Data [Character Animation].

Structure

Structure type of the C3D rig you're importing: Mocap Biped or Mocap Biped Fingers.

Pose Map

Loads a C3D pose file (.c3dpose) which is a skeleton structure rig that is in a zero or base pose, which is the T-pose.

Enable Symmetry

Uses symmetry when you move the black spheres of the structure rig to fit them inside the mocap markers of the mocap data.

Import C3D

Loads a C3D mocap data file with animation. This applies the animation on the markers in this file to the structure rig. This file needs to have been created with the same set of markers from the same mocap session as were used to create the pose map file.

Refit

Defines how to constrain the structure rig to be driven by the mocap data on the yellow mocap markers.

Save Pose Map

Saves the structure rig's current pose in a .c3dpose file.

Save Motor File

Saves the C3D animation on the rig as a normalized motion file (.motor) that can be applied to any tagged rig of the same structure.
