Pose Constraint Property Editor

| Pose | Scaling | Options

Constrains an object's position, orientation, and scaling values (all transformation values together, known as a pose) to those of a constraining object.

For more information, see Pose Constraints.

To apply: Select an object, Constrain Pose, then pick the object to which the selected object will be constrained.

To redisplay: Select the constrained object, choose Explore Constraints on the MCP, and click the constraint's icon.

Pose

Active

Toggles the activeness of this constraint.

Blend Weight

Blends the weight of this constraint with other constraints on the selected object.

Scaling

X/Y/Z

Sets the size of the constrained object in local space.

Rotation

X/Y/Z

Sets the rotation of the constrained object in local space.

Position

X/Y/Z

Sets the position of the constrained object in local space.

Scaling

Scaling Orientation (Shear)

If shearing exists for an object, another object's pose can be constrained to this object, taking its shearing into account. Shearing can be obtained if scaling a hierarchy with rotated objects while in Classical scaling mode (from the object's Local Transform Property Editor).

The Scaling options on this property page are the same as the ones on the Pose page.

Options

Constrain

Position

Activates/deactivates any position constraint between the constrained object and the constraining object. For example, if you deselect this option, the constrained object does not change position.

Orientation

Activates/deactivates any rotation constraint between the constrained object and the constraining object.

Scaling

Activates/deactivates any scaling constraint between the constrained object and the constraining object.

Position Affected by Constraining Object?

Affected by Orientation

Changes the constrained object's position when the constraining object is rotated.

Affected by Scaling

Changes the object's position when the constraining object is scaled.
