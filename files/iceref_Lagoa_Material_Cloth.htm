Lagoa Material Cloth

Creates a cloth-like material using geometry that is created or the point cloud that is emitted. The elastic structure that is created is very tight with a high structure breaking point.

Plug this compound's Execute output into an Execute port of the Lagoa Phase node.

The parameters exposed for this compound are a subset of those in the Lagoa Main Material compound, mainly in the Pressure, Inelastic, and Elasticity groups.
