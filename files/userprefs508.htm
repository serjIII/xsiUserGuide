Install Add-On Dialog Box

Installs a .xsiaddon file containing various customizations such as scripts, presets, custom commands, toolbars, layouts, and so on.

To display: Choose File Add-Ons Install.

Filename

The name and location of a .xsiaddon file. Type a path or use the Browse button.

Items to Install

Type

The kind of element (for example, layout, preset, or library) you want to install from the add-on. A list of elements matching the type you selected appears in the Items box. You can install elements of different types at the same time.

Items

The elements of the selected type available in the add-on to install. Check the corresponding box to include an element. By default, all items are checked.

In some cases you cannot select or deselect individual items that are dependents of another item. For example, if you select a preset for installation, you will not be able to deselect any of the SPDL files that are associated with it.

Select All

Select all elements of the selected type for installation.

Select None

Deselect all elements of the selected type.

Destination

Select the desired Root Location from the list.

Advanced

Click to display the Add-On Information dialog box, which may contain additional information provided by the add-on author. See Add-On Information Dialog Box.

Install

Installs the selected Items to the specified Location.
