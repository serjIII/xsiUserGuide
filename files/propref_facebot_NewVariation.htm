New Variation Dialog Box

Create a new variation of the viseme library used for lip syncing in Face Robot.

To display:

On the Act panel in the Face Robot layout, click the Library > LipSync tab and choose Setup Visemes. Then in the Visemes Dialog Box, click the Variation New button.

For more information, see Creating Variations of the Visemes [Face Robot].

Name

The name of a new variation of visemes to be added to list of Viseme variations in the Visemes Dialog Box.

For example, if you enter "yelling" as the Variation name, the viseme names will have this as a suffix (such as 0x026a0061_I_yelling) when you import them.

Based On

The name of a viseme library upon which this new variation is to be based.
