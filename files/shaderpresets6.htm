Fast Lens Effects

DEPRECATED. This shader is unsupported, but it may still get installed with Softimage to provide compatibility with older scenes that use it. It is recommended that you replace unsupported shaders in your scenes with equivalent shaders from the current Softimage shader library.

| Lights | Flares | Render Tree Usage

Category: Lens

Shader Type: Lens

Output: Color (RGB) value

Creates a quick lens effects or enhances other effects such as a lens flare.

NOTE:This shader can only work with spotlights and point lights. It has no effect on an infinite light.

Name

The shader's name. Enter any name you like, or leave the default.

Lights

Light List

Creates a light list to specifiy which lights should produce the effect.

Flares

Render Flare Only

Renders only the flare on a black background. Saves rendering time and makes image composition easier later on.

Safe Overlay

Takes the underlying color into account and tries to ensure that the result does not exceed the maximum color intensity. If both the rendered flares and the underlying scene do not exceed this intensity, the result is guaranteed not to exceed either.

Saturation Multiplier

Controls the intensity of the flare colors.

At 1, the flare changes to a gray scale. Very useful for defining the "dazzle" of a flare.

Brightness

Defines the scene's luminance.

Lens Guard Margin

Defines a rectangle around the view plane with the width equal to the margin. The flare intensity falls off linearly within this region. This parameter blocks out direct rays from light sources outside the view angle. In screen space, this means that light(s) outside the view area should not cause lens effects. In reality, a lens guard isn't perfect, and when a light is nearly within view it causes lens effects.

Focus Pull

Allows a lens flare to be animated. The effect's individual components rotate and translate to simulate the realignment of the camera optics. You can also use this parameter to select a specific configuration in the flare pattern.

Render Tree Usage

This shader can be applied directly to the camera in the lens shader stack.
