Lagoa Advect Points

Uses one cached point cloud to drive a second point cloud to make bubbles, debris, and so on, in a fluid simulation.

This compound should be applied to a point cloud that is reading from cache files.

To apply: See Applying Cached Post Sim Effects [Lagoa Multiphysics Simulations].

Tasks: Lagoa/Cached Sim Post Effect

Output Ports: Execute

Remove Outside Polygonizer

Removes points that are outside of the mesh connected to the Polygonized Cloud port.

Polygonized Cloud

The mesh outside of which points are removed.

To avoid a dependency cycle, this should be polygonized mesh generated from a different cloud than the cloud that this compound is applied to. Both clouds can read from the same cache files.

Random Delete Bubbles

Deletes points from the fluid simulation so that only random bubbles or debris particles remain.

Random Size

Randomizes the size of particles.

Size Base Value

The base value for size, when Random Size is on.

Size Variance

The variance used for size, when Random Size is on.

Size Turbulence Scale

The scale of the noise field used to randomize particle size when Random Size is on.

Remove Bubbles

The proportion of particles to delete when Random Delete Bubbles is on.

Random Placement

Jitters the position of particles.

Place Variance

The variance used for jittering particle positions, when Random Placement is on.

Place Turbulence Scale

The scale of the noise field used for jittering particle positions, when Random Placement is on.

Place Animated

Whether the noise field used for jittering particle positions is animated or not, when Random Placement is on.

Place Animation Speed

The animation speed of the noise field used for jittering particle positions, when Place Animated is on.

Stretch

Clones points along their velocity to give the appearance of being stretched.

True Ratio

The proportion of points to clone, when Stretch is on. Note that this value is relative to the original number of points, so it must be higher than Remove Bubbles to have any effect.

Distance to Main Bubble

Adjusts the distance of cloned points from the original points, when Stretch is on.

Drop Formation

Scales down the size of cloned points to give the appearance of drop-like shapes, when Stretch is on.

Shape

The shape to use for points in the cloud.
