Get Goal Group Index

Returns the index of the goal group assigned to each actor in a CrowdFX simulation. For more about goal groups, see Creating Groups of Goals.

This compound can be used only in an ICE tree on the crowd simulation Point_Cloud object.

Tasks: Crowds/Goals, Crowds/Getters

Output Ports: Value
