Syflex Gravity

This compound applies a directional force that defines an acceleration over time. It moves simulated objects based on their mass and existing velocity.

This compound is used inside the Syflex Cloth and Syflex Curve compounds.

To apply: See Using the Syflex Gravity Force.

Tasks: Syflex/Forces

Output Ports: The Force output plugs into the Force ports on the syflexISimul node, which controls the Syflex cloth or curve simulation.

Gravity

Defines the XYZ direction of the gravity and its strength.

The strength of the gravity should be set in accordance with the Softimage unit scale and frame rate used in the scene. See Using the Syflex Gravity Force for information on how to calculate this value.
