DXShadowReceiver

DEPRECATED. This shader is unsupported, but it may still get installed with Softimage to provide compatibility with older scenes that use it. It is recommended that you replace unsupported shaders in your scenes with equivalent shaders from the current Softimage shader library.

Shader Type: RealTime

Output: RealTime

NOTE:This shader is no longer supported, and scenes using it will not display correctly.

This shader is designed to author content for DirectX. It is used to render shadows on one or more shadow receiving objects. It is meant to be used with the DXShadowMap shader, which should be applied to the shadow casting object.

ShadowBuffer

The frame buffer to which shadows are rendered. This value should match the ShadowBuffer value in the DXShadowMap shader applied to the shadow casting object.

TextureStage

Defines the texture stage to which the received shadows are rendered.
