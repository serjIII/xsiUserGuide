Perlin Turbulences

This compound uses three Perlin noise values giving three scalars that are output as an array of 3D vectors. This compound is generally useful only with the Curl-Noise compound (it's also used within that compound).

Plug this compound's psi_i output into any port that accepts vectors. The most likely port in which to plug this is the Potential port of the Curl-Noise Framework compound.

Tasks: Particles/Curl-Noise

Output Ports: PSI_i

The parameters here are the same as in the Perlin Noise Base Layer group in the Curl-Noise compound.
