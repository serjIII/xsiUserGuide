Extruding Polygons in ICE

There are several nodes and compounds for extruding polygons.

  • Extrude Polygon Island is a basic node that provides some of the options of the non-ICE operator.
  • Apply Extrude Polygon Along Axis is a compound that contains the Extrude Polygon Island node and provides more options for specifying polygon indices and transforming the extruded polygons.
  • Create Extrusion Along Strands is a compound that creates a new topology based on the strands of an input point cloud.

Using the Extrude Polygon Island Node

Use the Extrude Polygon Island node to modify a topology value before setting the Topology attribute.

Using the Apply Extrude Along Axis Compound

The Apply Extrude Polygon Along Axis compound sets the Topology attribute for you, so it can be connected directly to the ICETree root for simple extrusions. You can use the Filter Mode options to specify the polygons to extrude in any one of a variety of ways, and also choose whether the transform is applied per subdivision or just once along the entire extrusion.

Creating an Extrusion Along Strands

You can generate a mesh topology by extruding an outline along the strands of a particle cloud using the Create Extrusion Along Strands compound. There are two ways do this:

  • Using a single tree.
  • Using two trees: one to create the geometry and another to deform it along the strands. This lets you freeze the first tree once you are satisfied with the topology, while keeping the deformation active.

TIP:The StrandToMesh_ExtrusionID attribute is useful for identifying and distinguishing between vertices that belong to extrusions on different strands. This allows you to process them further in your tree.

Extruding Along Strands with One Tree

  1. Create a point cloud with strands. The Create Extrusion Along Strands compound requires that all StrandPosition arrays have the same size, so you can use Create Strands but not Generate Strand Trails. For more information see Creating Strands.

    The Create Extrusion Along Strands compound uses the StrandSize attribute control the width of the extrusions, or if that attribute does not exist then it uses the base particles' Size. Make sure that at least one of those attributes exists and is not zero.

    There will be one subdivision in the extrusion per strand segment, so adjust the number of strand segments accordingly.

  2. If you want to extrude a cross-section other than the default circle, create a simple polygon mesh to extrude. For example, you can draw a one-polygon mesh in the top view using the Add/Edit Polygon tool (N key) â€” see Adding and Editing Polygons (Drawing).

    Draw the polygon around the scene origin so that its points are not too offset from its object center. The center is the point that is extruded along each strand.

  3. Get an empty polygon mesh and create a non-simulated ICE tree.

  4. Connect Create Extrusion Along Strands to the ICETree root, and specify the point cloud.

  5. If you are using a custom mesh for the shape of the cross-section, set Base Shape Type to Custom and connect the name of the mesh to Custom Shape Name.

Extruding Along Strands with Two Trees

  1. Follow the steps above to extrude along strands with one tree.

  2. Turn on Mute Deformation.

  3. Create another ICE tree, and apply Deform Strand Extrusion using the same point cloud.

  4. Make sure that Remove Strand Position and Index are set to the same value in both Deform Strand Extrusion and Create Extrusion Along Strands.

  5. Use the other options in Deform Strand Extrusion to control the rotation along the length of the strand.
