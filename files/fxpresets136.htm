Grayscale

Allows you to take an RGB input image and output it as a grayscale Alpha image.

Requires an Input (RGB).

Produces an Output (A).

Grayscale

Red Weight

Sets the degree to which the red channel contributes to the output image.

Green Weight

Sets the degree to which the green channel contributes to the output image.

Blue Weight

Sets the degree to which the blue channel contributes to the output image.

Output

Controls this operator's Output Options. Note that this operator uses limited output options.
