Locking Shapes

When you're working in the shape manager, you may want to lock certain shapes so that you don't modify or delete them, accidentally or otherwise. You can, however, duplicate a locked shape. When a shape is locked, the shape-animated object is also locked.

For general information on locking objects and parameters, see Locking and Tagging Scene Elements.

To lock shapes

Do either of the following:

  • Select a shape in the shape list and choose Edit Lock Add.

    or

  • Select one or more shapes in the shape list, then right-click one of the shapes and choose Lock Add.

    In the shape list, you can see that each locked shape has a (Locked) label appended to its name.

To unlock shapes

Do either of the following:

  • Select a locked shape and choose Edit Lock Remove.

    or

  • Select one or more locked shapes in the shape list, then right-click one of the shapes and choose Lock Remove.

To lock the shape weight's animation

  • On the Animate tab, right-click on a shape weight slider's animation icon and choose Locks Animation Locked. This prevents the shape weight slider's values from being keyed.
