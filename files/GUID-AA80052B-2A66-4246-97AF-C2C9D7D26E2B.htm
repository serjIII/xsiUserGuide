Get Topology

Returns the Topology attribute of the specified polygon mesh object.

Categories: Topology

Tasks: Topology/Getters

Output Ports: Topology

In Name

Connect the name of a polygon mesh object, or a prefix to use with Reference.

Reference

Specify a polygon mesh object. If anything is connected to In Name, it gets prefixed to the name specified here.

Transform Mode

  • No Transform performs to transformation on the input point positions.
  • From Topo Input transforms the point positions by the input's global kinematic values.
  • Use Other Object transforms the point positions by the global kinematic values of the object specified below.

In Null Name

Connect the name of a null or other object to use to transform the point positions.
