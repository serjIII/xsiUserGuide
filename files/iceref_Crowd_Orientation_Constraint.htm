Crowd Orientation Constraint

Sets the orientation of a deformer in a proxy rig in a CrowdFX simulation to a specific rotation.

This compound can be used in the Post-Simulation region of the Point_Cloud object before Set Shape Instance Bones, or in the Simulation region after Set Locomotion.

For more information, see Constraining the Orientation or Scale.

Tasks: Crowds/Constraints

Output Ports: Execute

Proxy Deformer Name

The name of the deformer in the actor's Rig_Proxy hierarchy to constrain.

Child Transform Compensation

Applies the corresponding transformation to the immediate children of the deformer.

Rotation

The new orientation.

Blend

Interpolates between the simulated and new orientation values, where 0.0 gives the simulated value and 1.0 gives the new value.

TIP:If the simulation includes multiple actor sources, you can plug multiple instances of this node into a Select Case node with its Condition port driven by Get Actor ID. This allows you to use different values for different actor types.
