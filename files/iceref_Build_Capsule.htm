Build Capsule

Builds an capsule shape between the specified positions. To use, connect the output into the Point Array port of Display Line. To transform the capsule's position, modify the output first, for example, by adding an offset or multiplying by a matrix.

Category: Debugging

Output Ports: Result1

Num Segments

The smoothness of the end caps.

Radius

The width of the capsule.

Height

The length of the capsule.
