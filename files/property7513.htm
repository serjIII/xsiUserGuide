Light Rig Builder Options Dialog Box/Property Editor

Light rigs are groups of infinite lights, parented to a null, whose direction, color, and intensity are calculated by analyzing the luminance of a .hdr or .pic image.

  • The options described below appear in the Light Rig Builder Options dialog box, which opens when you create a new light rig, in which case they are used to configure the new rig.

  • When you open the Light Rig Builder Options property editor, it displays the options used to create the most recent light rig in the scene.

Image File

Specifies the source image file used to create the light rig.

Number of Layers

Specifies the number of layers of luminance extracted from the source image.

Layer Luminance Extraction

Specifies the amount of luminance to be extracted from each layer in order to create lights.

Light Distance

Specifies the distance between lights in image space.
