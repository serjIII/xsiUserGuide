Freehand/Bezier Shapes

Point Shape | Wireframe Color

These options allow you to control individual Freehand or Bezier garbage matte shapes that you created using the tools in an operator's Mask options. You can adjust the shape's dimensions, position, center of rotation and angle of rotation. You can also invert a shape.

Point Shape

Name

The shape's name. You can change it.

Move

Pos X, Y

Sets the position of the shape in X and Y. values of 0 and 0 will place the shape in its original positions Positive or negative values create positive or negative offsets relative to that position.

Scale

Scl X, Y

Applies a scaling factor to the shape in X and Y. Values of 1 and 1 keep the shape at its original size, while values of 2 and 2 cause it to double in size, and so on.

Pin

Pin X, Y

Adjusts the position of the pin marker, which is the shape's rotational center, in X and Y. The shape's position may change to compensate.

Rotate Angle

Adjusting this value rotates the shape about its pin marker. Values range from -180 degrees to 180 degrees.

Invert Color

Inverts the masking properties of the shape. If the shape was hiding an effect, clicking this button will reveal the effect within the shape's boundaries, and vice-versa.

Wireframe Color

Use Wireframe Color

Uses the wireframe colors specified here instead of the defaults.

Unselected Color

The RGB values of the shape wirerframe when it is unselected.

Selected Color

The RGB values of the shape wirerframe when it is selected.
