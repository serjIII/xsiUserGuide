Deinterlace

Deinterlaces the fields of a sequence. The dominant field of the first frame is interpolated or replicated into the other field, producing frame 1, then the other field of the first frame is interpolated or replicated into the dominant field, producing frame 2. This is repeated for each frame, producing a sequence of twice the original length.

Requires an Input (Any).

Produces an Output (Any).

Deinterlace

Source Clip Range

Specifies the frames in the source to be used as Start and End frames when retiming.

Filter Type

Specifies whether additional frames created by the deinterlace operation are interpolated or replicated.

Field Dominance

Sets the dominant field. Select Odd or Even.
