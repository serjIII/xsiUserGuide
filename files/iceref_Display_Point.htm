Display Point

Displays points at the specified positions. The points are displayed if the output of the Result port is pulled, or alternatively you can connect the Execute output port.

Category: Debugging

Output Ports: Result, Execute

Display

Toggles the display on and off.

Select Color

Selects a predefined color.

Point

Connect a position in the object's local reference frame.
