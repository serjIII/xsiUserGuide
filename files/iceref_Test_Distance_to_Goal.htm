Test Distance to Goal

This compound uses the distance to an object defined as a particle goal as a trigger for an effect.

Plug its Result output into a Trigger port on a State compound or in a Condition port of an If node.

For more information on states, see ICE Particle States [ICE Particle Simulations]; for more information on goals, see ICE Particle Goals [ICE Particle Simulations].

Tasks: Particles/Conditionals

Output Ports: Result

Test Distance

The distance in Softimage units to the goal object.

Type

Defines the condition for the Test Distance value to be triggered: Equal To; Not Equal To; Less Than; Less than or Equal to; Greater Than; or Greater Than or Equal To.

Epsilon

Widens the range of acceptable values by a small amount so as to reduce rounding errors.
