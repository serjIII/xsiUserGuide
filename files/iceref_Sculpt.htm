Sculpt

The ICE-based Sculpt deformation uses a weight map to move points along a their normals or another vector.

To apply: See Sculpt [ICE Deformations].

Tasks: Deformation/Effects

Output Ports: Result

Mute

Toggles the deformation on and off without affecting the values of other parameters.

Sculpt Factor

How far points are moved. This value is multiplied by the weight map's value at each point.

Mode

Select the vector along which to move points:

  • Normal uses the points' normals.

  • X, Y, and Z use the objects local axes.

  • Custom allows you to use the Custom Direction option below.

Custom Direction

The vector along which to move points when Mode is "Custom", expressed in the object's local reference frame.

Reference

The weight map to use.

Smooth

Iterations

The number of times to smooth the weight map's values.

Smooth Factor

The amount of smoothing to apply on each iteration.
