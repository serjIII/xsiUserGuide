Selecting ShotClips

When you select one or more ShotClips, all appropriate commands are applied to them, such as copying, scaling, moving, and so on. Selected ShotClips have a white border.

The Select Tools

There are two tools in the Camera Sequencer Editor for selecting ShotClips:

  • Select Tool
  • Rectangle Select Tool

By default, the Select Tool is active, which makes it easy to immediately select and move ShotClips around after they are created.

To activate the Select Tool

  • Press the Space bar or choose Edit Select Tool from the menu bar.

When over a ShotClip, the cursor looks like a double-headed arrow when the Select Tool is active.

NOTE:The double-headed arrow appears only when the cursor is on ShotClips. On empty Shot Tracks, the appearance of the cursor changes to single arrow.

You can switch to the Rectangle Select Tool to make sure the clips do not move at all.

To activate the Rectangle Select Tool

  • Press the Q key or choose Edit Rectangle Select Tool from the menu bar.

The cursor looks like a single arrow when the Rectangle Select Tool is active.

To select ShotClips using either the Select Tool or Rectangle Select Tool, you can do any of the following

  • Click a ShotClip.
  • Shift+click to select multiple ShotClips.
  • Ctrl+click to toggle-select multiple ShotClips.
  • Drag over any part of one or more ShotClips to select them.
    NOTE:If the Select Tool is active, start dragging in an empty area of the Shot Track so that you do not select and move the ShotClips as you drag.
