Add/Remove Columns Dialog Box

Modifies the columns displayed in the browser. See The Browser.

To display:

Right-click in an empty area of the browser's Content pane.

Available Fields

Lists the columns that you can display. Click to select a column, or double-click to move it to the other list.

Displayed Fields

Lists the columns to display. Click to select a column, or double-click to move it to the other list.

Add All

Moves all the columns to Displayed Fields.

Add

Moves the selected column to Displayed Fields.

Remove All

Removes all columns except Name from Displayed Fields.

Remove

Removes the selected column from Displayed Fields. You cannot remove Name.

Move Up

Moves the selected column one up in the Displayed Fields list. This changes the order in which columns are displayed in the browser.

Move Down

Moves the selected column one down in the Displayed Fields list. This changes the order in which columns are displayed in the browser.
