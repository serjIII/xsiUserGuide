Shape

The shape of particles.

This attribute is used for rendering and display.

This attribute is set by the Emitters compounds, and is exposed in their property editors as the Shape List parameter.

It is also used by the Simulate Rigid Bodies node for determining collisions.

You can quickly get and set the Shape attribute with the Get Particle Shape and Set Particle Shape compounds.

Data Type

shape

Belongs To

points

Geometry Types

predefined
