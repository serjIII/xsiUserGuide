Creating a Toon Render Pass

You can create a toon render pass as you would any other rendering pass. Creating a toon pass ignores any shaders that you've attached to your scene objects and shades them with Toon Paint and Host shaders.

NOTE:If you want to add inking to your toon pass, you must add a Toon Ink Lens shader to the pass camera's lens shader stack. Open the pass' property editor (Pass Edit Current Pass) and open the Lens Shaders tab of the Camera property editor.

To create a toon pass

  • From the Render toolbar, choose Pass Edit New Pass Toon.

    A new toon pass is created and a Toon Paint and Host shader is applied to your scene objects.
