Setting the Output Directory

Location:

  • The Render Manager > Scene Render Options

  • Choose Render > Render > Scene Options from the Render toolbar.

  • Tab: Scene Globals [Properties Reference]

  • Options: Scene Output

The Scene Output options allow you to set the default output directory for the rendered sequence. Each pass' file name is appended to the scene output path to produce a fully resolved output path for the rendered sequence.

For information about setting a pass' path and file name, see Setting Pass Output Paths and File Names.

To set the output directory

  • Build the output directory path using Output Path Templates (see Tokens and Templates).

    or

  • Enter an explicit path in the text box. Use the Browse (...) button to choose a location.
