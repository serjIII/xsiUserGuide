Stroke Shape Property Editor

| Shape | Mask | Transform | Display

To Display: Select a stroke shape in the FX Viewer or in the explorer and press Enter.

Allows you to edit the properties of existing strokes painted in a vector paint operator with the paint brush tool.

When you open this property editor for a given stroke shape, an instance of the Brush Property Editor, which you can use to set the stroke's properties, is appended to this property editor.

Shape

Opacity

Controls the opacity level of the paint stroke. The higher the value, the more opaque the stroke.

Color

Defines the stroke's color. Use the sliders to set any RGBA value.

Mask

Paint To

RGB

When active, the stroke is painted on the image's RGB channels.

Alpha

When active, the stroke is painted on the image's alpha channel.

Obey Paint Mask

The paint mask is a matte or grayscale image that defines which areas of an image are paintable, and which areas aren't (sort of like an obey matte for painting). You can set any operator as the paint mask from the operator's contextual menu.

Enable

When active, parts of the image that correspond to black parts of the mask are protected from the stroke.

Invert

When the Obey Paint Mask > Enable option is activated, activating this option inverts the mask so that parts of the image that correspond to white parts of the mask are protected from the stroke.

Transform

Move

X/Y

Translates the stroke from its initial position in X and/or Y. The values are in pixels.

Scale

X/Y

Scales the stroke in X and/or Y.

  • If the Use Pivot option is active, scaling is relative to the pivot point.

  • If the Use Pivot option is not active, scaling is relative to the center of the stroke's bounding box.

Pivot

Use Pivot

Activates the pivot for scaling and rotation. When the pivot is active, all changes to the stroke's scale and rotation are performed relative to the pivot point.

X/Y

Translates the pivot from its initial position at the center of the stroke's bounding box.

Rotate

Angle

Specifies, in degrees, the stroke's rotation.

  • If the Use Pivot option is active, rotation is relative to the pivot point.

  • If the Use Pivot option is not active, rotation is relative to the center of the stroke's bounding box.

Display

Name

Displays the shape's name in the explorer. You can change it by typing a new name in the text box.

Visible

Controls the shape's visibility. Disabling this option mutes the stroke in the FX Viewer.
