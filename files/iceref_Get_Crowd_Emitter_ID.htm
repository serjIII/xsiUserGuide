Get Crowd Emitter ID

Returns the index of each actors' emitter in a CrowdFX simulation. The emitter's index can be set to a unique value in Set Emitter Data.

For more information, see Crowd Particle Emitters.

Tasks: Crowds/Getters

Output Ports: Value
