Randomize Op Property Editor

Randomly displace points.

To apply: Select one or more objects, clusters, or points and choose Deform Randomize from the Model or Animate toolbar.

To redisplay: Select the deformed object then choose Edit Properties Modeling Properties and click the Randomize Op tab.

Mute

Toggles the deformation on and off without affecting the values of other parameters.

Displacement

X, Y, Z

The maximum displacement of points along each axis in SOFTIMAGE units.

Repetitions

The number of times the randomize calculation is iterated.

Random Source

Type

Choose one of the following:

  • Seed. A seed is used to generate a pseudo-random number. Although this is not truly random, it can be reproduced in case you need to re-create the exact same effect. This option uses the Seed parameter.

  • Pure random. The effect is not predictable and cannot be reproduced exactly.

Seed

A value used to generate a pseudo-random number if Type is Seed. Different values produce different numbers, but the same value always produces the same number.
