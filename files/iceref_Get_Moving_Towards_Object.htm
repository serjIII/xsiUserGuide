Get Moving Towards Object

This compound gets the distance from the current particle position to an object and the particle's velocity. You can output the Resulting scalar value.

Tasks: Particles/Getters

Output Ports: Result

Global Pos

The global position of an object to which the particles move toward. Set the XYZ values in global space here or plug in 3D vector data from another node to get that information.
