AngularVelocity

The angular velocity of a point.

The Simulate Rigid Bodies node uses this attribute to update Orientation, and sets it based on collisions. It is also used to render rotational motion blur for instanced shapes.

Data Type

rotation

Belongs To

points

Geometry Types

  • point cloud

  • polygon mesh

  • NURBS surface

  • NURBS curve

  • lattice

Attribute Type

predefined

R/W or RO:

read/write
