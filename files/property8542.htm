Tangent Property Editor

A Tangent property is created on the selected polygon mesh object. It uses as input a color at vertices (CAV) property where the tangents are stored and a texture projection property.

To apply: choose Get Property Tangent from any toolbar.

To redisplay: Select the object, click the Selection button on the Select panel, and then click the Tangent icon located under the associated object node.

UVs

Specifies the texture projection used to compute the tangents. To use a different projection, click the Pick button and pick the projection from an explorer.

Tangents

Specifies the color at vertices (CAV) property that stores the tangents. To use a different CAV property (or to connect to a renamed CAV property), click the Pick button and pick the CAV property from an explorer.
