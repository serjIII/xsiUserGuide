Cluster Property Editor

To create a cluster, select some components such as points or polygons then click the Cluster button on the Edit panel.

To redisplay: Select the cluster and press Enter.

Name

The name of the cluster as it appears in the explorer. If you change this value, the new name appears on the property editor tabs the next time you display these parameters.

Cluster Display Color

Red, Green, Blue

Sets the display color of the cluster's components when the cluster is not selected. If a component belongs to more than one cluster, it uses the color of the last cluster created.

These controls have no effect on automatic envelope clusters. To change the display colors of automatic envelope clusters, use the weight editor or the weight paint panel. See Modifying Envelope Weights [Character Animation].

sort_order

When clusters with local materials overlap, this value controls the order in which they're drawn. The lower a cluster's sort_order value, the earlier it's drawn. Thus the cluster with the highest sort_order value is always drawn on top of any other overlapping clusters.

When you open scenes created using a version of Softimage where sort_order was not assigned to clusters, all of the clusters in the scene are assigned a sort_order value based on the order in which they were loaded from the file.
