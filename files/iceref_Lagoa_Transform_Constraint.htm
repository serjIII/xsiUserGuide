Lagoa Transform Constraint

Locks points to nulls so that the points are pinned during a Lagoa simulation or deformation.

To apply: See Pinning Regions with Lagoa Transform Constraint [Lagoa Multiphysics Simulations].

Tasks: Lagoa/Constraints

Output Ports: Execute

Enable

Enables this compound.

In Name Null

The name of the null or group of nulls to pin points to.

Display Attachment Position

Displays the attachment positions made between the elements and the null for the purpose of testing and debugging.

Time Setup

Update Mode

Controls when points get bound to the constraining nulls:

  • Once at Frame binds points to the constraining nulls on the frame specified by Set Constraint at Frame. Other points that come near the constraining nulls later in the simulation are unaffected.

  • Every Frame binds all points that come near the constraining nulls during the simulation.

Set Constraint at Frame

The frame at which to bind points to the nulls, when Once at Frame is set to Once at Frame.

Global Time

If this option is on, Set Constraint at Frame is relative to the scene's timeline. If it is off, the frame is relative to the simulation range set in the Simulation Time Control Property Editor.

Attachment

Search Type

Controls which points are bound to the constraining null:

  • Radial binds points that are closer than the null's size multiplied by Radius Multiplier.

  • Null Shape binds points that are inside the null's primary icon.

Radius Multiplier

A multiplier for the null's size when Search Type is Radial.

Tension

Controls the amount of stretching near constrained points. Higher values result in less stretching.

Consider Scale

A Boolean value that allows the transform constraint to consider the scaling of the In Name Null to be a deformation of the initial state. If this is True, then scaling the null deforms the constrained points, as opposed to affecting which points get constrained (when Update Mode is Every Frame).
