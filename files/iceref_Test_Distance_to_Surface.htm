Test Distance to Surface

This compound uses the distance to an object as a trigger for an effect.

Plug its Result output into a Trigger port on a State compound or in a Condition port of an If node.

For more information on states, see ICE Particle States [ICE Particle Simulations].

Tasks: Particles/Conditionals

Output Ports: Result, Location

Surface1

The object whose distance from the particles you want to use as a trigger. This must be an object with a surface. Plug the object's Value output into this port.

Enable Cutoff Distance

Toggles the activeness of the Cutoff Distance option.

Cutoff Distance

The maximum distance (in Softimage units) in which the particle considers the object.

Test Distance

The distance in Softimage units from a particle to the object.

Type

Defines the condition for the Test Distance value to be triggered: Equal To; Not Equal To; Less Than; Less than or Equal to; Greater Than; or Greater Than or Equal To.

Epsilon

Widens the range of acceptable values by a small amount so as to reduce rounding errors.
