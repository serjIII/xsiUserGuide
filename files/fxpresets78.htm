Mosaic

The Mosaic effect renders the image as if it was composed of small squares or tiles laid down on a flat surface. The tiles appear to be lit from the side, enhancing their texture.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

Tile Size

Set lower values to make each tile smaller, or set higher values to make the tiles larger. Tile Size is measured in pixels.

Grout Width

Set lower values to decrease the amount of space or "grout" between tiles, or set higher values to increase the amount of grout. Grout Width is measured in pixels.

Lighten Grout

Set lower values to darken the grout between tiles, or set higher values to lighten the grout.
