Vector2 To Vector4

Category: Processing > Conversion

Shader Family: Texture

Output: 4D Vector

Converts 2D vector input to 4D vector output.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Specifies the 2D vector input to be converted to 4D vector output.

Render Tree Usage

This shader is simply placed between 2D vector output and 4D vector input in order for the values to connect.
