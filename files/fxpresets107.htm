Paint Daubs

The Paint Daubs effect can render an image in a wide variety of paintbrush styles. A range of brush sizes, from large to small, can be specified to smooth and simplify images, or to capture and accentuate fine details.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

Brush Size

Set higher values to make the brush larger, or set lower values to make it smaller.

Brush Type

Click Brush Type and select an option from the drop-down menu.

Sharpness

Set higher values to make the image sharper, or set lower values to make it smoother.
