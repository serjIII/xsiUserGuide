Units Preferences

To display: Choose File Preferences and click Units.

Color Space

Specifies the default color model used by color controls when a property editor is opened.

  • RGB selects the Red-Green-Blue color space.

  • HLS selects the Hue-Lightness-Saturation color space.

  • HSV selects the Hue-Saturation-Value color space (also known as HSB, or Hue-Saturation-Brightness).

No matter which space you select as the default, you can always cycle through the available options by clicking on the button below the color swatch in any individual color control.
