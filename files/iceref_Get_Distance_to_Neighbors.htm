Get Distance to Neighbors

This compound gets the distance from the current particle position to its closest neighboring particles.

Tasks: Particles/Getters

Output Ports: Average_Distance

Enable Cutoff Distance

Toggles the activeness of the Cutoff Distance option.

Cutoff Distance

The maximum distance (in Softimage units) within which the particles consider each other.

Enable Max Number Points

Toggles the activeness of the Max Number Points option.

Max Number Points

The maximum number of neighboring particles that the each particle considers.
