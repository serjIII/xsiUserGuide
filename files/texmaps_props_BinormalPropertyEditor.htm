Binormal Property Editor

A Binormal property is created on the selected polygon mesh object. It uses as input a color at vertices (CAV) property where the binormals are stored and a texture projection property.

To apply: choose Get Property Binormal from any toolbar.

To redisplay: Select the object, click the Selection button on the Select panel, and then click the Binormal icon located under the associated object node.

UVs

Specifies the texture projection used to compute the binormals. To use a different projection, click the Pick button and pick the projection from an explorer.

Binormals

Specifies the color at vertices (CAV) property that stores the binormals. To use a different CAV property (or to connect to a renamed CAV property), click the Pick button and pick the CAV property from an explorer.
