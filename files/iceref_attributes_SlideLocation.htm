SlideLocation

Determines where on an input object the particles will slide.

It is used internally by the Slide on Surface compound.

You can quickly get the SlideLocation attribute with the Get Particle Slide Location compound.

Data Type

location

Belongs To

points

Geometry Types

  • point cloud

Attribute Type

custom

R/W or RO:

read/write
