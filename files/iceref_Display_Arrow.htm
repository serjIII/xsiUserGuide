Display Arrow

Displays an arrow shape between the specified positions. The shapes are displayed if the output of the Result port is pulled, or alternatively you can connect the Execute output port.

Category: Debugging

Output Ports: Result, Execute

Display

Toggles the display on and off.

Select Color

Selects a predefined color.

Point Array

Connect an array containing the start and end positions of the arrow in the object's local reference frame.

Make Line Thick

Draws extra lines to make the arrow shaft more noticeable. Make sure that Thickness is not too high for the length of the arrow or the arrow shape will degenerate.

Thickness

The distance between the extra lines that are drawn when Make Line Thick is on.

Arrowhead Size

The relative size and width of the arrowhead.

View Direction

The direction that the flat side of the arrowhead is oriented toward.
