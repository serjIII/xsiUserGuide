Merge Polygon Data

This node should be plugged into the Execute on Merge port of the Merge Polygon Meshes compound. For more information, see Merging Topologies in ICE.

Tasks: Topology/Copy and Merge

Output Ports: Execute_on_Merge

Reference

Specify an attribute name to store the merged data on "self".

Value

A data set to merge. Connect additional data sets into New (Value). Use data sets from the same meshes that are plugged into the Merge Polygon Meshes compound, and connect them in the same order.
