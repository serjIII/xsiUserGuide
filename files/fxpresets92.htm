Palette Knife

This effect renders an image with rough smears of color that appear to have been made with a palette knife. Although the patches of color are irregular in size and shape, the result is strongly representational.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

Stroke Size

Set higher values to increase the size of the knife strokes, or set lower values to decrease the stroke size.

Stroke Detail

Set higher values to make the knife strokes more detailed, or set lower values for less detail.

Softness

Set higher values for a softer image with less jagged edges, or set lower values for a sharper and more jagged result.
