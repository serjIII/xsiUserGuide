Tint

| Tint | Mask | Output

Tints the image with a selected color. You can control the opacity of the

tinted image over the original image.

Requires an Input (RGB or RGBA).

Produces an Output (RGB or RGBA).

Tint

Tint Color

Sets the color that the image will be tinted with. Select the required color from the color chooser, or pick it from the Input.

Tint Mode

Sets the color model that will be used to calculate the tinting. Select HSV or HLS. Using HLS retains the white levels from the original image.

Mix Level

Sets the opacity of the tinted image over the original image. Set a value between 0 and 1 using the slider.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
