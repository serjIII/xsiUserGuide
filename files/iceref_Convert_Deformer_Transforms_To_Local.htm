Convert Deformer Transforms To Local

Convert the local transforms of deformers into local space.

Global Transforms

Array of 4x4 matrices representing the global transform of the deformers of actors.

The input should come from the CrowdFX_Actor_Deformers ICE attribute or from the Get Self Pose node.

Root Index

Index of the root deformer in the 4x4 matrix array. The input should come from the Get Actor Rig Hierarchy compound.

Parent Indices

Array of Integer representing the indices of parent deformers. The input should come from the Get Actor Rig Hierarchy compound.
