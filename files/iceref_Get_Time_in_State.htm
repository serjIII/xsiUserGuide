Get Time in State

This compound gets the amount of time that the particle spends in a particular state. This is used in conjunction with the Start Timer and Init Timers compounds. You can output the Time in Seconds as a scalar value.

For more information on states, see ICE Particle States; for information on timers, see Setting Particle Timers.

Tasks: Particles/States

Output Ports: Time_in_Seconds
