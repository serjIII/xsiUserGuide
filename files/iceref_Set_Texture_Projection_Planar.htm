Set Texture Projection Planar

Calculates the UV coordinates of a planar projection, and stores it as a per-polynode 3D vector attribute on the "self" object that can be used as a texture projection when rendering.

Tasks: Topology/Setters

Output Ports: Execute

Mode

How to define the plane of projection:

  • Use Null uses the position and +Y direction of a null or other object.
  • Use Custom Plane uses the Custom Plane Settings.

In Null Name

Connect the name of a null or other object to use when Mode is set to Use Null.

Custom Plane Settings

Origin

The position of the plane

Normal

The normal of the plane

Scale U, Scale V

The size in Softimage units of one unit in texture space for each of the U and V directions..

Attribute Name

Reference

Specify the name of an attribute to store the 3D vector texture coordinates per polynode.
