Modifying Configuration Files with User Tools

You can use the User Tools application to edit Softimage configuration files and view the installation log from one common graphical interface. The User Tools application is installed by the Softimage Setup program. You can access it as a standalone application after you have installed the Softimage software.

To open the User Tools application on Windows

  • Choose Start Programs Autodesk[version of Softimage] > UserTools.

To open the User Tools application on Linux

  1. Open a terminal window.

  2. Source the Softimage environment by typing the following:

    source ~/.xsi_2013

  3. Start User Tools by typing the following:

    $XSI_BINDIR/UserTools

To modify configuration files

  1. In the following dialog box, select a file and click Edit File, or double-click the file name.

    User Tools always looks in the default location for a file. If the file is not located in the default location, a warning is displayed showing the default location and a browser opens to allow you to search for the file.

  2. Make the modifications to the file, then click Save and Quit.
