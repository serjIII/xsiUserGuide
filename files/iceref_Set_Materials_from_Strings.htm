Set Materials from Strings

Sets the "self" object's Materials attribute with the specified strings. You can specify which material in the Materials array to use using the MaterialID attribute per polygon. To add more strings, right-click on a port and choose one of the Insert commands, or connect a string to the New (Material Name) port.

Tasks: Topology/Setters

Output Ports: Execute
