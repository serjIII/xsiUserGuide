Choosing a Scene Renderer

Location:

  • The Render Manager > Scene Render Options

  • Choose Render > Render > Scene Options from the Render toolbar.

  • Tab: Scene Globals [Properties Reference]

  • Options: Scene Renderer

The Scene Renderer options allow you to choose whether your scene is rendered using mental ray or the Softimage Hardware Renderer. If you are using a third-party integrated renderer then its name will show up in the Scene Renderer drop-down list for you to select from as well.

You can also choose the renderer to be used to render each pass. By default, every pass is set to Use Scene Render Options. However, you may specify that either mental ray or the Hardware renderer be used for the pass, irrespective of the scene-level setting. See Setting the Pass Renderer.
