Lattice Property Editor (Deformation)

Lattices let you deform objects by moving points in the lattices' "scaffolding."

To apply: See Lattices [Data Exchange].

To redisplay: Select an object with a lattice deformation, click Selection on the MCP, and click the Lattice icon.

Mute

Toggles the deformation on and off without affecting the values of other parameters.

Deformation

Scope

Controls which points on the object are deformed:

  • Falloff Relative to Lattice's BBox. Only the points that are near the lattice's bounding box are affected. You can define a falloff so that points do not suddenly jump when they enter the lattice's bounding box.

  • All Points (SI3D). All the points on the object are deformed no matter where they are.

Amplitude

Controls the strength of the deformation.

Scaling

Mode

Controls how the object is deformed when the lattice object is scaled:

  • No Scaling. The deformed object is not scaled when the lattice object is scaled. However, the deformed object is still affected as the lattice's points move when scaled.

  • Treat Scaling as Deformation. The deformed object is scaled before the lattice deformation is applied.

  • Apply Scaling to Geometry (SI3D). The lattice deformation is applied first, then the deformed object is scaled.

Falloff (Scaling of Lattice's BBox)

Pre-deformed BBox

Affects points on the deformed object depending on whether or not they fall within the lattice's bounding box before the lattice was deformed.

Deformed BBox

Affects points on the deformed object depending on whether or not they fall within the lattice's actual bounding box, including any deformations applied to the lattice.

Start

Defines the start of the falloff, as a multiple of the lattice's bounding box. If this value is greater than one, the deformation starts to fall off outside the lattice's bounding box. If this value is between 0 and 1, the deformation starts to fall off within the lattice's bounding box.

End

Defines the end of the falloff, also as a multiple of the lattice's bounding box.

Falloff Profile

The shape of the falloff curve from the Start to the End.

You can edit the profile using the mouse and the same keyboard commands as the animation editor, or right-click to display a menu.
