Turbulence

Category: Texture Generators

Shader Family: Texture

Output: Scalar

Used with other texture generator tool shaders to produce a distorted texture. It creates a turbulent scalar pattern in a unit cube.

Name

The shader's name. Enter any name you like, or leave the default.

Coordinate

Defines the X, Y, and Z texture coordinates.

Mode

Anisotropic: Creates anisotropic noise.

Lines: Creates lines.

Cylinders: Creates cylinders.

Spacing

Defines the noise density.

Strength

Defines the distortion strength weight.

Power

Defines the degree of turbulence.

Iteration

Sets the number of turbulence passes. Maximum is 8.

Render Tree Usage

Use this shader to drive a parameter with a texture effect anywhere in a render tree. This shader generates a texture that can be used as an Intensity value, a Scale value, or to drive any scalar Surface shader parameter (IOR, Bumpmap Factor, etc.).

Connect a Texture Space Controller shader to drive the Coordinate value of this shader.
