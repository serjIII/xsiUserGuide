Lagoa Point Duplicator

Clones points and jitters their positions to create larger or more irregular shapes.

This compound should be applied to a point cloud that is reading from cache files.

To apply: See Applying Cached Post Sim Effects [Lagoa Multiphysics Simulations].

Tasks: Lagoa/Cached Sim Post Effect

Output Ports: Execute on Cached Cloud

Model

Controls how points are cloned:

  • Rough Points creates smaller clones resulting in a rougher combined shape.

  • Random Assemblies creates clones of the same size.

Iterations

Controls the number of cloned points.

Variance

Controls the variance of the cloned points' positions.

Relative Size

Controls the initial size of the cloned points in relation to its parent point.
