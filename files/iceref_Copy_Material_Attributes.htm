Copy Material Attributes

This node is used inside the Create Copies from Polygon Mesh and Clone Polygon Mesh compounds to transfer the Materials and MaterialID attributes based on the modulo of the polygon indices. It can be useful if you want to create your own compounds for copying topology.

Tasks: Topology/Copy and Merge

Output Ports: Execute

In Name

Connect the name of the source polygon mesh, or a prefix to use with Reference.

Reference

Specify the source polygon mesh. If anything is connected to In Name, it gets prefixed to the name specified here.
