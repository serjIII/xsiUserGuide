Ripple

The Ripple effect adds randomly spaced ripples to the image's surface, making the image appear as if it were under water which is being rippled by the wind.

For images composed of small elements, use lower settings for Ripple Size and Ripple Magnitude. For images composed of large elements, use higher settings.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

Ripple Size

Set lower values to make the ripples smaller, or set higher values to make the ripples larger.

Ripple Magnitude

Set lower values to give the ripples less distortion, or set higher values to give more distortion.
