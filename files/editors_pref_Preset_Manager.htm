Preset Manager Preferences

Stores the last-used settings for the preset manager. These values are used the next time the preset manager is opened.

To display: Choose File Preferences, expand Editors, and click Preset Manager.

Category Shelf

Library Mode

Controls whether the preset manager shows Tools or Tasks (when showing Operator Nodes).

Show

Controls whether the preset manager shows Materials, Operator Nodes (ICE nodes), or Shaders.

Preset Shelf

Thumbnail Size

Controls the size of thumbnails displayed in the preset manager.
