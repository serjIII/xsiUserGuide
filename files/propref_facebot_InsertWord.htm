Insert Word Dialog Box

Insert a word on the track in the lip sync view.

To display:

In the Lip Sync view in the Face Robot layout, right-click in an empty area in the speech track and choose Insert Word.

For more information, see Adding Words [Face Robot].

Word Text

The word you want to add in the empty space on the speech track.

Start Frame

Frame at which the new word starts.

End Frame

Frame at which the new word ends.

Make sure that this value is higher than the Start Frame value.

These frames are in global time, the same as in the scene's and lip sync view's timeline.

First Phoneme

The first phoneme that will be used for the new word.

This is the only phoneme that is used when you create the new word clip. To add more phonemes to the word, you need to split this one, and then switch those phonemes for ones you want â€” see Splitting Phonemes [Face Robot].

Ripple (Push Forward Other Words)

Pushes existing word clips that are to the right of the word insertion point further down (to the right) the speech track.

If you don't select this option, you may overwrite frames of other word clips that the new word overlaps on the speech track unless you make sure that the Start Frame and End Frame are between clips.
