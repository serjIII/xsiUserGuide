Current Simulation Sample

Returns the current simulation sample (subframe). This value is cumulative so if there are 10 samples per frame then at the start of the second frame of the simulation, the current simulation sample is 20. You can set Subframe Sampling in the Simulation Settings.

Categories: Data Access

Output Ports: Integer
