Text to Curve Converter Property Editor

Converts text to 3D curves. Text is defined using the Text Property Editor.

To apply: Choose any item from the Create Text menu on the Model toolbar.

To redisplay: Select the generated curves or polygon mesh, click the Selection button on the Select panel, then click the Text to Curve Converter icon (which may be under the Curve to Mesh icon).

Text vs. Font Size

Method

Controls how the text size in points is converted to Softimage units.

  • Ratio: Points per Softimage Unit. Sets the size of text based on a fixed ratio of points to Softimage units.

  • Fit: Text Height Defined in Softimage Units. Sets the absolute size of the text in Softimage units, and scales the different point sizes proportionally to fit. If the text includes multiple lines, all lines are made to fit within the defined height.

Ratio

The conversion factor of points to Softimage units when Method is Ratio.

Fit Size

The absolute size of text when Method is Fit.

Spacing

Character

Adjusts the spacing between individual characters. Positive values increase spacing and negative values decrease it.

Line

Adjusts the spacing between multiple lines of text. Positive values increase spacing and negative values decrease it.

Create Partition Clusters

Creates a separate subcurve cluster per character. This makes it easier, for example, to select subcurve clusters and extract new curve objects using Create Curve Extract from Subcurves.

The clusters are automatically named based on the character position and the character itself, as in Char_1_X, Char_2_S, Char_3_I, and so on.

If you change the text after creating partition clusters, they are not updated automatically. You must re-create the partition clusters again.

Explode Curves

Creates a hierarchy of curve objects from the text. There is one curve for each character, and all curves are parented under a common null.
