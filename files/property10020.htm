Hair Recomb Operator Property Editor

Grooms each guide hair along the emitter's surface in the approximate direction in which the hair is already going (using the root-to-tip vector).

To apply: Select at least one point on each of the guide curves you want to recomb, and then choose Modify Comb Recomb on the Hair panel. If you select the entire hair object instead of points before combing, all guide hairs are combed.

To display: Select the hair object, click the Selection button on the Select panel, and then click the Hair Recomb Operator icon.

For more information, see Combing and Recombing Hair [ Hair ].

Mute

Toggles the deformation on and off without affecting the values of other parameters.

Recomb Amount

The amount of combing, where 1 is fully combed and 0 is the previous state without any combing.
