syflexISpring

Categories

  • Syflex Force

Description

Defines the springs that are used to link a cloth or curve object's set of points.

This node is used in the Syflex Cloth Springs, Syflex Curve Springs, Syflex Spring 2Pt, and Syflex Zipper compounds.

For information on Syflex ICE springs, see Using the Syflex Spring Forces.

Ports

Inputs

Stiffness

Defines the stiffness (stretch) of the springs that define the cloth or curve deformation.

Damping

The damp coefficient, which controls the intensity of the damping effect on the spring.

Length

The spring's relative rest length in Softimage units. The length is set to the initial distance between the points when the spring is created.

Zip

Allows you to open/close the points along the disconnected cloth edge that creates a zipper. See Syflex Zipper for information.

Points

The points (vertices) of the input simulated object.

In Position

The global XYZ position of the input simulated object.

Outputs

Force

The output force.
