Get Particle Speed

This compound gets the particle's current speed data as defined by its PointVelocity attribute. You can output the Speed as a scalar, which gives you only the rate of change over distance, but without the velocity vector that determines direction.

If you want to output both the Speed and a Vector, you can use the Get Particle Velocity compound.

For more information, see ICE Particle Speed [ICE Particle Simulations].

Tasks: Particles/Getters

Output Ports: Speed

Reference

The particle's PointVelocity attribute.
