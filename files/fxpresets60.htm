Transform 3D

| Transform | Source Rect | Fill Color | Output Crop | Mask | Output |

Performs a 3D transformation using any combination of translation, scaling, and rotation about X, Y and Z axes.

Requires an Input (Any).

Produces an Output (Any).

Transform

Move

Sets the movement in X, Y and Z from the original position.

Click the mouse pointer icon to display the Move manipulator in an FX viewer. Use the manipulator to move the image interactively instead of using the sliders.

Click the R button to reset all of the Move values to 0.

Scale

Sets the scaling in X, Y from the original size, about the origin. Scaling in Z currently has no effect.

Click the mouse pointer icon to display the Scale manipulator in an FX viewer. Use the manipulator to scale the image interactively instead of using the sliders.

Click the R button to reset all of the Scale values to 0.

Rotate

Sets the rotation in X, Y and Z from the original angle, about the origin.

Click the mouse pointer icon to display the Rotate manipulator in an FX viewer. Use the manipulator to rotate the image interactively instead of using the sliders.

Click the R button to reset all of the Rotate values to 0.

Origin

Sets the position of the origin in X, Y and Z relative to the bottom left-hand corner of the image. The origin is the point about which scaling and rotation are calculated.

Click the mouse pointer icon to display the Origin manipulator in an FX viewer. Use the manipulator to move the origin interactively instead of using the sliders.

Click the R button to reset all of the Origin values to 0.

Lens Angle

Sets the vertical lens angle, or field of view, of a notional camera. The default value of 30 allows the whole, un-transformed image to be seen, regardless of size.

Filtering

Sets the method by which the pixel content of the transformed image is calculated. Choose between Point, Bilinear, EWA, Lanczos, Mitchell, B-Spline, and Cardinal.

Source Rect

Source Rect

Sets the area of the input image that will be used to create the output image when Source Crop is turned on. You should normally display the Input when setting the Source Rectangle.

Source Crop

Sets whether or not the Source Rectangle will be applied. You can turn Source Crop on or off at keyframes; the setting at a keyframe will apply to all subsequent frames until the next keyframe is encountered.

Fill Color

Background Col

Sets the background color, that is, the color of the regions in the output image that are not made up from the transformed input image, and would otherwise appear black.

Output Crop

Crop Rectangle

Sets the area which will be output when Apply Crop is turned on; regions of the output image not contained within the Crop Rectangle will be rendered as black (not as the Background Color).

Apply Crop

Sets whether or not the Crop Rectangle will be applied. You can turn Apply Crop on or off at keyframes; the setting at a keyframe will apply to all subsequent frames until the next keyframe is encountered.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
