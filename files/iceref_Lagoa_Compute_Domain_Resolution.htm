Lagoa Compute Domain Resolution

Computes the resolution per point. This node is useful if you need to find out what the resolution of a point is during the simulation.

This node is used internally to define the resolution of the simulation and is exposed for development reasons in case you need to know the math for computing the resolution of a point.

Plug this compound's Execute output into a port of the ICETree node or the Execute port of the Simulation Root node.

Tasks: Lagoa/Tools

Output Ports: Execute

Resolution

Plug a scalar node into this port to control the resolution per point.

Tetrahedra

Select this option if the particles are emitted in a tetrahedral formation.
