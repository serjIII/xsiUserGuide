Closest Goal

Sets every actors' goal to the closest object in the Goals group of a CrowdFX simulation. The exact position for each actor is randomized within the goal's radius, set by Min Distance in the CrowdFX Goal Property Editor.

This node should be plugged into the Execute Goal port of Simulate Collision Avoidance.

For more information, see Moving Toward Goals.

Tasks: Crowds/Goals

Output Ports: Result
