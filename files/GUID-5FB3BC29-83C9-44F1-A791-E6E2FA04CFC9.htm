ClosestEdgeFromNull ClosestPolygonIndex

Custom attributes set by Get Closest Edge Index from Null:

  • ClosestEdgeFromNull is a single integer that stores the index of the edge that is closest to the null.
  • ClosestPolygonIndex is a single integer that stores the index of the polygon that is closest to the null.
