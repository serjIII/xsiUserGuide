Get Moving Towards Surface

This compound gets the distance from the current particle position to the surface of an object and the particle's velocity. You can output the Resulting scalar value.

Tasks: Particles/Getters

Output Ports: Result

Surface1

The object to which the particles move toward. This must be an object with a surface. Plug in the object's Value output into this port.

Enable Cutoff Distance

Toggles the activeness of the Cutoff Distance option.

Cutoff Distance

The maximum distance within which the particles consider the object.
