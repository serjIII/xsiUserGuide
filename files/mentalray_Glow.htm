Glow

Category: mental ray > Materials

Shader Family: Surface Material

Output: Color

Glow is a material shader designed to simulate an object with internal lighting. With Glow, you can create glow maps that respond to light levels and overexposures in a realistic, photographic manner.

Name

The name of the shader node displayed in the render tree. Enter any name you like, or leave the default.

Glow

The color of the internal light.

Brightness

Sets the intensity of the internal light. Lower values show little glow, while higher values show a bright glow.

Mix Diffuse

The glow is normally added directly to the input Surface Material color. This parameter, however, uses the light and dark areas of the underlying Diffuse color to vary the brightness of the glow.

Surface Material

An input color computed by an illumination shader and modified by the Glow shader.

Diffuse

The same color used by the illumination shader.

Transparency

The multiplier used to mix the input Surface Material color and the glow effect.
