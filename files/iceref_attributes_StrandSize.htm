StrandSize

The sizes of shapes along a strand.

This attribute is exposed as the Strand Size Profile option in the Create Strands and Generate Strand Trails compounds.

If this attribute is not defined, the base particles' Size is used instead.

If the size of this array does not match StrandPosition, it is ignored.

For point and segment shapes, ShapeSizeAbsolute controls whether this value is interpreted as Softimage units or pixels.

Data Type

array of scalars
