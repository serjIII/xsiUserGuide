Biovision Import Dialog Box

Specifies which Biovision file (BVH) to import and how it is to be imported.

The BVH file contains hierarchy information and the base pose of the skeleton. This information is used to create a hierarchy of nulls (the skeleton) within a model when imported into Softimage.

The BVH file also contains the motion capture data as keyframed animation that can be applied to the skeleton.

To display: Choose File Import Biovision File from the main menu bar.

For more information, see Importing Biovision Files [Character Animation].

File Name

Specify the path and file name for the Biovision (BVH) file to be imported. Click the Browse (...) button to select a specific location and file name from a browser.

Skeleton Description Mode

Bone Chains

Creates a standard skeleton hierarchy made of bones. You can then set the Bones Skeleton Options to define the skeleton setup.

Hierarchy of Nulls

Creates a skeleton hierarchy of nulls.

Bones Skeleton Options

If you selected Bone Chains as the Skeleton Description Mode, you can choose how you want the skeleton's bones to be connected and what type of chain structure to create.

Hierarchy

Choose how you want the skeleton's bones to be connected:

  • Connect with unanimated bones: All bones, animated or not, are connected as part of the skeleton hierarchy.

  • Leave unconnected: Bones that are not animated are not connected to the skeleton hierarchy.

Chains

Select which structure you want for the resulting skeleton hierarchy:

  • SI3D Mode: Creates SOFTIMAGE|3D-style chains whose effectors are children of the last bone of the chain.

  • XSI Mode: Creates default Softimage-style (XSI) chains whose effectors are children of the chain root.

Add Motion Action to Mixer

Adds the accompanying keyframed motion as an action clip in the animation mixer.

Group Animated Objects

Creates a group that includes all animated objects in the skeleton hierarchy.
