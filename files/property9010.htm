Texture Projection Geometry to Texture Operator

Edits a spatial texture projection property applied to a geometric object.

Compute in World Space

When on, the spatial projection is calculated in world space. As a result, the object will appear to "swim" in the texture whenever the object is transformed.

When off, the spatial projection is calculated in object space. As a result, the object will appear to "swim" in the texture whenever the object's geometry is modified.

Mute

When on, the texture projection will be temporarily frozen. Disable this parameter to resume editing the texture projection.
