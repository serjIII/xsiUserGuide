Scalar to Rotation

Categories

  • Conversion

  • Rotation

Description

Outputs a new rotation built from the input X, Y, Z and angle components.

Ports

Inputs

X

The X component for the new rotation.

Y

The Y component for the new rotation.

Z

The Z component for the new rotation.

Angle

The angle component for the new rotation.

Outputs

Rotation

The new rotation.
