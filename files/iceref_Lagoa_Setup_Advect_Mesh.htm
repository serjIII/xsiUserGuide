Lagoa Setup Advect Mesh

Sets up a Lagoa mesh advection to deform a mesh based on a particle simulation.

To apply: See Advecting Geometry with Lagoa [Lagoa Multiphysics Simulations].

Tasks: Lagoa/Presets

Output Ports: Execute

Emitter In Name

Plug in an emitter object's Out Name output here.

You can also plug a group node here, with each emitter object defined in the group.

Passive Body In Name

Plug in the Out Name output of the object with which the points will collide. This object can be animated (deformation or transformation) or not.

Passive objects are not affected by any type of force in the simulation environment.

You can define each collision object's properties in its own Lagoa Set Collision Data compound.

Null In Name

Plug in a null object's Out Name output here.
