Test Timer

This compound uses a certain amount of time passing as defined in a Start Timer compound as a trigger for an event.

Plug its Result output into a Condition port of an If node (for example) or any node or compound that causes an execution.

If you're using states, plug its Result output into a Trigger port on a State compound or the Simulation Root node.

You also need to use the Init Timers compound to initialize the timers before you can use them.

For more information on states, see ICE Particle States; for more information on timers, see Setting Particle Timers.

Tasks: Particles/Conditionals,Particles/Timers

Output Ports: Result

Select Timer

Lets you choose a timer via the Start Timer compound. Make sure it matches with the timer name of the corresponding Start Timer compound you want to use.

Global Time

Uses the scene's Global In/Out setting (as displayed in the timeline range) for the time value.
