Import Visemes Dialog Box

Import a library of viseme poses for lip syncing in Face Robot.

To display:

On the Act panel in the Face Robot layout, click the Library > LipSync tab and choose Setup Visemes. Then in the Visemes Dialog Box, click the Import Visemes button.

For more information, see Importing Visemes [Face Robot].

Import

Imports the variations of viseme poses that you select here into the Face Robot head model's Mixer > Sources > Visemes folder.

  • All Variations: imports all visemes in the library.

  • Normal Variation Only or (Current) Variation Only: Imports only those visemes tagged with the _Normal or _(current variation) suffix.

    For example, if you have created an "Drunk" variation, you can choose to import only those visemes by selecting this option.

Overwrite Existing Visemes

Overwrites any viseme poses of the same name in the Face Robot head model's Mixer > Sources > Visemes folder with the visemes variations you've selected here.

Note that this doesn't affect the viseme poses in your current library.
