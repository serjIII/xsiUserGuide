Setting Pass Output Paths and File Names

Location:

  • The Render Manager > (select a pass)

  • Set the current pass and choose Render > Render > Pass Options or Pass > Edit > Edit Current Pass from the Render toolbar.

  • Tab: Output [Properties Reference]

  • Options: Render Channels Output (Framebuffers)

For each render pass, you need to set file name of each render channel that you wish to output. The file names are appended to the scene output path to produce a fully resolved output path for the rendered sequences.

For more information about setting the scene output path, see Setting the Output Directory.

To set file names for a pass' render channels

In the Render Channels Output options, do the following for each render channel whose file name you wish to set:

  1. In the render channels output grid, click the channel's Filename field to make the file name editable.

  2. Build the file name using Tokens and Templates, or simply type in a file name.

    See Setting the Frame Padding for details on how frame padding is resolved for file names.

  3. If you wish to set additional option, make sure that the channel is still selected and click the Edit button to open the Render Channel property editor.

  4. Set the following Output Format options as needed:

    • Select a format for the image files from the image format list.

      For a list of the formats supported by Softimage, see the Render Channel (Framebuffer) Property Editor.

    • Choose a bit depth for your render. Your options are 8-bit, 16-bit, and float, depending on the Image Format.

      NOTE:Because not all image formats can be output at all channel widths, the Channel Width menu only displays available widths for the selected format.
      TIP:Right-click on a row in the Render Channel Output table and choose Browse to open the output folder in a Softimage browser window.
