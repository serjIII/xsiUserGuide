Bend Strands

This compound bends particle strands along the particle's local orientation axis or along a global axis.

Create strands, then plug this compound's Execute output into a PostSimExecute port on the Simulation Root node or in a port on the ICETree node.

For information, see Shaping the Strands.

Tasks: Particles/Strands

Output Ports: Execute

Select Bend Method

Select how accurately and quick you want the bending to be:

  • Slow and Accurate uses the actual angle value that you specify in Bend Angle, but is slower to calculate.

  • Fast and Approximate uses an approximation of an angle that you specify in Bend Angle, but it's faster to calculate.

Bend Profile

This curve defines how the angle is applied to the strand, starting from the strand's base (key on the left) to the tip (key on the right). By default, not much bend is applied to the strand base, but then the full amount is applied by the tip of the strand.

Bend Angle

The angle in degrees of the bending to apply to the strand. How accurately this angle is reflected in the shape of the strand depends on which Bend Method you selected.

Select Bend Axis Direction

The direction in which the strand bends according to the option you select:

  • Bend Using Axis Vector in World Space bends all the strands in a global direction that you define with the Bend Axis values.

  • Bend Using Axis Vector Local to Particle Orientation bends each strand in the direction according to its particle's local orientation. If you change the orientation of the particles (such as in the Emit compound or with an Align compound), the strand's bending changes to reflect this.

Bend Axis

Bends the strand along the X, Y, or Z axis you set here according to the Bend Axis Direction option you selected.
