Envelope Operator Property Editor

Controls the envelope deformation. It is created when you apply an envelope, but the property editor is not displayed automatically.

To display: Select an envelope then choose Edit Properties Animation Properties and click the Envelope Op tab.

Mute

Toggles the deformation on and off without affecting the values of other parameters. For example, you can temporarily mute the envelope operator while changing the reference pose.
