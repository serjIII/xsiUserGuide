Lagoa Material Foam

Creates foam using the point cloud that is emitted. The surface tension and external pressure values are very high to create nice sticky globs of particles that cling together.

The parameters exposed for this compound are a subset of those in the Lagoa Main Material compound, mainly in the Pressure group.

Plug this compound's Execute output into an Execute port of the Lagoa Phase node.

For more information on setting up a Lagoa ICE tree, see Creating a Lagoa Effect from Scratch.

Tasks: Lagoa/Material Type

Output Ports: Execute
