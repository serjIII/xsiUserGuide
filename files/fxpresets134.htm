Lens Flare

| Flare | Star | Glow | Lights | Mask | Output

This operator applies a lens flare to your image or sequence. This is useful for adding the lens flare effect to pre-rendered passes. You can control the brightness and dimensions of the basic flare effect, it's "star" and its glow. You can also set up to three lights to act as sources for the flare(s).

Requires an Input (Any).

Produces an Output (Any).

Flare

Enable

Toggles the lens flare effect on and off.

Flare

Displays the Flare Preset dialog box from which you can select a flare type.

Intensity

Controls the brightness of the flare.

Scale

Controls the size of the flare.

Aspect Ratio

Specifies the ratio between the flare's width and height.

Composite Method

Specifies how the flare is added to the image or sequence:

Add: the flare's lights contribute to the overall lighting.

Screen: the flare's lights do not contribute to the overall lighting.

Star

Enable

Specifies whether the operator creates a star-flare effect.

Number of Rays

Controls the number of rays per star.

Rays Length

Defines the length of the star's ray.

Rays Length Jitter

Controls the seemingly random variations in the star ray's lengths.

Rays Thickness

Defines the thickness of the star's ray.

Rays Thickness Jitter

Controls the seemingly random variations in the star's rays' thicknesses.

Rotation

Specifies the rotation of the flare's star effect.

Auto Rotate

When on, the star effect will automatically rotate as the flare's lights are moved.

Brightness

Controls the brightness of the star effect.

Twinkle

This parameter controls the random flickering of the star over time.

Glow

Glow

Applies a glow or halo within the flare effect.

Size

Defines the size of the halo.

Brightness

Controls the overall brightness of the flare's glow.

Falloff

Defines the halo's attenuation from the center of the flare.

Lights

Light 1,2,3

Enables the light. You can enable up to three lights and configure the following parameters for each.

Color

Each light's color sliders set the light's color.

Position

Sets the position of the light in X and Y. values of 0 and 0 will place the flare in the image's lower-left corner. Both X and Y position values can range from -2048 to 2048.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
