Get Actor ID

Returns the index of the actor source used by an actor in a CrowdFX simulation, where 0 is the first actor added, 1 is the second, and so on. Note that this is the ID of the corresponding mesh, not the rig if you are using a single animation source.

This compound can be used only in an ICE tree on the crowd simulation Point_Cloud object.

For more information, see Loading Actors for the Crowd Simulation.

Tasks: Crowds/Getters

Output Ports: Value
