Mask Shapes

Defines the basic properties of a Mask Shapes operator. A Mask Shapes operator is like a vector paint operator, except that it affects only the alpha channel. Mask Shapes operators are useful for painting garbage mattes.

Can accept an input (Any).

Produces an Output (Any).

Vector Paint

Bit Depth

Specifies the bit depth of the output image.

8 bit outputs an 8-bit image.

16 bit outputs a 16-bit image.

Width/Height

Specify the resolution of the operator's output image.

Operator

Operator Name

The operator's name in the explorer and the FX Tree. You can change this to any name.

Active

Toggles whether or not the operator is active. Deactivating this option mutes the operator.
