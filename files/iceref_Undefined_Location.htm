Undefined Location

Categories

  • Geometry Queries

Description

Outputs an undefined location. This node is often useful when initializing attributes on geometry of type location. When connecting a location port into the source port of a Get Data node, if the location is undefined, the element is removed from the data set executed for that branch.

Ports

Outputs

Location

The undefined location.
