Lagoa Match Node Type

Tasks: Lagoa/Tools

Output Ports: Result

Second

Type of node:

  • Real nodes are moved by the simulator.

  • Embedded nodes do not get moved by the simulator, but other points will react to them.

  • Virtual nodes are completely skipped by the simulator.
