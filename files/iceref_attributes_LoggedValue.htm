LoggedValue

A custom attribute that is used by the Graph Value Over Time compound to store to draw the graph.

If you use more than one Graph Value over Time compound on an object, change the name so that the attributes do not interfere with each other.

Data Type

scalar

Belongs To

anything

Geometry Types

  • any

Attribute Type

custom

R/W or RO:

read/write
