Compound Clip Property Editor (Action)

To display: Select a compound action clip in the animation mixer then press Enter and click on the tab with the compound action clip's name.

Controls

Auto Transition

Transition curves are automatically created for new action clips instantiated within the compound action clip.

Time Span

Start Time

The start time of the compound action clip in the local time of its parent. This value is provided for information only and cannot be modified from this property editor. To change it, move or scale the clip in the animation mixer.

Duration

The duration of the compound action clip in the local time of its parent. This value is provided for information only and cannot be modified from this property editor. To change it, move or scale the clip in the animation mixer.
