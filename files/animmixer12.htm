Find Cycles Dialog Box

To display: Select a cycled clip and its driven object, then choose Clip Find Cycles from the animation mixer's command bar.

For more information, see Finding Animation Cycles in Action Clips [ Nonlinear Animation in the Animation Mixer ].

Algorithm

Energy Analysis (rotations): If you have selected objects with rotation parameters, this algorithm is already selected. If a period is found, the cycle operation is performed.

Frequency Analysis (marked parameters): If you have selected objects with marked parameters, this algorithm is already selected. If more than one period is found, the operation will not be performed because multiple cycles cannot be calculated.

Cycle Estimate

Duration

The minimum cycle duration in frames: All cycles found with a duration of fewer frames than this are rejected.

Operation

Trim Clip

Trims clips based on the cycle found while attempting to keep as many periods as possible.

Add Markers

Adds markers to the initial time and then at regular intervals separated by the period found.

Marker Value

Enter a name for your markers.
