Set Size Relative to Emitter

This compound lets you set the particle's size relative to its emitter. This may be useful if the emitter is changing size over time, or if you have the same point cloud being emitted from several different emitters and you want the particle size to be different for each.

Plug its Value output into the Size port of an Emit compound to set this value upon emission.

Tasks: Particles/Setters

Output Ports: Value

Relative Size

Sets the particle's Size attribute relative to the emitter's size. A value of 1 uses a 1:1 ratio of the current particle Size and emitter size.

A value greater than 1 makes the particles larger relative to the emitter; and a value less than 1 makes the particles smaller.
