Paste Action Dialog Box

To display: Select an action source and choose Actions Apply Paste Action on the Animate toolbar.

For more information, see Pasting Animation from Action Sources [Nonlinear Animation in the Animation Mixer].

Source Action

These parameters let you specify the range of frames you want to copy from the selected action source.

Source

The name of the selected action source, which is the source that will be pasted.

Nb Fcurves

Displays how many function curves are contained in the source. This information is for display only and cannot be changed.

Start and End

The first and last frames that define the range of frames in the source to be copied.

By default, this is the first to last key in the source. Note that this is the source's frames, not necessarily what is shown in the mixer because any clip instanced from the source could have its timing altered.

Target

These parameters let you set up the target model and source to which you're pasting the animation copied from the Source Action.

Target

The name of the model whose source you're pasting the animation.

New Start and End

The first and last frames that define range in the target source to which the animation will be pasted. This lets you offset and scale the animation as you paste it.

Ripple

Ripples any existing animation in the target source that is in the New Start and End range you define here. The existing animation is pushed further ahead in time.
