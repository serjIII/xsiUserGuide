Searching and Replacing Strings in Paths

You can modify the file paths of several files at once using the external files manager's Search and Replace tool.

To modify many file paths at once

  1. From the main menu, choose File External Files to open the external files manager.

  2. If desired, select the rows for the paths that you want to modify from the external files manager grid.

    If you don't select any paths, the search and replace operation will affect all of the paths currently visible in the external files manager grid, unless you specify otherwise (see step 4)

  3. In the Actions section, click the Search and Replace button to open the Search and Replace dialog box.

  4. In the Search and Replace dialog box, choose your settings:

    • In the File Type text box, select the type of files whose entries you want to modify (for example, All, Pictures, Audio, Synoptic).

      Changing the file type updates the list of files to be modified, which appears at the bottom of the dialog box.

    • In the String to Search text box, enter the text string to search for in the existing path and file names.

    • In the String to Replace text box, enter the replacement text.

    • Check the Case Sensitive check box if you want to search and replace using the specific capitalization you've entered in the String to Search box.

    The grid control in the dialog box lists all the external files that will be affected by the search and replace operation. If you selected files from the external files manager's grid before opening this dialog box, the selected files are listed. Otherwise, all files are listed.

  5. Click OK.
