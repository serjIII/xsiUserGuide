Rough Pastels

This effect renders an image using strokes of colored pastel chalk on a textured surface. In areas of bright color, the chalk is thick and little of the underlying texture is apparent. In other areas, the chalk appears to have been scraped off, revealing more of the underlying texture. You can choose from a variety of textured surfaces, or you can create your own.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

Stroke Length

Set higher values to make the strokes longer, or set lower values to make the strokes shorter.

Stroke Detail

Set higher values to make the pastel strokes more detailed and more apparent, or set lower values to make the pastel strokes less detailed and less apparent.
