Set Locomotion

Sets the transformations of the deformers for each actor instance in a pedestrian crowd simulation. This compound calculates the pose state based on the particles' speed, and then blends between the idle, walk, and run animation cycles.

This compound should be applied in the Simulation region of the Point_Cloud object of a CrowdFX simulation, after the particles' positions and velocities have been set.

For more information, see How Action Sources Are Used in the Crowd Simulation.

Tasks: Crowds/Animation Cycle

Output Ports: Execute
