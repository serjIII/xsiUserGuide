Lip Stiction Property Editor

Adjust how the lips stick together for a Face Robot head.

To display:

On the Act panel in the Face Robot layout, click the Library > Tools tab and choose Lip Stiction.

For more information, see Solving the Sticky Lips Problem [Face Robot].

Active

Activates the lip stiction feature.

Amount

Right/Left

Sets the location of where the lips stick on the right and left side of the mouth. These locations are represented by two little red dots that appear at either corner of the lips.

Make sure that the animation controls are toggled on for display so that you can see the red dots.

Width

Defines the proper width between the lips for the deformation. A lower value closes the gap between the lips, creating more stickiness.

Paint

Stiction Influence

Activates the weight paint brush so that you can paint where you want the sticking to happen.

Seal Falloff

Seal Falloff

This curve controls the shape of how the lips stick together, moving from the corners of the lips toward the center like zipping up the lips.
