Set Actor Global Transform From Cycle

Sets the transformations of each actor instance's deformers to the values from the specified frame in an imported action. Values between full frames are interpolated. This may be useful when building or debugging custom crowd setups.

This compound can be used in an ICE tree on the Point_Cloud object of a CrowdFX simulation.

Tasks: Crowds/Animation Cycle

Output Ports: Execute

Mode

How to specify the frame:

  • Current Frame in Timeline: Uses the scene's current frame.
  • Current Frame in Cycle: Uses CrowdFX_Actor_CurrentFrameInCycle.
  • Frame Input: Uses the Frame value specified below.

Note that in all cases, the Initial Frame value set by Initialize Locomotion Data or Initialize Stadium Data is added.

Cycle Index

The action to use, where 0 represents the first action imported, 1 is the second, and so on.

Frame

The frame to use when Mode is "Frame Input".
