Bend

The ICE-based Bend compound bends an object's geometry.

To apply: See Bend [ICE Deformations].

Tasks: Deformation/Effects

Output Ports: Result

Mute

Toggles the deformation on and off without affecting the values of other parameters.

Axis

The axis of the object's local coordinate space that is bent.

Mode

Select which part of the object to bend.

Bend Direction

The direction in which the axis is bent, in degrees. For example, if the Y axis is bent in the direction 0 degrees, it bends toward the X axis; if it is bent in the direction 90 degrees, it bends toward the Z axis.

Angle

The angle of the bend in degrees.

Radius

The radius of an imaginary circle that describes the sharpness of the bend. The bend follows the circumference of this circle.

Offset

Offset of the deformation center from the object center in the object's local space.

Helper Display

Displays visual cues for the axis and radius used for bending.
