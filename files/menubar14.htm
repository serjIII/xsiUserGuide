Wavefront Export Dialog Box

| File Output | Geometry Options

Exports scene data to the Wavefront Object (*.obj) format. You can export a single object, multiple-selected objects, or all objects in the entire scene (if no selection is made when exporting).

To export: Select objects to be exported. Choose File Export Obj File from the main menu.

File Output

File

Filename

Specify the path and filename for the wavefront OBJ file to be exported. Choose the Browse (...) button to select a specific location from a browser.

Hierarchy

Choose how objects are exported to the OBJ file:

  • Merge all objects into one file: All selected objects are exported to a single OBJ file. (On by default)

  • Separate files per object: Each selected object is exported to its own OBJ file.

Sequence

Separate Files Per Frame

When on, exports one OBJ file per frame defined in the sequence as specified by the start frame, end frame and step frame. (Off by default)

Start

Specify the first frame of the sequence.

End

Specify the last frame of the sequence.

Step

Specify the increment between frames in the sequence. This allows you to skip frames.

Geometry Options

Geometry

Polygon Meshes

Exports polygon mesh information.

User Normals

Exports user-defined normals from UserNormals user data maps.

Nurbs Meshes

Not implemented.

Curve List

Not implemented.

Triangulation

Export Render Tesselation

Not implemented.

Vertices

Exports vertices in Local Coordinates or Global Coordinates.

Material

Export Material

Exports material information to a material library (*.mtl) file which is referenced by the OBJ (*.obj) file.

Export Current UV Set

Exports UV topology information.
