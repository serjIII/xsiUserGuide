Comp Adjust

Individually adjusts any primary (red, green, blue) or secondary (cyan, magenta, yellow) components of a clip, using a graph.

Requires an Input (RGB or RGBA).

Produces an Output (RGB or RGBA).

Graph

Graph

The horizontal axis represents the selected component values in, and the vertical axis represents the selected component values out.

Component

Sets the component to be adjusted. Select Red, Green, Blue, Cyan, Magenta, Yellow or Alpha, as required.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
