Get Set Sum

Categories

  • Math > Statistics

Description

Outputs the sum of all values in the data set. For example, if the data set contains the values of the weight map on each point on a polygon mesh, Get Set Sum will return the sum of all the weight values in that weight map.

Ports

Inputs

Value

The data set from which to calculate the sum.

outputs

Result

The sum of all the values in the set.
