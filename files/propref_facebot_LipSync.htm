Lip Sync Dialog Box

Generate a speech clip based on the viseme library and audio file for lip syncing in Face Robot.

To display:

On the Act panel in the Face Robot layout, click the Library > LipSync tab and choose Create LipSync.

For more information, see Creating a Speech Clip from an Audio File [Face Robot].

Language

Select the language of the audio file and text file to use for generating the lip sync: English or Japanese.

Audio

The audio file that you want to use for lip syncing. Click the browse button (â€¦) to open a browser in which you can select a file.

See Supported Audio File Formats [Nonlinear Animation in the Animation Mixer] for information on formats.

Silence Duration

The longest allowable length of silence (in seconds) in the audio file. If the silence lasts longer than this value, then the speech recognition analysis process stops, as if the audio had ended.

Text

  • If an accompanying text file exists for the audio file in the same folder, it is automatically loaded. You can fix any typographical errors here and click the Save button to update it before loading.

  • If no text file is automatically loaded with the audio file, click the browser button for Text to open a browser in which you can select the text file.

  • If no text file exists for the audio file, you must first create an empty .txt file with the same name as the audio file and located in the same folder. Then you can load that .txt file and enter the dialogue in the text box here. Click the Save button to overwrite the empty .txt file with your changes.

Save

Saves any changes you've made to the text file.

Reload

If you made changes to the text, but don't want to keep them, click this button to reload the original text file before saving it.

Clip Offset

Starts the speech clip at the frame that you specify. For example, to start the lip sync clip at frame 45, enter 45 here.

Current

Starts the speech clip at the current frame in the timeline. This updates the value in the Clip Offset text box to that frame.

Create LipSync

Generates a speech clip based on the viseme library and audio file that you've selected. See Creating a Speech Clip from an Audio File [Face Robot] for more information.
