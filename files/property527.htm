3D Object Property Editor

To display: Click on the top icon of a 3D object in an explorer. To view and edit a different property set of an object, expand the object in an explorer and click on one of the child icons. Alternatively, click Selection on the Select panel and click on one of the child icons there.

Name

The name of the object as it appears in the explorer. The name is also used to refer to the object in the text boxes on the Select panel, as well as by the scripting commands. Spaces and other illegal characters are automatically converted to underscores. If there is already an object with the same name, a number is appended automatically.

Didn't find what you are looking for? If you are looking for help on parameter in a particular property set, make sure to click the help button (?) associated with the desired property page in the body of the property editor. Alternatively, activate the corresponding tab at the top of the property editor before you click the main help button at the top of the property editor.

If you are looking for help on concepts and procedures, choose Help User's Guide from the main menu.
