File Cache Source Property Editor

General | Info

A file cache source is created automatically for every cache file that gets created.

For information on creating cache files, see Caching Animated Deformations and Simulations.

To open the property editor:

Do one of the following:

  • Right-click on a file cache clip in the animation mixer and choose Source.

  • In an explorer, click the file cache source node under Sources > FileCache in the Sources/Clips scope, or under the model's Mixer > Sources > FileCache in the Scene Root scope.

General

<path>

The path and file name of the cache file.

Frames

The number of frames in the cache file. This information is provided for information only and cannot be changed.

Info

Name

The name of the cache source. The default name is based on the cache file name.

Object Name

The scene object on which the cache is applied.

Path

The path and file name of the cache file.

Clip Color

Sets the color of all clips in the animation mixer that are instantiated (created) from this source.

Click the color chip and select a color from the color editor, or drag the color sliders to select a color.

NOTE:To share a clip color between sources, drag the color chip from one source property page and drop it on a color chip in another source property page.

For general information about setting colors, see Defining Color Properties.

Sequence Info

The number of frames, first frame, and last frame in the cache file. This information is provided for information only and cannot be changed.
