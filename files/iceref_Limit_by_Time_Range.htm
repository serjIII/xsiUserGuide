Limit by Time Range

This compound limits the particle emission to occur only within a certain range of frames. These values must be between the Offset and Duration values that are currently set in the Simulation Time Control property editor.

Plug this compound's Enable Emitter output into the Enable port of the Emit compound.

For more information on emissions, see Setting Up Any Type of ICE Particle Emission [ICE Particle Simulations].

Tasks: Particles/Emission Control

Output Ports: Enable_Emitter

Use Start Frame

Uses the Start Frame that you specify. If you don't use the Start Frame, the particles are emitted from the first frame of the simulation.

Start Frame

The first frame on the timeline that you want to use for the particle emission range. Remember that particles are actually only emitted the frame after this frame number.

Use End Frame

Uses the End Frame that you specify. If you don't use the End Frame, the particles continue to be emitted until the end of the simulation.

End Frame

The last frame on the timeline that you want to use for the particle emission range. Particles are not emitted anymore after this frame.
