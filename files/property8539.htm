Custom Color Property Editor

This is a simple custom property containing just a single color parameter. It is provided as a built-in property so that you can easily add a custom color to objects. For example, you can reference the custom color in a realtime programmable shader.

To apply: Choose Get Property CustomColor from any toolbar.

You can generate your own, more sophisticated custom property objects using the CPSet Wizard.
