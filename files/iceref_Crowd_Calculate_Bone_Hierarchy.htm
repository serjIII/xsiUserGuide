Crowd Calculate Bone Hierarchy

Categories

  • Crowds

Description

Calculates the transforms of the actor deformers in global space. This node is used by the Convert Deformer Transforms to Global compound .

Parameters

Local Transforms

The local transforms of the deformers

Ports

Inputs

Parent Indices

The indices of the parent deformers. Connect this input from the Get Actor Rig Hierarchy node.

Outputs

Global Transforms

The global transforms of the deformers
