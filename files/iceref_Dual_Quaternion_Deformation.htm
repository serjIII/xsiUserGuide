Dual Quaternion Deformation

Deforms an envelope using dual quaternion skinning, or a mixture of dual quaternion and linear blend skinning. See Dual Quaternion Skinning [ICE Deformations]

Tasks: Deformation/Skinning

Output Ports: Execute

Deformation Method

Specifies the deformation method to use: Dual Quaternion Skinning, Linear Blend Skinning, or Use Blend Input to interpolate between the two methods.

Blend Input

Controls the interpolation between linear blend and dual quaternion skinning when Deformation Method is set to Use Blend Input. A value of 0.0 uses linear blend skinning, a value of 1.0 uses dual quaternion skinning, and values in-between use a weighted average. You can drive this port with a weight map to vary the interpolation across the surface.

As an alternative to this parameter, you can also drive Deformation Method directly with a scalar value between 0.0 and 1.0.

Deformer Group

The group of deformers that was used to envelope the geometric object.

Deformer Group In Name

A scene reference to prepend to the group specified by Deformer Group to fully resolve the group's name.

Envelope Object

The name of the enveloped mesh. The default is "Self" but you can change it to apply the deformation to a different object.

Envelope Object In Name

A scene reference to prepend to the object specified by Envelope Object to fully resolve the object's name.

Envelope Cluster Property

The element with the Envelope_Weights and EnvelopeDeformerIndices attributes. The default should work in most cases but you can change it if the property has been renamed.
