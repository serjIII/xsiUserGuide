Basic Crowd Simulation

Calculates the new actor position from velocity.

Tasks: Crowds/Behavior

Output Ports: Simulate. Connect this port to the Behavior Core compound.

Velocity Input

Connect this port to compounds with Velocity output. These compounds calculate and set the actor velocity (PointVelocity).
