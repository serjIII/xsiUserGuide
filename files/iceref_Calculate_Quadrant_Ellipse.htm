Calculate Quadrant Ellipse

Returns the radius corresponding to the input angle on a shape composed of a portion of different ellipses in each quadrant.

Category: Trigonometry

Output Ports: Radius

Angle

The angle for which to calculate the radius.

Pos X

The maximum X value of the shape.

Neg X

The minimum X value of the shape.

Pos Y

The maximum Y value of the shape.

Neg Y

The minimum Y value of the shape.
