Current Frame

Categories

  • Data Access

Description

Returns the global or local frame index at which the ICE tree is being evaluated.

There can be a difference between the local and global frame only with simulated ICE trees. In this case, the local frame corresponds to the number of frames since the beginning of the simulation. The beginning of the simulation is defined in the Simulation Time Control Property Editor under the simulation environment.

Ports

Inputs

Global Time

Specifies whether to return the global frame index (selected) or the local frame index.

If you want to play a cache file using the frames in the scene timeline, select this option so that global time is used. This can prevent the frame offset issue that often occurs when the cache file's local frames are used for playback.

Outputs

Frame

The frame index at which the ICE tree is being evaluated.
