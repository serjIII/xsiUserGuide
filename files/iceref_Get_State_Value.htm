Get State Value

This compound gets the State ID for the state that you select. You can output the Result value as an integer.

For more information on states, see ICE Particle States.

Tasks: Particles/States

Output Ports: Result

State Value

Select the State ID for the state whose value you want to get. The State ID is a unique identifier defined in each State compound.
