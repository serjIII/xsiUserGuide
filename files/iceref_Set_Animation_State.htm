Set Animation State

This node is used to set the active Animation State for an Actor. It is also possible to define the current animation of an actor using multiple Animation States blended together. By animating the weights, it is possible to blend from one Animation State to another.

Tasks: Crowds/Behavior

Output Ports: Execute. Connect this port to the Animation State group of the Behavior Core compound.

State

Name of the Animation State.

Weight

Weight of the associated Animation State.
