Create Strand from Curves

This compound creates continuous particle shape segments along one or more curves. If you alter the curve's shape or number of points, the strands are updated to reflect these changes.

Plug its Execute output into a port on the ICETree node in an ICE tree that can be in the Modeling, Simulation, or Post-Simulation region of a point cloud.

Tasks: Particles/Strands

Output Ports: Execute

Nb Segments

The number of segments in each strand along the curve. Note that this doesn't affect the overall strand length, just how the strand is divided.

Size

The Size of the particle that is extruded along the curve.

Shape

The particle Shape that is repeated for each segment along the strand.

Segment Mode

The mode to use for displaying and rendering strands that use the Segment or Point shape types:

  • Softimage Unit: Draws the strand thickness in actual Softimage units. This sets the ShapeSizeAbsolute attribute.

  • Pixel Unit: Draws the strand thickness with a constant raster space width; that is, strands far away from camera are rendered with the same raster space thickness as strands that are close to the camera.

Display Strand Volume

Displays an approximation of the strand's volume as a wireframe for visual feedback without needing to render the strands.

Loft Shape Along Strand

Extrudes the selected particle shape in Y along the strand's length so that it appears to be one continuous segment. This option uses the StrandDeform attribute.

The Box, Cylinder, Capsule, and Cone shape types are currently supported for lofting, as well as instanced shape geometry. It's not necessary for the Segment shape.

If this option is off, the particle shape is repeated separately at each segment (the StrandPosition attribute).

NOTE:Remember that you need to draw a render region to see the shapes lofted along the strand in the viewport.

Profile

This profile curve defines the size and shape of the strand, starting from the strand's base (point on the left) to the tip (point on the right). A value of 1 is the same Size as the particle shape.

Select Strand Color Method

The color channels from which you can choose to be used for the strand color gradient: Set New Color, Lightness, Hue, or Alpha.

Strand Color Gradient

Sets the color of the strand using a gradient scale that represents the strand.

Note that the color gradient is displayed in the render region only if you have an appropriate color gradient shader set up for the particles in the render tree, such as the Particle Density shader or the Particle Strand Gradient Compound.

  • You can set the color in each box below the gradient by selecting it and then using the color sliders.

  • You can add more boxes by clicking anywhere beneath the gradient, or remove boxes by selecting them and pressing Delete.

  • Set the position of each box by dragging it, or enter a value between 0 and 1 in the Pos text box below.

  • Drag the small circles above the gradient to set the range of each color defined in a box.

Curve Name

Plug in the Output name output of a curve. A strand is created along this curve.
