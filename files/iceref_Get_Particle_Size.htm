Get Particle Size

This compound gets the particle's current size data as set by its Size attribute. You can output the Value as scalar data.

For more information, see ICE Particle Size [ICE Particle Simulations].

Tasks: Particles/Getters

Output Ports: Value

Reference

The particle's Size attribute.
