Lagoa Emit Mesh

Emits particles at the vertices of a mesh and sets initial data for using a Lagoa particle simulation to deform the mesh. This compound is used inside the Lagoa Setup Advect Mesh compound.

To apply: See Advecting Geometry with Lagoa [Lagoa Multiphysics Simulations].

Tasks: Lagoa/Element Creation

Output Ports: Execute

Enable

Enables this compound.

Emitter

Emitter In Name

The geometry to deform with particles.

Emit at Frame

The frame on which particles are emitted.

Global Time

If this option is on, Emit at Frame is relative to the scene's timeline. If it is off, the frame is relative to the simulation range set in the Simulation Time Control Property Editor.

Fitting

Filter Mesh Attribute

Filters the particle emission and geometry deformation by the specified attribute, such as a weight map's Weights attribute.

Invert

Inverts the attribute used to filter the emission and deformation.

Display

Color

Sets the color of emitted particles for display purposes.

Shape

Sets the shape of emitted particles for display purposes.

Simulator Settings

Node Type

Make sure this is set to Real. The other values are available for future functionality.

Cluster ID

Sets the cluster for points emitted from the connected object. Points in clusters with different IDs will not attach to each other during the simulation.

Constrainable

Determines whether points can make contact links or be constrained.

Phase

Sets the phase for points emitted from the connected geometry. Different phases on different geometries allows you to use different Lagoa materials.

Initial Force

The local XYZ components of the initial force for points on the connected geometry.

Execute on Emit

Any node plugged in here is executed only at emission.

Resolution Map

How size is computed for collisions, etc.

  • Automatic computes the size based on the average distance between neighbors.

  • Manual uses the Resolution value below.

Resolution

The size to use when Resolution Map is set to Manual.
