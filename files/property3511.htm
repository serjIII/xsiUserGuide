Clips Property Editor

Controls the resolution of all image clips that have effects enabled. For example, you can scale all clips to Half Size to reduce memory for previews.

NOTE:Any scaling performed from this property editor does not affect source images.

To display: Choose Explore Sources/Clips and then click a Clips folder icon.

Scaling

Type

Defines the scaling to apply to all image clips that have effects enabled.

To define a specific resolution, select Custom and use the Resolution X and Y sliders to determine a resolution.

To use the rescaling values set in individual clips, select None.

Resolution X, Y

Defines an X and Y resolution for image clips. The slider values are used only when Type is set to Custom.
