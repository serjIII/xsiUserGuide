StrandDeform

Whether shapes are lofted along the length of the strand (true) or placed individually at the discrete StrandPosition values (false).

This attribute is exposed as the Loft Shape Along Strand option in the Create Strands and Generate Strand Trails compounds.

Lofting is supported only for Box, Capsule, Cone, and Cylinder shapes. It is not necessary for segments.

For instanced particle shapes, if StrandDeform is true, the instances are deformed in Y along the length of the strand (unless ShapeInstanceTime is defined, for example, if you used the Set Instance Geometry compound instead of the Instance Shape node). If it is false or undefined, then the instances are simply placed at the discrete StrandPosition values.
