Geometry Property Editor (Icosahedron)

Controls the subdivisions of an icosahedron.

To apply: Choose Get Primitive Polygon Mesh Icosahedron.

To redisplay: Select the icosahedron, click the Selection button on the Select panel, then click the Geometry icon.

Geodesic Frequency

Number of subdivisions for each edge of the basic polyhedral shape.

Subdivision Method

How edges are subdivided:

  • Linear uses equilinear subdivision.

  • Angular uses equiangular subdivision.
