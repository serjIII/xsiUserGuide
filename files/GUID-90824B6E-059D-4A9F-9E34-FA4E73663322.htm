Shot Track Contextual Menu

When you right-click on a selected Shot Track, the contextual menu provides options to perform various tasks on the Shot Track.

Option Description
Insert ShotClip Inserts a ShotClip on the Shot Track. On a single track, you can add as many Shot Clips you want.
Add Shot Track Adds a Shot Track below the selected track.
Insert Shot Track Adds a Shot Track above the selected track.
Select All Clips Selects all Shot Clips on a track. The selected Shot Clips appear with a white border.
Delete Track Deletes the selected track.
Track Properties Displays the preferences to turn on or off the Ripple, Mute, and Solo Shot Track controls. See Shot Track Controls.
Display Options Contains the preferences to toggle the display of Shot Track and Shot Clip information. See Displaying Shot Tracks and Track Names, Displaying ShotClips Information, Defining the Default Length of a ShotClip.
