Build Cross

Builds intersecting axes similar to the default null shape for display. To use, connect the output into the Point Array port of Display Line.

Category: Debugging

Output Ports: Result

Position

The position of the shapes' centers in the object's local reference frame. Connect an array of positions for multiple shapes.

Orientation

The shapes' rotation.

Dimensions

The shapes' scaling.
