syflexISpringGeo

Categories

  • Syflex Force

Description

Defines the geometry on which the springs are created for the cloth or curve deformation.

This node is used in the Syflex Cloth Springs and Syflex Curve Springs compounds.

Ports

Inputs

Type

The type of springs that are created:

  • 1 = Stretch springs

  • 2 = Shear springs

  • 3 = Bend springs

Geometry

The geometry of the simulated object: polygon meshes for cloth and linear curves for curve deformations.

Outputs

Points

The array of points to be connected by springs.
