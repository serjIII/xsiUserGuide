Lin2Log

Cineon/DPX images use a logarithmic color space, so to work with them in the FX tree, you first have to convert them to linear space. Once you've applied the desired effects, you will likely want to convert them back to logarithmic space. This operator allows you to convert linear images back to log.

NOTE:You can set the FX tree to automatically "de-log" Cineon/DPX images when they are read and "re-log" them when they are written. These options are in the FX Tree preference dialog box (choose File Preferences from the FX tree menu).

Requires an Input (Any).

Produces a Output (Any).

Cineon Setup

Gamma

Performs a gamma correction. Using the gamma correction value used by your monitor ensures that the converted image will look correct on film.

Film Gamma

Adjusts the film gamma.

Refblack

The "Black Reference". This value maps the minimum linear black value of 0 to the log black reference value.

Refwhite

The "White Reference". This value maps the maximum linear white value to the log white reference value.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
