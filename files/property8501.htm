Snapping Preferences

Controls the targets to which objects and components will snap.

To display: Choose Snap Snap Options from the Snap panel.

Snap To

Reference Plane (Grid)

Activates snapping to viewport grids as well as reference planes.

Points

Activates snapping to targets with no dimensions, like points.

Segments

Activates snapping to targets with a single dimension, like curves.

Facets

Activates snapping to targets with two dimensions, like surfaces.

General Options

Use Region for Snapping

Controls how elements snap to targets:

  • If this option is on, elements snap to targets only when they are inside a box surrounding the mouse pointer.

  • If this option is off, elements snap to the closest target no matter how far away it is on screen. Note that this can slow down snapping on large scenes because all visible objects must be tested for snapping.

Region Size (pixels)

Controls the size of the snapping region around the mouse pointer in pixels. If a target falls within the region, it will be snapped to. This value is also used when welding vertices on polygon meshes interactively.

You can also change the size of the snapping region interactively by using the mouse wheel while pressing the Shift key. This works only if you are not using the mouse wheel to zoom â€” see Mouse Wheel.

Snap to Unselected Objects Only

Prevents an object from snapping to itself. You can activate this option, for example, if you want to snap an object's center to a point on another object while preventing the center from accidentally snapping one of the object's own points.

Snap to Tagged Components Only

Restricts the targets to the last set of components of a given type that were tagged (selected) on an object. You can tag components such as points on the target object beforehand, then when you move an element it will snap only to those points.

This option works only with tagged points, edges, and polygon. You can always snap to other types of target if they are enabled.

Use 2D Snapping in Orthographic Views

Snaps an element only in its own plane with respect to an orthographic camera. When this option is on, an element's Y position is not affected when snapping in the Top view, its Z position is not affected when snapping in the Front view, its X position is not affected when snapping in the Right view, and so on.

Snap to Closest Segments/Midpoints Only

Prevents snapping to Midpoints, Edges, and Knot Curves on back-facing components (determined by the normal direction) on closed shapes.

Snap to Visible Grid in Camera Views

Snaps to the floor grid instead of the view plane grid when snapping to the grid in Camera views.

Show Snap Targets

Toggles the display of points and other targets for snapping.

Point Options

Points

Vertices on polygon meshes, as well as control points on curves, surfaces, and lattices.

Knots

Knots on curves and surfaces.

Centers

Centers of 3D objects.

Midpoints

Midpoints of polygons and edges.

Segment Options

Edges

Edges on polygon meshes.

Curves

Curve objects.

U, V Knot Curves

Knot curves in the corresponding direction on surface objects.

Boundaries

Boundaries on surface objects.

When snapping to knot curves or boundaries, the center of the snapping region must be over the surface.

Facet Options

Faces

Polygons on polygon mesh objects.

Surfaces

Surface objects and subsurfaces.

Object Options

Filters let you further refine the type of snapping targets. When a filter is active, only components on the specified types of objects are considered as snapping targets.

All Objects

Components on any type of object.

Nulls/Chains

Null and chain objects. This works only with Centers.

Polygon Meshes

Components on polygon mesh objects.

NURBS Curves

Components on curve objects.

NURBS Surfaces

Components on surface objects.

Lattices

Lattice objects. This works only with Points and Centers.

Particles

Particle systems. This works only with Points and Centers.

Hair

Hair objects. This works only with Points and Centers.
