Geometry Property Editor (Cone)

Controls the number of points on a primitive cone.

To apply: Choose Get Primitive Polygon Mesh Cylinder or Get Primitive Surface Cylinder.

To redisplay: Select the cone, click the Selection button on the Select panel, then click the Geometry icon.

Subdivisions

U, V, Base

The number of segments in the corresponding directions. Higher subdivision values produce greater resolution and denser, heavier geometry.

Extent (Angles)

The Start and End U parameters let you open the cone. Values are in degrees.

Close

The Close Bottom option caps the bottom of the cone.
