GeoShader Property Editor

Controls the geometry shaders attached to a geometry shader primitive.

To get: Choose Get Primitive Geometry Shader Primitive from any toolbar.

To redisplay: Select the GeoShader object and press Enter. Alternatively, click on the GeoShader's icon in an explorer.

Shader

Lists the geometry shaders currently applied to the primitive.

Add

Applies a geometry shader to the primitive.

Delete

Removes the geometry shader selected in the Shader list from the primitive.

Inspect

Opens a property editor for the geometry shader selected in the Shader list.
