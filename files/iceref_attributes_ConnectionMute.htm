ConnectionMute

A Boolean value that deactivates spring and damper connections if set to true. You can use an array of Booleans to mute each connection in ConnectionIndices individually, or you can use a single Boolean to mute all connections.

If the ConnectionMute array is larger than the ConnectionIndices array, then any overflow is ignored. If it is smaller, the last value in the array is repeated for the remaining connections.

For more about ICE springs, see ICE Springs and Dampers [ICE Particle Simulations].
