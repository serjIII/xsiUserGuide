Stained Glass

The Stained Glass effect creates an image that appears to be a piece of stained glass, lit from behind. The strength of the light shining through the glass can be varied from a soft, even glow to a focused brilliance.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

Cell Size

Set higher values to increase the size of the stained glass pieces, or set lower values to decrease the size.

Border Thickness

Set higher values to increase the thickness of the lead borders, or set lower values to decrease the thickness.

Light Intensity

Set higher values to increase the intensity of the light source, or set lower values to decrease the intensity.

Background Color

Sets the color for background regions of the effect.

Set a color using the sliders.
