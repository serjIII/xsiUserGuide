AllowExecution

Used by the Execute Once compound to control execution of the connected branches.

If you use more than one Execute Once compound on an object, change the name so that the attributes do not interfere with each other.

Data Type

Boolean

Belongs To

anything

Geometry Types

  • any

Attribute Type

custom
