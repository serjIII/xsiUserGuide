misss_call_shader

Category: mental ray > Materials

Shader Family: Surface Material

Output: Color

This shader is a utility "pass through" shader. It allows you to pass shaders through as parameters to material Phenomena for items such as environments, photons, and displacement.

Name

The name of the shader node displayed in the render tree. Enter any name you like, or leave the default.

shader

The shader to be called.

default_shader

The shader that is called if shader is not specified.

mode

The shader calling mode, where 0 is "automatic". Any other number maps to a shader calling mode miShader_type.
