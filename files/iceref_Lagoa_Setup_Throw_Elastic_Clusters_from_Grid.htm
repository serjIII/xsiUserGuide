Lagoa Setup Throw Elastic Clusters from Grid

Contains all the necessary Lagoa nodes to create a particle emission in which particles are clustered together in groups at each emission frame, and are not blended together and solved as one element. The default results are individual "soft body" clusters of particles being emitted every nth frame and bouncing off one or more collision objects.

Plug this compound's Execute output into a port on the ICETree node or in an Execute port on a Simulation Root node.

Tasks: Lagoa/Presets

Output Ports: Execute

This compound is the same as Lagoa Setup Throw Elastic Clusters except that you need to use a null as the emitter object instead of a volume object.

As well, there are some additional exposed ports.

Null In Name

Plug in a null object's Out Name output here. The particles are emitted from this object.

If no object is plugged in here, the particles are emitted from a global position.

You can also plug a group node here, with each null emitter object defined in the group.

You can change the shape of the null in its property editor to change the emission volume.
