Defining the Default Length of a ShotClip

By default, the length of the newly added ShotClip is equal to the total duration of the Sequence Time.

To define the length of a the newly added ShotClip using the Preferences setting

  1. From the View menu, select Preferences.

    The Camera Sequencer Editor Preferences window appears.

  2. Click the Clips tab, and choose Absolute Duration from the Define Length of Newly Created Shot Clip drop-down list.

    The Number of Frames field is activated.

  3. Set the number of frames that you want to define as the duration of a newly added ShotClip.
