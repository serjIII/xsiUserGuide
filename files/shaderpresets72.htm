Toon Soft

DEPRECATED. This shader is unsupported, but it may still get installed with Softimage to provide compatibility with older scenes that use it. It is recommended that you replace unsupported shaders in your scenes with equivalent shaders from the current Softimage shader library.

Shader Type: Material (Soft3D)

Output: Color (RGB) value

Used with the Toon Lens shader, you can render objects with ink lines drawn over the standard material.

Name

The shader's name. Enter any name you like, or leave the default.

Ink Color

Sets the color for the ink.

Thickness

Defines the ink width. This may cause artifacts if two intersecting objects have very different line widths.

Illuminate Ink

Varies the color of contours according to the amount of light reaching the surface beneath. To switch on, use a non-zero value: values greater than 0 brighten the contour, while values less than 0 darken it; each is proportional to the color of light.
