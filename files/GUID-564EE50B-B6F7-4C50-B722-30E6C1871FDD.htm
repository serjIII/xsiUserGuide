Rotation to Euler

Categories

  • Conversion

  • Rotation

Description

Outputs a new 3D vector built by converting the input rotation according to the specified Rotation Order parameter.

Parameters

Rotation Order

Rotation order

Ports

Inputs

Rotation

The rotation to convert.

Outputs

Rotation XYZ

The new 3D vector.
