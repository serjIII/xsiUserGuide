Init Strand Dynamics

This compound initializes certain Strand attributes on the particle strands so that you can apply dynamics to them. When strands have dynamics applied, they can respond to collisions and other forces, as well as to the velocity and rotational velocity of the particle emitter. This lets you create hair-like effects with strands.

This compound is used inside the Emit Strands compound so that the strand attributes are initialized directly after the particles are emitted and the strands are created. You can use this node to help create your own custom effect.

Plug this compound's Execute output into a Port on the ICETree node.

For more information on using dynamics on strands, see Creating Strand Dynamics [ICE Particle Simulations].

Output Ports: Execute
