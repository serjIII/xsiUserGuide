Weight Map Property Editor

Defines a weight map. Weight maps can be used to modulate parameters such as the amplitude of a deformation across an object or cluster.

To apply: Select an object or cluster and choose Get Property Weight Map.

To redisplay: Select the object then choose Explore Property Maps on the Select panel and click the icon of the corresponding weight map. If the weight map is connected to a parameter, you can also display these parameters by clicking the red connection icon to the right of the connected parameter's slider on the corresponding property page.

Name

The name of the weight map.

Rename

Click to rename the weight map. See Rename Dialog Box.

Map Color

Red, Green, Blue

These three values determine the color used to display weights in the viewport. To display weights, make sure that the view mode is set to Constant or Shaded and that Show Weight Maps is on.
