Turbulize Value by Range

This compound applies noise (4D) to add turbulence to any parameter using any value within a range of possible values.

Plug its Value output into different ports of any compound, such as the Speed, Mass, and Size ports of the Emit compound, but there are many others that work.

For more information on turbulence, see Turbulizing Particle Values [ICE Particle Simulations].

Tasks: Particles/Modifiers, Deformation/Modifiers

Output Ports: Value

Min Value

Defines the minimum value of the range in which the turbulence is generated. A larger range of values yields a more obvious turbulence effect.''

Max Value

Defines the maximum value of the range in which the turbulence is generated.

Turbulence Scale

The amount of turbulence applied to the parameter value.

Animated

Causes the noise field to evolve continuously over time.

Animation Speed

The speed at which the noise evolves.

Complexity

Adds a fractal-like complexity to the noise which increases the level of detail in the noise pattern.
