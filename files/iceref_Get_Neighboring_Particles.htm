Get Neighboring Particles

This compound gets the particle ID and point position data from the particle's neighboring particles. You can output the Number of Points, Points, and Point Positions values.

Tasks: Particles/Getters

Output Ports: Number_Points, Points, Point_Positions

Enable Cutoff Distance

Toggles the activeness of the Cutoff Distance option.

Cutoff Distance

The maximum distance (in Softimage units) in which the particle considers its neighbors.

Enable Max Number Points

Toggles the activeness of the Max Number Points option.

Max Number Points

The maximum number of neighboring particles that each particle considers.

Filter by Cone Angle

Toggles the activeness of the Cone Angle option.

Select Cone Direction

The direction in which the Cone Angle area is projected: along the particle's local XYZ axis or along the particle's velocity vector.

Cone Angle

The maximum area in degrees in which the particle considers its neighbors.

Display Debugging

Displays the force vectors between points.
