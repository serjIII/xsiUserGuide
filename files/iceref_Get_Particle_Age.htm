Get Particle Age

This compound gets the particle's current Age data as set by its Age attribute. The age is the number of seconds that a particle has existed since it was emitted. You can output this Age data.

For more information, see ICE Particle Age [ICE Particle Simulations].

Tasks: Particles/Getters

Output Ports: Age

Reference

The particle's Age attribute.
