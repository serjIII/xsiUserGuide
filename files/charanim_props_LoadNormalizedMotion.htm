Motor File Property Editor

Loads a .motor (normalized motion) file onto a tagged rig or skeleton. Any type of animation can be retargeted and saved in a .motor file.

To display: Select a tagged rig or skeleton and choose Tools MOTOR Load Motion from the Animate toolbar.

For more information, see Applying Normalized Motion (.motor) Files to a Rig [Character Animation].

Motor File

The name of the .motor file you want to load. Click the browser button (...) to open a browser in which you can select the .motor file.

Load Setting - Start

Sets the start frame at which you want to load the .motor file.

Apply

Applies the animation from the selected .motor file to the target rig.

Adjust

Opens the Retarget Adjust Property Editor in which you can adjust the normalized motion on the target rig.

Remove

Removes the normalized motion from the target rig.

Plot

Opens the see Plot Retarget Dialog Box in which you can plot the normalized motion on the target rig into fcurves.
