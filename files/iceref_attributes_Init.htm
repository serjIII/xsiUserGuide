Init

False if a point was created on the current frame, true otherwise. This attribute is automatically set by the Add Point and Clone Point nodes, as well as by compounds that contain them, such as the Emitters and Spawning compounds.

You can use this attribute to distinguish between new and old particles. For example, you can set data only on new particles even in compounds that don't have access to the On Execute ports of Add Point and Clone Point.
