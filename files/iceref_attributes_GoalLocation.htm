GoalLocation

Available to store a location on a surface towards which you want a point to move.

This attribute is used by the Goals compounds.

This attribute is not considered by the Simulate Particles node. There must be other nodes in a graph to set the Force or PointVelocity to get a point moving toward its goal.

You can quickly get and set the GoalLocation attribute with the Get Particle Goal and Set Particle Goal compounds.

Data Type

location

Belongs To

points

Geometry Types

  • point cloud

  • polygon mesh

  • NURBS surface

  • NURBS curve

  • lattice

Attribute Type

predefined

R/W or RO:

read/write
