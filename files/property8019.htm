Split Edge Op Property Editor

Splits edges on polygon mesh objects.

To apply: See Splitting Edges [Modeling].

To redisplay: Select the polygon mesh, click the Selection button on the Select panel, then click the SplitEdge Op icon.

Ratio Edge

The position of the new point along the old edge, from 0 to 1.

Curvature Continuity

Modifies the position of the new point(s) based on the surrounding surface curvature. A value of 0 gives a flat result (no curvature), while 1 gives full curvature continuity. You can enter smaller and larger values for an exaggerated effect.

Parallel Edge Loop

Adds a new point at the same position on all parallel edges.

Connect

Connects the new points to each other. This option works only when Parallel Edge Loop is on.
