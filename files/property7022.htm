Symmetry Constraint Property Editor

Constrains the transformations of two objects in symmetry according to a specific global plane or a reference object.

For more information, see Symmetry Constraints [ Animation ].

To apply: Select an object, Constrain Symmetry, then pick the object to which the selected object will be constrained.

To redisplay: Select the constrained object, choose Explore Constraints on the MCP, and click the constraint's icon.

Active

Toggles the activeness of this constraint.

Plane of Symmetry

Constrains the objects to be symmetrical according to a global plane: XY, YZ, or XZ.

Scaling

Symmetrize Scaling

Toggles on/off the scaling of the symmetrical transformations.

X, Y, Z

Sets the symmetry scaling offset in global space on each axis.

Rotation

Symmetrize Rotation

Toggles on/off the rotation of the symmetrical transformations.

X, Y, Z

Sets the symmetry orientation offset in global space on each axis.

Position

Symmetrize Position

Toggles on/off the translation of the symmetrical transformations.

X, Y, Z

Sets the symmetry position offset in global space on each axis.
