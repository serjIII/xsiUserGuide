mib_map_get_color

The mib_map_get shaders can be used to drive shader parameters from named properties of the currently rendered particle.

For example, individual colors of the particles may be stored in a field called color. If this shader uses the color field name as the input parameter and is connected to the material shader of the particle object, then each particle will be rendered in its specific color.

For information on this shader, see the mental ray Documentation at www.autodesk.com/mentalray-help-2013-enu.
