mib_map_get_integer_array

The mib_map_get shaders can be used to drive shader parameters from named properties of the currently rendered particle. This shader is like the mib_map_get_integer shader except that it outputs color data instead of an integer.

For information on this shader, see the mental ray Documentation at www.autodesk.com/mentalray-help-2013-enu.
