Vector To Scalars

| Render Tree Usage

Category: Processing > Conversion

Shader Family: Texture

Multiple Outputs: Scalars

Converts vector input information into separate scalar outputs for each XYZ axis.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Specifies the vector inputs to be converted to a scalar output.

Render Tree Usage

This shader is placed between a vector output and scalar inputs in order for the values to connect.
