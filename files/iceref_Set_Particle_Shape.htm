Set Particle Shape

This compound lets you set the particle's shape, using the particle's Shape attribute as a reference.

Plug its Execute output into the Execute on Emit port of an Emit compound to set this value once, or plug it into the ICETree node to update at every frame.

For more information on particle shape, see ICE Particle Shapes [ICE Particle Simulations].

Tasks: Particles/Setters

Output Ports: Execute

Shape

Sets the particle's shape for how the particle shape is drawn on screen. You can choose from these shapes: Point, Segment (Trails), Disc, Rectangle, Sphere, Box, Cylinder, Capsule, Cone, and Blob.

Reference

The particle's Shape attribute.
