LocalTime

The time that has passed since the start of the simulation. Objects in different environments can have different LocalTime values.

Data Type

scalar

Belongs To

objects

Geometry Types

  • any object with a Simulation region in the construction stack

Attribute Type

inherent

R/W or RO:

read-only
