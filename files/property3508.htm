Cache Properties Property Editor

You can set the size and format of the cached images the flipbook will display. These settings do not affect the format of exported flipbooks.

To display: Open the image clip viewer and choose File Cache Properties from its command bar.

General

Playback Zoom

Sets the scale of the displayed images with respect to the size of the cached images.

Cache Format

Sets the color format of the displayed images.

Cache Scale

Sets the scale of the cached images with respect to the original image source (the viewport or the loaded video file).

Cache Disabled

Images are not cached in memory.
