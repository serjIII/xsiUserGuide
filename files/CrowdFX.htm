CrowdFX Property Editor

Custom property that sets data and create and read cache files for this Crowd model in a CrowdFX simulation.

To display: Select a CrowdFX icon in the explorer (under a Crowd model node).

Set As Current Crowd

Click this button to set this Crowd model as the current crowd, if multiple crowds are in the scene.

When you choose CrowdFX commands from the ICE toolbar, they are applied only to the current crowd.

When you create a new crowd, it is automatically set as the current crowd.

Caching

Writes to and reads from an .icecache file that is created for this Crowd model's simulation point cloud. See Caching or Exporting a CrowdFX Simulation for more information.

Simulation Cloud File Path

Enter the path and name of the cache file to be saved, or from where it is to be read. Click the browse (â€¦) button to select a different path and name.

The [Frame] token and .icecache file extension are automatically appended to the cache file name upon creation, so you don't need to specify them in the file name.

If you're reading a cache file, remove the [Frame] token and .icecache file extension from the cache file name before loading it.

Note that only the .icecache file format is supported.

Start Frame

The first frame in the crowd simulation that will be plotted to the cache file.

If you're reading the cache file, this is the first frame of the cache file that is used to create the cache clip.

End Frame

The last frame in the crowd simulation that will be plotted to the cache file.

If you're reading the cache file, this is the last frame of the cache file that is used to create the cache clip.

Attributes

If you don't enter anything in this text box, Softimage automatically makes sure that all required ICE attributes are included. These are the attributes that are used in the point cloud's ICE tree and in the render tree for rendering.

Any attributes you enter here, such as custom attributes, are appended to the required attributes. Enter the attributes in a comma-separated list with no spaces between the comma and the next attribute name.

Cache Point Cloud Simulation

Click this button to cache the current crowd's point cloud simulation to the cache file in the path that you have defined above.

Use Cache

Applies the cache file that is specified in the path above to this crowd's simulation point cloud.

This crowd's point cloud must use the same instanced envelope geometry as was used by the point cloud from which the cache file was created.

Selecting this option creates a hidden point cloud named Point Cloud Cached that contains a cache clip in its Mixer. This cache clip drives the simulation of the point cloud over the frames that are contained in the cache clip - see Using Cache Files in the Animation Mixer for more information.

Deselect this option to return to the live simulation on the point cloud.
