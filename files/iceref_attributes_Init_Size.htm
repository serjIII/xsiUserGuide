Init_Size

A particle's original Size value when it was emitted.

This attribute is set by the Emitters compounds, and is exposed in their property editors as the Size parameter.

It can also be used by Modify Particle Size or your own compounds.

Data Type

scalar

Belongs To

points

Geometry Types

  • point cloud

Attribute Type

custom

R/W or RO:

read/write
