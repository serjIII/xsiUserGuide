About Intranets and Netview

You can create an intranet to share presets, textures, scripts, and other data with your workgroup using Netview.

You can use any text or HTML editor to create your intranet pages. Some high-end HTML editors offer features that let you write, test, and debug scripts, which you may find useful.

The sections that follow describe some techniques that allow HTML pages in Netview to interact with Softimage. It is assumed that you are already familiar with HTML and scripting for the Web.

For information about scripting for Softimage in general, see Script Development [SDK Guide].

For information about using Netview, see Netview.

NOTE:Files that require browser plug-ins, including PDF, Flash animation, and so on, will not work within Netview. However, you can include links to these files that open in a new browser window.
