Channel Target Special Values

This is a list of the commands available for a channel's Target when you select Special as the Action Type.

Target Command

TIMELINE

Drives the timeline.

TIME_START

Goes to the first frame.

TIME_END

Goes to the last frame.

TIME_STEP_FORWARD

Goes to the next frame.

TIME_STEP_BACK

Goes to the previous frame.

TIME_STOP

Stop the playback.

TIME_PLAY_NORMAL

Plays back from current frame (real-time playback).

TIME_PLAY_FRAME

Plays back from first frame (real-time playback).

TIME_PLAY_SLIDE

Plays from current frame (all frames).

TIME_SPECIFY,frame

Goes to the specified frame.

ACTIVATE,driver_ID

Activates the specified device driver.

DEACTIVATE,driver_ID

Deactivates the specified device driver.
