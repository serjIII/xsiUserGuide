Set Goal By Direction

Sets goal for actors to move towards a direction.

Tasks: Crowds/Behavior

Output Ports: Goal. Connect this port to the Goal group of the Set Velocity Using Collision Avoidance compound.

Direction

Scalar value describing the angle of the direction from Z axis.

Coordinate System

The coordinate in which the direction is calculated:

  • Emitter local space
  • Scene global space
