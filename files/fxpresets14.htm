Curtain Wipe

| Wipe | Mask | Output

Wipes any clip to any other clip, vertically, horizontally or at any angle. The edge of the wipe can be softened.

Requires an Input 1 (RGB), and an Input 2 (RGB).

Produces an Output (RGB).

Wipe

Wipe Trans

Sets the proportion of each clip to be displayed. Set a value between 0 and 1 using the slider. 0 is all of the second clip, 1 is all of the first clip.

Softness

Sets the softness of the edge of the wipe.

Direction

Sets the direction of the wipe.

Angle

Sets the angle of the wipe when the Direction is set to Use Angle. Set a value in degrees between 0 and 360 using the slider 0 degrees is vertical, and positive values move counter-clockwise.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
