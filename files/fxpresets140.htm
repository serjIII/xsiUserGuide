Pinch

Pinches the image in towards, or pushes it out from a user-defined point. The amount of distortion and the area affected can be controlled.

Requires an Input (Any).

Produces an Output (Any).

Pinch

Amount

Sets the amount of swirl. Set any floating-point value. Negative values swirl the image anti-clockwise, positive values swirl it clockwise.

Center X, Y

Set the center of the effect as a normalized proportion of the image. Set any floating-point values.

Diameter

Sets the diameter of the circle as a normalized proportion of the vertical size of the image. Set any floating-point value.

Interpolate

Sets whether or not transformed areas will be smoothed.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
