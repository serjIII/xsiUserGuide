Add Forces

This compound is used as a type of "hub" when applying forces to particles or simulated objects. You plug other force outputs into its force ports and then it adds up all those forces and applies them to the particles.

Note: If forces have already been applied to particles, this compound adds a new force to the existing forces.

Plug this compound's Execute output into a port on the ICETree node or into a State compound, if you are using particle states.

For information, see Creating and Applying ICE Forces to ICE Simulations [ICE Guide].

Tasks: Particles/Forces

Output Ports: Execute

Force1

Defines a force value to be applied to the particle or object. You can connect any of the force compounds into this port, or you can type in XYZ values to create a general force if nothing is plugged in here.
