Randomize Actor ID

Returns a random integer per particle, between 0 and the maximum possible ID given the number of different actor sources in a CrowdFX simulation (i.e. the number of sources minus 1). You can plug the output into the Actor ID port of Initialize Locomotion Data or Initialize Stadium Data.

Tasks: Crowds/Initialize

Output Ports: integer

Seed

The seed for the random number generator. Change this value to generate a different random pattern.
