ShotClip Property Editor

Provides the option to change the active camera of a ShotClip.

To open the ShotClip Property Editor

  • Select a ShotClip and choose Clip Clip Properties from the Camera Sequencer Editor menu bar.

    or

  • Choose Edit Properties.

    or

  • Right-click on a ShotClip and choose Time Properties.

ShotClip Option

Active Camera

Name

Choose a camera from the drop-down list to make it active for the selected ShotClip.
