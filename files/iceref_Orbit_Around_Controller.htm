Orbit Around Controller

This compound makes particles orbit around an object.

Plug its Execute port into a Port on the ICETree node.

For more information, see ICE Particles Orbiting Around an Object [ICE Particle Simulations].

Tasks: Particles/Motion Control

Output Ports: Execute

Use Input Orbit Axis

Toggles the use of the Input Orbit Axis values that you enter below.

You need to select Set Velocity Using Orbits Per Second Value as the Select Velocity Method option below in order to use this option.

Input Orbit Axis

The global XYZ axis around which the particles orbit around the controller.

You need to activate the Use Input Orbit Axis option above to use these settings.

Set Orbit Altitude

Toggles the use of the Orbit Altitude value that you enter below.

You need to select Set Velocity Using Orbits Per Second Value as the Select Velocity Method option below in order to use this option.

Orbit Altitude

Determines the size of the orbiting particles' circumference around the controller object. This value is in Softimage units.

You need to activate the Set Orbit Altitude option above to use this option.

Select Velocity Method

The type of velocity used to calculate how the particles orbit around the object:

  • Use Current Velocity uses the particle's current velocity.

If you select this option, you cannot use the Orbits Per Second, Orbit Altitude, or Input Orbit Axis values. The particles orbit around the controller on the global Z axis. The orbit altitude is determined by the distance of the controller object from the particle emission point: the greater the distance, the larger the orbit.

  • Set Velocity Using Orbits Per Second Value uses the Orbits Per Second (below) values that you enter.

    If you select this option, you can also use the Input Orbit Axis and Orbit Altitude options.

Orbits Per Second

If you selected Set Velocity Using Orbits Per Second Value, you need to set the number of times per second that the particles revolve around the object. This changes the particles' velocity.

Controller Object Name

The object around which the particles orbit. Plug the object's Output Name output into this port.
