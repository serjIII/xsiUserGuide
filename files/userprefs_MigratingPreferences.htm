Migrating Preferences and Other Customizations

You can migrate your preferences and other customizations from a previous installation of Softimage on your computer. All items that are saved and stored in your users folder are migrated. These include your custom toolbars, layouts, keymaps, commands, events, presets, scripts, and SPDL files.

Migrating from within Softimage

  1. Choose File Migrate Preferences.

    A list of available Softimage installations on the computer appears.

  2. Select an installation and click OK.

Migrating from the Command Line

You can migrate preferences using the -migrate command line option: 

xsi -migrate <previous user preferences path>

The -migrate option takes as argument a path containing preferences to be migrated to the currently running Softimage (the one executing the -migrate option).

  • If migration failed because you did not have administrative privileges to register the preferences. Log on as administrator and use the -register_only option at the command line to complete the migration of preferences.

  • If migration failed because default preferences already exist for the current Softimage installation, use the -overwrite switch at the command line to force the migration process and overwrite any existing preferences.
