CrowdFX Actor Proxy Property Editor

Lets you load the action sources from the source model to be used for the actor in the crowd simulation.

To display: Do either of the following:

  • Select a CrowdFX_ActorProxy icon in the explorer (under a Crowd model > Actors_Proxies > actor proxy node).
  • In a CrowdFX Actors Proxies Property Editor, click the Inspect Animations button.

For more information, see Loading Action Sources.

To display:

Action Sources

List

List of all action sources loaded in the original model's Mixer node.

Select one or more from the list, then click the Move >> button to move them to the Selected Actions list to be loaded for the actor to use in the crowd simulation.

Selected Actions

List

List of all action sources that are ready to be loaded for the actor proxy using the buttons below.

There is a maximum of three actions that are used in the crowd simulation by default, but you can add more later. If you use fewer than three actions, you'll need to adjust the pose states in the ICE tree.

To remove an action source from this list, select it and click the Move << button to move it back to the Action Sources list.

Reorder

The action sources should be in top-down order from slowest to fastest speed (for locomotion-type actions) so that the blending between them is smooth. Use the Up and Down buttons to reorder them.

Apply Actions > All Above

Loads all action sources in the Selected Actions list on the actor for it to use in the crowd simulation.

Apply Actions > Selected

Loads only the selected action sources in the Selected Actions list.

For Global Z Locomotion > Use Center of Gravity

Uses the actor proxy's COG (center of gravity) deformer to calculate the horizontal speed (in the global positive Z direction) of the actor for locomotion-type actions in the crowd simulation.

If you're using either stationary-type actions or actions that are animated in local space (such as an "in place" walk cycle), deselect this option.

See Should You Use the COG or Not? for more information.

Actor Source Model

Select

Selects the source model from which the actor is created.

Show/Hide

Toggles the display of the source model in the viewport.

Actor Proxy Model

Select

Selects the actor proxy model in the crowd simulation.

View Rig

Opens an Object view window that displays the actor's rig proxy - see Modifying the Actor's Rig.
