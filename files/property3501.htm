Duplicate Options Property Editor

Options | Transform

Defines how objects are copied, cloned, or instantiated. You can choose whether or not to pass on constraints, animation, hierarchical links, and other characteristics that belong to the copied object to its instances.

To display: Choose Edit Duplicate/Instantiate Options from the main menu bar.

Options

Resulting Topology

This option applies to duplicates, clones, and instances.

Hierarchy

Specifies whether or not the copy shares the same parent as the original. You can also set the copied object to have no parent, or set a new null as its parent.

Duplicated Items

These options apply to duplicates.

Polygons

Choose Extrude to connect duplicated polygons to the original mesh.

Choose Duplicate to leave duplicated polygons unconnected.

This option simply controls whether the Duplicate Polygons option is initially activated or not in the Extrude Op Property Editor (Components along Axis) [Properties Reference].

History

Choose Copy to give the duplicated object the same operator history as its original.

Choose Current to give the duplicated object no operator history.

Property Sets

Choose Copy to give the duplicated object copies of the properties applied to the original.

Choose Ignore to ignore the properties applied to the original. The duplicated object inherits any applicable properties from its parent or scene root.

Animation

Controls whether animation is copied when duplicating elements.

  • Copy animation and share mixer sources copies the animation (fcurves, expressions, constraints, clips) from the original to the duplicate. When duplicating an entire model, the mixer sources are shared.

  • Copy animation and copy mixer sources copies the animation (fcurves, expressions, constraints, clips) from the original to the duplicate. When duplicating an entire model, the mixer sources are duplicated as well.

  • Ignore does not apply any animation to the duplicate.

Constraints

Choose Copy & Share Input to apply the same constraints acted upon the original to the duplicated object.

Choose Ignore to apply no constraints to the duplicated object.

Texture Supports

Controls whether and how texture supports are copied when duplicating textured elements.

  • Only if Selected duplicates the texture support only if it was selected along with the original object(s). If the texture support is not selected, the duplicate object shares the original object's texture support.

  • Always duplicate duplicates the original object's texture support whether the support is selected or not. The duplicate support appears in the same position in the duplicate object's hierarchy as the original support appears in the original object's hierarchy.

  • Always share does not duplicate the original object's texture support. Rather, the duplicate object(s) share the original object's support.

  • Freeze projections duplicates the original object, but freezes the texture projection (and geometry) of the duplicates. As a result, no duplicate texture support appears.

Resulting State

This option applies to duplicates, clones, and instances.

Selection

Specifies which objects are selected once the duplication operation is complete. You can specify that only the original is selected, only the copies are selected, or both.

Track Placement

Switches on or off the object placement option.

Placement

Sets the method of placement of the copies. Choose Same Global Xform to place the copies directly over the original. Choose Apply/Repeat Xform to place the copies in a position equal in value to the latest transformation of the original object.

Transform

If multiple copies are created and Apply Repeat Xform is selected, then the values set here will be applied to copies in an additive fashion.

Scaling

X/Y/Z

The incremental scaling values of copies.

Rotation

X/Y/Z

The incremental rotations of copies, in degrees.

Translation

X/Y/Z

The incremental rotations of copies, in Softimage units.
