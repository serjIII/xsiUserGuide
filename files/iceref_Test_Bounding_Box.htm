Test Bounding Box

This compound uses a certain number of particles within the particle's bounding box as a trigger for an effect.

Plug its Result output into a Trigger port on a State compound or in a Condition port of an If node.

For more information on states, see ICE Particle States [ICE Particle Simulations].

Tasks: Particles/Conditionals

Output Ports: Result

Condition

The minimum/maximum number of particles in X/Y/Z within the particle's bounding box that is used as a trigger.
