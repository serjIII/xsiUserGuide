Transform per Copy

Controls the transformation applied to specific copies of a polygon mesh. This node should be plugged into the Execute on Copy port of the Create Copies from Polygon Mesh compound to provide additional control over the point positions of each copy.

Tasks: Topology/Copy and Merge

Output Ports: Execute_on_Copy

Copy Filtering

Comparison Index

The value to compare each copy's index with.

Type

The type of comparison. The transform is applied to the point positions of copies with indices that match this.

Transform

Specifies the Scaling, Rotation, Translation, and corresponding variances to apply to the point positions of the copies that match the criterion.
