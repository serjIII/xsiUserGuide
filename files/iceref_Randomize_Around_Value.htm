Randomize Around Value

This compound adds randomness to any parameter using a mean value around which the random values are generated according to the variance.

Plug its Value output into different ports of any compound, such as the Speed, Mass, and Size ports of the Emit compounds, but there are many others that work.

For more information, see Randomizing Particle Values [ICE Particle Simulations].

Tasks: Particles/Modifiers,Deformation/Modifiers

Output Ports: Value

Object Name

The object you can use for a reference.

Object Name Reference

Current object's name.

Seed

Defines the sequence of random numbers used. If you require that two nodes generate different sets of values with the same parameters, simply assign them different seeds.

Animated

Toggles the animation of the randomness; that is, whether it varies over time or not.

You can select this option when using an unsimulated ICE tree (such as if the point cloud's or object's ICETree node is in the Modeling region).

Base Value

The mean value around which the random Value is calculated in any XYZ direction.

For example, if this value is 2 and the Value is 1, the random value possibilities would be any value between 1 and 3.

Value

The amount of variation of the randomness on either side of the Base Value.
