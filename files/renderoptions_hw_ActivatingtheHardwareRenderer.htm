Activating the Hardware Renderer

You may activate the hardware renderer for the entire scene, or for specific passes.

To activate the hardware renderer for the entire scene

  1. Do one of the following:

    • The Render Manager > Scene Render Options

      or

    • From the Render toolbar, choose Render > Render > Scene Options

  2. On the Scene Globals tab, set the Scene Renderer to Hardware Renderer.

To activate the hardware renderer for a pass

  1. Do one of the following:

    • The Render Manager (select a pass)

      or

    • Set the current pass and choose Render > Render > Pass Options or Pass > Edit > Edit Current Pass from the Render toolbar.

  2. On the Output tab, set the Pass Renderer to Hardware Renderer.
