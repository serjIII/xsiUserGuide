Extract Edge Loop Property Editor

Creates a curve object from selected edges on a polygon mesh. Disconnected edge loops on the same polygon mesh object become subcurves of a single curve object. Edge loops on different polygon mesh objects become separate curve objects.

To apply: Select one or more edges on a polygon mesh and choose Create Curve Extract from Edges on the Model toolbar.

To redisplay: Select the curve object, click the Selection button on the Select panel, then click the Extract Edge Loop icon.

Mesh Subdivision Level

The level of subdivision used to extract the curve. Higher values produce smoother, denser curves, as if the original polygon mesh was subdivided. To make sure that the curve follows the original mesh exactly, set this parameter to the same values as on the Polygon Mesh tab of the original object's Geometry Approximation Property Editor.
