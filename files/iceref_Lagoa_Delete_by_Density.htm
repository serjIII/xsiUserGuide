Lagoa Delete by Density

Removes points based on how their density compares to a given value. This compound requires that density calculations are enabled in the Lagoa simulation.

This compound should be applied to a point cloud that is reading from cache files.

To apply: See Applying Cached Post Sim Effects [Lagoa Multiphysics Simulations].

Tasks: Lagoa/Cached Sim Post Effect

Output Ports: Delete

Type

The type of operation to determine how to use the Density value: Equal To, Not Equal To, Less Than, Less Than or Equal To, Greater Than, or Greater Than or Equal To.

Density

The minimum density that it will take for points to be deleted.
