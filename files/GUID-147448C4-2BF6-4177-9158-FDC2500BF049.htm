Get Distance to Surface

This compound gets the distance from the current particle position to the surface of an object. You can output the Distance and Distance Vector values.

Tasks: Particles/Getters,Particles/Surface Interaction

Output Ports: Location, Distance, Distance_Vector

Surface1

The object that you want the particles to consider. This must be an object with a surface. Plug in the object's Value output into this port.

Enable Cutoff Distance

Toggles the activeness of the Cutoff Distance option.

Cutoff Distance

The maximum distance (in Softimage units) within which the particles consider the object.

Proximity Method

The geometry component of the object that is used to calculate the distance: Closest Surface, Closest Vertex or Knot, or Closest Smoothed Surface.
