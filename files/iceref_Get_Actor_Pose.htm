Get Actor Pose

Allows you to view a specific animation cycle in a CrowdFX simulation without needing to playback the scene.

The easiest way to use this compound is to apply it in an ICE tree above the Simulation region on the Point_Cloud object, set the Cycle Index, and then adjust the Frame parameter while the current frame on the timeline is set to the first frame of the simulation.

Alternatively if you apply this compound in the Simulation region, make sure that it is applied after Set Locomotion or Set Stadium Animation, because it depends on values set by the Set Cycles Attributes sub-compound. You may also need to advance a frame to see changes, depending on the Play Mode set in the Simulation Time Control Property Editor.

Tasks: Crowds/Debugging

Output Ports: Execute

Cycle Index

The ID of the animation cycle to show, where 0 is the first imported action source, 1 is the second, and so on.

Frame

The frame of the cycle to show.
