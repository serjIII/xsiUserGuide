Marker Display Options Dialog Box (Animation Mixer)

These options are currently unavailable.

Switches the display of various marker properties in the animation mixer. Some of these options can also be set on the animation mixer's View menu.

To display: Right-click on a marker and choose Display Options.

Marker Name

The name displayed at the top of markers.

Maker Value

The value displayed at the bottom of markers.

Tooltips

Pop-ups displaying the name when the mouse hovers over a marker.
