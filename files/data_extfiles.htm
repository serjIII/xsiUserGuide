Managing External Files

A typical Softimage scene has a variety of external files associated to it. These files can be texture files, audio files, actions, material libraries, and so on. As you develop the scene, you'll probably need to perform some cleanup and management operations on its external files. For example, you might need to update some paths or locate a missing image. You can do all this, as well as perform other file management tasks, using the external files manager.
