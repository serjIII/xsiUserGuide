Create New Library Dialog Box

Create a new viseme library used for lip sync for Face Robot.

To display:

On the Act panel in the Face Robot layout, click the Library > LipSync tab and choose Setup Visemes. Then in the Visemes Dialog Box, click the Create New Library button.

For more information, see Creating a New Viseme Library.

Source Library

Path

The viseme library that you want to use as a base for a new library. Click the browse button (…) to open a browser in which you can select the library.

The Face Robot viseme library path is located in:

Softimage install path\FaceRobot\Data\FaceRobot\Viseme

Copy Viseme Files

Copies the visemes from the Source Library into the Destination Library.

Destination Library

Path

The new viseme library that you want to create from the Source Library. Enter a library path and name or click the browse button (…) to open a browser in which you can select a library.

When you click OK, this new library is created.

Overwrite Existing Files

Overwrites any existing visemes of the same name in the Destination Library.
