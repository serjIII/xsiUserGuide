Add-On Information Dialog Box

Provides optional, extra information about an add-on.

To provide info about your add-on: From the Package Add-On Dialog Box, click Advanced.

To display info about the add-on you are installing: From the Install Add-On Dialog Box, click Advanced.

To display info about the add-on you are uninstalling: From the Uninstall Add-On Dialog Box, click Properties.

Version

Softimage

Identifies the version of Softimage on which the add-on was created. This value is supplied automatically. This value is displayed for information only and cannot be modified.

Add-On Version

Identifies the version of the add-on format. This value is displayed for information only and cannot be modified.

General

Title

A name given to the add-on by the author.

Category

Indicates the general category of the add-on.

Description

Summary

A short summary of the add-on.

Description

A long description of the add-on.

Usage

Brief instructions for using the add-on.

Author

Author Name

The name of the add-on author.

Author E-mail

The e-mail address of the add-on author.

Company

The company name of the add-on author.

Platforms

Indicates which platforms the add-on runs on.

Languages

Indicates all of the languages used for the components of the add-on.

Additional Info

Provides a URL that points to additional information available on the web about the add-on.
