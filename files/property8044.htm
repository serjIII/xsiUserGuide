Automatic Cage Assignment Property Editor

Controls the weight assignment of points on a caged object to the points or polygons of the cage deformers.

To apply: See Cage Deformations [Data Exchange].

To redisplay: In an explorer, click the Envelope_Weights icon under the appropriate cluster (by default CageWeightCls).

Use Polygon Deformer Components

Assigns points on the deformed element to the polygons of the cage deformers. If this option is off, the deformee's points are assigned to the cage deformer's points. This option applies only for deformers that are polygon meshes.

Strength, Falloff

These two parameters determine the weight values of points. If Strength is relatively high, deformed points are weighted more strongly to the nearest components on the deformer, with a sharp falloff as the distance increases. If Falloff is relatively high, the weight values are more spread out.
