Cluster Center Op Property Editor

Constrains the center of a cluster to the center of a deforming object. When you transform the deformer, the cluster moves with it.

To apply: Tag points or select a cluster, choose Deform Cluster Center from the Animate or Model toolbar, and pick a deforming object like a null.

You can also use Edit Properties Create Cluster with Center to create a cluster, create a null at the cluster's geometric center, and apply a cluster center deformation with one command.

To redisplay: Select the object with the cluster, click the Selection button on the Select panel, then click the Cluster Center Op icon.

Mute

Toggles the deformation on and off without affecting the values of other parameters.

SI3D Rotation

When on, rotations are performed about the deforming object's center.

When off, rotations are performed around the cluster's reference frame (the geometric center before the Scaling, Rotation, and Translation offsets below are taken into account).

Amplitude

The amplitude of the deformation. When this value is greater than 1, the cluster's center overshoots the deformer's center resulting in an exaggerated deformation.

Scaling X, Y, Z

Scaling factors that are applied to the cluster center. The deformer's scale values are multiplied by these factors.

Rotation X, Y, Z

Rotation offsets that are applied to the cluster center. The deformer's rotation values are offset by these amounts. Values are in degrees.

Translation X, Y, Z

Translation offsets that are applied to the cluster center. The deformer's translation values are offset by these amounts. Values are in Softimage units.
