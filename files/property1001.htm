Flare Property Editor

| Effect | Flare | Glow | Star

Defines the type of flare you can assign to a light object and how it will react with the camera. For more information, see Applying a Lens Flare Property.

To Display: Expand the light node in the explorer view and select the light's flare icon (it will only be visible if a flare property has been applied to the light).

Effect

Enabled

Switches the flare effect on or off.

Flare

Flare File

Specifies a flare file preset from the drop-down menu.

Click and scroll through the list to see more flare file selections.

Brightness

Controls the overall brightness of the flares. The brightness and visibility of the light also affect the flare brightness.

Size

Defines the flare's size. Default value is 1.

Aspect Ratio

Defines how "squashed" the flare fill be. The default value of 1 causes the flare to be round.

Glow

Glow

Applies a glow or halo within the flare effect.

Brightness

Controls the overall brightness of the flare's glow.

Size

Defines the size of the halo. A value of 1 creates a halo to cover the entire image.

Falloff

Defines the halo's attenuation from the center of the flare.

Star

The light will create a starry shape on the lens. The number of rays as well as its angular distribution can be controlled. The thickness and length (size) of each ray can be jittered.

Stars

Defines whether the shader creates a star-flare effect on the camera's lens. The star's rays can be defined with the following parameters:

Brightness

Defines the overall brightness of the stars.

Number of Rays

Controls the number of rays per star.

Length

Length

Defines the length of the star's ray.

Jitter

Controls the seemingly random variations in the star ray's lengths.

Angle

Auto Rotate

When on, the star effect will automatically rotate as the light source is moved.

Resolution

Defines the resolution at which the auto rotate effect will be rendered.

Twinkle

This parameter controls the random flickering of the star over time. Humming "Twinkle, twinkle little star" is optional.
