Using the Output Rectangle

The options on the Output Rect page of the Tracking property editor allow you to output a specific region of the image.

To output a specific region

  1. In the FxTree, double-click the Tracking operator to open its property editor and preview it in the Fx Viewer.

  2. Click the Output Rect button to display the Output Rect page, select Apply to activate the output rectangle.

  3. Set the Output Rect Left/Bottom/Top/Right values to define the boundaries of the output rectangle in pixels.
