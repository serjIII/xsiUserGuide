EnvelopeWeightsPerDeformer

The percentages to which a point is weighted to all deformers. This array includes the weights for every deformer in the deformer group in the order they are indexed.

Data Type

array of scalars

Belongs To

points

Geometry Types

  • envelope weights property

Attribute Type

inherent

R/W or RO:

read/write
