DragCoefficient

The factor by which points are affected by drag.

This value is used by the Wind, Fan, and Eddy force objects (see Types of Forces [ Simulation and Effects]) in conjunction with the Size to determine the drag force.

Data Type

scalar

Belongs To

points

Geometry Types

  • point cloud

  • polygon mesh

  • NURBS surface

  • NURBS curve

  • lattice

Attribute Type

predefined

R/W or RO:

read/write
