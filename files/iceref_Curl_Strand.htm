Curl Strand

This compound curls the strand starting from its base at the particle. This creates a curl around the strand, using the particle's local orientation axis as the vector for curling the shape.

You may want to use this compound to shape the strands when using an unsimulated ICE tree, such as if the ICETree node is in the point cloud's Modeling region.

Create strands, then plug this compound's Execute output into a PostSimExecute port on the Simulation Root node or in a port on the ICETree node.

Tasks: Particles/Strands

Output Ports: Execute

For information, see Shaping the Strands [ICE Particle Simulations].

Curl Distribution Profile

This curve defines how the curl amount is distributed along the strand, starting from the strand's base (key on the left) to the tip (key on the right).

Curl Amount

The number of degrees in which the strand is rotated around the particle's local orientation axis. A value of 360 is a full rotation.

Curl Size Profile

This curve defines how the curl size is distributed along the strand, starting from the strand's base (key on the left) to the tip (key on the right). The is the profile of the curl's shape.

Curl Size

The extent of the curl. The higher the value, the "tighter" the curl.
