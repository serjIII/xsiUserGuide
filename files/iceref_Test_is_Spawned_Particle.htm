Test is Spawned Particle

This compound uses the fact that the particles are spawned as a trigger for an effect.

Plug its Result output into a Trigger port on a State compound or in a Condition port of an If node.

For more information on states, see ICE Particle States [ICE Particle Simulations]; for more information on spawning, see Spawning ICE Particles [ICE Particle Simulations].

Tasks: Particles/Conditionals,Particles/Spawning

Output Ports: Result
