Cosine (Expression Function Reference)

The Function Trigonometry Cosine command in the expression editor inserts the token cos( at the insertion point in the editing pane. This token evaluates the cosine function of an expression. Enter an expression after the opening bracket ( then enter a closing bracket ).

Example

cos(10*Fc)

Returns the cosine of the expression 10*Fc (10 times the current frame number).
