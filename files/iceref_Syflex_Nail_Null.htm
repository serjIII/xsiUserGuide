Syflex Nail Null

This compound lets you attach a simulated cloth or curve's vertices to a constraining object using a null to determine which vertices are nailed.

To apply: See Nailing Cloth or Curve Vertices to an Object.

Tasks: Syflex/Constraints

Output Ports: The Force output plugs into a Force port on the Syflex Cloth or Syflex Curve node.

The parameters for this constraint are the same as for Syflex Nail except that you use this port for the input null:

Null

The null object that determines which cloth or curve vertices are nailed to the constraining object. Plug its Get Data node's Out Name output into this port.
