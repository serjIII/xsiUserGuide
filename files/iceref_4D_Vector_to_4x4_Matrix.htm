4D Vector to 4x4 Matrix

Categories

  • Conversion

Description

Outputs a new 4x4 matrix built by arranging the four input 4D vectors according to the method specified in the Row or Column parameter.

Parameters

Row or Column

Specifies whether the input vectors become rows or columns in the matrix.

Ports

Inputs

Vector 1

The first 4D vector.

Vector 2

The second 4D vector.

Vector 3

The third 4D vector.

Vector4

The fourth 4D vector.

Outputs

Matrix

The new 4x4 matrix.
