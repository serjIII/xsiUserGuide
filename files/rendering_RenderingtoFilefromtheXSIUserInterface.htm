Rendering to File from the Softimage User Interface

Once you've set the render options for the render passes in your scene, you can render those passes directly from the Softimage interface. Since the options are set, all you need to do is start the render. This section tells you how.

If you haven't done so already, set the render options for the passes you wish to render, as described in Pass Rendering Options.

To render passes from the render manager

  • Click Render Pass Current button to render only the current pass. See Setting the Current Pass.

  • Click Render Pass Selected button to render only the passes you have selected in the explorer.

  • Click Render Pass All Passes to render all the passes you have defined and their active render channels.

To render passes from the render toolbar

  • Render Render Current Pass to render only the current pass. See Setting the Current Pass.

  • Render Render Selected Passes to render only the passes you have selected in the explorer.

  • Render Render All Passes to render all of the scene's passes.
