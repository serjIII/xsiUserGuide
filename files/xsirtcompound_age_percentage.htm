Age Percentage Compound

Category: Particle

The Age Percentage compound controls a parameter's values using a profile curve. The values on this curve are mapped to the particle's lifetime using the particle's Age and AgeLimit attributes.

Remember that you must set the particle's AgeLimit attribute in the point cloud's ICE tree for this shader compound to work - see Limiting the Particle's Age for more information.

For general information about ICE particle shaders, see ICE Particle Shading.

Parameters

Name

The compound's name. Enter any name you like, or leave the default.

Curve

Controls the profile curve for the parameter values and applies it to the lifetime of the particle. Enter values on the Y axis according to the parameter that you are controlling.

Render Tree Usage

Plug the output into any scalar port of a shader in a point cloud's shader tree, such as the ports on the Particle Density shader, or Particle Shaper and the Particle Renderer compound. For example, plug this compound into the Irregular Shape port on the Cigarette Smoke compound to make the particle's fractal shape intensity change over the particle's lifetime.
