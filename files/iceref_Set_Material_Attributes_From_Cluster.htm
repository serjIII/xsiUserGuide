Set Material Attributes From Cluster

Use this compound to convert local materials applied using clusters to materials applied using ICE. It appends the material applied to the specified cluster to the Self.Materials array, and then sets the MaterialID of the polygons in the cluster to the corresponding index.

Tasks: Topology/Setters

Output Ports: Execute

Cluster Name

Connect the Out Name of a Get Data node that references the cluster.

Library Name

The name of the material library.

Reference

A reference to the material used on the cluster.
