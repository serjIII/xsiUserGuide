EnvelopeWeights

The percentages to which a point is weighted to its deformers. This array includes the weights for just the deformers to which the point is weighted, in the same order as stored in EnvelopeDeformerIndices.

Data Type

array of scalars

Belongs To

points

Geometry Types

  • envelope weights property

Note that the Calculate Deformer Weights and Cleanup Painted Weights compounds also set a custom attribute with the same name directly on geometric objects. These attributes can be used by custom skinning compounds.

Attribute Type

inherent

R/W or RO:

read/write
