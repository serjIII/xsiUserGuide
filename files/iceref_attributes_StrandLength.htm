StrandLength

Set and used by the Strand compounds to keep track of the total length of strands.

This attribute is exposed as the Strand Length option in the Create Strands compound.

Data Type

scalar

Belongs To

points

Geometry Types

  • point cloud

Attribute Type

custom

R/W or RO:

read/write
