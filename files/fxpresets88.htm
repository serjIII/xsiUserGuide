Diffuse Glow

The Diffuse Glow effect renders an image as if it is being viewed through a soft diffusion filter. Bright areas glow with a diffuse brightness, while other areas are muted with a soft granularity.

Name

Specifies the operator's name.

Opacity

Sets the opacity of the effect over the input image. A value of 0 makes the effect completely transparent, while a value of 100 makes it completely opaque.

Use Alpha As Mask

Uses the input image's alpha channel as a mask for applying the effect. The effect is fully applied to areas of the image where the alpha channel is white, and not at all to areas where it is black.

Run Effect On

Enable the RGB and/or Alpha options to specify whether the effect is applied to the RGB channels, the Alpha channel, or both.

Graininess

Set higher values to add more grain to the image, or set lower values for less grain.

Glow Amount

Set higher values to have more area of the image affected by the glow, or set lower values to have less area affected.

Clear Amount

Set higher values to increase the area of the image that is left clear, or unaffected by the glow. Set lower values to decrease the area that is left clear and have more of the image affected by the glow.

Foreground Color

Sets the color for foreground regions of the effect. Set a color using the sliders.
