Get Closest Location on Geometry

This compound gets the geometry component data on the object that is plugged in here. You can output the Location, Position, and Normal data from the object.

Tasks: Particles/Getters,Particles/Surface Interaction

Output Ports: Location, Position, Normal

Geometry1

The object that you want the particles to consider. This must be an object with a surface. Plug in the object's Value output into this port.

Enable Cutoff Distance

Toggles the activeness of the Cutoff Distance option.

Cutoff Distance

The maximum distance (in Softimage units) within which the particles consider the object.

Proximity Method

The geometry component that is used as the closest location: Closest Surface, Closest Vertex or Knot, or Closest Smoothed Surface.
