Extract Curve Segment Property Editor

Creates a new curve based on an arbitrary segment of an existing one.

To apply: See Extracting Curve Segments [Surface and Curve Modeling].

To redisplay: Select the generated curve, click the Selection button on the Select panel, then click the Extract Curve Segment icon.

Inputs

Hide/Unhide

Hides all the inputs for the generated object to simplify the display of your scene in the 3D views, or unhides the inputs when you need to modify them.

Delete

Freezes the generator and deletes the inputs. This simplifies the scene once you are satisfied that you won't need to modify the inputs again.

Invert

Reverses the direction of the U parameterization on the generated curve.
