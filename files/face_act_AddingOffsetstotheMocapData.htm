Adding Offsets to the Mocap Data

Once you have applied mocap data to the face, you can key offsets on top of this data to adjust it. The offset is carried through to the mocap data.

To add offsets to the mocap data

  1. Select some animation controls and move them to where you want.

  2. Set keys for these controls by pressing the K key.

  3. Plot the animation to bake the mocap data and the offset animation into fcurves (see Plotting (Baking) the Mocap and Keyframe Data).

See also Adding Offsets to Mocap Data [Character Animation] for more information on different methods for adding offsets.
