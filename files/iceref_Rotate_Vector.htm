Rotate Vector

Categories

  • Math > Vector

  • Rotation

Description

Outputs a new 3D vector made from applying the input rotation to the specified vector.

Ports

Inputs

Vector

The vector to rotate.

Rotation

The rotation to apply.

Outputs

Result

The rotated vector.
