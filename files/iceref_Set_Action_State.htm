Set Action State

Sets CrowdFX_ActionState_ID, which is used by Action State to override the transformations for specific deformers on specific actors.

This compound should be used on the Point_Cloud object of a CrowdFX simulation.

For more information, see Modifying Deformer Transformations for the Animation States.

Tasks: Crowds/States

Output Ports: Result

Trigger

Controls whether the new action state ID gets set. Connect a per-particle Boolean to change state on a per-actor basis.

New State ID

The new action state ID to set when Trigger is true.
