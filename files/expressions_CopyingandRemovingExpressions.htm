Copying and Removing Expressions

You can copy and remove expression animation in the same way as other low-level animation — see Copying Animation and Removing Animation.

Expressions can be duplicated between different parameters for the same object or for different objects. For example, you can copy an expression from the X rotation of one object and paste it onto the Y scaling of another object.

TIP:The this and this_model tokens are useful when using the same expression on many different objects — see Entering Object Names.
