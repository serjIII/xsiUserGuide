Toolbar Wizard

Creates a new toolbar.

To display: On the Tree tab of the Plug-in Manager, choose File New Toolbar, or right-click on a specific Toolbars folder and choose New Toolbar.

Toolbar Name

The name of the toolbar.

Populate with Commands from this location

Automatically adds buttons for all commands in the corresponding add-on location.

Output Directory

The directory to store the .xsitb file.
