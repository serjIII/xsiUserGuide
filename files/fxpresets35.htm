Difference Extract

| Graph | Mask | Output

Extracts a matte based on the difference in color values between two images. The matte can be adjusted using a graph.

Requires an Input 1 (RGB), and an Input 2 (RGB).

Optionally takes a Combine Matte (A).

Produces an Output (A).

Graph

Graph

The horizontal axis represents the selected chrominance values in, and the vertical axis represents the matte values out.

Difference

Sets the criteria used to calculate the difference. Select On RGB, On Red, On Green, On Blue, On HSV, On Hue, On Sat or On Val.

Gain

Sets the tint for the output image.

Make Cutout

Forces the output image to RGBA mode.

Premultiply RGB

Premultiplies the output image by the generated matte.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
