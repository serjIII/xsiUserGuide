Curve Point Adaptor Property Editor

Identifies a point on a curve. This property is created automatically when you perform certain modeling operations (e.g., Add Knot, Remove Knot).

To display: Select a curve that has been modified, click Select on the Select panel, expand a Remove crv knot op, Insert crv knot op, or Curve Shift, then click the Curve Point Adaptor icon.

Position

The position of the point along the curve in terms of the parameter U.

Subcurve Index

The index of the disjoint subcurve that includes the point. This parameter is useful only if a single-curve object is composed of discontinuous segments as can happen when, for example, a curve is generated from the intersection of two surfaces.
