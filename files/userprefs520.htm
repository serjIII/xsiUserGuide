Output Format Dialog Box

Allows you to specify which scene elements to update with settings from the Output Format Preferences.

Update the Camera(s) Format settings

Activate this option to update the settings of all of the cameras in the scene.

Update the Render Option(s) Format and Image Resolution settings

Activate this option to update the settings of all of the render passes in the scene.

Update the FxTree(s) Image Defaults settings

Activate this option to update the settings of all of the Fx trees in the scene.

Update the Play Control settings

Activate this option to update the settings in the Play Control property editor.
