Chroma Extract

Extracts a matte based on chrominance. The matte can be adjusted using a graph.

Requires an Input (RGB).

Optionally takes a Combine Matte (A).

Produces an Output (A).

Graph

Graph

The horizontal axis represents the selected chrominance values in, and the vertical axis represents the matte values out.

Make Cutout

Forces the output image to RGBA mode.

Premultiply RGB

Premultiplies the output image by the generated matte.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
