Add Verlet Forces

Adds multiple forces to points, with an option to specify whether each force is in global or local space. This node is used internally by Verlet Framework.

Tasks: Deformation/Verlet Forces

Output Ports: Execute

Force1

A vector defining the force to add.

Global Space1

Controls whether the corresponding Force is interpreted in global space (true) or local space (false).
