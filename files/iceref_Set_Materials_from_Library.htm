Set Materials from Library

Sets the "self" object's Materials attribute with the string names of the connected materials. You can specify which material in the Materials array to use using the MaterialID attribute per polygon.

To use this node, connect it to the ICETree root or another execute-type port. Drag materials from the Material Manager or the Materials scope of an explorer into the ICE tree, and connect the Out Name ports into the Material Name inputs.

Tasks: Topology/Setters

Output Ports: Execute
