Accessing Render Options

You can access the various types of render options from a number of different places:

  • The Render Manager: the render manager is a dedicated view for editing scene, pass, and renderer options. It also contains useful tools like a built-in explorer view, quick access to pass rendering, rendering and output preferences, and a copy manager for you render settings.

    • Choose Render Render Manager from the Render toolbar.

    • Choose View Rendering/Texturing Render Manager from the main menu.

  • Render Toolbar: you can access the scene, pass, and renderer options property editors from the Render toolbar.

    • Choose Render Scene Options for the scene render options.

    • Choose Render Pass Options for the current pass render options.

    • Choose Render Renderer Options for the mental ray or Hardware renderer options (depending on which renderer is active for the current pass).

  • The Explorer: press 8 to open an explorer and the press P to set the scope to Passes or press U to set the scope to Current Pass. From there you can click the icons for the scene, pass, or renderer nodes to open the corresponding property editors.
