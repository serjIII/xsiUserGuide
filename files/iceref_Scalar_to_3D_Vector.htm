Scalar to 3D Vector

Categories

  • Conversion

Description

Outputs a new 3D vector built from the input X, Y and Z components.

Ports

Inputs

X

The X component for the new 3D vector.

Y

The Y component for the new 3D vector.

Z

The Z component for the new 3D vector.

Outputs

Vector

The new 3D vector.
