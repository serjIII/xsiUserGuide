Get Timer Values

This compound gets the particle state's timer values as set with the point cloud's Timer1_Start and Timer1_End attributes. This is used in conjunction with the Start Timer compound that lets you determine how long a particle stays in a particular state.

You can output the Start Time, End Time, Elapsed Time (that the particles are in the state), and Elapsed Percentage data.

You also need to use the Init Timers compound to initialize the timers before you can use them.

For more information, see Setting Particle Timers.

Tasks: Particles/Getters,Particles/Timers

Output Ports: Start_Time, End_Time, Elapsed_Time, Elapsed_Ratio

Select Timer

Select one of several timers that you have created.

Global Time

Uses the scene's Global In/Out setting (as displayed in the timeline range) for the time value.
