Lagoa Init Geometry

Sets data required by the Lagoa framework for deforming geometry. This compound is used inside the Lagoa Setup Cloth Mesh compound.

To apply: See Deforming Geometry Directly with Lagoa [Lagoa Multiphysics Simulations].

Tasks: Lagoa/Element Creation

Output Ports: Execute

Enable

Enables this compound.

Emitter

Init at Frame

The frame on which data is initialized. You can use this to control the start frame of the simulated deformation.

Global Time

If this option is on, Emit at Frame is relative to the scene's timeline. If it is off, the frame is relative to the simulation range set in the Simulation Time Control Property Editor.

Simulator Settings

Node Type

Make sure this is set to Real. The other values are available for future functionality.

Cluster ID

Sets the cluster for points on the connected object. Points in different clusters will not attach to each other during the simulation.

Constrainable

Determines whether points can make contact links or be constrained.

Phase

Sets the phase for points on the connected geometry. Different phases on different geometries allows you to use different Lagoa materials.

Initial Force

The local XYZ components of the initial force for points on the connected geometry.

Topology Map

Resolution Map

How size is computed for collisions, etc.

  • Automatic computes the size based on the average distance between neighbors.

  • Manual uses the Thickness value below.

Thickness

The size to use when Resolution Map is set to Manual.
