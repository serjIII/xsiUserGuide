Up Direction in the C3D Data Dialog Box

Specifies the axis for the Up direction for the C3D data when you import it.

To display: Select the tagged target rig to which you want to apply the C3D mocap data, then choose Tools MOTOR Mocap to Rig from the Animate toolbar.

In the Mocap to Rig property editor, select C3D as the file type, then click the Import C3D button.

For more information, see Retargeting Mocap Data from C3D or BVH Files to a Rig [ Character Animation ].
