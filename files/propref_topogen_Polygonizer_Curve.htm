Polygonizer_Curve Property Editor

| Main | Fcurves

A custom property that controls a curve's contribution to the density isofield used by a Polygonizer operator. For more information about using Polygonizer, see Meshing Point Clouds (and Other Things) [Data Exchange].

To display: Select the curve, choose Selection on the Select panel, and click the Polygonizer_Curve icon.

Main

Active

Controls whether the object contributes to the Polygonizer's isofield. Turn this off to temporarily mute the object's contribution to the polygonized mesh without permanently removing the object as an input.

Use

Controls what components of the object contribute to the isofield.

LOD

Controls sampling along particle strands, curves, and polygon faces. This value has no effect unless Use is set to an option that includes LOD.

Size and Influence

Isofactor

The base contribution of the object's components to the isofield. Note that you can use a negative Isofactor value to make an input remove volume from the resulting mesh.

Radius

The distance at which the isofield is equal to the Isofactor value.

Falloff

The thickness of the falloff region beyond the radius.

Falloff Type

Controls how the isofield value decreases in the falloff region. For example, "Square" gives an inverse-square falloff similar to many physical phenomena.

Fcurves

When Use is set to "Use Knots" or "Use Curve and LOD", you can use an fcurve to control the isofactor, radius, and falloff values along the length of the curve.

Multiply by FCurve

Multiplies the value set of the corresponding parameter on the Main tab by the fcurve.

For information about editing fcurves in the window, see The Profile Curve Editor [Animation].

FCurve Window

Controls the size of the fcurve editing control in the property editor: Small or Large.
