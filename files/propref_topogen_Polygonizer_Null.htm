Polygonizer_Null Property Editor

A custom property that controls a null's contribution to the density isofield used by a Polygonizer operator. For more information about using Polygonizer, see Meshing Point Clouds (and Other Things) [Data Exchange].

To display: Select the null, choose Selection on the Select panel, and click the Polygonizer_Null icon.

Active

Controls whether the object contributes to the Polygonizer's isofield. Turn this off to temporarily mute the object's contribution to the polygonized mesh without permanently removing the object as an input.

Size and Influence

Isofactor

The base contribution of the null to the isofield. Note that you can use a negative Isofactor value to make an input remove volume from the resulting mesh.

Radius

The distance at which the isofield is equal to the Isofactor value.

Falloff

The thickness of the falloff region beyond the radius.

Falloff Type

Controls how the isofield value decreases in the falloff region. For example, "Square" gives an inverse-square falloff similar to many physical phenomena.
