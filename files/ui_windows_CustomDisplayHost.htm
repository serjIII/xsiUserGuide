Custom Display Host

The Custom Display Host allows you to run other applications within Softimage. These applications can interact in real time with Softimage scene data.

If you've created or installed custom displays, you can open them in viewports by choosing them from the Custom Displays sub-menu of the viewport's Views menu. You can also open them in a floating window by choosing View General Custom Display Host from the main menu.

There is an example of a custom display, called VertexColorExample. There are other examples in the SDK Examples workgroup. For more information, see Custom Display Host [SDK Guide].
