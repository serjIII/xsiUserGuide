Chain from Curve Dialog Box

To display: Select a curve and choose Create Skeleton Create Chain from Curve from the Animate toolbar. Set the option as desired, then click OK and use the left mouse button to pick a curve to use as a path.

For more information, see Creating Chains from Curves [ Character Animation ].

Number of Bones The number of bones the resulting chain will have.

Constrain Chain to Curve

Specifies whether the chain will remain constrained to the curve, allowing you to change the chain's joint rotations by modifying the curve. Inverse kinematics will continue to work on the chain, but not forward kinematics.
