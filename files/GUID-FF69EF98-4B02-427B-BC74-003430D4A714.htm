ShotClip Contextual Menu

When you right-click on a ShotClip, the contextual menu provides options to perform various tasks on the ShotClip.

Option Description Shortcut Key
Split (Current Frame) Splits a ShotClip at the current frame. See Splitting ShotClips. N
Trim Before (Current Frame) Cuts the ShotClip portion, which is before the current frame. [
Trim After (Current Frame) Cuts the ShotClip portion, which is after the current frame. See Trimming ShotClips. ]
Toggle Lock Locks a ShotClip. When this option is active, you cannot horizontally move or resize the ShotClip. However, you can move the ShotClip from one track to another. See Locking ShotClips.  
Delete Deletes a ShotClip. See Deleting ShotClips.  
Clip Properties Displays the options to change the active camera, first and last frames of the Scene and Sequencer timeline.  
Time Properties Displays the options to change the first and last frames of the Scene and Sequencer timeline.  
Display Options Contains the preferences to toggle the display of Shot Track and Shot Clip information. See Displaying Shot Tracks and Track Names, Displaying ShotClips Information, and Defining the Default Length of a ShotClip.
