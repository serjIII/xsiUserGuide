ICE Attribute Editor

The new ICE Attribute Editor displays an alphabetical list all the available ICE attributes of the selected scene object. The attributes list changes depending on the object on which the ICE tree is currently affecting.

Using the ICE Attributes Editor, you can create, rename, delete attributes. You can also mark one or all attributes to get forcefully evaluated by Softimage during the ICE tree optimization process. In the previous versions of Softimage, you can only perform these attribute tasks using the SDK.

To view the attribute editor

  • From the Softimage main menu, choose View ICE ICE Attribute Editor.

or

  • Click the Attr. Editor button on the ICE Tree editor tool bar.

A

Menu bar with options to view attributes based on their category and edit the data type, context, and structure values of attributes. See Viewing Attributes Based on Their Category and Setting Custom Static Attribute Variables.

B

Update — Refreshes the list to display attributes of the object that you select.

C

Add — Creates a new attribute and adds it to the list.

NOTE:The category of new attributes is always Custom Static.

D

Delete — Deletes the selected attribute from the list. See Deleting Custom Static Attributes.

E

Show ICE Tree for selected object — Displays the ICE Tree editor with the corresponding ICE tree for the selected scene object.

F

A selected attribute on which you can perform various tasks.

You can select more than one attribute by pressing the Ctrl or the Shift key.

G

An attribute is marked to evaluate always. See Marking Attributes to Always Evaluated.

The editor also provides more information about each attribute, such as the Category, Data Type, Context, and Structure. For more information about Data Type, Context, and Structure, see Connecting ICE Nodes.

Viewing Attributes Based on Their Category

By default, the All Attributes option is active. This option lists all (hidden, system, and custom) attributes in the editor. You can list attributes of a particular category by choosing the one of the following options.

Hidden Attributes

Displays only the hidden attributes, which are represented by a double underscore at the beginning.

Hidden attributes can include attributes of System and Custom categories.

System Attributes

Displays only the System attributes (System Writable and System Readonly).

Custom Attributes

Displays only the Custom attributes (Custom Dynamic and Custom Static).

Warning when Attempting to View Attributes of two Objects

The ICE Attribute Editor displays a warning message when you attempt to view the attributes of more than one object.

For example:

  1. Create a polymesh.
  2. Create a basic emission.
  3. In the Explorer, select both the polymesh and pointcloud objects.
  4. Press Alt+9 to open the ICE Tree Editor.
  5. Click the Attr. Editor button in the ICE Tree Editor tool bar.

Setting Custom Static Attribute Variables

The following sub-menus within the Edit menu include different data types, contexts, and structures that you can choose to set for Custom Static attributes.

Set Data Type to

Choose a data type (Scalar, Integer, Color, and so on).

Set Context to

Choose a context (per Edge, per Object, per Point, and so on).

Set Structure to

Choose the data structure (Single or Array).

Alternatively, the ICE Attribute Editor provides a drop-down list for the Data Type, Context, and Structure columns of the Custom Static attribute. From the drop-down list, you can choose your own data type, context, or structure for the Custom Static attribute.

Converting Custom Dynamic Attribute to Custom Static

You can convert a Custom Dynamic attribute to Custom Static by choosing Edit Convert to Custom Static. By converting to Custom Static, you can rename, delete, and choose your own data type, context, or structure for it.

NOTE:This option is enabled only when you select a Custom Dynamic attribute.

Marking Attributes to Always Evaluated

From the attributes list, you can choose to forcefully evaluate one or more attributes. You can enable forceful evaluation upon all attributes, irrespective of their category (Hidden, System, or Custom)

To enable or disable forceful evaluation

  1. Select one or more attributes from the attribute editor.
    NOTE:Press Ctrl or Shift keys to select more than one attribute.
  2. Choose Edit Toggle Always Evaluated or toggle click on the selected attribute(s).

Renaming Custom Static Attributes

NOTE:You can only rename the Custom Static attributes.

To rename a Custom Static attribute

  1. Double-click the Name field.
  2. Change the current name.
  3. Press Enter or click anywhere outside the text box.

Deleting Custom Static Attributes

NOTE:You can only delete the Custom Static attribute that is not referred by any other nodes used in the ICE tree.

To delete a Custom Static attribute

  1. Select the Custom Static attribute from the list.
  2. Click the Delete button () from the tool bar.
