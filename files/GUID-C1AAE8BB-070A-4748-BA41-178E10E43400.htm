Adding ShotClips on a Shot Track

You can add any number of ShotClips to a Shot Track.

To add a ShotClip at the current frame on the selected Shot Track

  • Click the Add Clip at Current Frame button () in the toolbar.
    NOTE:
    • If there is already a ShotClip at the current frame on a Shot Track, then it searches for another existing Shot Track whose current frame contains space to add a ShotClip.
    • If the current frames in all Shot Tracks contain ShotClips, then it creates a new Shot Track at the bottom with a ShotClip at the current frame.

    or

  • Right-click and select Insert ShotClip.

By default, the length of the newly added ShotClip is equal to the total duration of the Sequence Time. For more information on changing the default length, see Defining the Default Length of a ShotClip.

NOTE:Adding a ShotClip is affected by rippling. This controls how other ShotClips on a Shot Track move in relation to the added ShotClip. For example, if Ripple is not active and you add a ShotClip between two other ShotClips where there is not enough room, then the clip you are adding will be trimmed to fit the available space.
