Get Cluster Attributes

This compound is a convenience for getting various cluster properties on the self object. The default names of properties are used as the default references, but you can change them if required.

Tasks: Deformation/Getters

Output Ports: User_Normals, Weights, UVWs, CAV

User Normal Property

The reference for the values returned from the User_Normals output port.

WeightMap Property

The reference for the values returned from the Weights output port.

UVW Property

The reference for the values returned from the UVWs output port.

Color At Vertices Property

The reference for the values returned from the CAV output port.
