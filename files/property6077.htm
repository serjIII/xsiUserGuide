Extract Curve Property Editor

Extracts a new curve from a NURBS surface.

To apply: See Extracting Curves from Surfaces [Surface and Curve Modeling].

To redisplay: Select the extracted curve, click the Selection button on the Select panel, then click the Extract Curve icon.

Inputs

Hide/Unhide

Hides all the inputs for the generated object to simplify the display of your scene in the 3D views, or unhides the inputs when you need to modify them.

Delete

Freezes the generator and deletes the inputs. This simplifies the scene once you are satisfied that you won't need to modify the inputs again.

Extract on Whole Mesh

Extracts the curve across multiple subsurfaces in the case of an assembled surface mesh.
