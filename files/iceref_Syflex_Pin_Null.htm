Syflex Pin Null

This compound lets you attach a simulated cloth or curve's vertices to a polygon mesh constraining object using a null to determine which vertices are pinned. When the constraining object is deformed or animated, the constrained vertices of the cloth or curve will follow.

You use the null's size, shape, orientation, and location (all of which can be animated) to determine the area of vertices that are pinned. All the cloth or curve vertices that are within the null object's space are pinned to the constraining polygon mesh object.

To apply: See Pinning Cloth or Curve Vertices to an Object.

Tasks: Syflex/Constraints

Output Ports: The Force output plugs into a Force port on the Syflex Cloth or Syflex Curve node.

The parameters for this constraint are the same as for the Syflex Pin compound except that you use this port for the input null:

Null

The null object that determines which cloth or curve vertices are pinned to the polygon mesh constraining object. Plug its Get Data node's Out Name output into this port.
