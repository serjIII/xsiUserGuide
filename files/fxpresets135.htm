Flare Ring

The Flare ring operator creates a fiery ring in the rendered frame. You can control the ring's dimensions, brightness and create random deviations in the width.

Requires an Input (Any).

Produces an Output (Any).

Flare Ring

Brightness

Controls the brightness and opacity of the ring effect. Higher values produce a more opaque, brighter ring.

Radius

Sets the radius of the ring. A value of 1 creates a ring that extends nearly to the frame's boundaries.

Width

Sets the width of the ring. A value of 1 causes the ring's inner edge to bleed as much as possible into the its center.

Center X, Y

Set the center of the effect as a normalized proportion of the image. Set any floating-point values.

Jitter Amount

Controls the strength of the seemingly random variations in the ring's width created when you set the seed parameter. For these variations to be visible, the jitter amount must be higher than 0.

Seed

Controls the pattern of random variations in the ring's width. Each seed value creates a different pattern. The strength of the pattern is controlled by the jitter amount.

Composite Method

Specifies how the flare ring is added to the image or sequence:

Add: the flare ring's light contributes to the overall brightness of the image.

Screen: the flare ring's light does not contribute to the overall brightness of the image.

Mask

Controls this operator's Mask Options.

Output

Controls this operator's Output Options.
