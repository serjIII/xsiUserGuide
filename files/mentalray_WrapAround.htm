WrapAround

Category: mental ray > Lens

Shader Family: Lens

Output: Color

This shader grabs a snapshot of the entire 360-degree environment surrounding the camera. This snapshot can be useful in a variety of ways: as a reflectivity map, spherically projected onto a dome as an environment map, etc. There are no parameters.

Name

The name of the shader node displayed in the render tree. Enter any name you like, or leave the default.
