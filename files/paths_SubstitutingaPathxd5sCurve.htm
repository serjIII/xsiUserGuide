Substituting a Path's Curve

After you have animated an object on a path, you can substitute the original curve with a new one to create a completely different animation.

  1. Select the curve that is currently the path.

  2. Choose Edit Operator Substitute Input from main command panel.

  3. Pick the curve to be the new path. Right-click to end the picking session.

    The object is assigned to the new curve.
