Self-Anti-Aliasing

DEPRECATED. This shader is unsupported, but it may still get installed with Softimage to provide compatibility with older scenes that use it. It is recommended that you replace unsupported shaders in your scenes with equivalent shaders from the current Softimage shader library.

| Render Tree Usage

Category: Lens

Shader Type: Lens

Output: Color (RGB) value

Performs a type of anti-aliasing without taking the background color into consideration by over-sampling the primary rays of an object. Especially useful for games developers who want to create "sprites" that aren't "contaminated" with any contour.

Filter Level

Defines the degree of over-sampling. High values take longer to render.

Filter Edges

When on, the filtering and over-sampling are not adaptive.

Render Tree Usage

This shader can be applied directly to the camera in the lens shader stack.
