Create Render Channel Dialog Box

Creates a custom (user-defined) render channel and adds it to the Available Render Channels tab of the Scene render options. Custom channels and preset channels are displayed as green rows in the grid. Built-in channels are displayed as gray rows.

To display:

  1. The Render Manager > Scene Render Options > Available Channels

  2. Click the Add... button below the Render Channels grid.

Name

Enter a name for the custom render channel.

Type

Choose a data type for the channel being created:

  • Color

  • Grayscale

  • Depth

  • Normal Vector

  • Vectors/Motion Vectors

  • Object Label
