Boolean Invert

Category: Processing > Math

Shader Family: Texture

Output: Boolean

Negates a boolean switch: What was on becomes off, and vice-versa.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Boolean is on.

Render Tree Usage

Used to connect a shader (with a Boolean output) to the Input parameter. The output will automatically be negated.
