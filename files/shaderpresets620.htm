Scalar To Integer

| Scalar To Integer

Category: Processing > Conversion

Shader Family: Texture

Output: Integer

Converts a scalar input value to a whole integer output value.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Defines the scalar input to be converted to an integer.

Render Tree Usage

This shader is simply placed between a scalar output and an integer input in order for the two values to connect.
