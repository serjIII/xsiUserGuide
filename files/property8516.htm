Symmetry Map Generator Property Editor

Generates a symmetry map. Symmetry maps establish correspondence between points on either side of a geometrical object for symmetrical manipulation.

To apply: Select an object and choose Get Property Symmetry Map from any toolbar.

To redisplay: Select the object, click the Selection button on the Select panel, expand the Clusters\SymmetryMapCls\<symmetry_map_name> node, then click the Symmetry Map Generator icon.

Symmetry Plane

The plane used for symmetry, relative to the object's own local coordinate system: YZ (X = 0), XZ (Y = 0), XY (Z = 0).

Point Tolerance

The positional tolerance for finding a corresponding point on the other side of the object, in Softimage units.
