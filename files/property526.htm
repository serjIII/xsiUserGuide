Wave Property Editor

| General Property Page | Profile Property Page

Wave control objects define animated wave deformations. To create a wave, choose Get Ctrl Object Wave from the Simulate toolbar or Get Primitive Control Object Wave from the other default toolbars.

To redisplay: Select the wave object and press Enter.

NOTE:To deform objects by a wave, select the objects, choose Modify Deform Wave from the Model toolbar (or Deform Deform Wave from the Animate toolbar), then pick the wave object.

General Property Page

Type

Determines how the wave propagates from its source:

  • Circular produces a circular pattern emanating from a point like a water drop on a pond.

  • Planar produces a planar pattern emanating from a line like a wave on the beach.

  • Spherical produces a spherical pattern emanating from a point like an explosion.

Displacement Direction

The direction in which the points of affected objects are displaced:

  • Up displaces points along the local Y axis of the wave object.

  • Direction displaces points in the direction in which the wave is moving.

  • Normal displaces each point along the normal of the deformed object at that point.

Periodicity

In Space

Repeats the shape of the wave's profile between the first and last key points indefinitely.

In Time

Repeats the entire wave effect, including the result of Amplitude and Spread (set in the Wave Op Property Editor). The wave deformation begins again after the End frame set on the Time Control tab of the operator's property editor.

Velocity

The initial speed at which the wave moves in units per second

Acceleration

The rate at which the wave's speed changes in units per second squared. Positive values cause the velocity to increase, and negative values cause it to decrease.

Vertical Falloff

Start/End

Sets a height range over which the wave's effect decreases along the local Y axis of the wave object. Distance is measured from the center of the wave object. Start specifies the height at which the effect begins to decrease; End specifies the height at which the wave has no effect at all; and between 0 and the Start value, the full amplitude is applied.

Profile Property Page

Amplitude Profile

The shape of the wave's cross-section. The amplitude of the deformation at any point is determined by this profile and the value of the Amplitude parameter of the Wave Op Property Editor of the affected object. The horizontal length of this profile can also be modified and animated with the Spread parameter of the Wave Op deformation.

You can edit the profile using the mouse and the same keyboard commands as the animation editor, or right-click to display a menu.
