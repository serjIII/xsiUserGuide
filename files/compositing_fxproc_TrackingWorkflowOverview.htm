Tracking Workflow Overview

Generally speaking, using the tracker involves the following basic steps:

  1. In the FxTree, get the images or sequences that you wish to use as the Tracking operator's input and, if you are doing a match move, foreground.

  2. Add a Tracker operator by doing one of the following:

    • From the FxTree menu, choose Ops Transform Tracking.

      or

    • Use the operator selector to pre-connect the Tracking operator before inserting it into the FxTree workspace, as described in Connecting Operators.

      The Tracking operator is found on the Transform tab of the operator selector.

  3. Activate and position the trackers, as described in Setting Up the Trackers.

  4. Create the motion paths by tracking through the sequences, as described in Creating Motion Paths.

  5. When the tracking process is finished, apply the desired effect by setting the tracking Mode to Match Move, Stabilize, or Destabilize.

    • Match Move is described in Applying a Match Move.

    • Stabilize and Destabilize are described in Stabilizing and Destabilizing.

  6. If desired, use the output rectangle to output only a selected region of the sequence, as described in Using the Output Rectangle.
