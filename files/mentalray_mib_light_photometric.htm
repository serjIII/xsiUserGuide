mib_light_photometric

Category: mental ray > Light

Shader Family: Light

Output: Color

This shader is a variation of a simple point light shader emitting light according to the distribution determined by an IES light profile. This shader also honors the decay rate.

Colors

[color]

Specifies the light color.

Shadows

Enable

When on, enables shadows. The light will be diminished or blocked by occluding objects between the light and the illuminated object.

Umbra

Only has an effect if shadows are enabled. Blends from the shadow color (0, the default) to the light color. Values greater than 0 (zero) let the light penetrate occluding objects. At 1, no shadows appear.

Distance Attenuation

Start

Specifies the distance from the light where attenuation begins.

Light Profile

[profile]

Specifies the intensity of the light using the values stored in an IES profile referenced by this shader. If no light profile is provided the light behaves like a uniform point light.
