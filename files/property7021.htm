Chain Root Property Editor

Adjusts a chain root's display type and size, including shadow icons.

For more information, see Changing the Shape and Size of Chain Elements [Character Animation].

To display: Select a chain root and press Enter.

Primary Display

Icon

Select from a list of shapes to use for the root's display — see Shapes for Effector, Root, and Null Icons and Their Shadows.

Size

Sets the size of the display icon for the root on all axes (uniform scaling).

When you change this value, the root's accompanying shadow always scales proportionally with it, regardless of the state of the Lock Shadow Size When Scaling option.

Shadow Display

A shadow is a visual reference that makes it easier to see and select chain elements. When you select a shadow, its chain element is also selected.

Icon

Select from a list of shapes to use for the root's shadow — see Shapes for Effector, Root, and Null Icons and Their Shadows.

Select an option other than None to see the following parameters.

Color

Sets the color of the shadow. Select the Custom option to set a custom color using the color sliders. The resulting color appears in the color chip.

You cannot change the color of the shadow if Wireframe is selected: the standard display color is used.

Position

Offsets (translates) the shadow from the root on the X/Y/Z axes.

Size

Specifies the scale of the shadow. The default value of 1 means a 1:1 ratio with the root.

Lock Shadow Size When Scaling

Keeps the shadow at a constant size when the root is scaled (using the Scale transform commands, not the Size parameter above).
