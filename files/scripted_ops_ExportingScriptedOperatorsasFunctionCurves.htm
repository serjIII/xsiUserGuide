Exporting Scripted Operators as Function Curves

You can export the plotted output of a scripted operator on a parameter to a raw function curve file (.fraw2 format). Autodesk Softimage plots the value of the output connections and saves a function curve file with a key at every frame. You can then open the saved .fraw2 file in the animation editor.

To plot and export a scripted operator as a raw function curve

  1. Make sure that Connections is selected in the list in the upper left of the scripted operator editor and that the desired output parameter connection is selected.

  2. In the scripted operator editor, choose File Save as FCurve.

  3. In the browser that opens, select a directory and file name and click OK.

    For information about opening function curves in the animation editor, see Saving and Loading Function Curves [Animation].
