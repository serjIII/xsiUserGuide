Lagoa Material Generic Liquid

Creates a basic viscous liquid using the point cloud that is emitted.

The parameters exposed for this compound are a subset of those in the Lagoa Main Material compound, mainly in the Pressure, Viscosity, and Inelastic Settings groups.

Plug this compound's Execute output into an Execute port of the Lagoa Phase node.

To use this material, choose the Particles Create Lagoa Particles Pouring Liquid command from ICE toolbar, which lets you create your own liquid particle emission. This command uses the Lagoa Setup Pouring Liquid compound, which contains the Generic Liquid material.

See Creating a Pouring Liquid for more information.

Tasks: Lagoa/Material Type

Output Ports: Execute
