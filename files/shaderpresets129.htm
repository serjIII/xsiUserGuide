Scalar Invert

| Render Tree Usage

Category: Processing > Image Processing

Shader Family: Texture

Output: Scalar

Inverts a scalar value.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Specifies the input value to be inverted.

Output = 1 minus input

Render Tree Usage

Used to the right of any type of scalar output shader to reverse (invert) its value.
