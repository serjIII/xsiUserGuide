Randomize Animation State Start Frame

An internal compound used by the Initialize Move Locomotion Data node to randomize the start frame for the Move Animation State.

Tasks: Crowds/Initialize

Output Ports: Execute. Connect this port to the Emit_and_Initialize ICE tree.

State

Name of the Move Animation State

Base Value

Base value for randomizing the initial frame

Variance

Variance for randomizing the initial frame

Seed

Seed for randomizing the initial frame

By setting the State and Speed parameters (Base Value, Variance, and Seed), you can control the speed of actors in the specific animation state in a crowd scene. By controlling the speed of actors, you can make them walk naturally, and not like in a military parade.
