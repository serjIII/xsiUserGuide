Overview of the Shape Manager

To display the shape manager

  • Choose Deform Shape Shape Manager from the Animate toolbar.

    or

  • Choose View Animation Shape Manager from the main menu.

    or

  • To anchor it in a viewport, choose Shape Manager from a viewport's Views menu.

If an object is selected when you choose this command, it is framed in the shape manager's viewer.

If nothing is selected when you choose this command, the viewer is empty. Select an object, then click the Update icon in the shape manager to have it appear in the viewer.

A

Create tab displays the Shape list which is all the shapes set for the selected object — see Creating Shapes.

B

Animate tab displays weight sliders for each shape, as well as commands and controls for keying — see Animating the Shapes.

C

Shape viewer displays the current state of the object's shape — see Setting Up the Shape Viewer.
