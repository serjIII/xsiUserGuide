De-interlacing Image Clips

You can specify the field type of image clips in your scene. This is important if you are using video files as textures and you intend to render your scene in fields instead of frames. When an image clip's field type is specified, the clip is de-interlaced at render time so that only one of the clip's fields appear in each of the scene's rendered fields.

NOTE:Field Type is a clip effect, requiring that the clip's Image Management mode be set to Load in Memory. Setting the field type subjects the clip to all of the caveats that clip effects entail. For details, see Applying Clip Effects.

For more information about field rendering in general, see Field Rendering [Rendering].

To set a video image clip's field type

  1. Edit the clip to open its property editor.

  2. From the Video Texture/De-Interlacing options on the Texturing tab, set the Field Type to None, Lower Field First / Even (NTSC and DV Formats), or Upper Field First / Odd (PAL and HD Formats).

    This specifies the field order for video images clips. When Field Type is set to none, the clip is not de-interlaced at render time.

    The Field Type setting should match the Field Order setting on the Scene Globals tab in the Scene Field Rendering options section of the Scene Render Options Property Editor [Properties Reference].
