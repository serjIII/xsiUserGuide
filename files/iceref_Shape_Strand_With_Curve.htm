Shape Strand With Curve

This compound shapes the strand according to the shape of the curve. The base of the curve matches the start of the first strand segment's position.

Create strands, then plug this compound's Execute output into a PostSimExecute port on the Simulation Root node or in a port on the ICETree node.

For information, see Shaping the Strands [ICE Particle Simulations].

Tasks: Particles/Strands

Output Ports: Execute

Curve

The curve object that you're using as the strand deformer. Plug the curve's Value output into this port.
