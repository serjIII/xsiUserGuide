syflexISpringBorder

Categories

  • Syflex Force

Description

Defines the springs that are used along a cloth object's disconnected edge (border) to link a set of points.

This node is used in the Syflex Zipper compound.

Ports

Inputs

Geometry

The geometry of the simulated object: only polygon meshes for cloth objects are allowed.

Point0

The point at which the zipper starts on one edge of the cloth.

Point1

The point at which the zipper ends on the same edge of the cloth as Point0.

Point2

The point at which the zipper starts on the opposite edge of the cloth as Point0 and Point1.

Outputs

Points

The array of points to be used for connecting the cloth object's edges by a spring.
