About Crosswalk for Max

Reliable interoperability is the most important part of a mixed-tool pipeline — Crosswalk for Max helps you seamlessly transfer Softimage content into and out of an Autodesk® 3ds Max® pipeline using the latest dotXSI (*.xsi) file format.

Crosswalk for Max is optimized for dotXSI version 6.0. Supported versions of 3ds Max can read the dotXSI format using the Crosswalk for Max plug-in. 3ds Max does not need to be installed on the same system as Softimage to enable data transfer.

Getting Crosswalk Updates

To get the latest updates to the Crosswalk for Max integration, you can download the Autodesk Crosswalk toolset for free from autodesk.com/softimage-crosswalk. The Crosswalk for Max plug-in is a component of this toolset and it can be installed on any system running a supported 3ds Max installation.

For information on how to enable the full Crosswalk capabilities for a 3ds Max installation, see Crosswalk Setup [Advanced Setup and Configuration].
