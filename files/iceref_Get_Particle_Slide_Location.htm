Get Particle Slide Location

This compound gets the particle's position and normal data at the point of contact with an obstacle when sliding using the Slide on Surface compound. This is defined by the particle's SlideLocation attribute.

You can output the particle's Location, Position, and Normal data.

For more information on sliding, see ICE Particles Sliding on Obstacles [ICE Particle Simulations].

Tasks: Particles/Getters

Output Ports: Location, Position, Normal

Reference

The particle's SlideLocation attribute.
