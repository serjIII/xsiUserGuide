Hair Attenuate Operator Property Editor

Attenuates the lengths of the guide curves based on the size of the polygons of the hair emitter. Guide curves become proportionately shorter where polygons are smaller. You can apply this operator multiple times to obtain a cumulative effect.

To apply: Select a hair object and choose Modify Length Attenuate from the Hair panel. If you select points, tips, or strands instead of a hair object, only the corresponding guide curves are affected.

To display: Select the hair object, click the Selection button on the Select panel, and then click the Hair Attenuate Operator icon.

For more information, see Attenuating the Length [ Hair ].

Mute

Toggles the deformation on and off without affecting the values of other parameters.
