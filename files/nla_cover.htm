Nonlinear Animation in the Animation Mixer

This section covers features about the many techniques and tools used for nonlinear animation in Autodesk® Softimage®. Nonlinear animation is a way of animating that does not restrict you to a fixed time frame. You can store different types of animation, animation layers, shapes, images, and caches into packages called sources, then load this package in the animation mixer.

Related Information

You can use the animation mixer for these types of data:

  • Fcurves, expressions, and constraints — see Actions

  • Shape animation — see Animating Shapes

  • Audio — see Audio and Animation

  • Animation layers — see Animation Layers

  • Cached animation and simulation sources — see Caching Animated Deformations and Simulations

  • Image Clips — see Manipulating Image Clips in the Animation Mixer
