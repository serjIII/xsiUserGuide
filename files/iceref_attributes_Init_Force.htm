Init_Force

A particle's original Force value when it was emitted.

This attribute is set by the Emitters compounds, and can be used by your own compounds.

Data Type

3D vector

Belongs To

points

Geometry Types

  • point cloud

Attribute Type

custom

R/W or RO:

read/write
