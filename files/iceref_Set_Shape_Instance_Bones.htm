Set Shape Instance Bones

Sets data that is used when displaying the Point_Cloud object of a CrowdFX simulation in OpenGL mode. In this mode, the Point_Cloud particles instantiate the actors' Mesh_Proxy, which uses a real-time shader to calculate the envelope deformation per instance. This may give faster previews in heavy crowd scenes. The results are visible in High Quality and Realtime Shaders OpenGL viewing modes.

This compound should be used in the Post-Simulation region on the Point_Cloud object of a CrowdFX simulation, after any constraints. It is disabled by default.

In addition to this compound on Point_Cloud, you must use Shape Instance Skinning on Mesh_Proxy.

For more information, see Viewing the Actor's Envelope.

Tasks: Crowds/Skinning

Output Ports: Execute

Enable

Enabling this compound temporarily overrides the viewport visibility of the Actor_Copies mesh, both to improve performance and so that you don't see both meshes superimposed.
