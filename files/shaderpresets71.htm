Toon Material

DEPRECATED. This shader is unsupported, but it may still get installed with Softimage to provide compatibility with older scenes that use it. It is recommended that you replace unsupported shaders in your scenes with equivalent shaders from the current Softimage shader library.

| Toon Paint | Toon Soft

Shader Type: Material (Soft3D)

Output: Color (RGB) value

Creates solid color contour shading on the surface of objects, as well as adjusting material-specific ink characteristics. You can use the Toon surface shader by itself or in conjunction with the Toon Lens shader -- which creates the ink lines -- to render objects with ink lines drawn over the standard material in Softimage.

Name

The shader's name. Enter any name you like, or leave the default.

Toon Ink

Ink (requires Cartoon)

Thickness

Modifies the width of inked contours. Line width is also affected by the global Thickness parameter of the Toon Lens shader. Setting very different line thickness for two overlapping objects can produce undesirable artifacts.

Color

Material Ambient

When enabled, the shader uses the material's ambient color.

Use Custom Ink Color

Lets you to use a custom color defined with the Custom Ink Color parameter.

Custom Ink Color

Sets the color for the ink.

Illuminate

Varies the color of contours according to the amount of light reaching the surface beneath. To switch on, use a non-zero value: values greater than 0 brighten the contour, and values less than 0 darken it; each is proportional to the color of light.

Outline (relative to color)

Saturation

Defines the color intensity of the outline.

Brightness

Defines the brightness of the outline color.

Ink between Lighting

Draws ink between a change in light.

Ink between Colors

Draws ink between a change in color.

Toon Paint

Highlight Size

Controls the size of the highlight.

Highlight Shadow Border

Creates a border region between highlight and shadow colors.

Width

Defines the amount of depth fading to apply to the object's ink thickness. For example, if you specify a width of 0.75, the thickness of the line is reduced to 25% of its original thickness as the object moves from the start to the end distance.

Color

Gradient

When enabled, the paint color will be displayed as a gradient.

Blend

When enabled, the paint color will be blended.

Global Ambience

Uses the color of the ambient lighting you have activated.

Shadows Only

Specifies that the global ambience color affects only the object's shadow.
