Layered Fog

| Render Tree Usage

Category: Volume

Shader Family: Volume

Output: Color

Creates a fog effect in a definable layer, with editable start and stop points.

Name

The shader's name. Enter any name you like, or leave the default.

Fog Color

Color

Defines the Red, Green, Blue, and Alpha channels that make up the fog's color.

Density

Defines the fog's density: 0 for invisible fog; 1 for opaque fog.

Fog Distance

Start, Stop

Specifies the distance between where the fog begins (completely transparent) and where it ends (completely opaque). Fog appears only between these two distances, and the fog density is linearly interpolated between these two distances.

Fog Height and Distance

Base Height

Defines at what height (in SOFTIMAGE units) the fog effect begins.

Thickness

Defines the thickness of the fog effect, starting from the base height.

Render Tree Usage

Used to apply this volume effect to an object (as opposed to a scene) by connecting this shader to the Volume input of the material node. Volume shaders cannot be expanded or built upon in the render tree.
