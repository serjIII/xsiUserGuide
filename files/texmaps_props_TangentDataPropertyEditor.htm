Tangent Data Property Editor

Holds the name of the color at vertices (CAV) property associated to the TangentOp3 operator.

This node acts as a tracking mechanism so that tangents and binormals do not compete for the same color at vertices property (overwriting each other every time TangentOp3 re-evaluates the data).

IMPORTANT:It is recommended that you do not modify any of the options in this property editor.
