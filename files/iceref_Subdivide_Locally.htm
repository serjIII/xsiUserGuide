Subdivide Locally

Categories

  • Topology

Description

Suibdivide one or more polygons.

Ports

Inputs

Topo

The topology you want to use. If not connected, the default is "Self.Topology".

Rule

The algorithm used for subdivision.

Polygon Index

The indices of any polygon to subdivide. You can use a single integer, array, or set.

Depth

Number of times that the subdivision process is iterated.

Outputs

Result

The modified topology.
