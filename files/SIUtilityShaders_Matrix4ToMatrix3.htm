Matrix4 To Matrix3

Category: Processing > Conversion

Shader Family: Texture

Output: 3x3 Matrix

Converts a 4-by-4 matrix input to 3-by-3 matrix output.

Name

The shader's name. Enter any name you like, or leave the default.

Input

Specifies the 4-by-4 matrix input to be converted to 3-by-3 matrix output.

Render Tree Usage

This shader is simply placed between 4-by-4 matrix output and 3-by-3 matrix input in order for the values to connect.
