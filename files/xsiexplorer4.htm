XSI Explorer Preferences

To display: Choose File Preferences, expand the Editors node, and click XSI Explorer.

Explorer

Track Selection

The explorer window automatically scrolls up or down to display the nodes of selected elements. If multiple elements are selected, the explorer will only keep the first selected element in view.

Current Scope

The current or last-selected scope of the explorer. This value is read-only and displayed for information only. To change the scope, select an option from the Scope menu of an explorer.

Default Viewer

The name of the viewer to display by default when the XSI explorer is opened. This is automatically set to the last item selected from the Viewer menu in the XSI Explorer.
