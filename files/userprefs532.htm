Polygon Preferences

To display: Choose File Preferences, expand the Tools node, and click Polygon.

Preserve Planarity During Creation

Toggles whether Softimage will allow you to create non-planar polygons.

When Connecting CW and CCW Polygons

Defines Softimage's default behavior when clockwise and counterclockwise polygons are connected.
