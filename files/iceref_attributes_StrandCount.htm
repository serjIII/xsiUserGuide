StrandCount

Set and used by the Strand compounds to keep track of the number of segments in a strand.

This attribute is exposed as the Num Segments option in the Create Strands compound and as the Max Num Segments in the Generate Strand Trails compound.

Data Type

integer

Belongs To

points

Geometry Types

  • point cloud

Attribute Type

custom

R/W or RO:

read/write
