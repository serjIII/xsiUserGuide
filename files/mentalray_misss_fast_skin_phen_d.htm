misss_fast_skin_phen_d

Category: mental ray > Material Phenomena

Shader Family: Material Phenomenon

Output: Custom

This phenomenon is an exact duplicate of misss_fast_skin_phen (refer to this shader for all parameter descriptions), with the exception that it also supports displacement mapping. The displace parameter allows you to pick a shader that displaces the surface.

This must be implemented as a separate node because a phenomenon with displacement will always attempt to displace the object.
