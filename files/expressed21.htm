Tangent (Expression Function Reference)

The Function Trigonometry Tangent command in the expression editor inserts the token tan( at the insertion point in the editing pane. This token evaluates the tangent function of an expression. Enter an expression after the opening bracket ( then enter a closing bracket ).

Example

tan(10*Fc)

Returns the tangent of the expression 10*Fc (10 times the current frame number).
