Shape Instance Skinning

Sets data that is used when displaying the Point_Cloud object of a CrowdFX simulation in OpenGL mode. In this mode, the Point_Cloud particles instantiate the actors' Mesh_Proxy, which uses a real-time shader to calculate the envelope deformation per instance. This may give faster previews in heavy crowd scenes. The results are visible in High Quality and Realtime Shaders OpenGL viewing modes.

This compound should be used in the Modeling region on the Mesh_Proxy objects of a CrowdFX simulation.

In addition to this compound on Mesh_Proxy, you must use enable Set Shape Instance Bones on Point_Cloud.

For more information, see Viewing the Actor's Envelope.

Tasks: Crowds/Skinning

Output Ports: Execute

Enable

Enables the shape instance envelope deformation.

Deformer Group

Reference

A reference to the group used as deformers for the envelope. Normally this is This_Model.Deformers_Proxy.

Deformer Group In Name

A name to prefix to the Deformer Group reference to fully resolve the name.

Envelope Object

Reference

A reference to the mesh to be instantiated and deformed. Normally this is This_Model.Mesh_Proxy.

Envelope Object In Name

A name to prefix to the Envelope Object reference to fully resolve the name.
