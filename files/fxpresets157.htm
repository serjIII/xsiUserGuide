Lightning

Creates a random lightning effect.

Lightning does not require an input.

Produces an Output (Any).

Generation

Branch Probability

Sets the likelihood that the lightning will have few or many branches. Set any floating-point value using the treadmill; 0 is no branches Â­ the higher the value, the more branches there are likely to be.

Age

Sets the proportion of the lightning to be displayed. Set a floating-point value between 0 and 1 using the slider; 0 is no lightning, 1 is the complete lightning bolt. The normal animation would be from 0 at the first frame to 1 at the last frame.

Seed

This is a base number upon which all random values are calculated. Set any integer value using the treadmill to achieve a different random effect.

Render Mode

Sets the quality of the rendered lightning. Select the required option from the pop-up menu:

Quick renders only the motion of the lightning, without color or volume.

Standard renders the full lightning without antialiasing. This is a useful setting for fine-tuning the animation.

Antialiased renders the full lightning with antialiasing. Use this setting when rendering to ensure that fine branches do not break up.

Glow

The glow is the outer portion of the lightning bolt.

Glow Radius

Sets the radius of the glow in pixels. Set a floating-point pixel value between 0.1 and 40 using the slider.

Glow Color

Sets the color of the glow.

Core

The core is the inner portion of the lightning bolt.

Core Radius

Sets the radius of the core in pixels. Set a floating-point pixel value between 0.1 and 10 using the slider.

Core Color

Sets the color of the core.

Output

Controls this operator's Output Options.
