Creating ICE Compounds

To create a compound

  1. In the ICE Tree view, select the nodes you want to include in the compound.

  2. Do one of the following:

    • Choose Compounds Create Compound.

      or

    • Right-click on one of the selected nodes (not over a port) and choose Create Compound.

    A new compound node named CompoundNode is created, containing all the previously selected nodes. Any connections to unselected nodes become connected ports on the new compound.

To modify the new compound, see the following sections:

  • Editing Compounds.

  • Managing Exposed Ports.

  • Setting Compound Properties.

TIP:If you want to make a compound out of a whole tree but there are several ports connected to the root node, first right-click on the root node (not over a port) and choose Insert Execute Node. This makes it easy to create a compound with a single output port.

Exploding Compounds

Exploding a compound "unpacks" it while leaving all contained nodes and connections intact.

To explode a compound

  • Select one or more compounds and choose Compounds Explode Compound.
