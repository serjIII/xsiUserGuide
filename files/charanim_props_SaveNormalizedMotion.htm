Save Normalized Motion Dialog Box

Saves any type of animation in a .motor (normalized motion) file. The animation can be retargeted C3D or Biovision mocap data, or it can be keyframes or any other type of animation.

The .motor files can then be applied to any tagged rig of the same structure (such as biped or quadruped).

To display: Select a tagged rig or skeleton with any type of animation and choose Tools MOTOR Save Motion from the Animate toolbar, or click the Save Motor File button in the Mocap to Rig property editor for saving C3D data in .motor files.

For more information, see Saving Animation Data in Normalized Motion (.motor) Files [ Character Animation ].

Range Settings

Start and End

The first and last frames of animation on the rig that you want to save in the .motor file.

After you click OK in this dialog box, a browser appears in which you can save the .motor file.

To load the .motor file on a tagged rig, select the rig and choose Tools MOTOR Load Motion from the Animate toolbar.
