Envelope Weights Property Editor

Sets the desired limit for the number of deformers to which an envelope point can be weighted. This number is displayed in the weight editor's command bar and used as the default when you click Enforce Limit. See Limiting the Number of Deformers per Point [Character Animation].

To apply: Click Enforce Limit in the weight editor.

Deformer Limit

The maximum number of deformers to which a point can be weighted.
