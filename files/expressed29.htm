Absolute (Expression Function Reference)

The Function Value Absolute command in the expression editor inserts the token abs( at the insertion point in the editing pane. This token returns the absolute (positive) value of an expression. Enter an expression after the opening bracket ( then enter a closing bracket ).

Example

abs(sphere.kine.local.posy)

Returns the absolute value of the local Y translation of sphere.
