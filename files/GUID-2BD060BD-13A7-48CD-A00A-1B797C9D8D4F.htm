Supported nCache Attributes

Point clouds are sent from Softimage to other Autodesk products using the nCache format. The following tables list the nCache attributes that can be included in the files from Softimage, and the ICE attributes that they correspond to.

Required Attributes

Softimage

Optional Attributes

Optional attributes can be set in the ICEFlowState property. Note that ICE attributes must be used in Softimage or they will not be calculated and therefore not cached. See Beware of ICE Optimizations.
