3D Vector to Scalar

Categories

  • Conversion

Description

Outputs the X, Y and Z components of the input 3D vector.

Ports

Inputs

Vector

The 3D vector to convert to three scalar values.

Outputs

X

The X component of the input 3D vector.

Y

The Y component of the input 3D vector.

Z

The Z component of the input 3D vector.
