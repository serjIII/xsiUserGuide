Setting the Pass Renderer

Location:

  • The Render Manager > (select a pass)

  • Set the current pass and choose Render > Render > Pass Options or Pass > Edit > Edit Current Pass from the Render toolbar.

  • Tab: Output [Properties Reference]

  • Options: Pass Renderer

The Pass Renderer option allows you to specify which renderer is used to render each pass. By default this option is set to Use Scene Render Options, which forces the pass to use the renderer specified for the scene (see Choosing a Scene Renderer). However, you may specify that either mental ray or the Hardware renderer be used for the pass, irrespective of the scene-level setting.

To choose the renderer for your pass

  1. Make the pass for which you want hardware rendering the current pass.

  2. Choose mental ray or Hardware Renderer from the Pass Renderer drop-down.

  3. Once you have chosen your renderer, you can decide to create a local set of mental ray or hardware renderer options for the pass. See Setting Global or Local Render Options.
